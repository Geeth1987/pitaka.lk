අවණ්ටඵලදායක වග්ග.   
# අවණ්ටඵලදායක ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කළාවූ, පස්මරුන් දිනූ, භාග්‍යවත්වූ, සතරංසි නම් සම්බුදු රජතෙම, විවේකය කැමති වන්නේ ගොදුරු ගමින් නික්මුන සේක.
# ´´මම (එසේ නික්මෙන්නාවූ) බුදුරජුන් දැක, ඵල ගත් අත් ඇත්තේ (උන්වහන්සේ වෙත) එළඹියෙමි. පහන් සිත් ඇත්තේ, සොම්නස් වූයේ, නැටි රහිත ගෙඩි වර්‍ගයක් දුනිමි.
# ´´මින් සිව් අනූවන කපෙහි මම යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අවණ්ටඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ලබුදජදදායක ථෙරාපදානය.
# ´´එකල්හි මම බන්ධුමතී නගරයෙහි ආරාමිකයෙක් (අරමෙහි වතාවත් කරන්නෙක්) වීමි. (එහිදි) අහසෙහි වඩින්නාවූ කෙලෙස් රජස් විරහිත බුදුරජුන් දුටිමි.
# ´´(දැක) මම දෙල් ගෙඩි ගෙන බුදුරජුන්හට දුනිමි. මහත් යසස් ඇති සර්‍වඥ තෙම අහසෙහි වැඩ සිටියේම එය පිළිගත් සේක.
# ´´(එය) මාගේ සතුට උපදවන්නෙකි. මෙලොවෙහිදීම සුව එලවන්නෙකි. පැහැදි සිතින් (එම) ඵල බුදුරජුන්හට දී,
# ´´එකල්හි ප්‍රීතියද, මහත්වූ උතුම් ප්‍රීතියද ලැබීමි. ඒ ඒ තන්හි උපන්නාවූ තමන්හට රුවන් නිපදේමය.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ලබුදජදදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උදුම්බර ඵලදායක ථෙරාපදානය.
# ´´චින්තා නදී තීරයෙහි පුරුෂෝත්තමවූ බුදුරජතෙම වැඩ විසූහ. (මම) සන්සුන්වූ, කෙළෙස් රජස් විරහිත , එකඟවූ බුදුරජුන් දිටිමි.
# ´´කෙළෙස් මල දෝනාවූ උන්වහන්සේ කෙරහි පහන් සිත් ඇත්තේ, මම දිඹුල් ගෙඩියක් ගෙන බුද්‍ධ ශ්‍රේෂඨයන් වහන්සේට දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උදුම්බරඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පිලක්ඛ ඵලදායක ථෙරාපදානය.
# ´´වනයෙහිදී මහත් කීර්ති ඇති අත්‍ථදස්සී බුදුන් දැක පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, පුලිල ගෙඩි දුනිමි.
# ´´එක් දහස් අටසියවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පිලක්ඛ ඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ඵාරුස ඵලදායක ථෙරාපදානය.
# ´´ලෝකයේ පූජාවන් පිළිගන්නාවූ, රන්වන්වූ, මාර්‍ගයෙහි වඩින්නාවූ බුදුරජුන් දැක (මම) උන්වහන්සේට බොරළු දමන (උගුරැස්ස) නම් ගෙඩි වර්‍ගයක් දුනිමි.
# ´´මින් එක් අනුවන කපෙහි යම් ඵලදානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඵාරුස ඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වල්ලි ඵලදායක ථෙරාපදානය.
# ´´සියළු ජනයෝ එක්ව එකල්හි වනයට ගියාහුය. එකල්හි ඵල සොයන්නාවූ ඔවුහු ඵල වර්‍ග ලැබුවාහුය.
# ´´එහිදී (මම) සියල්ල තෙමේම අවබෝධ කරගත්තාවූ, පස්මරුන් දිනූ සම්බුදුරජුන් දිටිමි. (දැක උන්වහන්සේට) මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, වැල්වල සෑදුනු ඵල වර්‍ගයක් දුනිමි.
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වල්ලි ඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කදලිඵල දායක ථෙරාපදානය.
# ´´කිණිහිරි රුකක් මෙන්ද, පසළොස්වක් පොහෝදා සඳු මෙන්ද බබළන, ඇවිලගත් ප්‍රදීප වෘක්‍ෂයක් වැනිවූ බුදුරජුන් දුටිමි.
# ´´මම කෙසෙල් ගෙඩියක් ගෙන, පහමන් සිත් ඇත්තේ, සොම්නස්වූයේ, බුදුරජුන්හට පිදීමි. (ඉ්කබිති) වැඳ ඉවත්ව ගියෙමි.
# ´´මින් එක් තිස්වන කපෙහි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කදලිඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පනසඵලදායක ථෙරාපදානය.
# ´´එකල්හි අර්ජුණ නම් සම්බුද්‍ධ තෙමේ හිමවතෙහි විසූහ. එම සර්‍වඥ තෙමේ පසළොස්චරණ ධර්‍මයන්ගෙන් යුක්තවූයේ සමාධියෙහි දක්‍ෂවූයේද වේ.
# ´´කළගෙඩි පමණවූ පනස (කොස්) හා ජීවජීවක නම් ඵලද ගෙනවුත් පර්ණච්ඡත්‍රයෙහි කුඩය වෙනුවට ඉස ලන කොළයෙහි තබා මම ශාස්තෘන් වහන්සේට දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පනසඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සොනකොටිවිස ථෙරාපදානය.
# ´´විපස්සී බුදුරජුන්ගේ සසුන පවත්නා කල්හි සිව් දිගින් වඩින සංඝයාට බන්ධුම රාජධානියෙහි මා විසින් ලෙණක් කරවන ලදී.
# ´´(එම) ලෙණෙහි බිම් වස්ත්‍රවලින් අතුරා (සංඝයාට) පිදීමි. එකල්හි ඔදවැඩි සිත් ඇත්තේ, සොම්නස්වූයේ, මෙබඳු ප්‍රාර්ථනාවක්ද කෙළෙමි.
# ´´මම බුදුරජුන් සතුටු කරන්නෙමි. පැවිද්දද ලබන්නෙමි. අනුත්තරවූ, උතුම් ශාන්ති නම්වූ නිවණද ස්පර්‍ශ කරන්නෙමි කියායි.
# ´´එම කුශලමූලයෙන්ම අනූ කල්පයක් සැරිසැරීමි. කරණ ලද පින් ඇති මම දිව්‍යවූයේ, මනුෂ්‍යවූයේ, බැබලීමි.
# ´´ඉක්බිති අවශේෂ කර්‍ම විපාකයෙන් අන්තිම භවයෙහි මේ චම්පා නගරයෙහි අග සිටුහුගේ එකම පුත්‍රයාව උපන්නේ වෙමි.
# ´´මා උපන් පමණින් (එය) අසා පියතුමාහට මෙබඳු අදහසක් විය. (එනම්) මම කුමරුවාහට නොඅඩුව විසි කෙලක් (ධනය) දෙවමි කියායි.
# ´´මාගේ පතුල් දෙකෙහි සියුම්වූ මොළොක් පහස් ඇති කපු පුලුන් වැනි යහපත් සිව් අඟුල් දිග ලෝමයෝ හටගත්තාහුය.
# ´´මින් සිව් අනූවන කපෙහි යම් (සමන් මල්) දමක් දැරූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. සුමන මල් දම් දැරීමෙහි මේ විපාකයයි.
# ´´මා විසින් සම්බුදු රජතෙම සතුටු කරණ ලදී. (මම) අනගාරිය නම්වූ සසුනෙහි පැවිදිවීමි. මා විසින් පැමිණෙන ලද රහත් බව ඇත්තේ, කෙළෙස් තැවුල් නිවන ලදුයේ, සිසිල්වූයේ වෙමි.
# ´´පටන්ගන්නා ලද වීර්‍ය්‍යය ඇති භික්‍ෂූන් අතුරෙන් අග්‍රයයි බුදුරදුන් විසින් (මම) දක්වන ලදුයේ වෙමි. මහත් ඎද්‍ධි ඇති, ෂඩභිඥා ඇති, ක්‍ෂය කළ ආශ්‍රව ඇති රහත් කෙනෙක්ද වෙමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ලෙන් දනෙහි විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. 
# [පරිවර්තනය නැත]
මෙහි මේ අයුරින් ආයුෂ්මත් සොනකොටිවිස තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පුබ්බකම්බ පිලොතිකා ථෙරාපදානය.
# ´´අනවතප්ත මහවිල් තෙරෙදී පැන විචාරණ ලද සොණකොටිවිස ස්ථවිර තෙමේ බික් සඟන ඉදිරියෙහි (එය) විසඳීය.
# ´´නොයෙක් රත්නයන්ගෙන් බබලන නානාවිධ සුගන්‍ධ පුෂ්පයන්ගෙන් ගැවසීගත් (හිමාලය) වනයෙහි අනවතප්ත විල සමීපයෙහි වූ සිත්කළු ගල් තලාවක මහත්වූ භික්‍ෂු සංඝයා විසින් පිරිවරන ලද බුදුරජාණන් වහන්සේ වැඩහුන් සේක් එහිදී තමන් වහන්සේගේ පූර්‍වකර්‍මය ප්‍රකාශ කළ සේක.
# ´´මහණෙනි, මා විසින් යම් කර්‍මයක් කරන ලදද (මාගේ ඒ කර්‍මය අසව) විපාකාවශේෂ කර්‍මය බුදු බැව් ලැබූ කල්හිද විපාක දේ.
# ´´පෙර මම අප්‍රකටවූ (ශුද්‍ර) ජාතියකට අයත් මුනාළී නම් ධූර්‍තයෙක් වීමි. (එකල්හි) ක්ලේශයන්ගෙන් දූෂ්‍ය නොවූ සරභූ නම් පසේ බුදුන් වහන්සේට නින්දාකොට බැන්නෙමි.
# ´´එම (අකුශල) කර්‍ම විපාකයෙන් නිරයෙහි බොහෝ කලක් හැසුරුනිමි. බොහෝ දහස් ගණන් අවුරුදු දුක් වේදනා වින්දෙමි.
# ´´එම කර්‍මයාගේ අවශේෂ විපාක වශයෙන් මේ අන්තිම භවයෙහි සුන්‍දරිකා නම් පරිබ්‍රාජිකාව නිමිතිකොට මා විසින් නින්දා අපහාස වචන ලබන ලදී.
# ´´සබ්බාභිභූ නම් බුදුරජුන්ගේ නන්‍ද නම් ශ්‍රාවකයෙක් විය. උන්වහන්සේට නින්දා කොට මා විසින් බොහෝ කාලයක් නිරයෙහි සැරිසරන ලදී.
# ´´අවුරුදු දසදහසක් පමණවූ දීර්‍ඝ කාලයක් නිරයෙහි හැසුරුනෙමි. මිනිසත් බව ලැබූ මම බොහෝ නින්දා ලැබීමි.
# ´´එම කර්‍මයාගේ ශේෂවූ විපාකයෙන් චිඤ්චමානවිකා තොමෝ ජන සමූහයා ඉදිරියෙහි මට අභූතයෙන් චෝදනා කළාය.
# ´´බොහෝ දෙනා විසින් සත්කාර කරණ ලද්දාවූද මම බහුශ්‍රුත බමුණෙක් වීමි. මහවෙනෙහි පන්සියයක් මානවකයනට මන්ත්‍ර හදාරවමි.
# ´´මහත් ඎද්‍ධි ඇති, පඤ්චාභිඥාලාභීවූ භයජනක ඎෂිවරයෙක් තෙම එහි පැමිණියේය. මම (එසේ) පැමිණියාවූ දුෂ්‍ය නොවූ ඔහු දැක නින්දා කෙළෙමි. (කෙසේදයත්)
# ´´ඉක්බිති මම ශිෂ්‍යයන්හට මේ ඎෂිතෙමේ කාමභොගියෙකැයි කීමි. මානවකයෝද එසේ කියන්නාවූ මාගේ (කථාවට) අනුව සතුටු වූවාහුය.
# ´´ඉක්බිති සියළු මානවකයෝ ගෙයක් ගෙයක් පාසා පිඬු පිණිස යන්නාවූ මේ ඎෂිතෙම කාමභෝගියෙකැයි මහජනයාට කීවාහුය.
# ´´ඒ සියළු මානවකයෝ මේ පන්සියයක් භික්‍ෂූහුම වෙති. එම කර්‍ම විපාකය කරණකොටගෙන ඔවුහු සුන්‍දරිකා පරිබ්‍රාජිකාව නිමිතිකොට නින්දා ලැබූවාහුය.
# ´´පෙර මම (එක පියෙකුට දාව අන් මව් කුසකින් උපන්) ෙවමාතික (කණිටු) සොහොයුරා ධනය හේතුකොටගෙන නැසීමි. ගිරි දුර්‍ගයකටද හෙළීමි. (හෙළා) ගලකින්ද ගැසීමි.
# ´´එම කර්‍ම විපාකය කරණකොටගෙන දෙව්දත් තෙමේ ගලක් පෙරලීය. (ඉන් නැගුණු) ගල් පතුරක් මාගේ පයෙහි මහපටැඟිල්ලට වැදුනේය.
# ´´පෙර මම (කුල) දරුවෙක්ව මහ මගෙහි ක්‍රීඩා කරමින් (එම) මගෙහි (වඩින) පසේ බුදුවරයන් වහන්සේ කෙනෙකු දැක ගල් කැබෙල්ලක් (උන්වහන්සේ දෙසට) විසි කෙළෙමි.
# ´´එම කර්‍ම විපාකය කරණකොටගෙන මෙම අන්තිම භවයෙහි දෙව්දත් තෙමේ මා මැරීම පිණිස දුනුවායන් යෙදවීය.
# ´´මම පෙර ඇතරුවෙක් වීමි. පිඬු පිණිස හැසිරෙන්නාවූ උතුම් පසේබුදුවරයන් වහන්සේ කෙනෙක් දැක ඇතා ලවා උන්වහන්සේ ගැටවීමි.
# ´´එම කමී විපාකය කරණකොටගෙන ගිරිබ්බජ (මගධරට පුරාණ රාජධානිය) නම් උතුම් නගරයෙහිදී (මත්වී) භ්‍රාන්තවූ නාලාගිරි ඇත්තෙමේ දරුණුවූයේ මා වෙත එළඹියේය.
# ´´මම පෙර පෘථිවීශ්වරවූ රජෙක් වීමි. (එකල්හි) ආයුධය (කඩුව) කින් (ගසා) පුරුෂයෙකු මැරීමි. එම කර්‍ම විපාකය කරණකොට ගෙන බොහෝ කලක් නිරයෙහි පැසුනෙමි.
# ´´එම කර්‍මයාගේ ශේෂවූ විපාකයෙන් මෙකල මාගේ පාදයෙහි සියළු සිවිය (ඇතුළත ලේ ගැලීම් නිසා) නරක්විය. කර්‍මය නැති නොවේ.
# ´´මම කෙවුල් ගමක කෙවුල් දරුවෙක් වීමි. (එකල්හි) නසන ලද මසුන් දැක සොම්නස ඉපදවීමි.
# ´´එම කර්‍ම විපාකයෙන් මාගේ හිසෙහි (දුක්) වේදනාවක් ඇතිවිය. යම් කලෙක්හි විඩූඪභ තෙමේ යුද කෙළේද, එකල්හි සියළු ශාක්‍යයෝද නැසුනාහුය.
# ´´ඵුස්ස බුදුරජාණන් වහන්සේගේ සසුනෙහි (එම) ශ්‍රාවකයන්හට ´´යව කව්´´ ´´හැල් අනුභව නොකරව්´´ යයි නින්දා කෙළෙම්ද,
# ´´එම කර්‍ම විපාකයෙන් බමුණෙකු විසින් ආරාධනය කරන ලදුයේ වෙරඤ්ජාවෙහි වසමින් එකල්හි තෙමසක් මුළුල්ලෙහි යව අනුභව කරණ ලදී.
# ´´(පෙර) මල්ල යුද්‍ධයක් පවත්නා කල්හි (ප්‍රති මල්ලව ආ) මල්ල පුත්‍රයෙක් පෙලීමි. එම කර්‍ම විපාකය කරණකොටගෙන මාගේ පිටෙහි (පවත්නා) දුක්ඛ වේදනාවක් විය.
# ´´මම පෙර වෛද්‍යවරයෙක් වීමි. (ධනය ගනු කැමතිව) එක් සිටු පුත්‍රයෙක් විරේක කරවීමි. එම කර්‍මය කරණකොටගෙන මට ලෝහිත පක්ඛන්දිකා (ලේ අතීසාරය) වැළඳුනේ වේ.
# ´´(පෙර) ජෝතිපාල මානවකවූ මම එකල්හි කාශ්‍යප බුදුරජාණන් වහන්සේට ´´හිස මුඩු කළහුට (සම්බෝධි) ඥානයක් කොයින්ද, (තොපට) (සම්බෝධි) ඥානය ඉතා දුර්ලභය´´ යි කීමි.
# ´´එම කර්‍ම විපාකය කරණකොටගෙන උරුවෙලායෙහි සාවුරුද්දක් බොහෝ දුෂ්කර ක්‍රියා කෙළෙමි. ඉක්බිති සම්‍යක් සම්බෝධියට පැමිණියෙමි.
# ´´පූර්‍ව කර්‍මයෙන් (සැප) වළක්වන ලදුයේ නොමගින් බුදුබව සොයන්නෙමි. මම (එම) නොමගින් උතුම් සම්‍යක් සම්බෝධියට නොපැමිණියෙමි.
# ´´කුසල් හා අකුසල් (රැස් කිරීමේ ශක්තිය) ක්‍ෂය කරණ ලදුයේ, සියළු (කෙළෙස්) තැවුල් බැහැරකරණ ලදුයේ, ශෝක රහිතවූයේ, වෙහෙසීම් රහිතවූයේ, ආශ්‍රව රහිතවූයේ පරිනිර්‍වාණයට පත්වන්නෙමි.
# ´´සියළු අභිඥා බලයනට පත් සර්‍වඥයන් වහන්සේ අනවතප්ත මහ විල් (තෙර) හිදී බික් සඟන ඉදිරියෙහි මෙසේ වදාළ සේක. මෙහි මේ අයුරින් භාග්‍යවතුන් වහන්සේ තමන් වහන්සේගේ පූර්‍ව චරිතය (දැක්වෙන) පුබ්බකම්මපිලොතික නම්වූ ධර්‍ම පරියායය වදාළ සේක.
