 කිඞ්කනි පුප්ඵිය වග්ග.   
# තිකිඞ්කණි පුප්ඵිය ථෙරාපදානය.
# ´´කිණිහිරි රුකක් මෙන් බබලන්නාවූ, කඳු අතරෙහි වැඩ හුන්නාවූ, ලෝක නායකවූ, කෙළෙස් විරහිත විපස්සී බුදුරජුන් දුටිමි.
# ´´කිංකණි පුෂ්ප තුණක් ගෙන (එම බුදුරජුන්හට) පිදීමි. සම්බුදුරජුන් පුදා දකුණු දිගට මුහුණ ලා යෙමි.
# ´´මම මැනවින් කරන ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි.
# ´´මින් එක් අනූවන කපෙහි යම් බුද්‍ධ පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් තිකිඞ්කනි පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# පංසුකූල පූජක ථෙරාපදානය.
# ´´හිමවතට නුදුරෙහි උදඞ්ගණ නම් පර්‍වතයක් විය. එහි ගස් මුදුනක එල්වන ලද පාංශුකූල චීවරය දුටිමි.
# ´´එකල්හි මම කිඞ්කණි පුෂ්ප තුණක් නෙලාගෙන සතුටුවූයේ, සතුටු සිතින්ම (එම) පංශුකූල චීවරයට පිදීමි.
# ´´මැනවින් කරන ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි.
# ´´මින් එක් අනූවන කපෙහිවූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, එම අර්හත් ධජය පුදා (ඒ හේතුවෙන්) දුගතියක් නොදනිමි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් මොරහත්‍ථිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කොරණ්ඩ පුප්ඵිය ථෙරාපදානය.
# ´´මම පෙර පිය මුතුන් පරපුරෙහි පටන්ම වනයෙහි කටයුතු කරන්නෙක් වීමි. සිව්පා සතුන් මැරීමෙන් ජීවත් වෙමි. මට කුසලයක් නැත.
# ´´මාගේ අසපුව අබියෙස පසැස් ඇති, ලෝකාග්‍රනායකවූ තිස්ස බුදුරජතෙමේ අනුකම්පාවෙන් පියවර තුණක් දැක්වූහ.
# ´´තිස්ස බුදුරජුන්ගේ තබනලද පියවර සටහන් දැක සතුටුවූයේ, සතුටු සිතින් පියවර සටහනෙහි සිත පැහැදවීමි.
# ´´පිපීගිය කරඬු ගසක් දැක, කොපුව සහිතවම (එම මල්) ගෙන ශ්‍රීපාද ලාංඡනය පිදීමි.
# ´´මම මැනවින් කරන ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි.
# ´´මම දිව්‍යවූ හෝ, මනුෂ්‍යවූ හෝ යම් යම් යෝනියක උපදිම්ද, ප්‍රභා සහිතවූ, කොරණ්ඩ පුෂ්ප වර්‍ණවූ සිවි ඇත්තේ වෙමි.
# ´´මින් දෙයානූවන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පියවර සටහන් පිදීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් සීහාසන වීජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කිංසුක පුප්ඵිය ථෙරාපදානය.
# ´´පිපීගිය කෑල ගසක් දැක, ඇඳිලි බැඳගෙන, බුද්‍ධ ශ්‍රේෂ්ඨයන් වන්සේ සිහිපත්කොට (ගසේ තිබියදීම) අහසෙහි පිදීමි.
# ´´මම මැනවින් කරණ ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි.
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් ති උක්කාධාරක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# උපඩ්ඪ දුස්ස දායක ථෙරාපදානය.
# ´´එකල්හි පදුමුත්තර බුදුරජාණන් වහන්සේගේ සුජාත නම් ශ්‍රාවකතෙමේ පංශුකූල (සිවුර සඳහා රෙදි කැබලි) සොයමින් කසල (එකතුකොට) ඇතිතන්හි හැසිරෙත්.
# ´´මම හංසවතී නගරයෙහි අනුන්ගේ (මෙහෙ කරන) බැලයෙක් වීමි. (ඒ මම උන්වහන්සේට) අඩවතක් (රෙදි කැබෙල්ලක්) දී හිස නමා වැන්දෙමි.
# ´´මම මැනවින් කරන ලද එම කර්‍මය කරණකොගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, මිනිස් සිරුර හැර, තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි.
# ´´තෙතිස් වරක් දේවේන්‍ද්‍රව දිව්‍ය රාජ්‍යය කරවීමි. මම සත් සැත්තෑ වරක් සක්විති රජ වීමි.
# ´´(කරවන ලද) මහත්වූ ප්‍රදේශ රාජ්‍ය (මෙතෙකැයි) ගණන් වශයෙන් ගිණිය නොහැකිය. අඩවත් දනින් (භය විරහිත බැවින්) අකුතොභයවූ නිවන පසක් කළේ වෙමි.
# ´´මම අද කැමතිවන්නේ නම් වන පර්‍වත සහිතවූ (සියල්ල) කොමු පිළිවලින් වහන්නෙමි. අඩවත් දනෙහි මේ විපාකයයි.
# ´´මින් සියක් දහස්වන කපෙහිවූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. අඩවත් දනෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් අක්කමන දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ඝත මණ්ඩදායක ථෙරාපදානය.

# ´´මනාකොට සිතන ලද්දාවූ, ලෝක ජ්‍යෙෂ්ඨවූ, නරශ්‍රේෂ්ඨවූ, වාතාබාධයකින් පෙළුනාවූ, වැඩහුන්නාවූ, මහාවීරවූ භාග්‍යවතුන් වහන්සේ දැක, සිත පහදවා, ඉතා පිරිසිදු ගිතෙල් ටිකක් පිළිගැන්වීමි. (මෙම පුණ්‍ය කර්‍මය) කරන ලද බැවින්, රැස් කරන ලද බැවින් මේ භාගීරසී ගඟද,
#
# ´´සිව් මහ සමුදුරද මට ගිතෙල් සපයති. අප්‍රමාණවූ අසංඛවූ මේ මහත් පෘථිවියද,
# ´´මාගේ අදහස දැන මී සකුරු (බවට පත්) මේ සිව් මහ දිවයිනෙහි වූ ඒ මේ වෘක්‍ෂ සමූහයද,
# ´´මාගේ අදහස දැන කල්ප වෘක්‍ෂයෝ වෙති. පණස් වරක් දේවෙන්‍ද්‍රව දිව්‍ය රාජ්‍යය කෙළෙමි.
# ´´මම එක් පණස් වරක් සක්විති රජ වීමී. (කරවණ ලද) මහත්වූ ප්‍රදේශ රාජ්‍යය ගණන් වශයෙන් (මෙතෙකැයි) ගිණිය නොහැකිය.
# ´´මින් සානූවන කපෙහි වූ එකල්හි යම් (ගිතෙල්) දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිරිසිදු ගිතෙල් දනෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් ඝත මණ්ඩදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# උදක දායක ථෙරාපදානය.
# ´´පදුමුත්තර බුදුරජුන්ගේ අනුත්තරවූ භික්‍ෂු සංඝයා කෙරෙහි පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, පැන් කළයක් පිදීමි.
# ´´පර්‍වත මුදුනෙහිදී හෝ, ගස මුදුනෙහිදීහෝ, අහසෙහිදී හෝ, පොළොවෙහිදී හෝ යම් කලෙක්හි පැන් කැමති වෙම් නම් වහා මට (පැන්) පහළ වේ.
# ´´මින් සියක් දහස්වන කපෙහිවූ එකල්හි යම් (පැන්) දනක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිරිසිදු ගිතෙල් දනෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් උදක දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# පුලින ථූපිය ථෙරාපදානය.
# ´´හිමවතට නුදුරෙහි යමක නම් පර්‍වතයක් විය. (එහි) මට මැනවින් කරන ලද අසපුවක් විය. පන්සැලක්ද මොනවට නිම කරන ලදී.
# ´´(මම) නම්න් නාරද නම්වූ, උග්‍ර තපස් ඇති ජටිලයෙක් වෙමි. තුදුස් දහසක් ශිෂ්‍යයෝ මා පිරිවරත්.
# ´´මම එකල්හි විවේකව හුන්නේ, මෙසේ සිතීමි. සියළු ජන තෙමේ මා පුදයි. මම (කිසිවෙකුට) කිසිවක් නොපුදමි.
# ´´මට කිසි ඔවා දෙන්නෙක්ද නැත. (කිසිවෙකුට කරන) කිසි වතාවතකුදු නැත. මම ආචාර්‍ය්‍ය, උපාධ්‍යාය විරහිතව වනයෙහි වෙසෙමි.
# ´´මම යමෙකු වෙත එළඹෙන්නේ නම් හේ (මා කෙරෙහි) ගුරුවරයෙක යන සිත උපදවා ගන්නේය. මට එබඳු ගුරුවරයෙකු නැත. වනයෙහි විසීම නිරර්ත්‍ථකය.
# ´´මම එසේ ගරුභාවනීයවූ උතුමෙකු සොයන්නෙමි. (එබඳු උතුමෙකු) ඇසුරු කෙළේ වසමි. එවිට (කිසිවෙක්) (මට) නොගරහන්නේය.
# ´´නොගැඹුරු ඉවුරු ඇති, මනාවූ කුඩා තොටුපොල ඇති, මනොඥවූ සුදෝ සුදු වැලිතලාවන්ගෙන් ගැවසීගත් කුඩා ගඟක් මගේ අසපුවට නුදුරෙහි විය.
# ´´(එය) අමරිකා නම්වූ නදියකි. එකල්හි මම (එතැනට) එළඹ, වැලි එකතුකොට වැලි සෑයක් (පුලින චෛත්‍යයක්) කෙළෙමි.
# ´´භවය කෙළවර කරන්නාවූ යම් ඒ සර්‍වඥ මුනිවර කෙනෙක් වෙත් නම් මෙබඳුවූ සෑයක් (තනන්නේ) උන්වහන්සේ නිමිතිකොට මම මේ සෑය කරමි.
# ´´වැලිගොඩෙහි සෑයක් කොට මම ස්වර්‍ණ නිර්‍මාණය කෙළෙමි. රන් කිකිණි මල් තුන් දහසක් පූජා කෙළෙමි.
# ´´හටගත් ප්‍රීති ඇත්තේ, බඳින ලද ඇඳිලි ඇත්තේ සවස හා උදයෙහි නමදිමි. සම්බුදුන් හමුයෙහිදී මෙන් පුලින චෛත්‍යය (වැලි සෑය) වඳිමි.
# ´´යම් කලෙක්හි ගිහිගෙට අයත් ක්ලේශ විතර්‍කයෝ හටගනිත්ද, එකෙණෙහි සර්‍වඥ ස්ථූපය සිහි කරමි. ප්‍රත්‍යවෙක්‍ෂා කරමි.
# ´´සාර්ත්‍ථවාහකවූ, විනායකවූ බුදුරජුන් ඇසුරුකොට වසන්නේ, ක්ලේශයන්හි ගැටී වසන්නෙහි නම් නිදුකාණෙනි, (එය) ඔබට නුසුදුසුය.
# ´´එකල්හි සෑය සිහිකළ කෙනෙහි මාගේ සිත්හි ගෞරවය ඇතිවේ. අකුස්සෙන් පෙළන ලද ඇතෙකු මෙන් මිත්‍ථ්‍යා විතර්‍කයන් දුරු කෙරෙමි.
# ´´මෘත්‍යුරාජයා මෙසේ වසන්නාවූ මා වැසීය. මම එහි කළුරිය කළේ, බඹලොවට ගියෙමි.
# ´´මම එහි දිවි ඇතිතාක් වසා, තාවතිංස දෙව්ලොවෙහි උපනිමි. අසූවරක් දේවේන්‍ද්‍රව දිව්‍ය රාජ්‍යය කරවීමි.
# ´´මම තුන්සිය වරක් සක්විති රජ වීමි. (කරවන ලද) මහත්වූ ප්‍රදේශ රාජ්‍යය ගණන් වශයෙන් (මෙතෙකැයි) ගිණිය නොහැකිය.
# ´´මම එම පිදූ කිංකිණි පුෂ්පයන්ගේ විපාකය අනුභව කරමි. භවයෙහිදී සියක් දහසක් කිරි මව්වරු මා පිරිවරත්.
# ´´සෑය පිළිදැගුම් කරන ලද බැවින් දැලි කුණු නොතැවරේ. සිරුරෙහි ඩහ නොවැගිරෙයි. මම ප්‍රභා සහිත (පැහැපත්) වූයේ වෙමි.
# ´´ආශ්චර්‍ය්‍යයකි, මා විසින් මැනවින් කරන ලද සෑයකි. (එසේම) ඇමෙරිකා නදියද මැනවින් දක්නා ලදී. වැලිතලාවෙහි සෑයක් කොට අචලපදය වූ නිවණට පැමිණියේ වෙමි.
# ´´කුසල් කරනු කැමැත්තාවූ (නිර්‍වාණ) සාරය ගන්නාවූ සත්ත්‍වයා විසින් (එයට) මෙය කෙතය, මෙය නොකෙතයයි (වෙන්කොට ගතයුත්තක්) නැත. ප්‍රතිපත්ති තොමෝම අර්ත්‍ථසාධක වන්නීය.

# ´´බලවත් පුරුෂයෙක් තෙම සාගරය තරණය කරන්නට උත්සාහ කරන්නේද, යම්සේ (හේ) කුඩා දඬු කැබැල්ලක් ගෙන මහ සයුරට පණින්නේ නම්, මම මේ දඬු කැබෙල්ල නිසා මහ සයුර තරණය කරන්නෙමැයි උත්සාහයෙන්ද, වීර්‍ය්‍යයෙන්ද (එම) මනුෂ්‍යතෙම සයුර කරණය කරන්නේය.

# ´´එසේම මා විසින් යම් කර්‍මයක් කරන ලද නම් (එය) කුඩාය. ඉතාම සුළුය. (එහෙත්) එම කර්‍මය නිසා සසර ඉක්මවීමි.
# ´´එම කුශල මූලයෙන් මෙහෙයන ලදුයේ, අන්තිම භවය පැමිණි කල්හි, පළමුකොටම සැවැත්නුවර ආඪ්‍යවූ මහාසාර කුලයෙහි උපන්නේ (වෙමි.)
# ´´මාගේ දෙමාපියෝ සැදැහැත්තාහුය. බුදුරජුන්ගේ සරණ ගියෝය. මේ දෙදෙනාම දක්නා ලද (නිර්වාණ) පදය ඇත්තෝය. ශාසනයටද අනුව පවතිත්.
# ´´(ඔවුහු) බෝධියෙහි පතුරක් ගෙන රන් සෑයක් කරවූහ. බුදුරජුන්ගේ හමුවෙහි උදේ සවස නමස්කාරද කෙරෙත්.
# ´´පොහෝ දිනයෙක්හි ස්වර්‍ණ ස්තූපය පිටතට ගන්නාහුය. බුදුරජුන්ගේ ගුණ කියමින් ත්‍රියාම රාත්‍රියම ගත කළාහුය.
# ´´මම (ස්වර්‍ණ) ස්තූපය දැක, ඒ සමගම පුලින චෛත්‍යය (අතීතයෙහි පිදූ වැලි සෑය) සිහි කෙළෙමි. එම අසුනෙහිම හිඳ රහත් බවට පැමිණියෙමි. 
#දෙවිසිවන බණ වරයි. 
# ´´එම වීරවූ ( බුදුරජුන්) සොයන්නේ, ධර්‍ම සේනාපතීන් වහන්සේ (සැරියුත් තෙරුන්) දුටිමි. ගිහිගෙන් නික්ම උන්වහන්සේගේ සමිපයෙහි පැවිදිවීමි.
# ´´උත්පත්තියෙන් සත් හැවිරිදි කල්හි රහත් බවට පැමිණියෙමි. පසැස් ඇති බුදුරජතෙම (මාගේ) ගුණ දැන (මා) උපසම්පදා කරවූහ.
# ´´මා විසින් දඟර කල්හිම (මාගේ) (නිවන් සඳහා කළයුතු) කටයුතු නිමකරන ලදී. ශාක්‍යපුත්‍රවූ බුදුරජුන්ගේ සසුනෙහි කටයුතු දෙය අද මා විසින් කරන ලදී.
# ´´මහා වීරවූ ඎෂීන් වහන්ස, සියළු මෛර හා භය ඉක්මවන ලදුයේ වෙමි. සියළු කෙලෙසුන්ද ඉක්මවන ලදුයේ වෙමි. (මම) ඔබ වහන්සේගේ ශ්‍රාවකයෙක් වෙමි. ස්වර්‍ණ පුෂ්ප පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් පුලින ථූපිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# නළකුටි දායක ථෙරාපදානය.
# ´´හිමවතට නුදුරෙහි හාරිත නම් පර්‍වතයක් විය. එකල්හි සියල්ල තෙමේම දත් නාරද බුදුරජතෙම රුක් මුලක වැඩ විසූහ.
# ´´මම බටවලින් කුටියක් කොට තෘණවලින් සෙවෙනි කෙළෙමි. (එසේම) මම සක්මනක්ද පිරිසිදු කරවා බුදුරජුන්හට දුනිමි.
# ´´මම මැනවින් කරන ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට ගියෙමි.
# ´´බටවලින් කළ කුටිය පිදීම හේතුකොගෙන එහි (දෙව්ලොව) මට මැනවින් කරන ලද නිර්මිතවූ විමානය සැට යොදුනක් උසය. තිස් යොදුනක් පුළුලුය.
# ´´මම තුදුස් කපක් මුළුල්ලෙහි දෙව්ලොව සිත් අලවා විසීමි. එක් සැත්තෑවරක් දිව්‍ය රාජ්‍යය කරවීමි.
# ´´මම සූතිස්වරක් සක්විති රජ වීමි. (කරවන ලද) මහත්වූ ප්‍රදේශ රාජ්‍යය ගණන් වශයෙන් මෙතෙකැයි ගිණිය නොහැකිය.
# ´´සියළු ආකාරයෙන් උතුම් උපමා ඇති ධර්‍ම නමැති ප්‍රාසාදයට නැගී ශාක්‍යපුත්‍රවූ බුදුරජුන්ගේ සසුනෙහි මම කැමතිතාක්ම වසමි.
# ´´මින් තිස් එක්වන කපෙහි වු එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බටින් කළ කුටියක් පිදීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් නළකුටි දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# පියාල ඵලදායක ථෙරාපදානය.
# ´´පෙර (මම) මුව වැද්දෙක් වීමි. එකල්හි වනයෙහි හැසිරෙන්නේ සියළු ධර්‍මයන්ගේ පරතෙරට ගිය කෙළෙස් රජස් විරහිත බුදුරජුන් දුටිමි.
# ´´මම පියල් ගෙඩියක් ගෙන පහන්වූයේ, සිය සිතින්ම පින්කෙත්වූ වීරවූ බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේට දුනිමි.
# ´´මින් එක්තිස්වන කපෙහි වූ එකල්හි යම් (පියල්) දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
