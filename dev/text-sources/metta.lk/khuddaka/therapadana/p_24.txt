# තුවරදායක ථෙරාපදානය.
# ´´මම පෙර මහ වනයෙහි මුව වැද්දෙක් වීමි. මම බරකොට ගෙනවුත් තුවර ඇට (බැද) සංඝයාට දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. තුවර දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තුවරදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# නාගකෙසරිය ථෙරාපදානය.
# ´´මම (මුවන් මැරීම සඳහා) දුන්න සරසාගෙන වනයට ඇතුළු වීමි. උඩට නැගුණු මල්පෙති පිපුණු කුඩා විලක් දැක
# ´´(එම මල්) දෝතින් ගෙන හිසෙහි ඇඳිලි බැඳගෙන ලෝකබන්‍ධූවූ තිස්ස බුදුරජුන්හට පිදීමි.
# ´´මින් දෙඅනූවන කපෙහි යම් බුද්ධ පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´තෙසැත්තෑවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති ´´කෙසර´´ නම් සක්විති රජකෙනෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් නාගකෙසරිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# නළිනකෙසරිය ථෙරාපදානය.
# ´´මම ස්වයංජාත විලක් මැද ජල කුක්කුටයෙක්ව වසමි. ඉක්බිති (එහිදී) අහසෙහි වඩින්නාවූ බුදුරජුන් දුටුවෙමි.
# ´´තුඩින් මල් ගෙන පැහැදි සිතින් ලෝක බන්ධුවූ තිස්ස බුදුරජුන්හට පිදීමි.
# ´´මින් දෙයානූවන කපෙහි යම් නෙළුඹු මල් පිදීමක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´තෙසැත්තෑවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති සතපත්ත නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් නළිනකෙසරිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# විරවි පුප්ඵිය ථෙරාපදානය.
# ´´ලෝකනායකවූ (සිද්‍ධාර්ත්‍ථ) බුදුරජාණන් වහන්සේ මග වඩින සේක. එකල්හි (මම) ශබ්ද පවත්වන කුසුම් වර්‍ගයක්වූ විරවි පුෂ්පයන්ගෙන් (එම) බුදුරජුන්හට පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් විරවි පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කුටිධූපක ථෙරාපදානය.
# ´´මම සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේගේ ගඳකිළියෙහි වතාවත් කරන්නෙක් වීමි. කලින් කල පහන් සිතැත්තේ සියතින් දුම් ගැස්වීමි.
# ´´මින් සිව් අනූවන කපෙහි එකල්හිවූ යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කුටිධූපක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පත්තදායක ථෙරාපදානය.
# ´´(කාය - වචී - මනො මක්ඛකාවන්ගෙන් තොර බැවින්) ඎජු ගුණ ඇති (අටලෝ දහමෙහි නොසැලෙන බැවින්) තාදී ගුණ යුත්, උතුම්වූ දමථ (සන්සිඳීම්) යෙන් යුත් සිද්‍ධාර්ත්‍ථ බුදුරජුන්හට මා විසින් පාත්‍ර දානයක් (පාත්‍රයක්) දෙන ලදී.
# ´´මින් සිව්අනූවන කපෙහි වූ එකල්හි යම් (පාත්‍ර) දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පාත්‍ර දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පත්තදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ධාතු පූජක ථෙරාපදානය.
# ´´ලෝකනාථවූ සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේ පිරිනිවි කල්හි තාදී ගුණ ඇති බුදුරජුන්ගේ එක් ධාතූන් වහන්සේ නමක් මා විසින් ලබන ලදී.
# ´´සූර්‍ය්‍ය බන්ධුවූ බුදුන්ගේ ඒ ධාතූන් වහන්සේ රැගෙන නරෝත්තමයන් වහන්සේ වැඩ සිටින්නාක් මෙන් පස් වසක් පූජා විධි පැවැත්වීමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් ධාතු පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ධාතූන්ට උවටැන් කිරීමෙහි පිදීමෙහි විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ධාතු පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සත්තලි පුප්ඵිය ථෙරාපදානය.
# ´´වෙස්සභූ බුදුරජුන් ලොව පහළවු කල්හි මම සත්තලි නම් මල් සතක් හිසෙහි කොට එම බුදුජුන්හට පිදීමි.
# ´´මින් එක් තිස්වන කල්පයෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සත්තලි පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# බිම්බි ජාලිය ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කළාවූ, අග්‍ර පුද්ගලවූ පදුමුත්තර බුදුරජතෙම චතුස්සත්‍යය දේශනා කරන සේක. අමෘත පදය (නිවණ) දක්වන සේක.
# ´´එකල්හි මම රත්කරව් මල් කඩා තාදී ගුණ ඇති විපදෝත්තමවූ පදුමුත්තර බුදුරජුන්හට පිදීමි.
# ´´මින් අට සැටවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති කිඤ්ජකෙසර නම් සිව් සක්විති රජ කෙනෙක් වූහ.
# ´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් බිම්බි ජාලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උද්දායක ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කළාවූ, පස්මරුන් ජයගත්, කකුධ නම් බුදුරජතෙමේ වැඩ වෙසෙන විහාරයෙන් නික්ම මහා නදියට වැඩි සේක.
# ´´මම එකල්හි ඇසල (මල්) ගෙන සන්සුන්වූ, ඎජු ගුණ ඇති බුදුරජුන්හට පහන් සිත් ඇත්තේ පිදීමි.
# ´´මින් එක් තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප දානයෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උද්දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
