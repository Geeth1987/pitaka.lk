 පදුමුක්ඛෙප වග්ග.  
# ආකාසුක්ඛිපිය ථෙරාපදානය.
# ´´ඇතුළු වීථියෙහි වඩින්නාවූ, ස්වර්‍ණ වර්‍ණවූ, නරශ්‍රේෂ්ඨවූ සිද්‍ධාර්ත්‍ථ බුදුරජුන් (දිටිමි.) (දැක) ජලයෙහි හටගන්නාවූ පියුම් උපුල් වැනි උතුම් මල් දෙකක් ගෙන උන්වහන්සේ වෙත එළඹියෙමි.
# ´´ඉන් එක මලක් ගෙන බුදුරජුන්ගේ ශ්‍රීපාදයට බිම දැමීමි. එකක් අහසට උඩ දැමීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප දානයාගේ මේ විපාකයයි.
# ´´මින් දෙතිස්වන කපෙහි පෘථිවීශ්වරවූ, මහත් බල ඇති අන්තලික්ඛවර නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආකාසුක්ඛිපිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තෙලමක්ඛිය ථෙරාපදානය.
# ´´නරශ්‍රෙෂ්ඨවූ සිදුහත් බුදුරජුන් පිරිනිවි කල්හි මම (උන්වහන්සේගේ) බෝධි වේදිකාවෙහි තෙල් ගැල්වීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප දානයාගේ මේ විපාකයයි.
# ´´මින් සූවිසිවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති සුච්ඡවි නම් වූ අට සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තෙලමක්ඛිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අඩ්භචන්‍දිය ථෙරාපදානය.
# ´´තිස්ස බුදුරජුන්ගේ උතුම් බෝරුක මුල මවිසින් අර්ධ චන්‍ද්‍රාකාරයෙන් නොයෙක් වර්‍ගයෙන් මල් පුදන ලදී.
# ´´මින් දෙයානූවන කපෙහි යම් පුෂ්ප පූජාවක් කරණ ලදද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් පස් විසිවවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති දෙවල නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අඩ්ඪචන්‍දිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අපණ්ණදීපිය ථෙරාපදානය.
# ´´එකල්හි දෙවියෙකු වූ මම පොළවට බැස (මිනිස් ලොවට අවුත්) පහන්වූයේ සියතින්ම පහන් පසක් පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් ප්‍රදීප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ප්‍රදීප දානයාගේ මේ විපාකයයි.
# ´´පස් පණස්වන කපෙහි පෘථිවීශ්වරවූ සමන්තචක්ඛු නම් මහත් බල ඇති සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අපණ්ණදීපිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# බිලාලිදායක ථෙරාපදානය.
# ´´හිමවතට නුදුරෙහි ´´රොමස´´ නම් පර්‍වතයෙක් විය. එම පර්‍වත පාදයෙහි පර්‍වතය කෙළවර කරණ ලද ඉඳුරන් ඇති භික්‍ෂුවක් විය.
# ´´මම බිළාලි නම්වූ අල ගෙනගොස් එම ශ්‍රමණයන් වහන්සේට දුනිමි. සියල්ල තෙමේම දත් මහා වීරවූ පස්මරුන් දිනූ එම බුදුරජතෙම අනුමෝදනාව කළ සේක. (කෙසේද යත්)
# ´´තා විසින් මට පැහැදි සිතින් බිළාලි අල දෙන ලදද (ඒ හේතුයෙන්) භවයෙහි උපදනා කල්හි තොපටද ඵල නිපදේවා´´ කියායි.
# ´´මම මින් සිව් අනූවන කපෙහි යම් බිළාලි දනක් දෙන ලදද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බිළාලි දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් බිළාලිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මච්ඡදායක ථෙරාපදානය.
# ´´මම එකල්හි චන්ද්‍රභාග නදී තීරයෙහි කබරුස්සෙක් (පක්‍ෂි විශෙෂයක්) වීමි. මම මත්ස්‍යයෙකු ගෙන සිද්‍ධාර්ත්‍ථ බුදුරජුන්ට දුනිමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් මත්ස්‍ය දානයක් දෙන ලදද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මත්ස්‍ය දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මච්ඡදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ජවහංසක ථෙරාපදානය.
# ´´එකල්හි මම චන්‍ද්‍රභාග නදී තීරයෙහි වැද්දෙක් වීමි. අහස් මගෙහි වඩින්නාවූ සිදුහත් බුදුරජුන් දිටීමි.
# ´´මම මහාමුනිවූ එම බුදුරජුන් බලමින්, ඇඳිලි බැඳ, සිය සිත පහදවා ලෝකනායකවූ උන්වහන්සේට වැන්දෙමි.
# ´´මින් සිව් අනූවන කපෙහි යම් බුදුවරයෙකු වැන්දෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. වන්‍දනාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ජවහංසක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සලලපුප්ඵිය ථෙරාපදානය.
# ´´මම එකල්හි චන්‍ද්‍රභාඝා නදී තීරයෙහි කිඳුරෙක් වීමි. (එහිදී) රශ්මි ජාලාවන්ගෙන් ගැවසීගත් විපස්සී බුදුරජුන් දුටුවෙමි.
# ´´මම පහන් සිතැත්තේ, සොම්නස්වූයේ, මහත්වූ ප්‍රීතියෙන් සලල මල් ගෙන (එම) විපස්සී බුදුරජුන්හට විසිරවීමි (පිදීමි.)
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙලෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සලලපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උපාගතාසය ථෙරාපදානය.
# ´´හිමවත මැද මැනවින් නිමාකළ විලක් විය. මම එහි අනුන් වෙහෙසෙන සුළු භයානක රකුසෙක් වීමි.
# ´´අනුකම්පාව ඇති, කරුණාවන්තවූ ඒ විපස්සී බුදුරජතෙම මා (එම දරුණු ජීවිතයෙන්) ගොඩලනු කැමැත්තේ මාගේ සමීපයට පැමිණි සේක.
# ´´මම එසේ පැමිණියාවූ, මහාවීරවූ, දෙවියනට දෙවිවූ, නරශ්‍රේෂ්ඨවූ (උන්වහන්සේ දුටිමි.) (දැක) සිටින තැනින් නික්ම ශාස්තෲන් වහන්සේට වැන්දෙමි.
# ´´මින් එක් අනූවන කපෙහි පුරුෂොත්තමවූ බුදුරජුන්හට යම් වැඳීමක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. වැඳීමෙහි මේ ප්‍රතිඵලයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උපාගතාසය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තරණීය ථෙරාපදානය.
# ´´දක්‍ෂිණාර්හවූ, ස්වර්‍ණවර්‍ණවූ, භික්‍ෂු සංඝයා විසින් පිරිවරණ ලද, ශාස්තෲවූ විපස්සී බුදුරජතෙම ගංතෙරෙහි සිටිසේක.
# ´´මහ මුහුදු තරණය කරණ නැවක් එහි නොවීය. මම ගඟින් නික්ම අවුත් ලෝකනායකවූ එම බුදුරජුන් එතර කරවීමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි බුදුරජුන් ගඟින් එතෙර කිරීම නම්වූ යම් පින්කමක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුදුරජුන් ගඟින් එතෙර කිරීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තරණීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
