# සෙරෙය්‍යක ථෙරාපදානය.

# ´´වේද හදාරන්නාවූ, මන්ත්‍ර දරන්නාවූ, වේදත්‍රයෙහි පරතෙරෙට ගිය (මම) එලිමහන් තන්හි සිටියේ, වනයෙහි හැසිරෙන්නාවූ, සිංහයෙකු වැනිවූද, තැති නොගන්නාවූද, ව්‍යාඝ්‍රයෙකු වැනිවූද, ත්‍රිමදගලිත මාතඞ්ග කුලයෙහිවූ හස්ති රාජයෙකු වැනිවූද, මහර්ෂීවූ ලෝකනායක බුදුරජුන් දුටිමි.
#
# ´´මම කටුකරඬු මල් ගෙන අහසෙහි දැමීමි. බුදුරජුන්ගේ ආනුභාවයෙන් (එම මල්) හැම අතින් පිරිවරත්.
# ´´මහා වීරවූ ලෝකනායක සර්වඥයන් වහන්සේ අදිටන් කළ සේක. (එම අදිටන් බලයෙන්) නරශ්‍රේෂ්ඨයන් වහන්සේට හාත්පස මල් වියනක් සේ විසිරුනාහ.
# ´´ඉක්බිති නැටි උඩුකුරුවූ ඒ පුෂ්ප කඤ්චුකය (මල් වියන) සතියක් මුළුල්ලෙහි සෙවෙනියක් කොට යලි අතුරුදහන් විය.
# ´´විෂ්මය ජනකවූ, ලොමු දැහැගන්නාවූ එම ආශ්චර්‍ය්‍යය දැක සුගතවූ, ලෝක නායකවූ බුදුරජුන් කෙරෙහි සිත පැහැදවීමි.
# ´´එම චිත්ත ප්‍රසාදය කරණ කොට ගෙන කුශල මූලයෙන් මෙහෙයන ලද්දේ මම කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නොඉපදුනෙමි.
# ´´එක් දහස් පන්සියවන කපෙහි මහත් බල ඇති චිත්කමාල නම් පස්විසි සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සෙරෙය්‍යක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පුප්ඵථූපිය ථෙරාපදානය.
# ´´හිමවතට නුදුරු තන්හි කුක්කුට නම් පර්‍වතයක් විය. එම පර්වතය අතුරෙහි මන්ත්‍රයන්හි පරතෙරෙට ගිය බ්‍රාහ්මණයෙක් විය.
# ´´හැම කල්හි මා පන්දහසක් ශිෂ්‍යයෝ පිරිවරත්. ඔවුහු හැමට පළමු උදූසන නැගිටින සුළුවූ, මන්ත්‍රයන්හි විශාරදයෝද වූහ.
# ´´එම ශිෂ්‍යයෝ මෙසේ කීහ. බුදුරජ තෙම ලොව පහළවිය. උන්වහන්සේ අසූවක් ව්‍යඤ්ජන ඇති සේක. දෙතිස් මහා පුරුෂ ලක්‍ෂණයක් ඇති සේක. එම ජිනෙන්‍ද්‍රයන් වහන්සේ හිරු මෙන් බබළන සේක. පින්වතුන් වහන්ස, අපගේ ඒ වචනය දැනගනු මැනව කියායි.
#
# ´´මන්ත්‍රයන්හි පරතෙරෙට ගිය බමුණු තෙමේ, ශිෂ්‍යයන්ගේ වචනය අසා, අසපුවෙන් නික්ම, ලොකනායකවූ මහා වීරයන් වහන්සේ යම් දිශාවක වැඩ වෙසෙන සේක් නම් ශිෂ්‍යයන්ගෙන් එම දිශාව විචාරයි. (කුමක් පිණිසද යත්?)
# ´´මම එම දිශාවටද අප්‍රති පුද්ගලවූ, පස් මරුන් දිනුවාවූ උන්වහන්සේටද නමස්කාර කරන්නෙමියි කියායි. ඉක්බිති ඔද වැඩි සිත් ඇත්තේ, සොම්නස්වූයේ, ඒ තථාගතයන් පිදීමි. (එසේ කොට ශිෂ්‍යයනට මෙසේ කීවෙමි.)
# ´´ශිෂ්‍යවරුනි, එව්. යන්නෙමු. ඒ තථාගතයන් දකින්නෙමු. ශාස්තෲන් වහන්සේගේ සිරිපා වැඳ, බුද්ධ ධර්‍මය අසන්නෙමුය කියායි.
# ´´නික්ම එක් දිනකින් මම රෝගී වීමි. රෝගයෙන් පෙළුනේ විසීම සඳහා ශාලාවකට ගියෙමි.
# ´´සියළු ශිෂ්‍යයන් රැස් කරවා තථාගතයන් කෙබඳුද? පරම බුද්‍ධිමත් ලෝකනාථයන් වහන්සේගේ ගුණ කෙබඳුදැයි ඔවුන්ගෙන් විචාළෙමි.
# ´´මා විසින් විචාරණ ලද ඔවුහු බුදුරජුන් දක්නා ලද අයුරින්ම විස්තර කළාහුය. මාගේ හමුයෙහි ඒ බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේ මැනවින් (විස්තර) කොට දැක්වූහ.
# ´´මම ඔවුන්ගේ එම වචනය අසා සිත පැහැදවීමි. මම එහි මල්වලින් සෑයක් කොට කළුරිය කෙළේ වෙමි.
# ´´ඔවුහු මාගේ සිරුර දවා බුදුරජුන් සමීපයට ගියාහුය. ඇඳිලි බැඳ ශාස්තෲන් වහන්සේට අභිවාදනය කළාහුය.
# ´´මම සුගතවූ, මහර්ෂීවූ බුදුරජාණන් වහන්සේට මල්වලින් සෑයක් කොට කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නොඉපදුනෙමි.
# ´´සතළිස් දහස්වන කපෙහි නම්න් අග්ගිසම නම් මහත් බල ඇති සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´විසිදහස්වන කපෙහි ඝතාසන නම්වූ පෘථිවීශ්වර අටතිස් රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පුප්ඵථූපිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පායාසදායක ථෙරාපදානය.
 
# ´´මහ ලොහො තලියක කිරිබතක් සාදාගෙන දනක් දෙනු කැමැත්තාවූ මම එම පූජාව දෙතිස් මහ පුරිස් ලකුණින් යුත් ස්වර්‍ණ වර්‍ණවූ භික්‍ෂු සංඝයා විසින් පිරිවරණ ලද අරණින් නික්මුණාවූ (විපස්සී) බුදුරජුන් වෙත් පැමිණවීමි.
#
# ´´ලෝක ජ්‍යෙෂ්ඨවූ, නරශ්‍රේෂ්ඨවූ භාග්‍යවත් තෙමේ එකල්හි සුදු වතක් වැනිවූ අහසෙහි (මවන ලද) සක්මනට නැගුනේ වේ.
# ´´විෂ්මය ජනකවූ, ලොමු දැහැගන්වන එම ආශ්චර්‍ය්‍යය දැක එම ලොහොතලිය (පසෙක) තබා විපස්සී බුදුරජුන්හට අභිවාදනය කෙළෙමි.
# ´´දෙවියන් සහිත ලෝකයෙහි ඔබ සර්‍වඥ දෙවි වෙහිය. මහාමුනිවූ බුදුරජාණන් වහන්ස, අනුකම්පා උපදවා මෙය පිළිගනු මැනව (කියායි.)
# ´´ලෝකයෙහි ශාස්තෘවූ, මහාමුනිවූ, ලෝකනායකවූ භාග්‍යවත් සර්‍වඥතෙමේ මාගේ අදහස දැන (එය) පිළිගත් සේක.
# ´´මින් එකානූවන කපෙහිවූ එකල්හි මම යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. කිරිබත් දනෙහි මේ විපාකයයි.
# ´´මින් එක්සාළිස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති (බුද්‍ධ) නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පායාසදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ගන්‍ධොදකිය ථෙරාපදානය.
# ´´පුෂ්ප ඵල පල්ලවයන්ගෙන් බබළන කුඹුක් ගසක් වැනි, මොහඳුරු දුරු කරන්නාවූ, පස් මරුන් දිනූ, විපස්සී බුදුරජාණන් වහන්සේ උතුම් පහයෙහි හිඳ දුටිමි.
# ´´පහයට නුදුරෙහි බුදුරජාණන් වහන්සේ වැඩම වන කල්හි සූර්‍ය්‍යයාගේ (ප්‍රභාව) මෙන් උන්වහන්සේගේ ප්‍රභාව විහිදෙයි.
# ´´සුවඳ පැන්ද ගෙන බුද්‍ධ ශ්‍රෙෂ්ඨයන් වහන්සේට ඉස්සෙමි. මම එම චිත්ත ප්‍රසාදයෙන් යුතුව කළුරිය කෙළෙමි.
# ´´මින් එක් අනූවන කපෙහි යම් සුවඳ පැනක් ඉස්සෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් එක් තිස්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති සුගන්‍ධ නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ගන්‍ධොදකිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සම්මුඛාත්‍ථවික ථෙරාපදානය.
# ´´මම විපස්සී බුදුරජුන් ලොව පහළවන කල්හි (ඇතිවූ නිමිති බලා) මෙසේ පැවසීමි. (කෙසේද යත්) ජන සමූහයා (කෙළෙස් ගින්නෙන්) නිවමින් බුදුරජ තෙම ලොව පහළ වන්නේය.
# ´´යම් උතුමෙකු උපදින කල්හි දශ දහසක් ලෝදා කම්පාවේද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි මහත් ආලෝකය වීද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි ගංගාවෝ නොගැලුවාහුද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි අවීචියෙහි ගිනි නොඇවිලුනේද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි පක්‍ෂි සමූහයා (ඒ මේ අත) නොහැසිරුනේද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි වාතස්කන්‍ධය නොහමාද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´යම් උතුමෙකු උපදින කල්හි සියලු රත්නයෝ බැබලුනාහුද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´උපන් කෙණෙහිවූ සර්‍වඥ තෙමේ සියලු දිගුන් බැලීය. අභීත වචන පැවසීය. මෝ බුදුවරයන්ගේ ධර්‍මතායි.
# ´´යම් උතුමෙකු උපදින කල්හි සප්තවිධ වික්‍රමයෝ වූවාහුද දැන් ශාස්තෘවූ පසැස් ඇති භාග්‍යවත් තෙමේ දහම් දෙසයි.
# ´´මෙසේ ජනතාව සංවේගයට පමුණුවා ලෝක නායකයන් වහන්සේට ස්තූතිකොට සර්‍වඥයන් වහන්සේට වැඳ නැගෙණහිරට මුහුණ ලා ගියෙමි.
# ´´මින් එකුන් අනූවන කපෙහි යම් බුද්ධ සතූතියක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ ස්තූතියෙහි මේ විපාකයයි.
# ´´මින් අනූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, සම්මුඛථවික නම් සක්විති රජෙක් විය.
# ´´මින් එකුන් අනූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, පඨවීදුන්‍දුභි නම් සක්විති රජෙක් විය.
# ´´මින් අටාසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, ඔභාස නම් සක්විති රජෙක් විය.
# ´´මින් සතාසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, සරිතච්ඡේද නම් සක්විති රජෙක් විය.
# ´´මින් සාසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, අග්ගිනිබ්බාපණ නම් සක්විති රජෙක් විය.
# ´´මින් පස්අසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, සතිපච්ඡේදන නම් සක්විති රජෙක් විය.
# ´´මින් සූ අසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, වාත්‍යම නම් සක්විති රජෙක් විය.
# ´´මින් තෙයාසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, රතනපජ්ජල නම් සක්විති රජෙක් විය.
# ´´මින් දෙයාසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, පදවික්කමන නම් සක්විති රජෙක් විය.
# ´´මින් එක් අසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, විලෝකන නම් සක්විති රජෙක් විය.
# ´´මින් අසූවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති, ගිරිසාර නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සම්මුඛාත්‍ථවික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කුසුමාසනිය ථෙරාපදානය.
 
# ´´මම එකල්හි නිඝණ්ඩු ශාස්ත්‍රය හා (කවීනට හිතවූ) කෙටුභය යන මේවා සහිත ලක්‍ෂණ ශාස්ත්‍රයෙහි ඉතිහාසයෙහිද දක්‍ෂවූ බමුණෙක් වීමි. වේදත්‍රයෙහි පරතෙරට ගියේ ශිෂ්‍යයන්ට මන්ත්‍ර කියවීමි.
#
# ´´එක තැනකට පමුණුවා, මම ඔවුනට පුදනු කැමැත්තේ වීමි. (ඒ සඳහා) මා විසින් උපුල් මිටි සතක් පුටුවක තබන ලදී.
# ´´එකල්හි නරශ්‍රේෂඨවූ විපස්සී බුදුරජ තෙම භික්‍ෂු සංඝයා විසින් පිරිවරණ ලදුයේ සියලු දිගුන් බබුළුවමින් වඩින සේක.
# ´´අසුනක් පණවා, ඒ මල් අතුරා, බුදුරජුන් පවරා සිය ගෙදොරට වැඩමවීමි.
# ´´ස්වකීය ගෘහයෙහි මා විසින් පිළියෙල කරන ලද යම් ආහාරයක් ඇද්ද, පහන් සිතැත්තේ එය මම සියතින්ම බුදුරජුන්හට දුනිමි.
# ´´එය වළඳා අවසන්වූ උන්වහන්සේට සුදුසු කළායි දැන මල් මිටක්ද දුනිමි. සර්‍වඥ තෙම අනුමෝදනා කොට උත්තරාභිමුඛව වැඩි සේක.
# ´´මින් සත් අනූවන කපෙහිවූ එකල්හි යම් පුෂ්ප දානයක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප දානයෙහි මේ විපාකයයි.
# ´´මින් අනතුරු කපෙහි (අනූවන) සත්රුවනින් යුත් මහත් බල ඇති වරදස්සන නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කුසුමාසනිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ඵලදායක ථෙරාපදානය.
# ´´වේද හදාරන්නාවූ, මන්ත්‍රධරවූ, වේදත්‍රයෙහි පරතෙරට ගිය මම හිමවතට නුදුරෙහි අසපුවක වසමි.
# ´´මාගේ අග්නි පූජාවක් ඇත. මා විසින් පියුම් ගෙඩි මුළක බැඳ, ගසක් මුදුනෙහි එල්ලන ලදී.
# ´´ලෝකයාගේ පූජාවන් පිළිගන්නාවූ නොහොත් පූජාවනට ගෙයක් බඳුවූ ඒ පදුමුත්තර බුදුරජතෙම (මා) සසරින් ගොඩනගනු කැමැත්තේ, පිඬු පිණිස හැසිරෙමින් මා වෙත එළඹි සේක.
 
# ´´මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, බුදුරජුන්හට එම ඵල දුනිමි. මාගේ ප්‍රීතිය ඇති කරන්නාවූ මෙලොවම සුව ඵල වන්නාවූ, ස්වර්‍ණ වර්‍ණවූ, ලෝවැස්සාගේ පූජාවන් පිළිගන්නාවූ, නොහොත් පූජාවනට ගෙයක් බඳුවූ, ශාස්තෘතෙමේ අහසෙහි වැඩසිටියේ මෙම ගාථාව වදාළ සේක.

# ´´මෙම ඵල දානය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නූපදී´´ කියායි.
# ´´එම කුශල මූලය කරණකොටගෙන සම්පත් අනුභවකොට ජය-පරාජය දෙක හැර, අචලස්ථානය (නිවණට) පැමිණියේ වෙමි.
# ´´මින් එක්සියවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති, සුමඞ්ගල නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ඤාණ සඤ්ඤක ථෙරාපදානය.
# ´´හිමවත් පව්වෙහි කඳු අතරක වෙසෙමි. ශෝභනවූ වැලි තලාවක් දැක, බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේ සිහි කළෙමි.
# ´´නුවණෙහි (සැසඳීමට) උපමාවක් නැත. ශාස්තෲන් වහන්සේට සංග්‍රාමයකුදු නැත. සියලු ධර්‍මයන් නුවණින් දැන (දුකින්) මිදේ.
# ´´පුරුෂ ශ්‍රෙෂ්ඨයන් වහන්ස, ඔබ වහන්සේට නමස්කාර වේවා. පුරුෂොත්තමයන් වහන්ස, ඔබ වහන්සේට නමස්කාර වේවා. උතුම් නුවණ යම්තාක්ද, (එතෙක්වූ) ඔබ වහන්සේගේ නුවණ හා සම කෙනෙක් නැත.
# ´´බුද්‍ධඤාණයෙහි (සිත) පහදවා කපක් මුළුල්ලෙහි මම ස්වර්‍ගයෙහි සතුටුවීමි. සෙසු කල්පයන්හි මා විසින් කුශල් කරන ලදී.
# ´´මින් එක්අනූවන කපෙහි වූ එකල්හි යම් ඥාණ සංඥාවක් ලැබූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ ඥාන සංඥාවෙහි මේ ප්‍රථිඵලයයි.
# ´´මින් තෙසැත්තෑවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති, පලිත පුෂ්ථිය නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඤාණ සඤ්ඤක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ගණ්ඨිපුප්පිය ථෙරාපදානය.
# ´´දක්‍ෂිණාර්‍හවූ, ස්වර්‍ණවර්‍ණවූ, විපස්සී බුදුරජාණන් වහන්සේ ශ්‍රාවකයන් විසින් පිරිවරණ ලදුයේ ආරාමයෙන් නික්මුණු සේක.
# ´´මම මොහඳුරු දුරු කරන්නාවූ, සර්‍වඥවූ, බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේ දැක, පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, ලද පස් මල් පිදීමි.
# ´´විපදෝත්තමවූ, තාදී ගුණ ඇති බුදුරජුන්හට (ඇතිකරගත්) ඒ චිත්ත ප්‍රසාදය කරණකොටගෙණ සතුටුවූයේ, සතුටු සිතින් නැවත තථාගතයන් වැන්දෙමි.
# ´´මින් එක්අනූවන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් එක්සාළිස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති, චරණ නම් සක්විති රජෙක් විය.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ගණ්ඨිපුප්පිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පදුම පූජක ථෙරාපදානය.
# ´´හිමවතට නුදුරු, නා නා වෘක්‍ෂයන්ගෙන් ගැවසීගත්, මහා භූත සමූහයාට වාසස්ථානවූ, ගොතම නම් පර්වතයක් විය.
# ´´ඒ මධ්‍යයෙහි නිර්මිත කරණ ලද ආශ්‍රමයක් විය. මම ස්වකීය ශිෂ්‍යයන් විසින් පිරිවරණ ලදුයේ අසපුවෙහි වෙසෙමි.
# ´´මාගේ ශිෂ්‍ය සමූහයෝ පැමිණෙත්වා, මට පියුම් ගෙන එත්වා. විපදෝත්තමවූ තාදී ගුණ ඇති බුදුරජාණන් වහන්සේට පූජා කරන්නෙමු.
# ´´ඔවුහු ´´එසේය´´ යි මට පිළිවදන් දී, පියුම් ගෙනාවාහුය. එසේම මම (බුදුරජාණන් වහන්සේ) නිමිතිකොට ඒ සර්‍වඥයන් වහන්සේට පිදීමි.
# ´´එකල්හි ශිෂ්‍යයන් රැස්කරවා මම මනාකොට අනුශාසනා කෙළෙමි. (කෙසේදයත්) ´තෙපි පමා නොවව්. අප්‍රමාදව සුව එලවන්නේ වෙයි´´ (යනුවෙනි.)
# ´´වනක්‍ෂමවූ එම ශිෂ්‍යයනට මෙසේ අනුශාසනා කොට එකල්හි මම අප්‍රමාදී ගුණයෙහි යෙදුනේ, කළුරිය කළේ වෙමි.
# ´´මින් එක්අනූවන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද (ඒ හේතුවෙන්) අගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´එක් පණස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති, ජලුත්තම නම් සක්විති රජ කෙනෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පදුම පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
