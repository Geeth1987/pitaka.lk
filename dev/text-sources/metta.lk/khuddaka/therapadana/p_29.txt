 චිතකපූජක වග්ග  
# චිතකපූජක ථෙරාපදානය.
# ´´එකල්හි මම අජිත නම් බමුණෙක් වීමි. පූජාවක් කරනු කැමැත්තේ නොයෙක් මල් රැස් කළෙමි.
# ´´සිඛී බුදුරජාණන් වහන්සේගේ ඇවිලෙන්නාවූ සෑය දැක මම එම මල් ගෙනවුත් සැයෙහි විසිරවීමි.
# ´´මින් එක් තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් සත් විසි වන කපෙහි මනුජාධිපතිවූ, මහත් බල ඇති, සුපජ්ජලිත නම් සක්විති රජ කෙනෙක් වූහ.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් චිතකපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පුප්ඵධාරක ථෙරාපදානය.
# ´´මම වැහැරි සිවුරු දරන්නාවූ, එකස්කොට අඳනා ලදවූ අඳුන් දිවි සම් ඇති, උපදවූ පංචාභිඥා ඇති, සඳ පිරිමැදුමෙහි සමත් (ඎද්‍ධිමත්) තවුසෙක් වීමි.
# ´´ලෝකයාට අලෝකයක්වූ, මා සමීපයට වැඩියාවූ විපස්සී බුදුරජුන් දැක, මම එම ශාස්තෲන් වහන්සේට පරසතු මල් (ගෙනවුත් කුඩයක් මෙන් කොට) දැරීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප (ඡත්‍රයක්) දැරූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (මල් කුඩය) දැරීමෙහි මේ විපාකයයි.
# ´´මින් සත් අසූවන කපෙහි පෘථිවීශ්වරවූ, මහත් බල ඇති සමන්තධාරණ නම් එක් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පුප්ඵධාරක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ඡත්තදායක ථෙරාපදානය.
# ´´එකල්හි මාගේ පුත්‍ර තෙමේ කසාවත් හැඳියේ පැවිදිවිය. හෙතෙම ශ්‍රාවක බෝධිය (රහත් බවට) පැමිණියේ, ලොකයා විසින් පුදනු ලැබූවෙක්ව පිරිනිවන් පෑහ.
# ´´මම සිය පුතා සොයමින් පස්සෙන් ගියෙමි. මම පිරිනිවන් පෑ රහතන් වහන්සේගේ සෑය වෙත ගියෙමි.
# ´´මම එහි ඇඳිලි බැඳගෙන සෑය වැඳ එකල්හි සුදු කුඩයක්ද ගෙන (සෑයට) නගා තැබීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් කුඩ (ඡත්‍ර) පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මේ ඡත්‍ර දානයාගේ විපාකයයි.
# ´´මින් පස් විසිවෙනි කපෙහි ජනාධිපතිවූ, මහත් බල ඇති මහාරහ නම් තුන් සක්විති රජකෙනෙක් වූහ.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඡත්තදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සද්දසඤ්ඤක ථෙරාපදානය.
# ´´හිරු උදාවන්නටත් පෙර ආලෝකය මහත් විය. (මෙය) මහර්ෂිවූ බුදුරජාණන් වහන්සේගේ ලොව පහළවීමයි.
# ´´(ගිලන් ඇඳෙහි වැතිර සිටි) මම එහි ශබ්ද ඇසීමි. ඒ බුදුරජුන් නොදිටීමි. මරණාසන්න අවස්ථාව පැමිණි කල්හිද බුද්‍ධ යන නාමය සිහි කෙළෙමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් බුද්‍ධ යන හැඟීමක් ලැබුයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ සංඥාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සද්දසඤ්ඤක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ගොසීස නික්ඛෙපක ථෙරාපදානය.
# ´´මා විසින් (භික්‍ෂු සංඝයා) අරම් දොරින් නික්මෙන (මඩ සහිත) මගෙහි ගව හිස් කබලක් අතුරණ ලදී. (ආජානීය අශ්ව වාහනාදිය ලැබීමෙන්) එකල තමා කළ ස්වකීය කර්‍මයේ විපාකය විඳිමි. පූර්‍ව කර්‍මයාගේ මේ විපාකයයි.
# ´´ආජානීය අශ්වයෝද, වාත වේගයෙන් දිවිය හැකි වාත ජව අශ්වයෝද, සෛත්‍ධව අශ්වයෝද යන සීඝ්‍ර වාහන (සම්පත්) අනුභව කරමි. මේ සියල්ල ගව හිස් කබල (සංඝයාගේ ගමන් පහසුව සඳහා) දැමීමේ විපාකයයි.
# ´´පුණ්‍යකර්‍මය ආශ්චර්‍ය්‍යයයි. මා විසින් සරු කෙතෙහි කරණ ලද (මෙය) උතුම් පුණ්‍ය කර්‍මයකි. සංඝයා විෂයෙහි කරණ ලද උපකාරය හා සසඳනා විට අනික් සත්කාරයක් (ඉන්) කලාවකටද නොඅගී.
# ´´මම මින් සිව් අනූවන කපෙහි යම් ගව හිස් කබලක් අතුළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ගමන් පහසුව ඇතිවන සේ සංඝයා සඳහා ගෙරි හිස් කබල ඇතිරීමේ මේ විපාකයයි.
# ´´පන් සැත්තෑවන කපෙහි මහත් බල ඇති, අතිශය තෙදවත් සුප්පතිට්ඨිත නම් එක් සක්විති රජකෙනෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ගොසීස නික්ඛෙපක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පාදපූජක ථෙරාපදානය.
# ´´එකල්හි (මම) හිමවත් පව්වෙහි කිඳුරෙක් වීමි. (එහිදී) පීත වර්‍ණ රැස් ඇති (උදා) හිරු මෙන් බබළන බුදුරජුන් දිටිමි.
# ´´(තමා සිටින තැනට) වැඩියාවූ, ලෝක නායකවූ, ඒ විපස්සී බුදුරජුන් (දැක) මම ශ්‍රී පාදයෙහි සඳුන්ද, තුවරලාද ඉස්සෙමි.
# ´´මින් එක් අනූවන කපෙහි යම් ශ්‍රීපාද පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ශ්‍රීපාද පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පාදපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# දෙසකිත්තක ථෙරාපදානය.

# ´´එකල්හි මම ´´උපසාල්හක´´ නම් බමුණෙක් වීමි. මහ වනයට පිවිසි, ලෝක ජ්‍යෙෂ්ඨවූ, නර ශ්‍රේෂ්ඨවූ, ලෝකයාගේ පූජාවන් පිළිගන්නාවූ බුදුරජුන් දැක ශ්‍රීපාදයන් වැන්දෙමි. පහන් සිත් ඇති මා ගැන දැන බුදුරජ තෙම අතුරුදන් විය.

# ´´වනයෙන් නික්ම බුද්‍ධශ්‍රේෂ්ඨයන් වහන්සේ සිහි කෙළෙමි. (බුදුරජුන් වැඩමවූ) එම දෙසට පසසා මම කල්පයක් මුළුල්ලෙහි සවර්‍ගයෙහි සතුටුවීමි.
# ´´මින් දෙයානූවන කපෙහි බුදුරජුන් වැඩමවූ යම් දෙසකට පැසසූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පැසසීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් දෙසකිත්තක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සරණගමණීය ථෙරාපදානය.
# ´´මම හිමවත් පව්වෙහි වැද්දෙක් වීමි. ලෝක ජ්‍යෙෂ්ඨවූ, නර ශ්‍රේෂ්ඨවූ, විපස්සී බුදුරජුන් දිටිමි.
# ´´මම සම්බුදු රජුන් වෙත එළඹ වතාවත් කෙළෙමි. විපදෝත්තමවූ, තාදි ගුණ ඇති බුදුරජුන්ගේ සරණද ගියෙමි.
# ´´මින් එක් අනූවන කපෙහි බුදුරජුන් සරණකොට ගියෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මේ සරණාගමනයෙහි විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සරණගමණීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# අම්බපිණ්ඩිය ථෙරාපදානය.

# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් අඹ දනක් දුනිම්ද, (ඒ හේතුවෙන්) අගතියක් නොදනිමි. අඹ දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අම්බපිණ්ඩිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ´´මම පිඬු පිණිස හැසිරෙන්නාවූ විපස්සී බුදුරජුන් දුටිමි. විපදෝත්තමවූ, තාදී ගුණ ඇති (එම බුදුරජුන්හට) සැන්දක් පමණවූ භික්‍ෂාව දුනිමි.
# ´´මම එකල්හි පහන් සිත් ඇත්තේ, සොම්නස්වූයේ වැන්දෙමි. උත්තමාර්ත්‍ථයවූ නිවනට පැමිණීම පිණිස බුදුරජුන්හට පැසසීමි.
# ´´මින් එක් අනූවන කපෙහි මම බුදුරජුන්හට පැසසීමි. (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පැසසීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අනුසංසාවක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
