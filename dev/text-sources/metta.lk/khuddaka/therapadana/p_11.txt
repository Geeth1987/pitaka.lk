භික්ඛාදායි වග්ග. 
# භික්ඛාදායි ථෙරාපදානය.

# ´´මම ස්වර්‍ණවූ, ලෝකයාගේ පූජාවන් පිළිගන්නාවූ (නොහොත් පූජාවනට ගෙයක් බඳුවූ) උතුම්වූ විහාරයෙන් (පිඬු පිණිස) නික්මුණාවූ, තෘෂ්ණාවෙන් නික්ම නිවණට පැමිණියාවූ, ප්‍රඥාවෙන් උපශාන්තවූ, තාදී ගුණ ඇති, මහා වීරවූ සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට සැන්දක් පමණවූ භික්‍ෂා දානයක් දුනිමි.
#
# ´´පියවරින් පියවර වඩිමින් මහා ජනයා නිවණට පමුණුවන කල්හි මට (ඒ) ආදිච්ච බන්‍ධුවූ (හිරුගොත්හි) බුදුරජුන් කෙරෙහි මහත්වූ සොම්නසක් ඇතිවිය.
# ´´මින් සිව් අනූවන කපෙහිවූ එකල්හි යම් දනක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. භික්‍ෂා දානයාගේ මේ විපාකයයි.
# ´´මින් සත්අසූවන කපෙහි සත් රුවනින් යුත් මහා රේණු නම් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් භික්ඛාදායී තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ඤාණ සඤ්ඤක ථෙරාපදානය.

# ´´මම ස්වර්‍ණ වර්‍ණවූ, උතුම් ආජානීයයෙකු මෙන් ශ්‍රේෂඨවූ, ත්‍රිමද ගලිත මෘතඞ්ග හස්ති කුලයෙහිවූ ඇත්රජෙකු වැනි, මහර්ෂීවූ, පුන් සඳු මෙන් සියළු දිගුන් බබුළුවන්නාවූ, වීථියට පිළිපන්, ලෝක ශ්‍රේෂඨවූ (සිද්‍ධාර්ත්‍ථ) බුදුරජාණන් වහන්සේ දුටිමි.
#
# උන්වහන්සේගේ නුවණෙහි සිත පහදවා, ඇඳිලි බැඳගෙන සොම්නස්වූයේ, පහන් සිත් ඇත්තේ, සිද්‍ධාර්ත්‍ථ බුදු රජුන්හට අභිවාදනය කෙළෙමි.
# ´´මින් සිව් අනූවන කපෙහිවූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ ඥානය පැහැදි සිතින් සිත්හිලා සැළකීමේ විපාකයයි.
# ´´මින් තෙසැත්තෑවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති නර ශ්‍රේෂ්ඨවූ සක්විති රජවරු සෙළොස් දෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඤාණ සඤ්ඤක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උපාලි ථෙරාපදානය.
# ´´තීව්රා නගරයෙහි වෙසෙන සුළුවූ මම එකල්හි මල් කරුවෙක් වීමි. (එහිදී) පහළවූ කෙළෙස් රජස් ඇති ලෝකනායකවූ සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේ දුටුවෙමි.
# ´´මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, (උන් වහන්සේට) (උපුල්) මල් මිටක් දුනිමි. එම කර්‍මයාගේ විපාකයෙන් යම් යම් භවයක උපදීද,
# ´´මනාවූ මල්ලයන් විසින් පිරිවරණ ලදුයේ, පෙර තමා කළ පින්කම්හි ඉෂ්ඨ විපාක අනුභව කරමි. පුෂ්පදානයාගේ මේ විපාකයයි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මෙම භවය හැර සිව් අනූවන කපෙහි පටන්ම එහි නජ්ජු නම් පන්සියයක් රජවරු වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උපාලි තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පදපූජක ථෙරාපදානය.
# ´´මම සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට සමන් මල් පිදීමි. (උන්වහන්සේගේ) ශ්‍රීපාදයෙහි මේ මල් සතක් සතුටින් විසිරවීමි.
# ´´එම පුණ්‍ය කර්මය කරණකොටගෙන අද දෙව් මිනිසුන් අතරෙහි අතිශයින් බබළමි. සම්බුදු සසුනෙහි අන්තිම ශරීරය දරමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප පූජාවෙහි මේ විපාකයයි.
# ´´මින් පස්වෙනි කපෙහි සිව්මහ දිවයින හා සිව් සයුර කෙළවරකොට ඇති පෘථිවියෙහි අධිපතිවූ, ජනාධිපතිවූ සමන්තගන්‍ධ නම්වූ තෙළෙස් සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පදපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# මුඪිපුප්ඵිය ථෙරාපදානය.
# ´´එකල්හි මම නම්න් සුදස්සන නම්වූ මල් කරුවෙක්වීමි. ලෝක ජ්‍යෙෂ්ඨවූ, නර ශ්‍රේෂ්ඨවූ, පහවූ කෙළෙස් මල ඇති බුදුරජාණන් වහන්සේ දුටිමි.
# ´´(මම) දෑසමන් මල් ගෙන පදුමුත්තර බුදුරජාණන් වහන්සේට පිදීමි. (ඒ හේතුවෙන්) පිරිසිදු ඇස් ඇත්තේ, සොම්නස්වූයේ, දිවැසද ලැබීමි.
# ´´මම මේ පුෂ්ප පූජාව කරණකොට ගෙනද, චිත්ත ප්‍රණිධීන් කරණකොට ගෙනද කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නීපදුනෙමි.
# ´´මින් සතිස්වන කපෙහි මහත් බල ඇති දෙවුත්තර නම්වූ සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මුඪිපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
#
# උදකපූජක ථෙරාපදානය.
# 
# ´´මම අහස් මගෙහි වඩින්නාවූ, රන්වන්වූ, ගිනිමෙන් දිලිසෙන, ඇවිලගත් අග්නිය වැනි බුදුරජාණන් වහන්සේට අතින් දිය ගෙන අහසෙහි උඩට දැමීමි. මහාවීරවූ, කාරුණීක ඎෂිවූ බුදුරජතෙම (එය) පිළිගත් සේක.
# ´´අහසෙහි වැඩ සිටියාවූ පදුමුත්තර නම් ශාස්තෘතෙම මෙම ගාථාව වදාළෝය.
# ´´මෙම ජල දානයෙන්ද, ප්‍රීතිය ඉපදවීමෙන්ද, (මෙතෙම) කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නූපදී. (කියායි.)
# ´´වීපදෝත්තමයන් වහන්ස, ලෝක ජ්‍යෙෂ්ඨයන් වහන්ස, නරශ්‍රේෂඨයන් වහන්ස, එම කර්මය කරණකොටගෙන ජය-පරාජය (දෙක) හැර නිශ්චල ස්ථානය (නිවණට) පැමිණියේ වෙමි.
# ´´සදහස් පන්සියවන කපෙහි සිව්මහ දිවයින හා සිව්මහ සමුදුරු කෙළවරකොට ඇති පොළොවේ අධිපතිවූ ජනශ්‍රේෂඨවූ ඒ සහස්සරාජ නම් සක්විති රජවරු තිදෙනෙක්ද වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උදක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# නලමාලිය ථෙරාපදානය.
#
# ´´තාදී ගුණ ඇති, වීපදෝත්තමවූ, ලෝක ජ්‍යෙෂ්ඨවූ, තණ ඇතිරියක වැඩ හුන්නාවූ පදුමුත්තර බුදුරජාණන් වහන්සේට බට මලක් ගෙන, විජිනියක් (වටාපතක්) බැඳ පිළිගැවීමි.
# ´´ලෝකනායකවූ බුදුරජාණන් වහන්සේ විජිනිය පිළිගෙන මාගේ අදහස දැන මෙම ගාථාව වදාළ සේක.
# ´´යම්සේ මාගේ සිරුර නිවේද, පරිළාහයක් නොවේද, එසේම ඔබගේ සිත (රාග-ෙවෂ-මෝහ) යන ත්‍රිවිධාග්නියෙන් (නිවේවා) කෙළෙසුන්ගෙන්ද මිදේවා කියායි.
# ´´වනය ඇසුරු කළාවූ යම් කිසි දෙවි කෙනෙක් වූවාහුද, ඒ සියලු දෙවිවරු දායකයාහට දේශනා කරන්නාවූ බුද්‍ධ වචනය අසමුයයි රැස්වූවාහුය.
# ´´එහි දිව්‍ය සමූහයා විසින් පිරිවරණ ලදුව වැඩ හුන්නාවූ භාග්‍යවතුන් වහන්සේ දායකයා සතුටු කරවමින් මෙම ගාථාවන් වදාළ සේක. (කෙසේද යත්)
# ´´මේ විජිනි දානය කරණකොට ගෙනද, චිත්ත ප්‍රණිධිය කරණකොට ගෙනද, නම්න් සුබ්බත නම්වූ සක්විති රජෙක් වන්නේය.
# ´´කුශල මූලයෙන් මෙහෙයන ලදුයේ, එම කර්මයාගේ ඉතිරි විපාකය වශයෙන් නම්න් මාළුත නම්වූ සක්විති රජෙක් වන්නේය.
# ´´මෙම විජිනි දානය කරණකොට ගෙනද, මහත්වූ සත්කාරය කරණකොටගෙනද, කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නූපදින්නේය, කියායි.
# ´´තිස්දහස්වන කපෙහි ඒ සුබ්බත නම්න් අටතිස් රජකෙනෙක් වූහ. එකුන්තිස් දහස්වන කපෙහි මාළුත නම්න් අට රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් නලමාලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ආසනු පඨාන ථෙරාපදානය.
# ´´(මම) අල්ප ශබ්ද ඇති, නිශ්චලවූ මහ වනයට පිවිස, තාදී ගුණ ඇති අත්‍ථදස්සී බුදුරජාණන් වහන්සේට මා විසින් සිහසුනක් පුදන ලදී.
# ´´මල් මිටක් ගෙන ඒ ශාස්තෲන් වහන්සේ පැදකුණුකොට ඇසුරුද කොට උතුරු දිගට මුහුණලා ගියෙමි.
# ´´වීපදෝත්තමයන් වහන්ස, ලෝක ජ්‍යෙෂ්ඨයන් වහන්ස, නරශ්‍රෙෂ්ඨයන් වහන්ස, එම (පුණ්‍ය) කර්මය හේතු කොට ගෙන තමා නිවණට පත් කරමි. සියලු භවයෝ නසන ලදහ.
