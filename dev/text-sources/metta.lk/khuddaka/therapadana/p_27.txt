සුවණ්ණබිම්බොහන වග්ග.  
# සුවණ්ණබිම්බොහනිය ථෙරාපදානය.
# ´´උත්තමාර්ත්‍ථයවූ නිවන් ලබනු පිණිස මම සියතින් එක් අසුනක් දුනිමි. බිම්බොහන (මෙට්ට) යක්ද දුනිමි.
# ´´මින් එක් අනූවන කපෙහි යම් බිම්බොහන දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බිම්බොහන දානයාගේ මේ විපාකයයි.
# ´´මින් තෙසැටවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති අසම නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුවණ්ණ බිම්බොහනිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිලමුට්ඨිදායක ථෙරාපදානය.
# ලෝක නායකවූ ශාස්තෘ තෙමේ මාගේ අදහස් දැන මනෝමයවූ කයින් හෙවත් චිත්ත ගතියට හැරවූ සිරුරින් යුතුව ඎද්‍ධියෙන් (මා වෙත) එළඹි සේක.
# ´´මම එසේ එළඹියාවූ පුරුෂෝත්තමවූ ශාස්තෲන් වහන්සේට වැඳ, පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, තල මිටක් දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් තිල දානයක් (තල දනක්) දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. තල මිටක් දීමෙහි මේ විපාකයයි.
# ´´මින් සොළොස්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති නන්දිය නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිලමුට්ඨිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# චඞ්ගොටකිය ථෙරාපදානය.
# ´´සිද්‍ධාර්ත්‍ථ (බුදුරජතෙම) මහ සයුර ඇසුරුකොට ඇති කඳු අතරක වැඩ වෙසේ. මම (මල්) ගොටුවක් සාදාගෙන උන්වහන්සේට පෙර ගමන්කොට එය පිළිගැන්වීමි.
# ´´මම සියලු සත්ත්‍වයනට හිතානුකම්පා ඇති සිදුහත් බුදුරජුන්හට මල් ගොටුව පිළිගන්වා (ඒ හේතුවෙන්) කල්පයක් මුළුල්ලෙහි ස්වර්‍ගයෙහි සතුටුවීමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි (එසේ) මල් ගොටුවලින් පිදූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මල් ගොටුවක් පිදීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් චඞ්ගොටකිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අබ්භඤ්ජනදායක ථෙරාපදානය.

# ´´වීත රාගීවූ, තාදී ගුණ ඇති, ආකාශය සමවූ (මෙත්) සිත ඇති, ප්‍රමාද විහරණ (කෙළෙස් ගතිවලින් යුතුව නොසැළකිලිමත්ව විසීමක්) නැති, ධ්‍යාන කරන්නාවූ, සියලු මුළාවීම ඉක්මවූ, සකල ලෝක හිතෛශීභාවයෙන් යුත්, වීපදෝත්තමවූ, භාග්‍යවත් කොණ්ඩඤ්ඤ බුදුරදුන්හට මා විසින් ආලේපන තෛලයක් දෙන ලදී.
#
# ´´මින් අසංඛ්‍ය කල්පයකට පෙර වූ එකල්හි යම් ආලේපන තෛලයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ආලේපන තෛල දානයාගේ මේ විපාකයයි.
# ´´මින් පසළොස්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති චිරප්ප නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අබ්භඤ්ජනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# එකඤ්ජලිය ථෙරාපදානය.
# ´´දිඹුල් ගස් සෙවනෙහි නියමිත කොළ ඇතිරියක වසන්නාවූ මහර්ෂී ශ්‍රමණයන් වහන්සේට මා විසින් විසුමට තැනක් දෙන ලදී.
# ´´ලෝකනාථවූ, තාදී ගුණ ඇති, තිස්ස බුදුරජාණන් වහන්සේට ඇඳිලි බැඳ මල් ඇතිරියක් අතුළෙමි.
# ´´මින් දෙයානූවන කපෙහි යම් මල් ඇතිරියක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මල් ඇතිරිය දීමෙහි මේ විපාකයයි.
# ´´මින් තුදුස්වන කපෙහි ජනාධිපතිවූ මහත් බල ඇති, එක අඤ්ජලික නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් එකඤ්ජලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පොත්‍ථකදායක ථෙරාපදානය.
# ´´අසමාන පුණ්‍යක්‍ෂෙත්‍රවූ බුදුරජුන්ද, ධර්‍මය හා සංඝයාද අරභයා මහර්ෂීවූ බුදුරජුන්හට මා විසින් නියඳ වස්ත්‍ර දානයක් දෙන ලදී.
# ´´මින් එක්අනූවන කපෙහි වූ එකල්හි (එම වස්ත්‍ර දානය කිරීම් වශයෙන්) යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (බුමුතුරුණු සඳහා දුන්) එම නියඳ වස්ත්‍ර දානයෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පොත්‍ථකදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# චිතකපූජක ථෙරාපදානය.
# ´´මම චන්ද්‍රභාගා ගං ඉවුරෙහි යටිගං බලා යමි. (එහිදී) මාලුව මල් සතක් ගෙන (වැලිවලින් සෑයක් කොට එම) චිතකයට පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් චිතක පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´මින් සත් සැටවන කපෙහි සත් රුවනින් යුත්, පටිජග්ග නම් ඇති සක්විති රජ කෙනෙක් වූහ.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් චිතකපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ආලුවදායක ථෙරාපදානය.

# ´´හිමවත් පව්වෙහි සුදර්ශනීයවූ මහාසින්ධු නම් නදියකි. පැහැපත් ඉතා සන්සුන් බැව්හි යෙදුනු සුද්ස්සන නම් රහතුන් වහන්සේ දැක විෂ්මිතවූ සිත් ඇත්තේ, පහන් සිත් ඇත්තේ, සියතින්ම උන්වහන්සේට අල (කැබෙල්ලක්) දුනිමි.
#
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි යම් ආලුව දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ආලුව (අල) දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආලුවදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# එකපුණ්ඩරීක ථෙරාපදානය.
# ´´(මම) එකල්හි නම්න් රොමස නම් වීමි. මනා වෘත ඇති බුදුරජ තෙමේද විය. මා විසින් පහන් සිතින් (උන්වහන්සේට) පියුමක් දෙන ලදී.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් පියුමක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුණ්ඩරීක දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් එකපුණ්ඩරීක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තරණීය ථෙරාපදානය.
# ´´පහසුවෙන් යා නොහැකි මාර්‍ගයක මවිසින් ලෝකයාගේ එතරවීම පිණිස සියතින්ම ඒදන්ඩක් (හෙයක්) කරවන ලදී.
# ´´මින් අනූඑක්වන කපෙහි මවිසින් ඒදණ්ඩ කරවන ලදී. දුගතියක් නොදනිමි. එදඬු දානයාගේ මේ ඵලයයි.
# ´´මින් පණස්වන කපෙහි සත් රුවණින් යුතු මහත් බල ඇති සමාගධ නම් සක්විති රජෙක් වීමි.
# සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තරණීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
