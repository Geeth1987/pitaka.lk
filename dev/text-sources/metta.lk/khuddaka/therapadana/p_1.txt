 8්‍ර9එයැර්ච්ා්බ්9ි

# ථෙරාපදානය. _____ 
# බුද්ධ වග්ගය. _____
# සාරීපුත්‍රථෙරාපදානය ඉක්බිති ස්ථවිරාපදානය අසව්.
# ´´හිමවතට නුදුරෙහි ´´ලම්බක´´ නම් පර්‍වතයකි. එහි මට අසපුවක් කරණ ලදී. පන්සලක් මැනවින් මවන ලදී.
# ´´මනාවූ (කුඩා) තොටුපලවල් ඇති, මනොඥවූ සුදෝ සුදු වැලි තලාවන්ගෙන් ගැවසී ගත් කුඩා ගඟ මාගේ අසපුව අවට නුදුරෙහි විය.
# ´´බොරළු රහිත, ගැඹුරු ඉවුරු නැති, මනා ජලයෙන් යුත්, දුර්‍ගන්‍ධ විරහිතවූ (එම) කුඩා ගංගා තොම එහි මාගේ අසපුව හොබවමින් ගලා බසින්නීය.
# ´´කිඹුල්ලුද, මෝරුද මුඛයෙන් සූ සූ නදින් (දිය බුබුළු උඩට නගන සුංසුමාර නම්වූ) චණ්ඩ මත්ස්‍යයෝද, කැසුබුවෝද එහිවූ මාගේ අසපුව හොබවමින් ගඟෙහි සැරිසරත්.
# ´´පෙටියෝද, පාවුස නම් (ලූල්ලු) මත්ස්‍යයෝද, වජල නම් (වලයෝ)මත්ස්‍යයෝද, මුඤ්ජ නම් (මොද්දු) මත්ස්‍යයෝද, රොහිත (රෙහෙරු) මත්ස්‍යයෝද, චග්ගුල නම් මත්ස්‍යයෝද පෙරළෙමින් මාගේ අසපුව හොබවත්.
# ´´එම ගඟ දෙපසෙහි (හැමකල්හි) පිපුණු මල් ඇත්තාවූද, හටගත් ගෙඩි ඇත්තාවූද, වෘක්‍ෂයෝ දෙතෙරෙහි පහළට නැමුණාහු, මාගේ අසපුව හොබවත්.
# ´´අඹ, සල්, මදට, පළොල්, නික යන මේ වෘක්‍ෂයෝ පිපීගිය මල් ඇත්තාහු, මාගේ අසපුවෙහි හාත්පස දිව සුවඳ හමත්.
# ´´සපු, සලල, කොළොම්, නා, දොඹ, වැටකේ යන වෘක්‍ෂයෝද, පිපීගිය මල් ඇත්තාහු, මාගේ අසපුවෙහි හාත්පස දිව සුවඳ හමත්.
# ´´මාගේ අසපුවෙහි යොහොඹු, හෝපළු, භගිනි, මාලා යන වෘක්‍ෂයෝ මල් පිපී ගියාහුය. එසේම, රුකගුණ, රත්කරවු යන වෘක්‍ෂයෝද සුපුෂ්පිත වූවාහුය.
# ´´වැටකේ, කඳල, ගොවුක, බොලිද්ද යන වෘක්‍ෂයෝ මාගේ අසපුව හොබවමින් දිව සුවඳ හමත්.
# ´´කිණිහිරිය, සිහින් මිදිය, පියාය, කුඹුක්ය යන මේ වෘක්‍ෂයෝ මාගේ අසපුව හොබවමින් දිව සුවඳ හමත්.
# ´´දොඹය, ගිරි දොඹය (ගිරිපුන්නාහ), කොබෝ ලීලය යන වෘක්‍ෂයෝ සුපුෂ්පිත වූවාහු මාගේ අසපුව හොබවමින් දිව සුවඳ හමත්.
# ´´ඇසළය, කෙළිඳය, කොළොම්ය, මුහුණ මල්ය යන බොහෝ වෘක්‍ෂයෝ මාගේ අසපුව හොබවමින් දිව සුවඳ හමත්.
# ´´ආලකය, ඉසිමුග්ගය, (වනමුං) කෙසෙල්ය, ලප්නාරන්ය යන මේ වෘක්‍ෂයෝ සුවඳ දියෙන් වැඩුනාහු ඵල දරත්.
# ´´(අසපුව අසලවූ) තඩාගයෙහි හැමකල්හි ඇතැම් පද්මයෝ පිපෙත්. ඇතැම් කෙසර පද්මයෝ හට ගනිත්. ඇතැම් පිපුණු පද්මයෝ ගිලිහුණු පත් හා කෙසුරු ඇත්තෝ වෙත්.
# ´´පද්මයෝ කැකුළුව හටගනිත්. නෙළුඹු අල (මඩ අතුරින් ඇත් දළ මෙන්) වැඩීමෙන් මහවිල් කටු (කටු දෙලු) පත්වලින් ගැවසීගත්තාහු හැමකල්හි (එම) විලෙහි බබළත්.
# ´´විල අසල පිපීගිය නයිතය අම්බ (පුෂ්ප) ගන්ධිය, උත්තාලීය, බඳුවද යන මේ ගස්හු හැම කල්හි දිව සුවඳ හමත්.
# ´´පෙටියෝද, පාවුසයෝද, වලප්ය, මුඤ්ජය, රොහිතය, සංගුලය, මංගුරය යන මත්ස්‍යයෝද එකල්හි තඩාගයෙහි වසත්.
# ´´කිඹුල්ලුද, තන්නිගාහ රකුසෝය, මගුහය, අජකාර යන මොවුහු එකල්හි මෙම තඩාගයෙහි වසත්.
# ´´පරෙවිය, රවිහංසය, සක්වා ලිහිණිය (නදීවරයාද), කොකිලය, ගිරාය, සාලිකය යන මේ පක්‍ෂීහු ඒ විල ඇසුරෙන් ජීවත් වෙති.
# ´´කුකුත්‍ථකය, කුලීරකය, වෙනෙහි වෙසෙන පොක්ඛර සාතකය, දින්දිභය, සුවපොතය (මලිත්තා) යන මේ පක්‍ෂීහු එම විල ඇසුරෙන් ජීවත් වෙති.
# ´´හංසයෝය, කොස්වා ලිහිණියෝය, මොණරු, කොව්ලෝය, හුන පුපුළෝය, ජිවඤ්ජිවයෝය යන මේ පක්‍ෂීහු මේ පක්‍ෂීහු එම විල ඇසුරුකොට ජීවත් වෙති.
# ´´බකමුහුණෝය, පොඨසීස නම් පක්‍ෂීහුය, කබරුස්සෝය, උකුස්සෝය, මහා කාල නම් පක්‍ෂීහුය යන බොහෝ පක්‍ෂීහු එම විල ඇසුරුකොට ජීවත් වෙති.
# ´´පසදමෘගයෝය, ඌරෝය, වෘකයෝය, කැනහිල්ලුය, රොහිච්ච නම් මෘගයෝය, සුග්ගදී පොතයෝය යන බොහෝ මෘගයෝ එම විල ඇසුරුකොට ජීවත් වෙති.
# ´´සිංහයෝය, ව්‍යාඝ්‍රයෝය, දිවියෝය, වලස්සුය, කරබානා වලස්සුය, තෙතැනකින් මද බිඳෙන (ත්‍රිමද ගලිත) ඇත්තුය ඒ විල ඇසුරුකොට වෙසෙත්.
# ´´කිඳුරෝද, වඳුරෝද, යලිදු වනයෙහි කර්‍මාන්තයන්හි යෙදුනෝද (මුවසේවටු ආදීහුද) දාසයෝද, වැද්දෝද, එම විල ඇසුරුකොට වෙසෙත්.
# ´´තිලිසිරිය, පිය, වැල්මීය (කාසුවාරියා) යන මේ වෘක්‍ෂයෝ මාගේ අසපුවට නුදුරෙහි නිරතුරුවම පල දරත්.
# ´´මනාව හටගත් පලයන්ගෙන් යුත් කොසුම්භය, සලලය, කොළොම්ය යන මේ වෘක්‍ෂයෝ මාගේ අසපුවට නුදුරු තන්හි නිරතුරුවම පල දරත්.
# ´´අරළුය, නෙල්ලිය, අබය, ජම්බුය, බුළුය, ඩෙබරය, බදුලුය, බෙලිය, යන මෙකී වෘක්‍ෂයෝද පල දරත්.
# ´´ආළුය, කලම්බය, බිලාලීය, තක්කලය, ජීවකය, සුතකය යන බොහෝ අල වර්‍ගයෝ මාගේ අසපුවෙහි වෙති.
# ´´අසපුවට නුදුරෙහි මැනවින් වැවූ, පිරිසිදු සිහිල් දිය ඇති, මනා තොටුපලවලින් යුත් මනරම්වූ තඩාගයෝ වූහ.
# ´´පුණ්ඩරීක (සුදු නෙලුම්) සහිත පියුම් (රතු නෙළුම්) උපුල්වලින් ආකුලවූ තඩාගයන්ගෙන් ගැවසී ගත්තාහුය. දිව සුවඳද හමයි. මදාරා ගස්වලින් යුක්තය.
# ´´මෙසේ සුපුෂ්පිතව පල දැරුවාවූ, සර්‍වාඞ්ග සම්පූර්‍ණ වණයෙහි මැනවින් කරණ ලද රම්‍යවූ ආශ්‍රමයෙහි මම හැම කල්හි වෙසෙමි.
# ´´සිල්වත්ව, වෘත සම්පන්නව ධ්‍යාන කරණ සුළු වූයේ, හැම කල්හි ධ්‍යානයෙහි ඇළුනේ, පඤ්චාභිඥා බලයට පැමිණියේ ´´සුරුචි´´ නම් තවුසෙක් වීමි.
# ´´සූවිසි දහසක් ශිෂ්‍යයෝ මට උවටැන් කළාහුය. මේ සියලු ශිෂ්‍යයෝ ජාතිමත්, කිර්තිමත් බ්‍රාහ්මණයෝ වූහ.
# ´´(ඔවුහු) අංගලක්‍ෂණ ශාස්ත්‍රයෙහිද, ඉතිහාසයෙහිද, වේදත්‍රයෙහිද, ස්වකීය බ්‍රාහ්මණ ධර්‍මයෙහිද, පරතෙරට ගියාහු, නාම පදාදියෙහි දක්‍ෂවූහ. එසේම ව්‍යාකරණාදියෙහිද දක්‍ෂයෝ වූහ.
# ´´උල්කාපාත, භූමිචලනාදිය පිළිබඳ වැඩ අවැඩ දැනගන්නා උත්පාත ශාස්ත්‍රයෙහිද, ශුභා ශුභ නිමිති දැනගන්නා ශාස්ත්‍රයෙහිද, ස්ත්‍රී පුරුෂයන්ගේ හා මහා පුරුෂයන්ගේ ලක්‍ෂණ දන්නා ශාස්ත්‍රයෙහිද, දක්‍ෂවූවාහු පෘථිවිය පිළිබඳවූද, අහස පිළිබඳවූද, ශාස්ත්‍රයන්හි මාගේ ශිෂ්‍යයෝ දක්‍ෂවූහ.
# ´´අල්පෙච්ඡවූ, අල්පාහාරවූ, ලොල්බව නොමැති, ප්‍රාඥවූ මොව්හු ලාභ අලාභ දෙකින්ම එකසේ සතුටුවූවාහු, හැමකල්හි මා පිරිවරත්.
# ´´ධ්‍යාන කරණ සුළුවූ, ධ්‍යානයෙහි ඇළුනු, ධෛර්‍ය්‍යවත්වූ, ශාන්තවූ, එකඟවූ සිත් ඇති, පළිබෝධ විරහිත බව කැමති වන්නාවූ එම තව්සෝ හැමකල්හි මා පිරිවරත්.
# ´´පඤ්චාභිඥාවන්හි පරතෙරට පත්, අනුන්හට වචනයෙන් කියා (විඤ්ඤත්තියෙන්) පිළියෙලකර නොගත් බුදුන් වහන්සේ විසින් අනුදන්නා ලද ආහාරයෙහි ඇළුනාහු, අහසින් යාම් ඊම් කරන්නාවූ, වීර ගති ඇති එම තවුසෝ හැම කල්හි මා පිරිවරත්.
# ´´සදොර සංවරණය කරගත්, තෘෂ්ණාවෙන් තොර, සුරැකි ඉඳුරන් ඇති, ගිහියන් සමග නොගැටෙන, ලංවිය නොහැක්කාවූ ඒ ධෛර්‍ය්‍යවත් මාගේ ශිෂ්‍යයෝ
# ´´ගැටීමට නුසුදුසුවූ මොවුහු පළඟ බැඳීමෙන්, හිඳීමෙන් හා සක්මන් කිරීමෙන්ද රාත්‍රිය ගෙවත්.
# ´´ලංවිය නොහැක්කාවූ මාගේ (ඒ) ශිෂ්‍යයෝ ඇලිය යුතු දෙයෙහි නොඇලෙත්. ගැටියයුතු දෙයෙහි නොගැටෙත්. මුළාවියයුතු දෙයෙහි මුළා නොවෙත්
# ´´යුගග්‍රාහයෙන් හෙවත් කලහ කිරීමෙන් ගැටීමට නුසුදුසුවූ ඒ මාගේ ශිෂ්‍යයෝ පෘථිවිය කම්පණය කරත්. නිරතුරුවම විකුර්‍වණ ඎද්‍ධි විමසමින් පවතිත්.
# ´´එම ශිෂ්‍යයෝ ඉදින් ක්‍රීඩා කරන්නාහු නම් ධ්‍යාන ක්‍රීඩාවම කරත්. ළංවිය නොහැකි තෙදින් යුත් ඔවුහු (දඹදිව ප්‍රසිද්ධ) දඹ රුකින් ගෙඩි ගෙනෙත්.
# ´´ලංවිය නොහැක්කාවූ ඔවුනතුරෙන් ඇතැමෙක් ප්‍රත්‍ය සෙවීම පිණිස අපරගොයානයට යෙත්. ඇතැමෙක් පූර්‍වවිදේහයටද, ඇතැමෙක් උතුරුකුරු දිවයිනටද යෙත්.
# ´´අහසින් යන්නාවූ මාගේ ශිෂ්‍යයෝ තවුස් පිරිකර කද යවති. ඔවුහු පසුව යෙත්. සූවිසි දහසක්වූ ඔවුන් විසින් අහස් තලය වසන ලද්දේ වේ.
# ´´ඔවුනතුරෙන් ඇතැමෙක් (පලාඵලපත්‍රාදිය) ගින්නෙන් පිස කති. ඇතැමෙක් නොපිස අමුවෙන්ම කති. ඇතැමෙක් දත්වලින්ම පොතු ගලවා කති. කිසිවෙක් වනෙහි ලා කොටාගෙන කති. (එසේම) ඇතැමෙක් ගලින් කොටා කති. ඇතැමෙක් ස්වයං පතිත ඵලාහාරයෙන් ජීවත් වෙති.
# ´´ලංවිය නොහැක්කාවූ මාගේ ශිෂ්‍යයෝ ශුද්‍ධිය කැමති වන්නාහු, දියෙන් තෙමා තෙමීම කරන්නාහු, උදැහැනද, සවසද දියෙහි ගිලෙති.
# ´´දුරාසදවූ මාගේ ශිෂ්‍යයෝ කැසි (කිසිලි) නිය ලොම් ඇත්තාහු, මල බැඳුණු දත් ඇත්තාහු, දූවිලි මැකුණු හිස් ඇත්තාවූ සීල සුගන්‍ධයෙන් සුගන්‍ධවත් වූහ.
# ´´එකල්හි උත්‍ර තපස් ඇති ජටාධරවුූ මෙම තවුසෝ උදෑසනම (තමාවෙත) රැස්ව (ඔවුනට ලැබුණු) කුඩා මහත් ලාභයන් කියා අහසෙහි යති.
# ´´(එසේ) යන්නාවූ ඔවුන්ගේ වැහැරි සිව්රුවලින් හටගත් මහා ශබ්ද පවතී. අලුත් දිවිසම් හඬින් දෙවියෝ සොම්නස්වූවාහු වෙති.
# ´´අහසෙහි හැසිරෙන්නාවූ එම ඎෂීහු දිසානුදිසාවනට ගමන් කෙරෙති. ස්වකීය ශරීර බලයෙන් හා ධ්‍යාන බලයෙන් යුත් ඔවුහු යම් යම් තැනකට යනු කැමැත්තාහු නම් ඒ ඒ තැනටම යෙත්.
# ´´මේ සියළුතන්හි කැමතිසේ හැසිරෙන මොවුහු හැමදෙනම පෘථිවි කම්පණයෙහි සමත්වූවෝද, අහසෙහි හැසිරීමෙහි දක්‍ෂවූවෝද, උග්‍ර තේජස් ඇත්තාහුද, අභිභවනය කළ නොහැක්කාවූද, සයුර මෙන් අනුන් විසින් නොකැළඹිය හැක්කේ වෙයි.
# ´´ගුණයන් කරණකොට ලංවිය නොහැක්කාවූ මාගේ ශිෂ්‍යයන්ගෙන් ඇතැමෙක් සිටීමය, සක්මන් කිරීමය යන ඉරියව් දෙකින් යුක්තයෝය. ඇතැමෙක් හිඳීම් ඉරියව් පමණක් ඇති ඎෂිවරයෝය. (එසේම) ඇතැමෙක් ස්වයං පතිත ඵල අනුභව කරන්නෝ වෙති.
# ´´මොවුහු සව්සත්නට හිතවත්වූවාුි, මෛත්‍රී විහරණ ඇත්තාහුය. සියල්ලෝම තමා උසස්කොට නොදක්වන්නේය. කිසිවෙකු හෙළා දකින්නෝද නොවෙති.
# ´´සිහ රජුන් මෙන් නිර්භීතව, ඇත් රජුන් මෙන් බලවත්ව, ව්‍යාඝ්‍රයන් මෙන් ලංවිය නොහැක්කන්ව (එම ඎෂීහු) මාගේ සමීපයට පැමිණෙත්.
# ´´විද්‍යාධරයෝද, දේවතාවෝද, නාග, ගාන්‍ධර්‍ව, රාක්‍ෂස, කුම්භාණ්ඩ, අසුර, ගුරුළු යන මොවුහුද ඒ විල ඇසුරෙන් වෙසෙත්.
# ´´ජටාවන්හි හා තවුස් පිරිකර කදවල බර උසුලන්නාවූ, අඳුන් දිවි සම් උත්තරාසඞ්ගයන් ඇති ආකාශචාරීහු හා සියළුම ඎෂීහු එම විල ඇසුරෙන් වෙසෙති.
# හැම කල්හි මනා පැවතුම් ඇති මෙම ඎෂීහු ඔවුනොවුන්ට ගෞරව සහිතවූවාහු සූවිසි දහසක්වූ මොවුන්ගේ කිඹිසිලි ශබ්දයකුදු නැත.
# ´´පියෙහි පිය නොතබන්නාහු නිශ්ශබ්දව මනා සංවරයෙන් යුත්ව සියල්ලෝම මා වෙත එළඹ හිස නමා වඳිත්.
# ´´ශාන්තවූ තපෝ ගුණ ඇති එම සිසුන් විසින් පිරිවරණ ලදුයේ මම එම අසපුවෙහි ධ්‍යාන කරණ සුළුව, ධ්‍යානයෙහි ඇළුනේ වෙසෙමි.
# ´´ඎෂීන්ගේ සිල් සුවඳින්ද, (මල්වල) සුවඳින්ද යන මේ උභය සුගන්ධයෝ හා ගෙඩිවල සුවඳින්ද අසපුව සුවඳවත්වේ.
# ´´රෑ දාවල් නොදනිමි. මාගේ සිත්හි කුසල් දහම්හි නොඇලීමක් නැත. මම සියළු සිසුන්ට අවවාද කරන්නේ වැඩියක්ම ප්‍රීතිය ලබමි.
# ´´පිපෙන්නාවූ මල්වලද, පැසෙන්නාවූ ගෙඩිවලද දිව්‍යමයවූ සුගන්ධයෝ මාගේ අසපුව හොබවමින් හමත්.
# ´´එකල්හි මම සමාධියෙන් නැගිට කෙලෙස් තවන වීර්‍ය්‍යය ඇත්තේ, සප්‍රාඥවූයේ, තවුස් පිරිකර කදෙහි බරද ගෙන වනයට පිවිසෙමි.
# ´´(උල්කාපාත භූමි චලනාදී භූත විකෘතීන් පිළිබඳ වැඩ අවැඩ දැනගන්නා උත්පාත ශාස්ත්‍රයෙහිද, සිහිනයන් පිළිබඳ ශුභා ශුභ දැනගන්නා සුපින ශාස්ත්‍රයෙහිද, අංග ලක්‍ෂණ බලා ජීවිතය පිළිබඳ තොරතුරු පවසන ලක්‍ෂණ ශාස්ත්‍රයෙහිද මැනවින් හික්මුනේ, මුළු දඹදිවම පවත්නා මන්ත්‍ර පදද දරමි.
# ´´ලෝකයෙහි ජ්‍යෙෂ්ඨවූද, නරශ්‍රේෂ්ඨවූද, සියල්ල දත්තාවූ, අනවමදර්ශි භාග්‍යවතුන් වහන්සේ විවේක සැප කැමති සේක් හිමවතට එළඹිසේක.
# ´´පුරුෂෝත්තමවූ, අග්‍ර කාරුණීකවූ, බුදුරජාණන් වහන්සේ හිමවතට පිවිස පළඟ බැඳ වැඩහුන් සේක.
# ´´මහනෙල් මලක් මෙන් ප්‍රභාව සහිතවූ, මනරම්වූද යාග ද්‍රව්‍ය අනුභව කරන්නාවූ, දිලිසෙන ගින්නක් වැනිවූ, බුදුරජාණන් වහන්සේ එහිදී දුටුවෙමු.
# ´´පහන් රුකක් මෙන් දිලිසෙන්නාවූද, අහසෙහි විදුලියක් වැනිවූද, මනාව පිපීගිය සල් රුකක් වැනිවූද, බුදුරජාණන් වහන්සේ දුටුවෙමි.
# ´´මේ අග නම්වූ, පව් නැති මහාවීරවූ, දුක් කෙළවර කරන්නාවූ බුදුරජාණන් වහන්සේය. මේ දර්‍ශනය නිසා (සත්ත්‍වයෝද) සියළු දුකින් මිදෙත්.
# ´´දෙවියන්ට දෙවිවූ උන්වහන්සේ දැක මම බුදුකෙනෙක් දෝයි ලක්‍ෂණ පරීක්‍ෂා කෙළෙමි. ඒකාන්තයෙන් පසැස් ඇති උතුමෙකැයි දකිමි.
# ´´උතුම්වූ ශ්‍රී පාදයන්හි දහසක් ඇති චක්‍ර ලක්‍ෂණයෝ පෙනෙත්. උන්වහන්සේගේ මහා පුරුෂ ලක්‍ෂණ දැක තථාගත බැව්හි නිශ්චයට පැමිණියෙමි.
# ´´එකල්හි මුස්න ගෙන හැමද මල් ගෙනවුත් බුද්‍ධ ශ්‍රේෂඨයන් වහන්සේ පිදීමි.
# ´´චතුරෝඝයන් තරණය කළාවූ, ආශ්‍රව රහිතවූ, සම්‍යක් සම්බුදුරජාණන් වහන්සේ පුදා, අඳුන් දිවි සම් සකස්කොට ලෝකනායකවූ තථාගතයන් වහන්සේට නමස්කාර කෙළෙමි.
# ´´ආශ්‍රව රහිතවූ බුදුරජාණන් වහන්සේ යම් කාරණයකින් යුක්තව වැඩ විසූ සේක් නම් යම් කාරණයකින් ඒ ඥානය කියන්නෙමි. කියන්නාවූ මාගේ ඒ වචනය අසව්.
# ´´අමතොදය හෙවත් අපමණවූ දියුණුව ඇති බුදුරජාණන් වහන්ස, මේ ලෝකවාසී සත්ත්‍වයා නගාලනු මැනවි. ඔබ වහන්සේගේ දර්‍ශනය නිසා සැකය නැමැති සැඩ පහර තරණය කෙරෙත්.
# ´´ඔබ වහන්සේ සත්ත්‍වයන්ගේ ශාස්තෘ වන සේක. (උසස් බැවින්) ජයකොඩියද, (ලෙව්හි ප්‍රකට බැවින්) ධර්‍මවිජයද, (තිලොවෙහිම උඩට නැගී පෙනෙන බැවින් ඔසවා බිම සිටුවන ලදී.) යාගස්තම්භයද වන සේක. ඔබ වහන්සේම දෙපා ඇතියනට උතුම් වන සේක. (මොහඳුර නසන බැවින්) ප්‍රදීපයද (භවසත්නට) උතුම් පිහිටද පිහිටන තැනද වන සේක.
# ´´සමුදුරෙහි ජලය නැළියකින් මිණුමට හැකිවුවද, ඔබ වහන්සේගේ සර්‍වඥතාඥානය මිණුමට නොහැක්කේමය.
# ´´මහපොළොව තරාදි තට්ටුවේ තබා කිරා බැලීමට හැකිවුවද, ඔබ වහන්සේ හෝ සර්‍වතාඥානය කිරා බැලීමට නොහැක්කේමැයි.
# ´´අහස රැහැනෙන් හෝ අඟුලෙන් හෝ මිණිය හැකිවුවද, ඔබ වහන්සේ ගේ සර්‍වතාඥානය මිණුමට නොහැක්කේමය. ු ´´බුද්‍ධඥානය තෙම මහා සමුද්‍රයෙහි අපමණවූ ජලයද, අප්‍රමාණවූ මහ පොළොවද ඉක්මවා යයි. බුදු නැණ නිසා උපමා වශයෙන් කිසි වස්තු කෙනෙක් නොයෙදෙත්මැයි. ු1 ´´පසැස් අතියාණන් වහන්ස, දෙවියන් සහිත සත්ත්‍ව ලෝකයෙහි යම් පමණ සත්ත්‍ව කෙනෙක් පිළිබඳව සිත පවතී නම් එපමණවු මේ සත්ත්‍ව සමූහයා ඔබ වහන්සේගේ නුවණ නැමැති දැලෙහි ඇතුලට පිවිසියාහු වෙති. ු2 ´´සර්‍වඥයන් වහන්ස, ඔබ වහන්සේගේ යම් නුවණක් කරණකොට ගෙන හුදු උතුම්වූ සම්‍යක් සම්බෝධියට පැමිණියෙහිද, එම ඥානය කරණකොට ගෙන අන්‍ය තීර්ත්‍ථකයන් මඩිනෙහිය. ු3 ´´ඒ සුරුචිනම් තවුස්තෙම (සෙසු අටුවාවන්හි සරද තවුස්යයි දක්වා ඇත.) මෙම ගාථාවන්ගෙන් (උන්වහන්සේට) ස්තූතිකොට පොළවෙහි අඳුන් දිවිසම එලා හිඳගත්තේය.
# ´´මහමේරු පර්‍වත රාජයා මහසමුදුරෙහි යොදුන් අසූහාරදහසක් ඇතුළට බැසගත්තේ එපමණට (යොදුන් අසූහාරදහසක්) (සමුදුරෙන්) උඩට ඇත්තේයයි කියනු ලැබේ.
# ´´දිගු පුළුලිනුදු එපමණම උස්ව නැගුනු මේරු පර්වතය කෙළ ලක්‍ෂගණන්වූ අණුභෙදයන් සුණු කරණ ලද්දේ,
# ´´(එම අණු නම් සියුම් කොටස්) අලගු බවට තැබූයේ නමුදු ඔබ වහන්සේ ගේ සර්‍වතාඥානය ප්‍රමාණ කරන්නට නොහැක්කේමය.
# ´´යමෙක් සියුම් ඇස් සිදුරු ඇති දැලක් දියෙහි දමාද එවිට ජලයෙහි යම්කිසි ප්‍රාණී කෙනෙක් වෙත් නම් (ඔවුහු) එම දැළ තුලට වදින්නාහුය.
# ´´මහා වීරයන් වහන්ස, එසේම යම්කිසි පෘථග්ජන තීර්ත්‍ථක කෙනෙක් දෙසැට මිසදිටු නැමැති (ගෙතුනුවන) ගහනයට පිවිසියාහු එම මිසදිටු දැඩිකොට ගැනීම් වශයෙන් මුලාවූවාහු වෙත්.
# ´නිරාවරණවූ දක්නා සුළුවූ ඔබ වහන්සේ ගේ පිරිසිදු ඥානය කරණකොටගෙන තමන් වහන්සේගේ නැණ ඇතුළට පිවිසියාවූ මේ තීර්ත්‍ථකයෝ ඔබ වහන්සේ ගේ නුවණ නොඉක්මවත්.
# ´´පස්මරුන් දිනූ මහා යසස්කවූ අනවමදර්ශී නම් භාග්‍යවතුන් වහන්සේ එසමයෙහි සමාධියෙන් නැගිට සියලු දිශාවලෝකනය කළ සේක. ු119 ු12 ´´අනවමදර්ශී නම් බුදුරජාණන් වහන්සේගේ නිසභ නම් ශ්‍රාවකතෙම සන්සිඳුනු සිත් ඇති (අට ලෝ දහම්හි නොසැලෙන) තාදී ගුණ යුත් ශුද්‍ධවූ ශඩභිඥා ඇති ධ්‍යාන කරණ සුළුවූ ලක්‍ෂ ගණන් රහතුන් විසින් පිරිවරණ ලදුයේ, බුදුරජාණන් වහන්සේගේ සිත දැන (ඒ) ලෝක නායකයන් වහන්සේ වෙත එළඹියේය.
# ´´එහි අහසෙහි සිටියාවූ එම පිරිස (බුදුරජාණන් වහන්සේ) ප්‍රදක්‍ෂිණා කළාහුය. බැඳැඳිලි ඇතිව නමඳිමින් සර්‍වඥයන් වහන්සේ වෙතට බැස්සාහුය.
# ´´ලෝකයේ ජ්‍යෙෂ්ඨවූ, පස්මරුන් දිනුවාවූ, අනවමදර්ශී භාග්‍යවතුන් වහන්සේ භික්‍ෂු සංඝයා ඉදිරියෙහි වැඩ හිඳ සිනා පහළ කළ සේක.
# ´´මහර්ෂිවූ සර්‍වඥයන් වහන්සේගේ ´´වරුණ´´ නම් උපස්ථායක තෙමේ සිව්ර එකස්කොට ගෙන ලෝක නායකවූ බුදුරජාණන් වහන්සේගෙන් මෙසේ විචාළෝය.
# ´´ භාග්‍යවතුන් වහන්ස, ශාස්තෘවූ ඔබ වහන්සේගේ සිනා පහළ කිරීමට හේතුව කුමක්ද? ඒ සර්‍වඥ බුදුවරයාේ අකාරණයෙන් සිනා පහළ නොකරති.´´
# ලෝක ජ්‍යෙෂ්ඨවූ, නරශ්‍රේෂඨවූ අනවමදර්ශී භාග්‍යවතුන් වහන්සේ සංඝයා මැද හිඳ මෙම ගාථාව වදාළ සේක.
# ´´යම් තව්සෙක් තෙම මා මලින් පිදීද, මාගේ (සර්‍වඥතා) ඥානයට නැවත නැවත ස්තූති කෙළේද, ඒ තවුසා (කවරෙක්දැයි) මම තොපට පැහැදිලි කරන්නෙමි. එය පවසන්නාවූ මාගේ වචනය අසව්.
# ´´ බුදුරජාණන් වහන්සේගේ වචනය දැන සියලු දෙවියෝ එක්වූහ. සද්ධර්‍මය අසනු කැමැත්තාවූ ඔවුහු බුදුරජාණන් වහන්සේ වෙත එළඹියාහුය.
# ´´සියලු ලෝක ධාතූන්හි මහර්ධි ඇති ඒ දේව සමූහයා සද්ධර්‍මය අසනු කැමැත්තාහු, බුදුරජාණන් වහන්සේ වෙත එළඹියාහුය.
# ´´ඇත්, අස්, රිය, පාබල යන සිව්රඟ සෙනග නිතර මා පිරිවරන්නාහුය. බුද්‍ධ පූජාවෙහි මේ ඵලයයි.
# ´´සැට දහසක් තූර්‍ය්‍යයෝද, අලංකාර කළ ගෙරීහුද, මට නිතර (වැඩීමෙන්) උවටැන් කරන්නාහුය. මෙය බුද්ධ පූජාවෙහි ඵලයයි.

# ´´මැනවින් අලංකාර කරණ ලද සොළොස් දහසක් ස්ත්‍රීහු, විසිතුරු ආභරණ ඇත්තාහු, එල්බන ලද මිණි කොඬොල් ඇත්තාහු, දිග ඇසි පිහාටු ඇත්තාහු, සිනා සහිත වූවාහු, මැනවයි හැඳින්විය යුතු ශරීරාවයව ඇත්තාහු, නිරතුරුවම මොහු පිරිවරන්නාහුය. බුද්ධ පූජාවෙහි මේ බලයයි.

# ´´උන්වහන්සේගේ ධර්‍මය දායාදකොට ඇති ඖරස බුද්‍ධපුත්‍රවූ, ධර්‍මයෙන්ම මවාලූවාක් වැනි නම්න් ශාරීපුත්‍ර නම්වූ අග්‍ර ශ්‍රාවකයෙක් වන්නේය.
# ´´හිමවතින් පටන්ගත් භාගීරථී ගංගා තොමෝ ජලස්කන්‍ධය බස්වමින් සාගරයට පැමිණෙන්නීය.
# ´´එසේම මේ ශාරීපුත්‍ර තෙමේ ස්වකීය (කුල පරම්පරාවෙහි) පවත්නාවූ ත්‍රිවේදයෙහි විශාරදවූයේ, ප්‍රඥාවෙන් ශ්‍රාවකපාරමීඥානයෙහි තෙරට ගොස් (සියලු) සත්ත්‍වයන් සන්තර්‍පණය කරන්නේය.
# ´´හිමාලය පර්‍වතයෙහි පටන් මහා ජලස්කන්‍ධය දරන්නාවූ සයුර දක්වා මේ අතරෙහි යම් පමණ වැලි කැට රැසක් ඇද්ද, එය ගණන් වශයෙන් අසඞ්ඛ්‍යවේ.
# ´´ඒ වැලි රැසද නිරවශෙෂයෙන් (සම්පූර්‍ණයෙන්) ගණන මෙපමණයයි ප්‍රමාණ කරන්නට හැකිවන්නේ නමුදු ශාරීපුත්‍ර ස්ථවිරයන් වහන්සේගේ ප්‍රඥාවෙහි කෙළවරක් නොවන්නේය.
# ´´අලගු තබන කල්හිද, ගංගා නම් නදියෙහි වැලි අවසන් වන්නේය. ශාරීපුත්‍ර ස්ථවිරයන් වහන්සේගේ ප්‍රඥාවෙහි කෙළවරක් නොවන්නේමැයි.
# ´´මහ සයුරෙහි රළ ගණන් වශයෙන් මෙපමණයයි සංඛ්‍යා කළ හැකි නොවේ. එසේම ශාරීපුත්‍ර ස්ථවිරයන් වහන්සේගේ ප්‍රඥාවෙහි කෙළවරක් නොවන්නේමැයි.
# ´´ශාක්‍යකුල ශ්‍රේෂ්ඨවූ ගෞතම සම්‍යක් සම්බුදුරජාණන් වහන්සේ සතුටුකොට, ප්‍රඥාවෙන් ශ්‍රාවකපාරමීඥානයෙහි තෙරෙට ගොස් අග්‍රශ්‍රාවකයෙක් වන්නේය.
# ´´තාදී ගුණ ඇති (අට ලෝ දහම්හි නොසැලෙන) (සර්‍වඥවූ) ශාක්‍ය පුත්‍රයන් වහන්සේ විසින් පවත්වන ලද ධර්‍ම චක්‍රය (සද්ධර්‍මය) මැනවින් දහම් වැසි වස්වමින් ඒ අනුවම පවත්වන්නේය.
# ´´ශාක්‍යකුල ශ්‍රේෂ්ඨවූ ගෞතම බුදුරජාණන් වහන්සේ මේ සියල්ල (වෙසෙසින්) නුවණින් දැන භික්‍ෂු සංඝයා මද්‍ධ්‍යයෙහි හිඳ අග්‍රස්ථානයෙහි තබන්නේය.
# ´´අනවමදර්ශී බුදුරජාණන් වහන්සේගේ සමීපයෙහි මා විසින් පුණ්‍ය කර්මයක් මැනවින් කරණ ලදී. යම් ඒ අනවමදර්ශී නම් බුදුවරයෙක් වීද, උන්වහන්සේ විෂයෙහි එකී පුණ්‍යසම්භාරය කොට සියලු ගුණ සමූහයෙහි පරතෙරෙට ගියේ වෙමි. ආශ්චර්‍ය්‍යයෙකි.
# ´´අපමණ කලක් මුළුල්ලෙහි මා විසින් කරණ ලද කුශල කර්‍ම ඵලය මේ පශ්චිමාත්මභාවයෙහිදී දැක්වීය. (දුනුදියෙන්) මැනවින් මිදීගිය හී වේගය මෙන් මම කෙළෙස් දැවීමි.
# ´´හේතු ප්‍රත්‍ය වශයෙන් සංස්කරණය නොවන බැවින් අසංස්කෘතවූ, නිශ්චලවූ, නිර්‍වාණ පදය සොයන්නේ සියළු තීර්ත්‍ථකයන් පරීක්‍ෂා කරන්නේ එකල්හි මම භවයෙහි සැරිසැරුවෙමි.
# ´´ලෙඩකින් පෙළෙන්නාවූ පුරුෂයෙක් තෙම ලෙඩින් මිදීම පිණිස යම් සේ ඖෂධයක් සොයන්නේද, සියළු වනය (ඖෂධ පිණිස) පරීක්‍ෂා කරන්නේද,
# ´´අසංස්කෘත (හේතු ප්‍රත්‍ය බලයෙන් සකස් නොවන) අමෘත පදයවූ නිර්‍වාණය සොයන්නේ, අවිච්ඡින්නව (නොකඩවා පිළිවෙළින්) පන්සියයක් ජාතීන්හි ප්‍රව්‍රජ්‍යාවෙන් පැවිදිවීමි.
# ´´මම තවුස් දළමඩුළු බරින් බරවූයේ, අඳුන් දිවි සමින් කළ උතුරු සළු පොරවන්නේ, අභිඥාවරුන්ගේ පරතෙරට පැමිණ බඹලොව ගියෙමි.
# ´´බුදුසසුන හැර ඉන් පිටත්හිවූ සසුනක ශුද්‍ධියක් නැත. බුද්ධිමත්වූ කිසියම් සත්ත්‍ව කෙනෙක් වෙත්නම් සර්‍වඥ ශාසනයෙහි ශුද්ධිය ලබත්.
# ´´මෙය තමා විසින් කරණ ලද පුණ්‍ය කර්‍ම බලයෙන් ජනිතවූවකි. මෙය මෙසේ වීයයි කියන ප්‍රළාපයක් නොවේ. මම නිවණ සොයන්නේ වැරදි තීර්ත්‍ථයන්හි හැසුරුනෙමි.
# ´´යම්සේ හර කැමැත්තාවූ පුරුෂයෙක් තෙම කෙසෙල් ගසක් කපා පලන්නේද, (හෙතෙම) එහි හරයක් නොලබන්නේයි. එය හරයෙන් තොර ගසකි.
# ´´එසේම ලෝකයෙහි තීර්ත්‍ථකයෝද, නා නා දෘෂ්ඨි ගත් බොහෝ දෙනාද නිවණින් හිස්වූවෝම වෙති. හරයෙන් හිස්වූ කෙසෙල්ගස් මෙනි.
# ´´අන්තිම භවය පැමිණි කල්හි මම බ්‍රහ්මබන්ධු හෙවත් බ්‍රාහ්මණ කුලයෙහි උපන්නෙක් වීමි. මහත්වූ භොකස්කන්‍ධය හැර අනගාර්‍ය්‍ය නම් ශාසනයෙහි පැවිදිවීමි. ( පළමුවෙනි බණවර නිමි. )
# ´´වේද හදාරන්නාවූ, මන්ත්‍රධරවූ, ත්‍රී වේදයෙහි පරතෙරට ගිය සඤ්ජය නම් බමුණෙක් විය. මම ඔවුන්ගේ සමීපයෙහි වසමි.

# ´´(ඔබ) පහන් පැහැයට සුදුසුව පැමිණෙන්නෙහිය. (හෙණ්ඩු) ආදියෙන් නිශ්චල බවට පත්කළ ඇතෙකු මෙනි. මනාව හික්මුණු කෙනෙකු මෙනි. බ්‍රාහ්මණය, (ඔබ) ශික්‍ෂිත ශික්‍ෂා ඇත්තේ සන්සිඳුනේ වෙහිද?
# ´´ශෝකය නමැති හුල දුරු කරන්නාවූ (ඇද දවන්නාවූ) අමෘතය (නිර්වාණය) මා විසින් අවබෝධ කර ගන්නා ලදී. ඔබද එය අවබෝධ කර ගනුව. (අපි) බුදුරජාණන් වහන්සේගේ සමීපයට යමු.´´
# ´´මාගේ හික්මුණාවූ ඒ යහළු තෙමේ මැනවයි පිළිවදන් දී, අතින් අත ගෙන ඔබ වහන්සේගේ සමීපයට පැමිණියෙමි.
# ´´ශාක්‍ය පුත්‍රයාණන් වහන්ස, ඔබ වහන්සේගේ සමීපයෙහි දෙදෙනම පැවිදි වන්නෙමු. ඔබ වහන්සේගේ සසුන නිසා ආශ්‍රව රහිතව වසමු.
# ´´කෝලිත තෙමේ ඎද්‍ධි දැක්වීමෙන් ශ්‍රේෂ්ඨය. මම ප්‍රඥාවගේ පරතෙරෙට ගියෙමි. දෙදෙනම එක්ව සසුන හොබවමි.
# ´´අවසන් නොවූ බලාපොරොත්තු ඇත්තේ නොයා යුතු මගෙහි හැසුරුණෙමි. ඔබ වහන්සේගේ දැකීම නිසා මාගේ බලාපොරොත්තුව සම්පූර්‍ණ විය.
# ´´පොළොවෙහි පිහිටි වෘක්‍ෂයෝ (හෙමන්ත) කාලයෙහි පිපෙත්. සියලු ප්‍රාණීන් සතුටු කරත්. දිව සුවඳ හමත්.
# ´´මහා වීරයන් වහන්ස, ශාක්‍ය පුත්‍රයන් වහන්ස, මහ යසස් ඇත්තාණන් වහන්ස, (එසේම) ඔබ වහන්සේගේ සසුනෙහි පිහිටා (රහත් මග නුවණින්) පිපෙන්නට සුදුසු කාලය සොයමි.
# ´´සංසාරයෙන් මුදවන්නාවූ විමුක්තිය නමැති පුෂ්පය සොයන්නේ (දැන්) එම විමුක්ති පුෂ්ප ලාභයෙන් සියලු සත්ත්‍වයන් සතුටු කරමි.
# ´´පසැස් ඇතියාණන් වහන්ස, බුද්‍ධ ක්‍ෂෙත්‍රය යම් පමණ වේනම් ඒ සියළු තන්හි (කෝටි ලක්‍ෂයක් සක්වළ) සම්‍යක් සම්බුදුරජාණන් වහන්සේ හැර ප්‍රඥාවෙන් ඔබ වහන්සේගේ පුත්‍රයාට (ශාරීපුත්‍රයන්ට) සමාන කෙනෙක් නැත.
# ´´ඔහුගේ ශිෂ්‍යයෝද සුවිනීතයෝය. පිරිසිඳ මැනවින් හික්මුණේය. උතුම් ශික්‍ෂාවෙහි දමනයවූවෝ හැමකල්හි හේ (ඒ ශාරීපුත්‍ර) පිරිවරත්.
# ´´ධ්‍යාන කරණ සුළුවූ, ධ්‍යානයෙහි ඇළුනාවූ, ධ්‍රෛ්‍යවත්වූ, ශාන්ත සිත් ඇති, සන්සුන්වූ, නැණවත් (මොන නම්වූ නුවණින්) යුත් භික්‍ෂූහු හැමකල්හි හේ පිරිවරත්.
# ´´අල්පෙච්ඡවූ, ප්‍රාඥවූ, ධ්‍රෛ්‍යයවත්වූ, ස්වල්පාහාර ගන්නාවූ, ලොල් නොවූ, ලාභ අලාභ දෙකෙහි සමව සතුටු වන්නාවූ, (භික්‍ෂූහු) හේ පිරිවරත්.
# ´´අරණෙහි වසන්නාවූ (ආරණ්‍යක ධූතාඞ්ගධාරීවූ) ධ්‍යාන කරන්නාවූ, රූක්‍ෂ චීවරධාරී (ගොරෝසු සිව්රු දරන්නාවූ) විවේකයෙහි ඇළුනාවූ, ධ්‍රෛ්‍යවත්වූ (භික්‍ෂූහු) හැමකල්හි හේ පිරිවරත්.
# ´´පිළිපන් සිව්මග ඇති, සතර ඵලයෙහි පිහිටියාවූ, සෙසු ඵල තුනින්ද යුත් මේ සීමිත ශෛක්‍ෂයෝ නිවණ බලාපොරොත්තුවන්නාවූ හේ හැමකල්හි පිරිවරත්.
# ´´යම් පිරිසිදුවූ සෝතාපන්න මාර්‍ග පලයනට පැමිණියේද, සකෘදාගාමීහුද, අනාගාමීහුද, රහත්හුද වෙත්නම් ඔවුහු හැමකල්හි හේ පිරිවරත්.
# ´´(සතර) සතිපට්ඨානයන්හි දක්‍ෂවූවෝද (සප්ත) බොද්‍ධ්‍යඞ්ග භාවනාවෙහි ඇළුනෝද, බොහෝවූ ඒ සියලු ශ්‍රාවකයෝ හැමකල්හි හේ පිරිවරත්.
# ´´(සතර වැදෑරුම්වූ) ඎද්‍ධිපාදයන්හි දක්‍ෂවූවෝද, සමාධි භාවනාවෙහි ඇළුනෝද, (සතර) සම්‍යක් ප්‍රධානයෙහි යෙදුනෝද හේ හැමකල්හි පිරිවරත්.
# ´´ත්‍රිවිද්‍යා ඇත්තෝද, ෂඩභිඥා ඇත්තෝද, ඎද්‍ධියෙන් හා ප්‍රඥාවෙන් පරතෙරට ගියෝද හේ හැමකල්හි පිරිවරත්.
# ´´මහා වීරවූ බුදුරජාණන් වහන්ස, මැනවින් හික්මුණාවූ, ලංවිය නොහැක්කාවූ, උග්‍රවූ තේජස් ඇති ඒ මෙබඳුවූ ඔබවහන්සේගේ ශිෂ්‍යයෝ හේ හැමකල්හි පිරිවරත්.
# ´´සංයතවූ, කෙළෙස් තවන ගුණ ඇති යම් ශිෂ්‍යයන් විසින් පිරිවරණ ලදුයේ සිංහයෙකු මෙන් නිර්භීත වෙහිය. චන්‍ද්‍රයා මෙන් බැබලෙහිය.
# ´´වෘක්‍ෂයෝ පොළොවෙහි පිහිටා උඩට නැගෙත්. (වැඩෙත්) මහත් බවටද පැමිණෙත්. ඒ වෘක්‍ෂයෝ පලද දක්වත්. (පල දරත්.)
# ´´(එසේම) ශාක්‍ය පුත්‍රයන් වහන්ස, මහා යසස් ඇත්තාණන් වහන්ස, ඔබ පොළොව මෙන් වෙහිය. ඔබ වහන්සේගේ සසුනෙහි පිහිටා ඒ උතුමෝ අමෘත ඵලය හෙවත් නිවණ ලබත්.

# ´´සින්‍ධුය, සරස්වතීය, චන්‍ද්‍රභාගීය, ගංගාය, යමුනාය, සරභූය, මහීය යන මේ ගංගාවෝ වෙත්ද, ගලා බස්නාවූ මේ ගංගාවන් සාගරය පිළිගනී. (සයුරට වැටීමෙන් පසු) එම ගංගාවෝ පළමු නම් අත්හරිත්. සාගරයම ලෙස ප්‍රකට වේ.
# ´´එසේම චතුර් වර්‍ණයටම අයත්වූවෝ (ක්‍ෂත්‍රිය, බ්‍රාහ්මණ, වෛශ්‍ය, ක්‍ෂුද්‍රයෝ) ඔබ වහන්සේගේ සමිපයෙහි පැවිදිව පළමු නම් හරිත්. බුද්‍ධපුත්‍රයෝයයි ප්‍රකට වෙත්.
# ´´යම් සේ අහසෙහි ගමන් කරන්නාවූ නිර්‍මලවූ චන්‍ද්‍රයා සියළු තාරකා සමූහය (ස්වකිය) ප්‍රභාවෙන් මැඩ ලෝකයෙහි අතිශයින් බබළාද,
# ´´එසේම මහා වීරයන් වහන්ස, ඔබ වහන්සේ දෙවි මිනිසුන් පිරිවරණ ලදුයේ මේ සියලු දෙන ඉක්මවා හැමකල්හි ඔබ වහන්සේම බබලන්නෙහිය.
# ´´ගැඹුරු දියෙහි නැගුනු රැළහු වෙරළ නොඉක්මවත්. ඒ සියලුම රළයෝ වෙරළෙහි හැපෙත්. කැඩීගිය ඒ රළහු විසිරී යත්.
# ´´එසේම ලෝකයෙහි නොයෙක් ලබ්ධි ඇති ඒ තීර්ත්‍ථකවූ බොහෝ දෙනා ධර්‍මය අසනු කැමැත්තාවූ ඒ බුදුරජාණන් වහන්සේ නොඉක්මවත්.
# ´´පසැස් ඇතියාණන් වහන්ස, ඉදින් ඔවුහු ඒ ලබ්ධි ගෙන විරුද්ධ වාදයෙන් (වාද කිරීමට) පැමිණෙත්ද, ඔවුහු ඔබ වහන්සේගේ සමීපයට පැමිණ සුණු විසුණු වෙත්.
# ´´මහමඩ කරණකොට ගෙන දියෙහි හටගත් කුමුදුද, මන්දුලක නම් පියුම්ද, අනිකුත් බොහෝ ජලජ පුෂ්පයෝද, යම් සේ දිය සමග තැවරී සිටිත් නම්,
# ´´එසේම ලෝකයෙහි උපන්නාවූ බොහෝ සත්ත්‍වයෝ මඩෙහි කුමුදු මෙන් රාගද්වේෂයින්ගෙන් හටගත් පිළිකුල් ඇත්තාහු උඩට නැගෙත්. (වැඩෙත්.)
# ´´යම්සේ ජලයෙහි හටගන්නාවූ පියුම් දිය මැද වැඩේද, ඒ පියුම් පිරිසිදු දිය සමග නොගැටේ.
# ´´එසේම ලොව පහළවූ මහා වීරවූ බුදුරජාණන් වහන්ස, ඔබ පියුමක් මෙන් ලෝකය නමැති ජලයෙහි නොතැවරෙන්නෙහිය.
# ´´යම්සේ ජලජ පුෂ්පයෝ ඉල් මාසයෙහි බෙහෙවින් පිපෙත්ද, ඒ මාසය නොයික්මවත්ද, (එම) මල් පිපීමට ඒ (සුදුසු) කාලය වේද,
# ´´එසේම ශාක්‍ය පුත්‍රයන් වහන්ස, ඔබ පුබුදුවූයෙහිය. ඔබ වහන්සේගේ ශිෂ්‍යයෝ (අර්හත් ඵල ඥානයෙන්) පුබුදුවූවාහුය. යම්සේ ජලජවූ පියුම් පිපෙන කාලය නොඉක්මවත්ද, (එසේම ඔවුහු) අවවාදානුශාසනය නොඉක්මවත්.
# ´´මැනවින් පිපුණාවූ ශාල වෘක්‍ෂ රාජයා දිව සුවඳ හමාද, අනික් සල් රුක් විසින් පිරිවරණ ලද ශාල වෘක්‍ෂ රාජයාම හොබනේ වේද,
# ´´එසේම මහා වීරයන් වහන්ස, ඔබ (වහන්සේ) බුදු නැණින් පුබුදුවූයෙහිය. භික්‍ෂු සංඝයා විසින් පිරිවරණ ලදුයේ ශාල වෘක්‍ෂ රාජයෙක් මෙන් හොබනෙහිය.
# ´´යම්සේ හිමවත් පර්‍වතය හැම සතුනට (අවශ්‍ය) ඖෂධ ඇත්තේ වේද, නාගයනටද, අසුරයනටද, දේවතාවනටද වාසස්ථාන වේද,
# ´´එසේ මහා වීරයන් වහන්ස, ඔබ ප්‍රාණීන්ගේ සසර ලෙඩ නැසීමට (අවශ්‍ය) ඖෂධ ඇත්තේ වෙහිය. ත්‍රිවිද්‍යා ඇත්තෝද, ෂඩභිඥා ඇත්තෝද, ඎද්ධියෙන් පරතෙරට ගියෝද (ඔබ වහන්සේ ඇසුරු කරන්නේ)
# ´´ මහා වීරයන් වහන්ස, කාරුණීකවූ ඔබ වහන්සේ විසින් අනුශාසනා කරණ ලද ඔවුහු ධර්‍මරතියෙන් (එසේ ඇලී) ඔබ වහන්සේගේ සසුනෙහි වෙසෙත්.
# ´´යම් සේ කේශර සිංහයෙක් ගුහාවෙන් නික්ම සිව් දෙස බලා තෙවරක් සිංහ නාද කෙරේද,
# ´´කේශර සිංහයාගේ ගර්ජනයෙන් සියලු සිව්පාවෝ තැතිගනිත්. ඒ එසේමය. ජාතිමත්වූ සිංහ තෙමේ හැම කල්හි සිව්පාවුන් තැතිගන්වයි.
# ´´ මහා වීරයන් වහන්ස, ගර්ජනා කරන්නාවූ ඔබ වහන්සේගේ නාදය පෘථිවිය කම්පා කරයි. (ඉන්) අවබෝධයට සුදුස්සෝ (ධර්‍මය) අවබෝධ කෙරෙත්. මාර පාක්‍ෂිකයෝ බිය වෙත්.
# ´´ මහා මුනීන් වහන්ස, නාදකරන්නාවූ (දහම් දෙසන) ඔබ වහන්සේගේ (නාදයෙන්) සියළු තීර්ත්‍ථකයෝ බිය වෙත්. (ඒ තීර්ථකයෝ) කවුඩු රැළක් මෙන් බිය වූවාහු කේශර සිංහයා කෙරෙන් පළායන සිවුපාවුන් මෙන් වෙත්.
# ´´ගණ ප්‍රධාන කෙනෙක් ලෙව්හි (කිසිවෙකුට) ශාස්තෘයයි කියත්ද, ඔවුහු පිරිසට (ඒ ශාස්තෘවරයාගෙන් පිළිවෙළින් ආ) පරම්පරාගතවූ ධර්‍මය දේශනා කරත්.
# ´´ මහා වීරයන් වහන්ස, ඔබ සත්ත්‍වයනට එසේ ධර්‍ම දේශනා නොකෙරෙහිය. හුදෙක් බෝධි පාක්‍ෂීකවූ සත්‍යයන් තෙමේම අවබෝධකොට ගෙන,
# ´´(සත්ත්‍වයන්ගේ) අජ්ඣාසය හෙවත් චරියාවන්ද, ථාවරගතවූත් මහත්බවට ගිය කෙළෙස්දැයි යන මේ අසයානුසයන්ද (ශ්‍රද්‍ධාදී) ඉන්‍ද්‍රියයන්ගේ බල දුබල බවද (ධර්‍මාවබෝධයට) (සමත් අසමත්) පුඟුලන්ද දැන (සර්‍වඥයන් වහන්ස, ඔබ වහන්සේ) චාතුර්ද්වීපික මහා මේඝයක් (සිව් දිගටම වසින්නාවූ මහ වැහි වළාවක්) මෙන් (ධර්‍ම දේශනා නමැති සිංහ නාදයෙන්) ගර්ජනා කෙරෙහිය.

# ´´සක්වල ගැබ කෙළවරකොට නා නා ලබ්ධීන්ට ගැතිව සිතන්නාවූ (පිරී) හුන් පිරිස් වන්නීය. සැක දුරු කිරීම පිණිස ඒ සියලු දෙනාගේ සිත් (මේ මේයයි) දැන (සුදුසු) උපමා කීමෙහි සමත්, බුදුරජාණන් වහන්ස, (ඔබ වහන්සේ) එකම ප්‍රශ්වාසයක් (විසඳා) කියමින්ම (සක්වල ගැබ පුරා සිටි) ප්‍රාණීන්ගේ සැක දුරු කෙරෙහිය.
# ´´(දිය වසා ඒ මතුපිට වැතිර සිටි) සෙවෙල් මෙන් වූ (පොළොව මතුපිට වැතිර සිටි) මිනිසුන්ගෙන් පෘථිවි තොමෝ පිරී ගියා වන්නීය. ඒ සියල්ලෝ බැඳැඳිලි ඇත්තාහු ලෝක නායකවූ ඒ බුදුරදුන් ගුණ කීවාහුය.
# ´´ඒ සියලු දිව්‍ය මනුෂ්‍යයෝ කපක් මුළුල්ලෙහිද නා නා ප්‍රකාර ලෙස ගුණ කියන්නාහු නමුදු (එම ගුණ) පමණ කරන්ට නොහැක්කාහුය. තථාගතවරයෝ අපමණ ගුණ ඇත්තාහුය.
# ´´ස්වකීය ශක්තිය පරිදි මා විසින් බුදුරජාණන් වහන්සේ වර්‍ණනා කරණ ලදී. කෝටි සංඛ්‍යාත කාලයක් මුළුල්ලෙහි (එසේම) ගුණ කීර්‍තණය කළාහුය.
# ´´ඉදින් මනාව හික්මුණාවූ කිසියම් දෙවියෙක් හෝ මිනිසෙක් හෝ (එම ගුණ) (මහ සමුදුරෙහි) පිරුණ ජලය අවසන් කිරීමට මනින්ට සිතන්නේ නම් හෙතෙම වෙහෙසෙටම පැමිණෙන්නේය.
# ´´මහා යසස් ඇත්තාණන් වහන්ස, ශාක්‍ය පුත්‍රයන් වහන්ස, (මම) ඔබ වහන්සේගේ සසුනෙහි පිහිටා ප්‍රඥාවෙන් පරතෙරට ගොස් ආශ්‍රව රහිතව වෙසෙමි.
# ´´තීර්ථකයන් මඩිමි. බුදු සසුන පවත්වමි. අද ශාක්‍ය පුත්‍රවූ බුදුරජාණන් වහන්සේගේ ශාසනයෙහි ධර්‍ම සේනාපති වෙමි.
# ´´මා විසින් අපමණ කලක් මුළුල්ලෙහි කරණ ලද කර්‍ම පලය මෙහිදී දැක්වීය. මැනවින් විද හරණා ලද ඊ පහරක් මෙන් මම කෙළෙස් දැවීමි.
# ´´යම්කිසි මිනිසෙක් තෙම හැම කල්හි හිසින් බරක් දරන්නේ නම් (එම) බර කරණකොටගෙන දුකින් පෙළුණෙක් වන්නේය. (එසේම) බරින් බරට පත්වූවෙක්ද වෙයි.
# ´´මම ත්‍රිවිධාග්නියෙන් (රාග, දෝෂ, මෝහ) දැවෙමින් සසරෙහි හැසුරුණෙමි. මහමෙර (හිසමත) ඔසවා තබන ලද කල්හි මෙන් භව බරින් බර වූයේ වෙමි.
# ´´මා විසින් මාගේ (ඒ භව) බර තබන ලදී. සියලු (භව) භවයෝ විධ්වංසනය කරණ ලදහ. ශාක්‍ය පුත්‍රවූ බුදුරජාණන් වහන්සේගේ ශාසනයෙහි සියලු කටයුතු කරණ ලදි.
# ´´සම්පූර්‍ණ බුද්‍ධක්‍ෂෙත්‍රයෙහි බුදුරජාණන් වහන්සේ හැර මම ප්‍රඥාවෙන් අග්‍ර වෙමි. මට සමාන කෙනෙක් නොපෙනේ.
# ´´සමාධියෙහි අතිශයින් දක්‍ෂ වෙමි. ඎද්ධියෙහි පරතෙරෙට ගියේ වෙමි. මම අද කැමති වන්නේ නම් දහසක් මවන්නෙමි.
# ´´අනුපූර්‍ව විහාරයට වසඟවූ සර්‍වඥයෝ මට සසුන වදාළහ. මාගේ සැතපීම (නම්) නිරෝධය (නිරෝධ සමාපත්තියට සමවැදීම) යි.
# ´´මාගේ දිවැස පිරිසිදුය. මම සමාධියෙහි දක්‍ෂ වෙමි. සමාධී ප්‍රධානයෙහි යෙදුනේ, බොධ්‍යඞ්ග භාවනාවෙහි ඇළුනේද වෙමි.
# ´´ශ්‍රාවකයෙකු විසින් කළයුතු සියල්ලක් මා විසින් කරණ ලදී. බුදුරජාණන් වහන්සේ හැර මට සමාන (අනික්) කෙනෙක් විද්‍යමාන නොවේ.
# ´´සමාපත්ති ක්‍රමයන්හි දක්‍ෂවූයේ (ලෞකීක) ධ්‍යානයන්ටද, (ලෝකෝත්තරවූ) විමොක්‍ෂයන්ටද, ශීඝ්‍රව සමවදින්නේ, බොද්‍ධ්‍යඞ්ග භාවනාවෙහි ඇළුනේ ශ්‍රාවක ගුණයන්ගේ පරතෙරෙට ගියේ වෙමි.
# ´´පහස්නා (ලබන) ලද ශ්‍රාවක ගුණයෙන්ද (පරිසුත්තමභාරවාද) බුද්‍ධියෙන්ද, ශ්‍රද්‍ධාවෙන්ද යුත් සිත හැම කල්හිම බ්‍රහ්මචාරීන් කෙරෙහි ඇත්තේ වෙමි.
# ´´උදුරා හළ විෂ (දළ) ඇති සර්‍පයෙක් මෙන්ද, කැඩුණු අං ඇති ගවයෙකු මෙන්ද, බැහැරකළ (ගෝත්‍ර මදාදී) මාන දර්‍ප ඇත්තේ සංඝයාවෙත ආදර බහුමානයෙන් එළඹෙමි.
# ´´ඉදින් මාගේ ප්‍රඥා තොමෝ මූර්තිමත් වන්නී නම් මහපොළොව සම නොවන්නීය. අනවමදර්ශී බුදුරජාණන් වහන්සේගේ නුවණට පැසසීමේ මේ ඵලයයි.
# ´´තාදී ගුණ ඇති ගෞතම බුදුරජාණන් වහන්සේ විසින් පවත්වන ලද ධර්‍මචක්‍රය මම මැනවින් ඒ අනුව පවත්වමි. (අනවමදර්ශී බුදුරජාණන්ගේ) නුවණ පැසසීමේ මේ ඵලයයි.
# ´´කිසි කලෙකත් පවිටු බවක්, කුසීත බවක්, පසුබට වීර්‍ය්‍ය ඇති බවක්, අල්ප ශ්‍රුත බවක්, ආචාර විරහිත බවක් මා සමග එක් නොවීය.
# ´´බහුශ්‍රුත බවද, නුවණැති බවද, සිව් පිරිසිදු කල්හි මැනවින් හික්මුණු බවද, චිත්ත සමථයෙහි යෙදුනු බවද (මාගේ) මස්තකයෙහි (මුදුනෙහි) සිටීවා.
# ´´මෙහි රැස්වූ පින්වත්නි, තෙපි අල්පෙච්ඡවූවෝද, ලද දෙයින් සතුටු වන්නෝද, ධ්‍යාන ඇත්තෝද, ධ්‍යානයෙහි ඇළුනෝද වව් යන (ඒ වචනය) තොපට කියමි.
# ´´යමෙක් මම පළමුකොට දැන කෙළෙස් රජස් නැති නිර්‍මල කෙනෙක් වීම්ද ඒ ධෛර්‍ය්‍යමත්වු මාගේ ආචාර්‍ය්‍ය තෙමේ අස්සජි නම් ශ්‍රාවකයන් වහන්සේය.
# ´´මම උන්වහන්සේගේ බලයෙන් අද ධර්‍ම සේනාපති වීමි. සියලු ගුණයන්හි පරතෙරට පැමිණියේ ආශ්‍රව රහිතව වෙසෙමි.
# ´´යම් අස්සජී නම් ශ්‍රාවකයෙක් තෙම මාගේ ආචාර්‍ය්‍ය වීද, උන් වහන්සේ යම් දිසාවක වෙසේ නම් මම ඒ දෙසෙට හිස (දමා විසීම) කරමි.
# ´´ශාක්‍යසිංහවූ ගෞතම බුදුරජාණන් වහන්සේ මාගේ (කුශල) කර්‍මය සිහිකොට භික්‍ෂු සංඝයා මැද හිඳ මා අග්‍ර ශ්‍රාවක ස්ථානයෙහි තැබිය.
# ´´මාගේ ක්ලේශයෝ දවන ලද්දාහුය. සියලු භවයෝ සමූහනනය (විනාශ) කරණ ලද්දාහුය. හස්තිරාජයෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
# ´´මේ සිවි පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී´´ මෙහි මේ අයුරින් ආයුෂ්මත් ශාරීපුත්‍ර ස්ථවිරයන් වහන්සේ මේ ගාථාවන් දේශනා කළ සේක. ( ශාරීපුත්‍ර ථෙරාපදානය නිමි. )

# මහා මොග්ගල්ලාන ථෙරාපදානය.
# ´´ලෝක ජ්‍යෙෂ්ඨවූ, නර ශ්‍රෙෂ්ඨවූ, අනොමදස්සි භාග්‍යවතුන් වහන්සේ දිව්‍ය සමූහයා විසින් පිරිවරණ ලදුව හිමවතෙහි විසීය.
# ´´එකල්හි මම නම්න් වරුණ නම්වූ නාග රාජයෙක්වීමි. කැමති රූපයන් මවන සුළුවූයේ ඎද්‍ධි විකුර්‍වණ්‍ය කරමි. මම සයුරෙහි විසීමි.
# ´´නිතර සිය පිරිවරවූ ස්වකීය නාග සමූහයා පිරිවරා මම තූර්‍ය්‍යයන් වාදනය කරවීමි. එකල්හි නා මෙනෙවියෝ (අනොමදස්සි) බුදුරජාණන් වහන්සේ පිරිවරා වාදනය කළාහුය.
# ´´(මනුෂ්‍ය- නාග) තූර්‍ය්‍යන් වාදනය කරනු ලබන කල්හි (චාතුර්මහාරාජික) දෙවියෝද (දිව්‍ය) තූර්‍ය්‍යයන් වාදනය කළාහුය. මේ දෙපිරිස (දෙවි මිනිසුන්ගේ තූර්‍ය) ශබ්දය අසා බුදුරජාණන් වහන්සේද එය දැනගත් සේක.
# ´´ බුදුරජාණන් වහන්සේට ආරාධනාකොට ස්වකීය නාග භවනයට එළඹියෙමි. මම අසුන් පනවා (දානයට සුදුසු) කාලය සැළ කෙළෙමි.
# ´´රහතුන් දහසක් විසින් පිරිවරණ ලද බුදුරජාණන් වහන්සේ සියළු දිසාවන් බබුළුවන සේක් මාගේ භවනයට එළඹි සේක.
# ´´එකල්හි මම (එසේ) එළඹියාවූ භික්‍ෂු සංඝයා සහිතවූ, මහා වීරවූ, දේවාති දේවවූ, නරශ්‍රේෂඨවූ බුදුරජාණන් වහන්සේ ආහාර පානයෙන් සන්තර්‍පණය කළෙමි.
# ´´(සියල්ල) තෙමේම අවබෝධ කළාවූ, මහා වීරවූ, අග්‍ර පුද්ගලවූ බුදුරජාණන් වහන්සේ භික්‍ෂු සංඝයා මැද හිඳ අනුමෝදනාව කළ සේක. මෙම ගාථාවෝද වදාරණ ලදහ.´´
# ´´යමෙක් තෙම ලෝක නායකවූ බුදුරජාණන් වහන්සේද, සංඝයාද පිදී නම් හෙතෙම එම සිත පැහැදවීම කරණකොට ගෙන දෙව්ලොවට (උත්පත්ති වශයෙන්) යන්නේය.
# ´´සත් සැත්තෑ වරක් දිව්‍ය රාජ්‍යයද කරන්නේය. අට සියක් වරෙක පෘථිවි රාජ්‍යයද කරමින් පොළොවෙහි වෙසෙන්නේය.
# ´´පස් පණස් වරක් සක්විති රජ වන්නේය. එකෙණෙහි ඔහුට අපමණ භෝගයෝ උපදින්නාහුය.
# ´´මෙයින් අපරිමෙය්‍ය කල්පයෙහි (එකාසංඛ්‍ය කල්ප ලක්‍ෂයකින් මතුයෙහි) ඔක්කාක නම් කුලයෙහි උපදින්නාවූ ගෞතම නම්වූ ශාස්තෲන් වහන්සේ ලෝකයෙහි පහළ වන්නේය.
# ´´හෙතෙම (වරුණ නාග රාජ තෙම) නිරයෙන් චුතව මිනිසත් බවට (උත්පත්ති වශයෙන්) යන්නේය. කෝලිත නම්වූ බ්‍රාහ්මණයෙක් වන්නේය.
# ´´හෙතෙම පසුව පැවිදිව කුශල මූලයන් විසින් මෙහෙයන ලදුයේ ගෞතම බුදුරජාණන් වහන්සේගේ වීතීය ශ්‍රාවක වන්නේය.
# ´´පටන් ගන්නා ලද වීර්‍ය්‍ය ඇත්තේ, නිවණට මෙහෙයූ සිත් ඇත්තේ, ඎද්‍ධියෙහි කෙළ පැමිණියේ, සියලු ආශ්‍රව ධර්‍මයන් දැන (දුරුකොට) ආශ්‍රව රහිතව (ක්ලේශ පරිනිර්‍වාණයෙන්) පිරිනිවන් පාන්නේය.´´
# ´´පාප මිත්‍රයන් ඇසුරෙන් කාමරාගයෙහි වසඟයට ගියේ දුෂ්ට සිත් ඇත්තේ මව් පියන්ද නැසීමි.
# ´´මම නිරෙයෙහි හෝ, නැතහොත් මිනිසුන් අතරෙහි හෝ, යම් යම් යෝනියක උපදිම් නම් (එකී) පාප කර්‍මය හා එක්වීමෙන් බිඳීගිය හිස් ඇත්තේ මැරෙමි.
# ´´මෙය මාගේ (සසරෙහි) අවසානයයි. අන්තිම ආත්ම භාවය වේ. මේ ආත්ම භාවයෙහිදීද මට මැරෙන කල්හි මෙබන්දක් වන්නේය.
# ´´ප්‍රකර්‍ශ විවේකයෙහි යෙදුනේ, සමාධි භාවනාවෙහි ඇළුනේ, සියලු ආශ්‍රවයන් දැන (දුරුකොට) ආශ්‍රව රහිතව වෙසෙමි.
# ´´පෘථිවියද ඉතා ගැඹුරුය. ඝනය. නොසෙල්විය හැක. (එහෙත්) ඎද්‍ධියෙහි කෙළ පැමිණියේ (එය) වම් පය මහපටැඟිල්ලෙන් සොළවමි.
# ´´අස්මී මානය (සීල සමාධි ප්‍රඥාදිය ඇත්තේ මටමය යන ඇඟීම) නොදකිමි. මට මානය නැත. මම සාමණේරයන් පටන්කොට සියලු භික්‍ෂු සංඝයා කෙරෙහි ගෞරව සිත (ඇති) කරමි.
# ´´මින් (එකා සංඛ්‍ය කල්ප ලක්‍ෂයකට) අපරිමෙය කල්පයකට පෙර (අගසව් බව පිණිස) යම් කර්‍මයක් රැස් කෙළෙම්ද, මම ඒ ශ්‍රාවක භූමියට පත්වීමි. ආශ්‍රව ක්‍ෂය නම්වූ නිවණට පැමිණියෙමි.
# ´´මාගේ ක්ලේශයෝ දවන ලද්දාහුය. සියලු භවයෝ සමූහනනය (විනාශ) කරණ ලද්දාහුය. හස්තිරාජයෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
# ´´මේ සිවි පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී´´ මෙහි මේ අයුරින් ආයුෂ්මත් මහා මොග්ගල්ලාන ස්ථවිරයන් වහන්සේ මේ ගාථාවන් දේශනා කළ සේක. ( මහා මොග්ගල්ලාන තෙරුන් වහන්සේගේ අපදානය නිමි. )

# මහාකාශ්‍යප ථෙරාපදානය.
# ´´තාදී ගුණ (අටලෝ දහම්හි නොසැලෙන ගුණ) ඇති පදුමුත්තර භාග්‍යවතුන් වහන්සේගේ (ශාසනයෙහි) ලෝකනාථවූ (ඒ පදුමුත්තර) බුදුරජාණන් වහන්සේ පිරිනිවන් පෑ කල්හි (දිව්‍ය මනුෂ්‍යයෝ) ශාස්තෲන් වහන්සේට පූජා කෙරෙත්.
# ´´ජන සමූහයා ඔද වැඩි සිත් ඇත්තාහු අතිශයින් ප්‍රමුදිත වූහ. ඒ බොහෝ සංවේගයන් පවත්නා අතරෙහි මට ප්‍රීතියක් ඇති විය.
# ´´නෑ මිතුරන් කැඳවා, ´´මහා වීරවූ බුදුරජාණන් වහන්සේ පිරිනිවන් පෑ සේක. ඒකාන්තයෙන් පූජාවක් කරමු´´ ය යන මේ වචනය කීමි.
# ´´ලෝකනාථ බුදුරජාණන් වහන්සේ විෂයෙහි පින් රැස් කිරීම කරන්නෙමුය´´ යි ඔවුහු පිළිවදන් දී මාගේ සතුට වැඩියක්ම ඇති කළාහුය.

# ´´එහිද මාහට දිගින් සූවිසි රියන්වූ, පුළුලින් දොලොස් රියන්වූ, විදුලියක් වැනිවූ භවනයක් උස්ව පැන නැංගේය.

# ´´දහසක් මැටි දුනිමි. (යෙදීමි.) මාගේ පහන්හු ත්‍රිලෝක බන්‍ධුවූ බුදුරජාණන් වහන්සේට සතියක් දැල්වී නිවී ගියහ.
# ´´එම චිත්ත ප්‍රසාදයෙන්ද, සිත්හිවූ ප්‍රාර්ත්‍ථනයන්ගෙන්ද මම මිනිස් සිරුර හැර දෙව් විමනක ඉපදුනිමි.
# ´´දෙව් අත්බැව්හි උපන් මට මැනවින් නිර්මිත විමානයක් විය. (එය) හාත්පසින් ඇවිලේ. මෙය පහන් පූජාවෙහි විපාකයයි.
# ´´එකල්හි මම හාත්පස යොදුන් සියයක් බැබලීමි. සියළු දෙවියන් අභිභවමි. මෙය පහන් පූජාවෙහි විපාකයයි.
# ´´ශක්‍රව තිස් කපක් මුළුල්ලෙහි දිව්‍ය රාජ්‍යය කරවීමි. කිසිවෙක් මා ඉක්මවා නොසිතත්. (අවමන් නොකෙරෙත්.) මෙය පහන් පූජාවෙහි විපාකයයි.
# ´´මම අටවිසි වරක් චක්‍රවර්ති රාජ වීමි. එකල්හි හාත්පස යොදුනක් තැන් දිවා රෑ දෙක්හිම (එකසේ) දකිමි.
# ´´(ප්‍රඥා) චක්‍ෂුසින් දහසක් සක්වල (ලෝක) දකිමි. සර්‍වඥ සාසනයෙහි (පැවිදිව) දිවැස ලැබූයේද වෙමි. මෙය පහන් පූජාවෙහි විපාකයයි.
# ´´මෙයින් කල්ප තිස් දහසකට පෙර සුමේධ නම් සම්බුදු රජ තෙම විය. මා විසින් උන් වහන්සේට පැහැදුනු සිතින් පහන් දෙන ලදී. 213 ´´මාගේ ක්ලේශයෝ දවන ලද්දාහුය. සියලු භවයෝ සමූහනනය (විනාශ) කරණ ලද්දාහුය. හස්තිරාජයෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 214 ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
# ´´මේ සිවි පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී´´ මෙහි මේ අයුරින් ආයුෂ්මත් අනුරුද්ධ ස්ථවිරයන් වහන්සේ මේ ගාථාවන් දේශනා කළ සේක. ( අනුරුද්ධ ස්ථවිරයන් වහන්සේගේ අපදානය සතරවැනියි. )

# ´´සංක්‍ෂිප්තව අසා විස්තර වශයෙන් දෙසීමි. එසේ කියන්නාවූ මට සියලු ශිල්පයෝ අසා සතුටුවූවාහි,
# ´´ස්වකීය දෘෂ්ටිය අත්හැර දමා බුදුන් කෙරෙහි සිත පැහැදවූහ. මම සංක්‍ෂිප්තවද දෙසමි. එසේම විස්තර වශයෙන්ද දෙසමි.

# උපාළි ස්ථවිරාපදානය.
# ´´හංසවතී නගරයෙහි රැස්කොට තැබූ අසූකෙළක් ධනය ඇති, සමෘද්ධවූ ධන ධාන්‍ය ඇති, වේද හදාරවන්නාවූ, මන්ත්‍රධරවූ, වේදත්‍රයෙහි පරතෙරට ගියාවූ (අත් පා ආදියෙහි ලකුණු බලා අනාගතය පවසන) ලක්‍ෂණ ශාස්ත්‍රයෙහිද, ඉතිහාසයෙහිද, ස්වකීය බ්‍රාහ්මණ ධර්‍මයෙහිද පරතෙරට ගියාවූ සුජාත නම් බමුණෙක් විය.
# ´´එකසිඛ (හිසැ එක් සිළුවක් ඇති) පරිබ්‍රාජකයෝද, (මතු ලොව පහළවන) ගෞතම බුදුරජාණන් වහන්සේගේ (අනාගත) ශ්‍රාවකයෝද, (තැනින් තැන හැසිරෙන්නාවූ) චරක තවුසෝද, එකල්හි පොළොවෙහි හැසිරෙත්.
# ´´ඔවුහුද ප්‍රසිද්ධ බ්‍රාහ්මණයෙකුයි කියා මා පිරිවරත්. බොහෝ දෙනා මා අදහත්. මම කිසිවෙකුට නොපුදමි.
# ´´යම්තාක් කල් තථාගතයන් වහන්සේ ලොව පහළ නොවන සේක්ද, ඒ තාක් කල් බුද්ධ යන වචනය පවා (ලෙව්හි) නැත. එකල්හි මම මානයෙන් තදවූයේ පිදිය යුත්තෙක් නොදකිමි.
# ´´රෑ දාවල් (බොහෝ හවුරුදු) ගෙවී යාමෙන් (පසු) පසැස් ඇති පදුමුත්තර නම් ලෝක නායක බුදුරජ තෙම සායලු අඳුරු දුරුකොට ලොව පහළවිය.
# ´´පැතුරුණු බොහෝ දෙනා විසින් මැනවැයි සම්මතයෙන්වූ (බුදු) සසුනෙහි (ආදී කතෘවූ) බුදුරජ තෙම එකල්හි හංසවතී නගරයට එළඹි සේක.
# ´´පසැස් ඇත්තාවූ ඒ බුදුරජාණන් වහන්සේ පියාණන්ටත් වැඩ පිණිස ධර්‍ම දේශනා කළ සේක. එකල්හි පිරිසද හාත්පස යොදුනක් (තන්හි පැතුරුණේ) විය.
# ´´මිනිසුන් අතර සුනන්‍ද නම්න් සම්මතවූයේ ඒ තවුස් තෙමේ එකල්හි යම්තාක් බුදු පිරිසද (ඒතාක් තැන) මල්වලින් වැසීය.
# ´´පුෂ්ප මණ්ඩපයෙහි සිට බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේ චතුස්සත්‍යය දේශනා කරණ කල්හි කෙළ ලක්‍ෂයක් (ප්‍රාණීන්) ට ධර්‍මාභිසමය විය.
# ´´බුදු රජ තෙම දිව රෑ සතක් (සතිසක්) උතුම් දහම් වැසි වස්වා අටවෙනි දිනය පැමිණි කල්හි බුදු රජ තෙම සුනන්‍දට මෙසේ පැවසූ සේක.
# ´´මෙතෙම සසරෙහි දෙව්ලොවෙහිද මනුෂ්‍යයන් අතරෙහිද සැරිසරන්නේ භවයන්හිදී සියල්ලන්ට උතුම්ව සැරිසරයි.
# ´´කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි ජනිතවන ගෝත්‍රයෙන් ගෞතම නම්වූ ශාක්‍යයන් වහන්සේ ලොව පහළවන සේක.
# ´´ඒ ගෞතම බුදුරජාණන් වහන්සේගේ ධර්‍මයට දායාද (ධර්‍මයට කොටස් කරුවෙක්) වූ ඖරස පුත්‍රවූ (දහම් අසා සෝවාන් ආදී මාර්‍ග ඵල පිළිවෙළින් රහත්වූයේ බුදු රජුන්ගේ ඖරස පුත්‍රයායි.) ධර්‍මයෙන්ම මවාලුවාක් වැනිවූ පුණ්ණමත්තානිපුත්ත නම් ශ්‍රාවකයෙක් වන්නේය.
# ´´එකල්හි සම්බුදු රජ තෙම සියලු දෙනා සතුටු කරමින් ස්වකීය බලයද දක්වමින් සුනන්‍ද තවුසාට මෙසේ පැවසූ සේක.
# ´´ජනයෝ බුදුරජාණන් වහන්සේ කෙරෙහි පුණ්‍ය කර්‍ම කොට තමාගේ ගතිය පිරිසිදු කෙළේයයි සුනන්‍ද තවුසාට කරණ ලද ඇඳිලි ඇත්තාහු නමස්කාර කෙරෙත්.
# ´´ බුදුරජාණන් වහන්සේගේ වචනය අසා එහි මාගේද අදහස විය. ගෞතම බුදුරජාණන් වහන්සේ යම් පින්කමක් කිරීමෙන් දැකිය හැකි වෙම් නම් මමද (එබඳු) පින්කමක් කරන්නෙමි.
# ´´මම මෙසේ සිතා මාගේ (ඒ සඳහා) කටයුතු (පින්කම්) ගැනද සිතීමි. නිරුත්තර පුණ්‍යක්‍ෂෙත්‍රයෙහි (රත්නත්‍රයෙහි) මම කවර පින්කමක හැසිරෙම්ද?
# ´´(සරභඤ්ඤ වශයෙන් ග්‍රන්ථයන් කියවන හෙයින්) පාඨික නම්වූ මේ භික්‍ෂු තෙම බුද්ධ ශාසනයෙහි සියලු මධුර ස්වර පාඨකයන් අතුරෙහිද විනයෙහිද අග්‍රස්ථානය තබන ලදී. මම ඒ තනතුර පතමි.
# ´´මෙය මාගේ සයුර මෙන් නොකැළඹෙන අප්‍රමාණවූ භෝගයයි. මම ඒ ධනයෙන් බුදුරජාණන් වහන්සේට ආරාමයක් කරවන්නෙමි.
# ´´නගරයාගේ නැගෙනහිර දිගෙහි සෝභන නම්වූ අරමකි. (එය) ලක්‍ෂයක් මිළට ගෙණ සංඝාරාමයක් කරවීමි.
# ´´කුළු ගෙවල්ද, ප්‍රාසාදයන්ද, මණ්ඩපයන්ද, සඳළු පන්නා කළ ගෙවල්ද, දොර නොලූ ගුහාවන්ද, සක්මන් මළුද මැනවින් කොට සංඝාරාමයක් කරවීමි.
# ´´ඩහගන්වන ගෙයද, ගිනිහලද, යලි පැන්හළද, නහන ගෙයද කොට මම සංඝයාට දුනිමි.
# ´´දික්හඟලා පුටුද, (කුඩා) පුටුද, පරිභෝග භාජනද, ආරාමිකයන්ට දෙන බෙහෙත්ද යන මේ සියල්ල මම දුනිමි.
# ´´සන්සුන් සිත් ඇති තාදී ගුණ ඇත්තාවුන්ගේ ඒ ගුණයනට කිසිවෙක් පීඩා නොකෙරේවායි ආරක්‍ෂා තබවා දැඩිකොට පවුරක් කරවීමි.

# ´´යම්සේ (සර්‍ප විෂයෙන්) දුකට පත් පුරුෂයෙක් තෙම විෂයෙන් පෙළෙන ලදුයේ හෙතෙම විෂ නැසීමේ උපායවූ බෙහෙතක් සොයන්නේය.
# ´´සොයන්නේ විෂ නසන්නාවූ බෙහෙතක්ද කියන්නේය. විෂයෙන් මිදීම පිණිස එය බී සැප ඇත්තෙක් වන්නේය.
# ´´එසේම මහා වීරයන් වහන්ස, මම විෂයෙන් නටුවාවූ පුරුෂයෙකු මෙන් අවිද්‍යාවෙන් පෙළුනේ සදහම් බෙහෙත සෙවීමි.
# ´´දහම් බෙහෙත සොයන්නේ සියළු ඖෂධයනට අග්‍රවූ, සියලු (රාගාදී) හුල් දුරු කරන්නාවූ ඒ බුද්‍ධ ශාසනය දුටුවෙමි.
# ´´මම දහම් ඔසුව බී සියළු (සසර) විෂ දුරු කෙළෙමි. අජරාමරවූ, සිසිල්වූ, නිවණ ස්පර්‍ශ කෙළෙමි.
# ´´යම් සේ භූතයෙකු විසින් පෙළන ලද පුරුෂයෙක් තෙම භූත ග්‍රහණයෙන් පීඩිතවූයේ භූතයාගෙන් මිදීම පිණිස භූත වෛද්‍යයෙකු සොයන්නේද,
# ´´සොයන්නේ භූත විද්‍යාවන්හි දක්‍ෂයෙකු දකින්නේය. හෙතෙම ඔහුගේ භූතයා නසන්නේය. සහමුලින්ම වනසන්නේය.
# ´´මහා වීරයන් වහන්ස, එසේම මම ක්ලේශාන්‍ධකාරයෙන් පෙළුනේ (එම) අඳුරින් මිදීම පිණිස ඥානාලෝකය (ප්‍රඥාව) සොයමි.
# ´´ඉක්බිති කෙළෙස් අඳුර දුරු කරන්නාවූ ශාක්‍ය මුනීන්වහන්සේ දුටුවෙමි. උන්වහන්සේ භූත වෛද්‍යයෙක් භූතාවේශයෙන් (දුරු කරන්නාක් මෙන්) මේ මාගේ ක්ලේශාන්‍ධකාරය දුරු කළ සේක.
# ´´සසර සැඩපහර සින්දෙමි. තෘෂ්ණාශ්‍රොතය වැළැක්වීමි. භූත වෛද්‍යයෙකු මෙන් සියළු භවය මුලින්ම නැසීමි.
# ´´යම්සේ ගුරුලෙක් තෙම තමාගේ ගොදුරවූ නාගයා වෙත වේගයෙන් බසීද, මහ සමුදුර හාත් පස යොදුන් සියයක් කළඹාද,
# ´´හෙතෙම යටිකුරුකොට නාගයා ගෙන පෙළන්නේ ඒ ගුරුළුතෙම කැමති තැනකට ගෙනයේද,
# ´´මහා වීරයන් වහන්ස, එසේම එම ගුරුලා නාගයා බිලිකොට ගන්නාක් මෙන් අසංස්කෘත සංඛ්‍යාත නීවරණ සොයන්නේ මම (ක්ලේශ) දොස් (සමුච්ඡේද වශයෙන්) සෝදා හැරීමි.
# ´´නිරුත්තර ශාන්ති පදයවූ, උතුම්වූ නිර්‍වාණ ධර්‍මය දක්නා ලද මම ගුරුළා නාගයෙකු මෙන් මෙය (ධර්‍මය) ගෙන වෙසෙමි.
# ´´චිත්‍රලතා වනයෙහි හටගන්නාවූ ආසාවතී නම් වැල ඒ වැලේ (වයසින්) හවුරුදු දහසකින් එක ගෙඩියක් හට ගනී.
# ´´එතෙක් දිගු කලක් ගොස් පල ඇතිවූ කල්හි ඒ පලය දෙවියෝ ඇසුරු කරත්. මෙසේ ඒ ආසාවතී නම් උතුම් ලතාව දෙවියන්ට ප්‍රිය වන්නේය.
# ´´කල්ප ලක්‍ෂයක පටන් මම ඒ බුදුරජාණන් වහන්සේ ඇසුරු කරමි. උදේ හවස නමඳිමි. දෙවියෝ ආසාවතී ලතාව ඇසුරු කරන්නාක් මෙනි.
# ´´ (බුදුරජාණන් වහන්සේට කරණ) පාරිචරියාව (වත් පිළිවෙත් කිරීම) නොවඳය. නමස්කාරය නොසිස්ය. සසර බොහෝ දුර සිට ආවාවූ මම ඇතිවූ මේ බුද්ධොත්පාදක්‍ෂණය මා ඉක්මවා නොගියේය.
# ´´මම පරීක්‍ෂාකොට බලන්නේ භවයෙහි ප්‍රතිසන්‍ධියක් නොදකිමි. (එසේම) මම (ස්කන්‍ධ කිලේස) උපධි රහිත වූයේ, සියලු කෙලෙසුන්ගෙන් මිදුනේ, සන්සිඳුනේ, හැසිරෙමි.
# ´´පියුම වනාහි යම් සේ හිරු රැස් කරණකොටගෙන පිපේද, එසේම මහා වීරයන් වහන්ස, මමද බුදු රැසින් පිපෙන ලදුයේ වෙමි.
# ´´යම්සේ බලාකායෝනියෙහි (කොක් ජාතියෙහි) හැම කල්හි පිරිමි සත්තු විද්‍යමාන නොවෙද්ද, ඒ කෙකින්නියෝ හැම විටම මේඝ ගර්ජනා කරණ කල්හි ගැබ් ගනිත්.
# ´´යම්තාක් කල් වැඩි කොටස්ද (ඒතාක්) දීර්‍ඝ කාලයක් නමුදු (කෙකින්නියෝ) ගැබ් දරත්. යම් කලෙක්හි වැසි වසීද, එකල්හි (ගැබ්) බරින් මිදෙත්.
# ´´ධර්‍මය නැමැති මේඝයෙන් ගර්ජනය කරන්නාවූ පදුමුත්තර බුදුරජාණන් වහන්සේගේ ධර්ම මේඝයගේ ශබ්දය කරණකොටගෙන මම දහම් ගැබ් ගනිමි.
# ´´කල්ප ලක්‍ෂයක පටන්ම මම (රැස්කළ) පින් නැමැති ගැබ් දරමි. ධර්‍ම නැමැති මේඝය ගර්ජනා නොකෙරන කල්හි (එම ගැබ්) බරින් නොමිදෙමි.
# ´´ශාක්‍ය මුනීන්‍ද්‍රයන් වහන්ස, යම් කලෙක්හි ඔබ වහන්සේ රම්‍යවූ කපිලවත්‍ථු නගරයෙහි ධර්‍ම මේඝයන් ගර්ජනය කෙරෙහි නම්, මම (මෙම පින් ගැබ්) බරින් මිදුනේ,
# ´´(ආත්ම දෘෂ්ටි විරහිත බැවින්) ශූන්‍යතවූ, (ක්ලේශ නිමිත්ත රහිත බැවින්) අනිමිත්තවූ, (තෘෂ්ණා ප්‍රණිධි රහිත බැවින්) අප්පණිහිතවූ, විමොක්‍ෂ ධර්මයන්ද, චතුර්විධ ශ්‍රමණ පලයිද, යන මේ සියලු ධර්මයන් (ඉපදවීමි.) මම මෙසේ සියලු ධර්මයන් පිළිබඳ අවුල් සමූහය නිරවුල් කෙළෙමි. ( දෙවෙනි බණවර නිමි. )
# ´´පමණ නොකළ හැකි කාලයක පටන් ඔබ වහන්සේගේ මේ සසුන පතමි. මා පැමිණෙන ලද එම (ප්‍රාර්ත්‍ථනා සංඛ්‍යාත) අර්ත්‍ථ ඇත්තේ පැමිණෙන ලද ශාන්ති පදයවූ නිවණද ඇත්තේ වෙමි.
# ´´(පදුමුත්තර බුදු සසුනෙහි) පාඨික භික්‍ෂුව මෙන් විනයෙහි තෙරපතුයේ මට සමාන අනිකෙකුදු නැත. මම සසුන (ඔවාදශාසනය) දරමි.
# ´´විනයෙහිද භික්‍ෂු භික්‍ෂණී සංඛ්‍යාත (උභතොවිභංගයෙහි) ඛන්‍ධකයෙහිද (චුල්ලවග්ග මහාවග්ග) දෙක්හි තිකසංඝාදිසේස තිකපාචිත්තියාදියෙහිද පස්වැනිවූ පරිවාරයෙහිද යන මෙහි (විනය පිටකයෙහි) අකුරක හෝ ව්‍යඤ්ජනයක හෝ මාගේ සැකයක් නැත.
# ´´(පාප භික්‍ෂූන්ට) නිග්‍රහ කිරීමෙහිද, (සාපත්තික භික්‍ෂූන්ට පරිවාස දානාදියෙන්) ප්‍රතිකර්‍ම කිරීමෙහිද, කරුණෙහි හා නොකරුණෙහි දක්‍ෂවූයේ, (තර්ජනීයාදී කර්‍මයන්ගේ පටිපස්සද්ධි වශයෙන් සසුනට ඇතුළුකරගන්නා) ඔසාරණයෙහිද, ඇවතින් නැගුටුවාලන වුඨාපනයෙහිද, යන මේ සියල්ලෙහි පරතෙරට පැමිණියේ වෙමි.
# ´´(උභතො විභඞ්ග නම්) විනයෙහිද, (චුල්ලවග්ග මහාවග්ග නම්) ඛන්‍ධකයෙහිද, මම සුදුසු පදය බහා (විභඞ්ග ඛත්‍ධක) දෙකින් නිරවුල්කොට (ක්‍රම දක්වා) කෘත්‍ය වශයෙන් සුදුසු අදහස ඇතුළත් කරමි.
# ´´නිරුක්තියෙහි (ව්‍යවහාර වචනයන්හි නියම අන්‍ර්‍ථ කීමේ ක්‍රමය) ඉතා දක්‍ෂවූයේ, අභිවෘද්ධිය පරිහාණිය යන මේ දෙක්හිද ඡෙකවූයේ (ත්‍රිපිටකයෙහි) මා විසින් දැන නොගන්නා ලද්දක් නැත. බුද්ධ ශාසනයෙහි (මමම) අසහායවූ අග්‍ර විනයධරයා වෙමි.
# ´´මම අද ශාක්‍ය පුත්‍රවූ බුදුරජාණන් වහන්සේගේ සසුනෙහි (විනය විනිශ්චය, විදර්‍ශනා නම්) රූප දර්‍ශනයෙහි සියලු සැක දුරුකරමි. සියලු කුකුස් සිඳීමි.
# ´´පෙර පසු පදද (එක එක) අකුරුද (සිථිලධනිතාදී) දශවිධ ව්‍යඤ්ජන විධානයද (තෙන සමයෙන) යනාදී නිදානයද, නිගමනයද යන මේ සියල්ලෙහි මම දක්‍ෂ වීමි.
# ´´යම් සේ බලවත් රජෙක් තෙම සතුරු රජහට නිග්‍රහකොට තවන්නේද, සංග්‍රාමය ජයගෙන එහි නුවරක් ඉදි කරන්නේද (වේ නම්)
# ´´තාප්පයද, දිය අගලද, ඉන්‍ද්‍රඛීලයද, දොරටුවද, විවිධ අටළුද යන බොහෝ දෙය (මේ) නගරයෙහිද කරවන්නේය.
# ´´සිව්මං ඇතුළු වීථිද, මැනවින් බෙදන ලද ආපන ශාලාවන්ද (එසේම) එහි වරද නිවරද විනිශ්චය කරන අධිකරණ ශාලාවද කරවන්නේය.
# ´´හෙතෙම පස මිතුරන් නැසීම පිණිස, දොස් නිදොස් දැනගැනීම පිණිසද, භට (සේනා) සමූහය රැකීම පිණිසද සෙන්පතියෙකු තබයි.
# ´´හෙතෙම මාගේ භාණ්ඩයන් විනාශ නොවේවායි රාජ භාණ්ඩයන්ගේ ආරක්‍ෂාව පිණිස (ධනය) තැන්පත් කිරීමෙහි දක්‍ෂවූ පුරුෂයෙකු භාණ්ඩාගාර ආරක්‍ෂකයාකොට තබයි.
# ´´යමෙක් රජුට මමත්‍වය ඇත්තේ වේද, යමෙක් ඒ රජුගේ අභිවෘද්ධිය කැමති වේද, ඒ මිත්‍රයාහට (ධර්‍මානුකූලව පිළිපැදීමට) අධිකරණය (විනිශ්චයකාර පදවිය) දෙමි.
# ´´හෙතෙම (උල්කාපාත දික්දාහාදිභූත විකෘතීත් පිළිබඳ වැඩ අවැඩ දැනගන්නා) උත්පාත ශාස්ත්‍රයෙහිද, මූසිකච්ඡින්න (මීයන් කැපීම) ආදියෙහි වැඩ අවැඩ දැනගන්නා නිමිති ශාස්ත්‍රයෙහි, (ස්ත්‍රී පුරුෂයන්ගේ අඞ්ග ලක්‍ෂණ බලා ජීවිතය පිළිබඳ තොරතුරු පවසන) ලක්‍ෂණ ශාස්ත්‍රයෙහිද, දක්‍ෂවූ (බොහෝ ශිෂ්‍යයන්ද) වේද හදාරවන්නාවූ (වේදත්‍රය සංඛ්‍යාත) මන්ත්‍ර දරන්නාවූ පුරුෂයෙකු පුරෝහිත තනතුරෙහි තබයි.
# ´´මේ අංගයන්ගෙන් යුක්තවූයේ ක්‍ෂත්‍රියයායයි කියනු ලැබේ. (ඒ සේනාවෝ) ස්වකීය ඤාතීන්වූ දුඃඛිනියන් රකිනා සක්වා ලිහිණියා මෙන් හැම කල්හි රජතුමා රකිත්.
# ´´මහා වීරයන් වහන්ස, එසේම ඔබ නසන ලද සතුරන් ඇති රජෙකු මෙන් දෙවියන් සහිත ලෝකයෙහි ධර්‍ම රාජ (දහම් රජ) යයි කියනු ලැබේ.
# ´´තීන්‍ර්‍ථකයන්ද, සේනා සහිතවූ මාරයාද නසා ඝන මෝහාන්ධකාරය දුරුකොට ධර්‍ම නගරය මැවීය.
# ´´ප්‍රාඥයන් වහන්ස, ඒ ධර්‍ම නගරයෙහි ප්‍රාකාරය ශීලයයි. දොරටු කොටුව ඔබ වහන්සේගේ ඥානයයි. ඉත්‍ද්‍රඛීලය ඔබ වහන්සේගේ ශ්‍රද්‍ධාවයි. වාරපාලයා නම් සංවරය වෙයි.
# ´´අටල්ල සතිපට්ඨානයයි. සර්‍වඥයන් වහන්ස, ඇතුළු වීථිය ඔබ වහන්සේගේ ප්‍රඥාවය. සිව්මංසල ඎද්‍ධිපාදයයි. (මේ ධර්‍ම නගරය) සත්තිස් බෝධිපාක්‍ෂික ධර්‍ම නැමැති වීථීන්ගෙන් මැනවින් නිර්මිත කරණ ලද.ී
# ´´සූත්‍රය, අභිධර්‍මය, විනය යන සම්පූර්‍ණ නවාඞ්ග බුද්‍ධ වචනය ඔබගේ 6නගරයෙහි) අධිකරණ සභාවයි.
# ´´(අනත්තානුපස්සනාවෙන් ලබන ලද) සුඤ්ඤත විහාරයද, (අනිච්චානුපස්සනාවෙන් ලබන ලද) අනිමිත්ත විහාරයද, (දුක්ඛානුපස්සනාවෙන් ලබන ලද) අප්පණිහිත විහාරයද, (නිශ්චලවූ සතර ශ්‍රාමාණ්‍ය පල සංඛ්‍යාත) අනෙජ විහාරයද, (සියලු දුක් වැළැක්වීම නම්වූ) නිර්‍වාණයද යන මේවා ඔබ වහන්සේගේ ධර්‍ම කුටීහු වෙති.
# ´´ප්‍රඥාවෙන් අග්‍රස්ථානයෙහි තබන ලද වැටහීමෙහි දක්‍ෂවූ නම්න් ශාරීපුත්‍ර (නම් ශ්‍රාවක) තෙමේ ඔබ වහන්සේගේ ධර්‍ම සේනාපතියා වෙයි.
# ´´සර්‍වඥයන් වහන්ස, චුතූපපාත ඤාණයෙහි දක්‍ෂවූ, ඎද්‍ධියෙහි පරතෙරට පැමිණි, නම්න් කොලිත (නම් ශ්‍රාවක) තෙමේ ඔබ වහන්සේගේ පෙරෙවියා (පුරෝහිතයා) වෙයි.
# ´´මුනීන්‍ද්‍රයන් වහන්ස, පෞරාණික වංශය දරන්නාවූ, උග්‍රවූ තේජස් ඇති, (ගුණ ධර්‍මයන්ගෙන්) ලංවිය නොහැක්කාවූ, ධුතවාදීවූ, ආශ්‍රීත ගුණයෙන් අග්‍රවූ, මහාකාශ්‍යප ස්ථවිරයන් වහන්සේ ඔබ වහන්සේගේ අක්ඛදස්ස (හෙවත් ව්‍යවහාර කරණයෙහි ප්‍රධානය.)
# ´´බුදුරජාණන් වහන්ස, ශාසනයෙහි බහුශ්‍රුත ධර්‍මධරවූ, සියලු අධ්‍යයනය කරන්නාවූ, භික්‍ෂූන්ට අග්‍රවූ, නම්න් ආනන්‍ද (නම් ශ්‍රාවක) තෙමේ ඔබ වහන්සේගේ ධර්‍ම (භාණ්ඩාගාර) ආරක්‍ෂකයා වෙයි.
# ´´භාග්‍යවතුන් වහන්සේ මේ සියලු ශ්‍රාවකයන් ඉක්මවා මා පමණ කළ සේක. විනයෙහි විනය දත් නැණවතුන් විසින් ප්‍රකාශ කරණ ලද දෝෂ විචාරණය මටම පැවරූ සේක.
# ´´යම්කිසි බුද්‍ධ ශ්‍රාවකයෙක් තෙම විනයධර්‍ම විෂයෙහි ප්‍රශ්නයක් විචාරාද, එහි මාගේ විමතියක් නැත. මම එම අදහසම කියමි.
# ´´බුද්‍ධ ක්‍ෂෙත්‍රය (ආණක්‍ෂෙත්‍රය) යම් පමණ තන්හි පැතිරේ නම්, එපමණ තන්හි ඒ බුදුරජාණන් වහන්සේ හැර විනය පිටකයෙහි (දක්‍ෂවූ) මා වැනි කෙනෙක් නැත. මට වැඩි කෙනෙක් කොයින් වන්නේද?
# ´´ගෞතම බුදුරජාණන් වහන්සේ භික්‍ෂු සංඝයා මැද හිඳ (උභතොවිභඞ්ග නම්) විනයෙහිද, (චුල්ල වග්ග, මහා වග්ග නම්) ඛන්‍ධකයෙහිද උපාලි භික්‍ෂුව හා සම කෙනෙක් නැතැයි මෙසේ සිංහනාද කරණ සේක.
# ´´බුදුරජාණන් වහන්සේ විසින් දේශනා කරණ ලද නවාඞ්ග ශාස්තෘ ශාසනය යම්තාක් නම් ඒ සියල්ල විනයෙහි ඇතුළත්ය. විනය මුත්කොට ඇත්තේයයි දකින්නාහට
# ´´ගෞතම නම් වූ ශාක්‍ය මුනීන්‍ද්‍රයන් වහන්සේ මාගේ (පූර්‍ව) කර්‍මය සිහිකොට භික්‍ෂු සංඝයා මැද වැඩ හිඳ, මා එතදග්‍රස්ථානයෙහි තැබූ සේක.
# ´´කල්ප ලක්‍ෂයක (පමණ කාලයක) පටන් මෙම තනතුරු පැතීමි. මා විසින් පැමිණෙන ලද එම අර්ත්‍ථ ඇත්තේ විනයයෙහි පරතෙරට පත්වීමි.
# ´´මම පෙර ශාක්‍යයනට සොම්නස ඇතිකරන්නාවූ කපුවෙකු වීමි. ඒ කුළය හැර මහර්ෂීවු බුදුරජාණන් වහන්සේගේ පුත්‍රයෙක් (බුද්ධ පුත්‍රයෙක්) වීමි.
# ´´මෙයින් (මේ මහා භද්‍ර කල්පයෙන්) දෙවැනි කල්පයෙහි, අපමණ තෙද ඇති, අනන්තවූ පිරිවර ඇති, මහා ධනවත්, පෘථිවිපාලකවූ ´´අඤ්ජස´´ නම් රජෙක් වීමි.
# ´´පරිවාර මදයෙන්ද, භෝග මදයෙන්ද, ජාති මදයෙන්ද මත්වූ මම එම රජුගේ පුත්‍රවූ ´´චන්‍දන´´ නම් ක්‍ෂත්‍රියයා වීමි.
# ´´සියළු (ඇත්) අබරණින් අලංකාර කරණ ලද්දාවූ, ත්‍රිමද ගලිතවූ, හස්ති කුලයෙහි උපන් ලක්‍ෂයක් පමණ ඇත්තු නිතර මා පිරිවරත්.
# ´´ස්වකීය භට සේනාවන් විසින් පිරිවරණ ලද මම උයනට යනු කැමැත්තේ එකල්හි ´´සිරික´´ නම් ඇතු පිට නැගී නුවරින් නික්මුණෙමි.
# ´´පසළොස් චරණ ධර්‍මයන්ගෙන් යුත් වසන ලද (චක්‍ෂුරාදී) දොරටු ඇති, මැනවින් රැකි කාය චිත්ත ඇති, ´´දෙවල´´ නම් පසේ බුදුරජාණන් වහන්සේ මාගේ ඉදිරිපසින් පැමිණි සේක.
# ´´එකල්හි ´´සිරික´´ හස්තියා යවා පසේ බුදුරජාණන් වහන්සේට කරදර කළෙමි. ඉන් හටගත් කෝප ඇති එම ඇත් තෙමේ පියනුගුළුවයි. (නිශ්චලව සිටියි.)
# ´´මම දූශ්‍ය සිත් ඇති ඇතු දැක බුදුන් කෙරෙහි කෝප කෙළෙමි. පසේ බුදුරජාණන් වහන්සේට පීඩාකොට මම උයනට ගියෙමි.
# ´´එහි ඒ (නිමිත්තෙන්) හිස ඇවිලගත්තෙකු මෙන් සුවයක් නොවිඳීමි. බිලිය ගිලි මසෙකු මෙන් පසුතැවිලි දැවිල්ලෙන් දැවෙමි.
# ´´මම සයුර කෙළවරකොට ඇති පෘථිවිය ඇවිලගත්තාක් මෙන් විය. (මම) පියාගේ සමීපයට එළඹ මේ වචන කීමි.
# ´´කිපුණු සර්‍පයෙකු මෙන්ද, පැමිණි ගිනි කන්‍දක් මෙන්ද, දළ ඇති මතැතෙකු මෙන්ද යම් පසේ බුදුවරයන් වහන්සේ කෙනෙකුට ගැටීම්ද,
# ´´අනිකෙකු විසින් නොගැටිය හැකි බැවින් ඝෝරවූ උග්‍ර තපස් ඇති පස් මරුන් දිනූ පසේ බුදු රජ තෙම මා විසින් ගටන ලදී. (ඉන්) සියලු දෙනම වැනසෙමු. (ඊට) පළමු ඒ පසේ බුදුරජාණන් වහන්සේ ක්‍ෂමා කොට ගතිමි.
# ´´ඉදින් චිත්ත දමනය කළ එකඟ සිත් ඇති ඒ පසේ බුදුරජාණන් වහන්සේ ක්‍ෂමා නොකරන්නෙමු නම් සත් දිනකින් මෙපිට මගේ රට විනාශ වන්නේය.
# ´´සුමෙචලය, කොසියය, සිග්ගවය, සත්තුකය යන මේ (සිව්) රජවරු ඎෂීනට ගටා (අනදර) කොට ඔවුහු ස්වකීය රටවල් සමග දුකට පත්වූහ.
# ´´කාය සංයමයෙන් යුත් බ්‍රහ්මචාරීවූ ඎෂීහු යම් කලෙක්හි කිපෙත්ද, (එකල) සයුරු පර්‍වත සහිතවූ දෙවියන් සහිත ලෝකය වැනසෙත්.
# ´´තුන් දහසක් යොදුන් තන්හි පුරුෂයන් රැස් කරවීමි. වරද පැවසීම (ඇවැත් දෙසීම) සඳහා පසේ බුදුරජාණන් වහන්ස් වෙත එළඹියෙමි.
# ´´තෙත්තූ වස්ත්‍ර ඇත්තාවූ, තෙත්වූ කෙස් ඇත්තාවූ, සියල්ලෝම බඳින ලද ඇඳිලි ඇත්තාහු, බුදුරජාණන් වහන්සේගේ පාමුල වැටී මේ වචන කීවාහුය.
# ´´මහා වීරයාණෙනි, ඔබ සමාවනු මැනවි. ජන සමූහයා එය විශේෂයෙන් ආයාචනය කරයි. පරිලාහය (දැවිල්ල) දුරුකරනු මැනවි. අප නොනසනු මැනවි.
# ´´දෙවි මිනිසුන් සහිතවූ, අසුරයන් හා රාක්‍ෂයන් සහිතවූද සියල්ලෝ හැම කල්හි යකුලින් මාගේ හිස පළන්නාහුය.
# ´´දියෙහි ගිනි නොසිටී. ගලෙහි බිජුවට පැළ නොවේ. බෙහෙතෙහි කෘමි තෙමේ නොසිටී. (එසේම) බුදුරජුන් කෙරෙහි කෝපය හට නොගනී.
# ´´යම් සේ පොළොව නොසෙල්විය හැකිද, මහ සයුර පමණ කළ නොහැකිද, අහස කෙළවරක් නැත්තේද, එසේම බුදුවරයෝ නොකැළඹිය හැක්කේය.
# ´´මහාවීරවූ බුදුවරයෝ හැම කල්හි ඉවසීම ඇත්තෝය. (අනුන්ගේ වැරදි) ඉවසන්නාහු තපස්වීහුය. ඉවසීමෙන් යුත්වූවන්ගේද, ඉවසන්නවුන්ගේද (ඡන්‍දාදීන්වූ) එබඳු (අගති) ගමනක් නොවේ.

# ´´යම් කෙනෙකුගේ කන් යොමු කිරීමක් ඇත් නම් කියන්නාවූ මාගේ (වචනය) අසව්. නිර්‍වාණ පදය මා දක්නා ලද පරිදි (එම නිර්‍වාණ සංඛ්‍යාත) අර්ත්‍ථය තොපටද කියමි.
# ´´ශාන්ත සිත් ඇති එකඟවූ ඒ පසේ බුදුරජාණන් වහන්සේට අවමන්කොට මම ඒ කර්‍මය හේතුකොට ගෙන පහත් (කපු) කුලයෙහි උපන්නෙමි.
# ´´තෙපි (මේ බුදේ‍ධා්ත්පාදය) ක්‍ෂණය නිශ්පල නොකරව්. ඒ එසේමැයි. (බුදේ‍ධා්ත්පාද) ක්‍ෂණය ඉක්මවූවෝ ශෝක කෙරෙත්. ස්වකීයාභිවෘද්ධියෙහි වෑයම් කරව්. තොප විසින් පැමිණෙන ලද බුදේ‍ධා්ත්පාද ක්‍ෂණ ඇත්තෝය.
# ´´ඇතැම් අයට වමණයද, සමහරුනට විරේචනයද, එසේම ඇතැමුන්හට හළාහල විෂයද ඖෂධ වේ.

# ´´චරක් පානය කළ හළාහල විෂය ජීවිතය විනාශ කෙරෙයි. (එසේම) සසුන හා විරුද්ධවූ (ඒ පවිටු මිනිසා) කප් කෙළ ගණනක් දැවේ.

# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
# ´´මේ සිවි පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී´´ මෙහි මේ අයුරින් ආයුෂ්මත් උපාලි ස්ථවිරයන් වහන්සේ මෙම ගාථාවන් දේශනා කළ සේක. ( උපාලි ථෙරාපදානය සවැනියි. )

# අඤ්ඤාත කොණ්ඩඤ්ඤ ථෙරාපදානය.
# ´´(සකළ සත්ත්‍ව) ලෝකයෙහි ජ්‍යෙෂ්ඨවූ (විශේෂයෙන් වින්‍යෙය සත්ත්‍වයන් සසර සයුරෙන් එතෙරට පමුණුවන බැවින්) විනායකවූ බුදු පදවියට පත් පදුමුත්තර සම්‍යක් සම්බුදුරජාණන් වහන්සේ මම පළමුකොටම දුටිමි.
# ´´බෝමුලෙහි යම් පමණ සත්ත්‍වයෝ රැස් වූවාහුද, ඒ සියල්ලෝ බැඳැඳිලි ඇත්තාහු බුදුරජුන් පිරිවරා වඳිත්.
# ´´මෙතෙම ලබන ලද බුදුබව ඇත්තේ මොහාන්‍ධකාරය දුරු කෙරේයයි සතුටු සිත් ඇති ඒ සියලු දෙවියෝ අහසෙහි හැසිරෙත්.
# ´´සම්‍යක් සම්බුදුරජාණන් වහන්සේගේ සසුනෙහි කෙළෙස් දවමුයයි සතුටින් යුත් මහා නාදය පවතී.
# ´´වචනය සමගම කියන ලද දෙවියන්ගේ ශබ්දය අසා මම සතුටුවූයේ සතුටු සිතින් ප්‍රථම භික්‍ෂාව දුනිමි.
# ´´මාගේ අදහස දැන ලෝකයෙහි අනුත්තරවූ ශාස්තෲන් වහන්සේ දිව්‍ය සමූහයා මැද හිඳ මෙම ගාථාවන් වදාළ සේක. 
# ´´අභිනිෂ්ක්‍රමණය කොට සතියකින් මම සම්‍යක් සම්බෝධියට පැමිණියෙමි. බ්‍රහ්මචාරීව මාගේ (සිරුර) යැපෙන මෙය පළමු දානයයි.
# ´´යමෙක් තුසිත දෙව්ලොව සිට මෙහි අවුත් මට භික්‍ෂාව පිළිගැන්වීද, මම ඔහු ගැන කියන්නෙමි. එසේ කියන්නාවූ මාගේ වචනය අසව්.
# ´´කල්ප තිස් දහසක් දිව්‍ය රාජ්‍යය කරන්නේය. සියලු දෙවියන් අභිභවනයකොට දෙව්ලොව වසන්නේය.
# ´´දෙව්ලොවින් චුතව මිනිසත් බවට යන්නේය. එහි දහස් වරක් රාජ්‍යය කරන්නේය.
# ´´(මින්) කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ, ගෝත්‍රයෙන් ගෞතම නම්වූ, ශාස්තෲන් වහන්සේ ලොව පහළවන සේක.
# ´´හෙතෙම දෙව්ලොවින් චුතව මිනිසත් බවට යන්නේය. ගිහිගෙන් නික්ම පැවිදිව සවසක් (දුෂ්කර ක්‍රියා කරමින්) වසන්නේය.
# ´´ඒ අභිනිෂ්ක්‍රමණයෙන් සත්වැනි වර්‍ෂයෙහි බුදුරජතෙම සත්‍ය (ධර්‍මය) දේශනා කරන්නේය. නම්න් කොණ්ඩඤ්ඤ නම් තැනැත්තේ පළමුකොට (ධර්‍මය) ප්‍රත්‍යක්‍ෂ (අවබෝධය) කරන්නේය.´´
# ´´(ගෘහාභිනිෂ්ක්‍රමණය කළ) බෝසතුන් හා (නික්ම) ඒ අනුව පැවිදිවීමි. මා විසින් වීර්‍ය්‍යය මැනවින් කරණ ලදී. කෙලෙස් දැවීම පිණිස (අනගාර්‍ය්‍ය නම්) සසුනෙහි පැවිදිවීමි.
# ´´දෙවියන් සහිත ලෝකයෙහි (සියළු දතයුතු ධර්‍මයන් අවබෝධ කොට) සර්‍වඥවූ බුදුරජ මිගදාය නම් ඉසිපතනයට එලඹ (එහිදි) අමා බෙරය ගැසීය.

# ´´අග්‍ර පුද්ගලවූ තෙමේම සියල්ල අවබෝධ කළාවූ පදුමුත්තර නම් බුදුරජාණන් වහන්සේ එකල්හි හිමවතට ඉදිරිපසවූ චිත්‍රකූට පර්‍වතයෙහි විසීය.

# ´´මනාව පිපුණු පියුමක් ඩැහැගෙන නරශ්‍රෙස්ඨවූ බුදුරජාණන් වහන්සේ වෙත එළඹියෙමි. සමාධියෙන් නැගී සිටියාවූ තථාගතයන් වහන්සේට (එය) පිදීමි.

# ´´(අපි බුදුන් වෙමුයි) කියන්නවුනට ශ්‍රෙෂ්ඨවූ බුදුරජතෙම පැමිණිසේක. අපි (උන්වහන්සේගේ) එම ධර්‍මය අසමුයයි සියලු දෙවියෝ සර්‍වඥයන් වහන්සේගේ වචනය දැන රැස්වූහ.
# ´´ලෝකනායකවූ, දීර්‍ඝ දර්ශීවූ, බුදුරජාණන් වහන්සේ ඒ ප්‍රීතියෙන් යුක්තවූ ඔවුන් ඉදිරියෙහි මාගේ ශ්‍රද්‍ධාව ප්‍රකාශ කළ සේක.
# ´´යමෙකු විසින් මේ පියුම දෙන ලදද, සිංහ නාදය කරණ ලදද, මම ඔහු පිළිබඳ තොරතුරු කියන්නෙමි. (එසේ) කියන්නාවූ මාගේ වචනය අසව්.
# ´´මෙයින් අටවෙනි කල්පයෙහි සත්රුවන් සහිතවු සතර මහා දිවයිනෙහි අධිපතිවූ සක්විති රජෙක් වන්නේය.
# ´´සූසැට ජාතියෙක පෘථිවියෙහි අධිපතිවූ (රජ) බව කරන්නේය. මහා බල ඇති නම්න් පදුම නම්වූ සක්විති රජ වන්නේය.
# කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ ගොත්‍රයෙන් ගෞතම නම් වන්නාවූ ශාස්තෲන් වහන්සේ ලොව පහළවන සේක.
# ´´(එකල්හි හෙතෙම) බ්‍රාහ්මණයෙක් වන්නේය. (ගෞතම) බුදුරජාණන් වහන්සේ දහම් දෙසූ කල්හි බ්‍රාහ්මණ කුලයෙන් නික්ම එකෙණෙහි පැවිදි වන්නේය.
# ´´වීර්‍ය්‍ය කිරීම පිණිස මෙහෙයන ලද සිත් ඇත්තේ උපශාන්තවූ (ක්ලේශ රහිත බැවින්) නිරූපධිවූ හෙතෙම සියලු ආශ්‍රවයන් ප්‍රහීණකොට ආශ්‍රව රහිතවූයේ පිරිනිවන් පාන්නේය.
# ´´චණඩ මෘගයන්ගෙන් ගැවසිගත් ජන සම්බාධ රහිත දුර අරන් සෙනසුන්හි (වැස) සියළු ආශ්‍රවයන් ප්‍රහීණකොට ආශ්‍රව රහිතව පිරිනිවන් පාන්නේය.

# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.

# ´´ඔබහට මැනවින් කළ නැවෙහි සටහන් ඇති විමානයක් (දෙව්ලොව) පහළ වන්නේය. හැම කල්හි අහසෙහි මල් වියනක්ද දරන්නේය.
# ´´(මින්) අට පණස් කපකින් මත්තෙහි සතර මහා දිවයිනට අධිපතිවූ, දිනනසුළුවූ, තාරක නම් සක්විති රජෙක් වන්නේය.
# ´´(මින්) සත් පණස්වන කපෙහි මහ බල ඇති චම්පක නම් ක්‍ෂත්‍රිය තෙමේ උදාවන්නාවු හිරු මෙන් බබලන්නේය.
# ´´කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ ගොත්‍රයෙන් ගෞතම නම් වන්නාවූ ශාස්තෲන් වහන්සේ ලොව පහළවන සේක.
# ´´හෙතෙම (එකල) දෙව්ලොවින් චුතව මිනිසත් බවට යන්නේය. නම්න් රේවතවූ බමුණෙක්ද වන්නේය.
# ´´කුශල මූලයන්ගෙන් මෙහෙයන ලදුයේ ගිහිගෙන් නික්ම ගෞතම බුදුරජාණන් වහන්සේගේ සසුනෙහි පැවිදි වන්නේය.
# ´´පැවිදිව පසුව හෙතෙම භවනායෝගයන්හි යෙදුනේ, විදර්‍ශනා කරන්නේ, සියලු ආශ්‍රවයන් (ප්‍රහාණ පරිඥාවෙන්) දැන, ආශ්‍රව රහිතව පිරිනිවන් පාන්නේය.
# ´´නිවාණ සංඛ්‍යාත යොග ක්‍ෂෙමයට පමුණුවන ධුරවාහකයා (ගැල්බර උසුලන ගවයා) මෙන් මාගේ වීර්‍ය්‍යය වෙයි. (මම මේ) සර්‍වඥ ශාසනයෙහි අන්තිම ශරීරය දරමි. (අන්තිම ආත්මයයි.)
# ´´මා විසින් කල්ප ලක්‍ෂයකට පෙර කරණ ලද කර්‍මය මෙහිදී විපාක දැක්වීය. දුනුදියෙහි මැනවින් මිදුණු හී වේගය මෙන් (එම කර්‍මය) මාගේ කෙළෙස් දැවීය.
# ´´ඉක්බිති ලෝකාන්තයට ගිය මහ නුවණැති බුදුරජාණන් වහන්සේ වන සෙනසුනෙහි ඇළුනාවූ මා දැක වනවාසී භික්‍ෂූන් අතුරෙන් අගතන්හි තබා වදාළ සේක.
# ´´මාගේ ක්ලේශයෝ දවන ලද්දාහුය. සියලු භවයෝ සමූහනනය (විනාශ) කරණ ලද්දාහුය. හස්තිරාජයෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
# ´´මේ සිවි පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී´´ මෙහි මේ අයුරින් ආයුෂ්මත් ඛදිර වනිය රේවත තෙරුන් වහන්සේ මේ ගාථාවන් වදාළ සේක. ( ඛදිරවනිය රෙවත ථෙරාපදානය නවවැනියි. ) __________ 
# අපදාන පාළිය. _____ 
# ථෙරාපදානය. _____ 
# බුද්ධ වග්ගය. _____
# ආනන්‍ද ථෙරාපදානය.
# ´´පදුමුත්තර බුදුරජතෙම අරම් දොරින් නික්ම අමා වැසි වස්වන්නේ මහාජනයා සන්සිඳවීය. (නිවණට පැමිණවීය.)
# ´´ෂඩභිඥා ඇත්තාවූද, මහා ඎද්‍ධි ඇත්තාවූද, ලක්‍ෂයක් පමණවූ ඒ ක්‍ෂීණාශ්‍රවයෝ සෙවනැල්ලක් මෙන් සර්‍වඥයන් වහන්සේ පිරිවරත්.
# ´´උතුම් සේසත (මාගේ මස්තකයෙහි දරන්නේ) ඇතුපිට හුන්නේ වෙමි. මනොඥවූ රූප ඇති බුදුරජාණන් වහන්සේ දැක මට ප්‍රීතිය උපන්නේය.
# ´´ඇතුපිටින් බැස නර ශ්‍රෙෂ්ඨයන් වහන්සේ වෙත එළඹියෙමි. මාගේ රුවන් මුවා ඡත්‍රය බුද්ධ ශ්‍රේෂඨයන් වහන්සේට දැරීමි.
# ´´පදුමුත්තර මහර්ෂී (බුදුරජ) තෙම මාගේ අදහස දැන, එම (තමන් වහන්සේගේ දේශනා) කථාව හැර මෙම ගාථාවන් වදාළෝය.
# ´´ඒ යමෙක් ස්වර්‍ණාලංකාරයන්ගෙන් සරසන ලද ඡත්‍රයක් දැරීද, ඔහු (පිළිබඳව) ප්‍රකාශ කරන්නෙමි. (එසේ) පවසන්නාවූ මාගේ (වචනය) අසව්.
# ´´මේ පුරුෂ තෙම මින් (චුතව) ගොස් තුසිතයෙහි වසන්නේය. දිව්‍යප්සරාවන් විසින් පිරිවරණ ලදුයේ සම්පත් අනුභව කරන්නේය.
# ´´සූතිස් වරක් දිව්‍ය රාජ්‍යය කරන්නේය. (ඉන් චුතව) සිව්රඟ සෙනගට අධිපතිව එකසිය අට වරක් (පෘථිවීස්වරව) පොළොවෙහි වාසය කරන්නේය.
# ´´අට පණස් වරක් සක්විති රජ වන්නේය. පෘථිවියෙහි මහත්වූ ප්‍රදේශ රාජ්‍යය කරවන්නේය.
# ´´කල්ප ලක්‍ෂයකින් මතුයෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ ගෝත්‍රයෙන් ගෞතම නම්වූ ශාස්තෲන් වහන්සේ ලොව පහළවන සේක.
# ´´ශාක්‍ය කුලයට ධජයක් වැනිවූ බුදුරජාණන් වහන්සේගේ ඤාතීවූ මහර්ෂීන් (බුදුන්) වහන්සේගේ උපස්ථායකවූ නම්න් ආනන්‍ද නම් වන්නේය.
# ´´කෙළෙස් තවන වීර්‍ය්‍ය ඇති, ප්‍රඥාවත්වූ, බහුශ්‍රුත භාවයෙහි දක්‍ෂවූ, නිහතමානී පැවතුම් ඇති, තද ගති නැති, සියල්ලන් හදාරවන (ත්‍රිපිටකධර) කෙනෙක් වන්නේය.
# ´´වීර්‍ය්‍ය කිරීමෙහි මෙහෙයන ලද සිතැති හෙතෙම, උපශාන්ත වූයේ, උපධි රහිත වූයේ, සියලු ආශ්‍රවයන් (ප්‍රහාණ පරිඥාවෙන්) දැන, ආශ්‍රව රහිතව පිරිනිවන් පාන්නේය.
# ´´(යම් සේ) අරණෙහි උපතින් සැට වසක් ඉක්ම ගිය, ත්‍රිමදගලිත, මාතංග හස්ති කුලයෙහි ජාතවූ, මහත් දළ ඇති, රාජ වාහනවූ, ඇත්තු වෙද්ද,
# ´´එසේම නොයෙක් ලක්‍ෂ ගණන් පණ්ඩිතවූද, මහා ඎද්‍ධිමත්වූද, රහත් නාගයෝ වෙත්. සියල්ලෝ බුද්ධ නාගයන් වහන්සේගේ ලක්‍ෂණය ගත් අය නොවෙති.
# ´´මුල්යම්හිද, මැදියම්හිද, පැසුළුයම්හිද, නමස්කාර කරන්නෙමි. පහන් සිත් ඇත්තෝ සතුටුවූයේ, බුද්ධ ශ්‍රෙෂ්ඨයන් වහන්සේට උවටැන් කෙළෙමි.
# ´´කෙළෙස් තවන වීර්‍ය්‍යයෙන් යුක්තවූයේ, ප්‍රඥාවත්වූයේ, මනා දැනුම් ඇත්තේ, සිහියෙන් යුක්තවූයේ, ශෛක්‍ෂ්‍ය භූමියෙහි දක්‍ෂවූයේ, සෝවාන් ඵලයට පැමිණියේ වෙමි.
# ´´මෙයින් කල්ප ලක්‍ෂයකට පෙර යම් කර්‍මයක් කොට (යමක්) ප්‍රාර්ත්‍ථනා කෙළෙම් නම්, මම ඒ පදවියට පැමිණියේ, සසුනෙහි නිශ්චලව සිටියේ වෙමි.
# ´´මාගේ ක්ලේශයෝ දවන ලද්දාහුය. සියලු භවයෝ සමූහනනය (විනාශ) කරණ ලද්දාහුය. හස්තිරාජයෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පමුණුවන ලද්දාහු බුදුරජාණන් වහන්සේගේ සසුනද කරණ ලදී.
