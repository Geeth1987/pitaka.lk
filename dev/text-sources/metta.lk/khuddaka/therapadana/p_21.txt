කණිකාරපුප්ඵිය වග්ග.  
# කණිකාරපුප්ඵිය ථෙරාපදානය.
# ´´මම එකල්හි පිපුණු කිණිහිරි රුකක් දැක, චතුරෝයන් තරණය කළාවූ, තාදී ගුණ ඇති තිස්ස බුදුරජාණන් වහන්සේට (එම මල් නෙලා) දුනිමි.
# ´´මින් දෙඅනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් පන්තිස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´අරුණපාණී´´ යයි ප්‍රකටවූ සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කණිකාරපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මිනෙලපුප්ඵිය ථෙරාපදානය.
# ´´හිරු මෙන් ප්‍රතාපවත්වූ, ස්වර්‍ණ වර්‍ණවූ භාග්‍යවත්වූ, මෙත් සිත් ඇති සිඛී බුදුරජතෙම සක්මණට නැගුනේමය.
# ´´පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, ශ්‍රේෂ්ඨ ඥාන ඇති බුදුරජුන්හට වැඳ මිනෙල පුෂ්පයන් ගෙන (එම) බුදුරජුන්හට පිදීමි.
# ´´මින් එක්තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´එකුන් තිස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´සුමෙධයස´´ යයි ප්‍රකටවූ සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මිනෙලපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කිඞ්කිණිකපුප්ඵිය ථෙරාපදානය.
# ´´රන් ඇගෑවක් මෙන් බබළන සර්‍වඥවූ ලෝකනායක බුදුරජතෙමේ සිසිල් දිය සහිත විලකට බැස ජල ස්නානනය කළ සේක.
# ´´මම කිකිණි මල් ගෙන ඔදවැඩි සිත් ඇත්තේ සොම්නස්වූයේ, විපදෝත්තමවූ, තාදී ගුණ ඇති විපස්සී බුදුරජුන්හට පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සත්විසිවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති භීමරථ නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කිඞ්කිණිපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තරණිය ථෙරාපදානය.
# ´´වීපදෝත්තමවූ, නරශ්‍රේෂ්ඨවූ භාග්‍යවත් අත්‍ථදස්සී බුදුරජාණන් වහන්සේ ශ්‍රාවකයන් විසින් පිරිවරන ලදුයේ ගංතෙරට එළඹිසේක
# ´´(මම) සමව මුවවිට දක්වා පිරුණු, කපුටුවනට පවා ඉවුරෙහි සිට පැන් පානය කළහැකිසේ දියෙන් පිරීගිය එම ගඟින් තරණයකොට එතරට යාම දුෂ්කරවූ වීපදෝත්තමවූ බුදුරජුන්ද, භික්‍ෂු සංයාද එතර කරවීමි.
# ´´එක්දහස් අටසියවන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (ගඟින්) තරණය කරවීමෙහි මේ විපාකයයි.
# ´´මින් එක් දහස් තෙසියවෙනි කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´සම්භාගව´´ නම් පස් සක්විති රජකෙනෙක් වූහ.
# ´´මම මේ පශ්චිම භවයෙහි බමුණු කුලයෙහි උපනිමි. යහළුවන් තිදෙනෙකු සමග ශාස්තෘ ශාසනයෙහි පැවිදිවීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තරණිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# නිග්ගුණ්ඩිපුප්ඵිය ථෙරාපදානය.
# ´´(මම) විපස්සී බුදුරජාණන් වහන්සේගේ ආරාමිකයෙක් වීමි. (ඒ මම) නිකමල් ගෙන බුදුරජාණන් වහන්සේට පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් පන්තිස්වන කපෙහි මහත් බල ඇති ජනාධිපතිවූ මහාප්‍රතාප නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් නිග්ගුණ්ඩිපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# උදකදායක ථෙරාපදානය.
# ´´මම පැහැදුණු ඉඳුරන් ඇති, කෙළෙස් කැළඹිලි නැති, වළඳන්නාවූ ශ්‍රමණයන් වහන්සේ කෙනෙක් දැක එම සිද්‍ධාර්ත්‍ථ බුදුරජුන්හට කලයෙන් දිය ගෙනවුත් දුනිමි.
# ´´මම අද පහවූ සැක ඇති පිරිසිදුවූ නිර්‍මල කෙනෙක් වෙමි. (ඒ හේතුවෙන්) භවයෙහි උපදිනා කල්හි යහපත් විපාක ඇතිවේ.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් ජල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. උදක දානයාගේ මේ විපාකයයි.
# ´´මින් සැටවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´විමල´´ නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උදකදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සලලමාලිය ථෙරාපදානය.
# ´´පර්‍වතයන් අතුරෙන් කිණිහිරි රුකක් මෙන් බබළමින් හුන් සියළු දිගුන් හෙලිකරන්නාවූ සිද්‍ධාර්ත්‍ථ බුදුරජුන් දුටිමි.
# ´´එකල්හි මම දුන්න නිසැකකොට හීය යොදා (විද) නටු සහිත මලක් කඩා බුදුරජුන්හට පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් එක් පණස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ජුතින්‍ධර නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සලලමාලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කොරණ්ඩපුප්ඵිය ථෙරාපදානය.
# ´´සක් ලකුණින් බබළණ තබන ලද ශ්‍රීපාද සටහන් දැක (මම ඒ) විපස්සී බුදුරජුන්ගේ පියවර සටහන් අනුව යන්නේ
# ´´පිපුණු කොරණ්ඩ වෘක්‍ෂයක් දැක මුල සිටම ඇති ඒ සියළු මල් මා විසින් (බුදුරජුන්හට) පුදන ලදී. සතුටුවූයේ සතුටු සිතින්ම උතුම් පා වැන්දෙමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පියවර සටහන් පිදීමෙහි මේ විපාකයයි.
# ´´සත් පණස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´වීතමල´´ නම් එක් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කොරණ්ඩපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ආධාරදායක ථෙරාපදානය.
# ´´ලෝකබන්ධුවූ සිඛී බුදුරජුන්හට මා විසින් ආධාරකයක් (පාත්‍රදාරයක්) පුදන ලදී. මම (ඒ හේතුවෙන්) ධනය සහිත මේ පොළොවම දරමි.
# ´´මාගේ සියළු කෙළෙස් දවන ලදී. සියළු භවයෝ වනසන ලදහ. සම්බුදු සසුනෙහි අන්තිම ශරීරයද දරමි.
# ´´මින් සත්විසිවන කපෙහි මහත් බල ඇති ´´සමන්තචරණ´´ නම් සිව් සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආධාරදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වාතාතපනිවාරිය ථෙරාපදානය.

# ´´දෙවියනට දෙවිවූ, තාදී ගුණ ඇති තිස්ස බුදුරජුන්හට පැහැදුනු සිතින් එක් කුඩයක් මා විසින් පුදන ලදී. (ඒ හේතුවෙන්) මාගේ පව් වැලැක්කේ විය. (තමා වෙත කුශලයන්ගේ පැමිණීම විය. අහසෙහි කුඩ දරත්. පූර්‍ව ඡත්‍ර දානමය) කර්‍මයාගේ මේ විපාකයයි.
#
# ´´මා විසින් සියළු භවයෝ නසන ලදහ. (මෙය) මාගේ අන්තිම භවයයි. සම්බුදු සසුනෙහි අන්තිම ශරීරයද දරමි.
# ´´මින් දෙයානූවන කපෙහි වූ එකල්හි යම් ඡත්‍ර දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඡත්‍ර දානයාගේ මේ විපාකයයි.
# ´´මින් දෙසැත්තෑවන කපෙහි ජනාධිපතිවූ මහානිදාන නම් අට සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වාතාතපනිචාරිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
