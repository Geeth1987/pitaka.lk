 මන්දාරව පුප්ඵිය වග්ග.   
# මන්දාරව පුප්ඵිය ථෙරාපදානය.

# ´´මඬගල නම් මානවකවූ මම මදාරා මල් ගෙන, තව්තිසා දෙව්ලොව සිට මෙහි පැමිණ, සමාධියෙන් වැඩහුන් විපස්සී බුදුරජුන්ගේ මස්තකයෙහි දැරීමි. (ඒ මල් කූඩයක් මෙන්) සතියක් (බුදුරජුන්හට) දරා සිට නැවත දෙව්ලොවට ගියෙමි.
#
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මන්දාරව පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කක්කාරු පුප්ඵිය ථෙරාපදානය.
# ´´(මම) යාම දෙව්ලොව සිට මෙහි පැමිණි ශ්‍රීමත්වූ ගෞතමයන් වහන්සේ (දුටිමි.) (දැක) කැකිරි මල් ගෙන (එම) බුදුරජුන්හට පිදීමි.
# ´´මින් දෙයානූවන කපෙහි යම් බුද්ධ පූජාවක් (කැකිරි මල් පිදිමක්) කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කක්කාරු පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# භිසමුළාලදායක ථෙරාපදානය.
# ´´සියළු ධර්‍මයන්ගේ තෙරපත් ඵුස්ස නම් සර්‍වඥවරයෙක් විය. සප්‍රාඥවූ, විවේකය කැමති උන්වහන්සේ මාගේ සමීපයට පැමිණි සේක.
# ´´පස් මරුන් ජයගත්, මහා කාරුණීකවූ එම බුදුරජාණන් වහන්සේ කෙරෙහි සිත පහදවා, මම නෙළුඹල සහ දඬු ගෙන බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේට දුනිමි.
# ´´මින් දෙයානූවන කපෙහි වූ එකල්හි යම් නෙළුඹල දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. නෙළුඹල දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් භිසමුළාලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කෙශරී පුප්ඵිය ථෙරාපදානය.
# ´´එකල්හි මම හිමවත් පව්වෙහි විද්‍යාධරයෙක් වීමි. (එහිදී) මහත් යසස් ඇති, කෙළෙස් රජස් නැති, සක්මන් කරන්නාවූ බුදුරජුන් දුටිමි.
# ´´කෙසුරු මල් (බක්මී) තුණක් හිස මත තබා වෙස්සභූ බුදුරජුන් වෙත එළඹ පිදීමි.
# මම තිස් එක්වන කපෙහි වූ එකල්හි යම් යම් කර්මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කෙශරී පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අඬ්කොල පුප්ඵිය ථෙරාපදානය.
# ´´එකල්හි චිත්‍රකූට පර්‍වතයෙහි පදුම නම් බුදුරජතෙම වැඩ වෙසෙන සේක. තෙමේම අවබෝධ කළාවූ උන්වහන්සේ දැක (එහි) එළඹියෙමි.
# ´´එකල්හි පිපුණ රුකඟුන ගසක් දැක, මම එම මල් නෙලා, පස්මරුන් දිනූ පදුම බුදුරජුන් වෙත එළඹ (එය) පිදීමි.
# ´´මින් තිස් එක්වන කපෙහි වූ එකල්හි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අඞ්කොල පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කදම්බ පුප්ඵිය ථෙරාපදානය.
# ´´ඇතුල් වීථියෙහි වඩින්නාවූ, ස්වර්ණවර්ණවූ, දෙතිස් ලකුණෙන් විසිතුරුවූ, කසුන් ඇගෑවක් වැනි
# ´´ලෝක නායකවූ බුදුරජුන් උතුම් පහයෙහි හිඳ දුටිමි. (මම) ඒ විපස්සී බුදුරජුන්හට කොළොම් මල් ගෙන පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කදම්බ පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# උද්දාල පුප්ඵිය ථෙරාපදානය.
# ´´අනොම නම් සම්බුදුරජ තෙමේ ගං ඉවුරෙහි විසූ සේක. මම එකල්හි ඇසල මල් ගෙන පස් මරුන් දිනූ උන්වහන්සේට පිදීමි.
# ´´මින් තිස් එක්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උද්දාල පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# එක චම්පක පුප්ඵිය ථෙරාපදානය.
# ´´උපසන්ත නම් සම්බුදුරජතෙමේ කඳු අතරෙහි වෙසෙන සේක. (මම) එක් සපු මලක් ගෙන නරොත්තමවූ එම බුදුරජුන් වෙත එළඹියෙමි.
# ´´පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, පස්මරුන් දිනූ පසේබුදුරජුන්හට (එම මල්) දොහොතින් ගෙන පිදීමි.
# ´´මින් තිස් එක්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් එක චම්පක පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිමිර පුප්ඵිය ථෙරාපදානය.
# ´´චන්ද්‍රභාගා නදී තීරයෙහි ගඟ පහලට මම යෙමි. (එකල්හි) පිපුනු සල් රුකක් වැනිව බබළන්නාවූ බුදුරජුන් මම දුටිමි.
# ´´පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, මම තිමිර මල් ගෙන, එම මල් පසේ බුදුරජුන්ගේ මස්තකයෙහි (උඩට දමා) පිදීමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිමිර පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සලලපුප්ඵිය ථෙරාපදානය.
# ´´එකල්හි මම චන්‍ද්‍රභාග නදී තීරයෙහි කිඳුරෙක් වීමි. එහිදී සක්මණෙහි යෙදෙන, දෙවියනට දෙවිවූ බුදුරජුන් දිටිමි.
# ´´මම සළල මල් නෙලා, එම බුදුරජුන්හට දුනිමි. බුදුරජතෙමේ දිව්‍යමය සුවඳ ඇති එම සළල මල් ආඝ්‍රාණය කළ සේක.
# ´´මින් එක් තිස්වෙනි කපෙහි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියත් නොදනිමි. මේ බුද්ධ පූජාවෙහි විපාකයයි.
# (ඉක්බිති) පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, බුදුරජුන් වැඳ, නැවතද ඇඳිලි බැඳ, පර්‍වතයට නැංගෙමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියත් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සලල පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
