 ආලම්බනදායක වග්ග.  
# ආලම්බනදායක ථෙරාපදානය.
# ´´අටලෝ දහම්හි කම්පා නොවන බැවින් තාදී ගුණ ඇති, ලෝකජ්‍යෙෂ්ඨවූ, විපදෝත්තමවූ අත්‍ථදස්සී බුදුරජුන්හට මා විසින් එල්බෙන පුවරුවක් දෙන ලදී.
# ´´මහත්වූ සයුර කෙළෙවරකොට ඇති පෘථිවිය රජ කරමි. (පෘථිවියෙහිවූ) ධනයටද, ප්‍රාණීන්ටද ආධිපත්‍යය පවත්වමි.
# ´´මාගේ කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මින් දෙසැටවන කපෙහි මහත් බල ඇති ´´එකාපස්සීත´´ නම් ඒ තුන් සක්විති රජෙක් වූහ.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආලම්බනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# අජිනදායක ථෙරාපදානය.
# ´´මින් තිස්එක්වන කපෙහි මම ශාස්තෲන් ගරු කළ පිරිසෙහි කෙනෙක් වීමි. කෙළෙස් රජස් විරහිත, ලෝකයාගේ පූජාවන් පිළිගන්නාවූ සිඛී බුදුරජුන් දිටිමි.
# ´´ලෝක බන්ධුවූ සිඛී බුදුරජුන්හට මා විසින් සම් කඩක් දෙන ලදී. විපදෝත්තමයන් වහන්ස, ලෝක ජ්‍යෙෂ්ඨයන් වහන්ස, නරශ්‍රේෂ්ඨයන් වහන්ස, එම කර්‍මය කරණ කොට ගෙන,
# ´´මම සම්පත් අනුභවකොට කෙලෙස්ද දැවීමි. සම්බුදු සසුනෙහි (මේ) අන්තිම ශරීරයද දරමි.
# ´´මම මින් එක්තිස්වන කපෙහි යම් අඳුන් දිවි සමක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. අඳුන් දිවි සම් දනෙහි මේ විපාකයයි.
# ´´මින් පස්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති සුදායක නම් වූ සක්විති රජෙක් වීමි
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අජිනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# විරතනීය ථෙරාපදානය.
# ´´මම පෙර මහ වනයෙහි මුව වැද්දෙක් වීමි. ලෝකයාගේ පූජාවන් පිළිගත්තාවූ, කෙලෙස්මල නැති බුදුරජුන් දිටිමි.
# ´´විපස්සී බුදුරජුන්හට මා විසින් මස් කැබැල්ලක් දෙන ලදී. මම (ඒ හේතුවෙන්) දෙවියන් සහිත ලෝකයෙහි ආධිපත්‍යය කරමි.
# ´´මෙම මස් දන කරණකොට ගෙන මට රුවන් උපදී. මේ රුවන් දෙක ලෙව්හි නිවන් ලැබීමට උපකාරීවේ.
# ´´මස් දනෙහි ශක්තිය කරණකොට ගෙන මම ඒ සියල්ල අනුභව කරමි. මාගේ ශරීරයද මෘදුය. සියුම් දෙය අවබෝධ කර ගත හැකි ප්‍රඥාවද වේ.
# ´´මින් එක් අනූවන කපෙහි මම යම් මස් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මස් දනෙහි මේ විපාකයයි.
# ´´මින් සිව්වන කපෙහි ජනාධිපතිවූ, මහත් බල ඇති මහා රෝහිත නම් වූ එක් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් විරතනීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ආරක්ඛදායක ථෙරාපදානය.
# ´´මා විසින් සිදුහත් බුදුරජුන්හට වේදිකාවක් කරණ ලදී. මා විසින් මහර්ෂීවූ බුදුරජුන්හට ආරක්‍ෂාවක්ද දෙන ලදී.
# ´´එම කර්‍ම විශෙෂය කරණකොට ගෙන භයජනක අරමුණක් නොදනිමි. කොහි උපන්නේ නමුදු මට ත්‍රාසයක් විද්‍යාමාන නැත.
# ´´මම සිව් අනූවන කපෙහි යම් වේදිකාවක් කරවූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. වේදිකා දනෙහි මේ විපාකයයි.
# ´´මින් සවනු කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති අපස්සෙන නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආරක්ඛදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# ආරක්ඛදායක ථෙරාපදානය.
# ´´මම විපස්සී බුදුරජුන්හට ගිනිහල් ගෙයක් දුනිමි. රෝගීන්හට නිවාස ස්ථානද, උනුදියද, කෙළගසන (පඩික්කම්ද) දුනිමි.
# ´´එම කර්‍මය හේතුකොට ගෙන මට මේ ආත්ම භාවය මවන ලදී. මම ලෙඩ කුමක්දැයි නොදනිමි. පුණ්‍ය කර්‍මයාගේ මේ විපාකයයි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් ශාලාවක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ගිනිහල් ගෙය දීමෙහි මේ විපාකයයි.
# ´´මින් සත්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති අපරාජිත නම් එක් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ආරක්ඛදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අඞ්කොල පුප්ඵිය ථෙරාපදානය.
 
# ´´නාරද යනු මගේ නමයි. මට කාශ්‍යපයයිද කියති. ශ්‍රමණයනට අග්‍රවූ, දෙවියන් විසින් සත්කාර කරණ ලද අනුව්‍යඤ්ජනයන් දරන්නාවූ, ලෝකයාගේ පූජාවන් පිළිගන්නාවූ විපස්සී බුදුරජුන් දිටිමි. මම රුකඟුණ මල් ගෙන (එම) බුදුරජුන්හට පිදීමි.
# 
# ´´මින් සූසැත්තෑවන කපෙහි ස්වර්‍ණ සිවිකා ආදී යානයන්ද, මහාමාත්‍ය සේවකාදී බලයන්ද, හස්ති අශ්ව රථ සංඛ්‍යාත වාහනයන්ද යන මොවුන්ගෙන් යුක්තවූයේ පැළඳි මාලාභරණ ඇත්තේ ´´රොමිස´´ නම් රජෙක් වීමි.
#
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අඞ්කොල පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වටං සකිය ථෙරාපදානය.
# ´´උද්‍යාන භූමියට නික්ම යන්නේ ලෝකනායකවූ සිඛී බුදුරජුන් දිටිමි. මැනවින් කොට නිමවන ලද ස්වර්‍ණ වර්‍ණවූ හිසේ පළඳන මාලාවක් ගෙන,
# ´´ඇතුපිට සිටි මම වහා ඉන් බැස ලෝක බන්ධූවූ සිඛී බුදුරජුන්හට එම මාලාව පිදීමි.
# ´´මින් එක් තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් සත්විසිවන කපෙහි ජනාධිපතිවූ, මහත් බල ඇති මහාපතාප නම් වූ සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වටං සකිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මිඤ්ජ වටං සකිය ථෙරාපදානය.
# ´´කථාකරන්නවුන් අතුරෙන් උතුම්වූ සිඛී බුදුරජුන් පිරිනිවි කල්හි හිසෙහි පළඳින මාලාවන්ගෙන් ගැවසීගත් බෝධි පූජාවක් කෙළෙමි.
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි යම් පුෂ්ප දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් සත්තිස්වන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ´´මෙගබ්භ´´ නම් වූ සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මිඤ්ජ වටං සකිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සුකතා වෙලිය ථෙරාපදානය.
# ´´එකල්හි මම අසිත නම් මල්කරුවෙක් වීමි. හිසෙහි පලඳින මාලාවක් රජුට දීම සඳහා ගෙනයමි.
# ´´රජු වෙත පැමිණීමට පෙර බුදුරජාණන් වහන්සේ දුටිමි. සතුටුවූයේ, සතුටු සිතින් (එය) බුදුරජුන්හට පිදීමි.
# ´´මින් එක්තිස්වන කපෙහි යම් පුෂප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් පස්විසිවන කපෙහි, මහත් බල ඇති වෙහාර නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුකතා වෙලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# එක වන්‍දිය ථෙරාපදානය.
# ´´ශ්‍රේෂ්ඨවූ, උතුම්වූ, පස්මරුන් ජයගත් වෙස්සභූ බුදුන්හට මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ වැන්දෙමි.
