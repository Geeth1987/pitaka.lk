 සද්ද සඤ්ඤක වග්ග.   
# සද්ද සඤ්ඤක ථෙරාපදානය.

# ´´මම පෙර මහ වනයෙහි මුව වැද්දෙක් වීමි. එහිදී දිව්‍ය සමූහයා විසින් පිරිවරනලද, චතුස්සත්‍ය දේශනා කරන්නාවූ, මහාජනයා සසර සයුරින් ගොඩනගන්නාවූ සම්බුදුරදුන් දිටිමි. (දැක කොවුල් හඬට හුරු මිහිරි වචන ඇසීමි.)
#
# ´´බ්‍රහ්ම ස්වරය ඇති, ලොක බන්ධූවූ සිඛී බුදුරජුන්ගේ හඬෙහි සිත පහදවා ආශ්‍රවක්‍ෂයට (රහත් බවට) පැමිණියේ වෙමි.
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි යම් කර්මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (බුද්ධ ස්වරයෙහි) පැහැදීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සද්දසඤ්ඤක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# යවකලාපිය ථෙරාපදානය.
# ´´එකල්හි (මම) අරුණවතී නගරයෙහි යව කෙත් රකින්නෙක් වීමි. මගදී බුදුරජුන් දැක (උන්වහන්සේට වැඩහිඳීම සඳහා) යව (කරල්) මිටියක් (බිම) ඇතිරීමි.
# ´´අනුකම්පාව ඇති, කාරුණීකවූ සිඛී බුදුරජ තෙම මාගේ අදහස දැන එම (යව) ඇතිරියෙහි වැඩහුන් සේක.
# ´´එසේ වැඩහුන්නාවූ, නිර්මලවූ, මහා ධ්‍යාන ඇති, අසහාය නායකවූ (බුදුරජුන්) දැක සතුට උපදවා මම එහි කළුරිය කෙළෙමි.
# ´´මින් සත් තිස්වන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. යව ඇතිරි දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් යවකලාපිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කිංසුක පූජක ථෙරාපදානය.
# ´´පිපුණු කෑල ගසක් දැක ඇඳිලි බැඳගෙන සිද්‍ධාත්‍ර්‍ඵ බුදුරජුන් සිහිපත්කොට (එම මල්) අහසෙහි (ගසේ තිබියදීම) (බුදුරජුන්හට) පිදීමි.
# ´´මින් සිවු අනූවන කපෙහි වූ එකල්හි යම් පුෂ්ප පූජාවක් කෙලෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කිංසක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සකොටක කොරණ්ඩ දායක ථෙරාපදානය.
# ´´සිඛී බුදුරජුන්ගේ තබන ලද පියවර දැක මම අඳුන් දිවි සම එකස්කොට (එම) ශ්‍රෙෂ්ඨ ශ්‍රීපාදය වැන්දෙමි.
# ´´පිපුණු කටුකරඬු රුකක් දැක (එය) කිණිති සහිතව (කඩා) පියවරෙහි (ශ්‍රීපාදයෙහි) චක්‍ර ලක්‍ෂණය පිදීමි.
# ´´මම තිස් එක්වන කපෙහි වූ එකල්හි යම් කර්මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ශ්‍රීපාදය පිදීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සකොටක කොරණ්ඩ දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# දණ්ඩදායක ථෙරාපදානය.
# ´´එකල්හි මම වනයට වැද හුණලීයක් කපා එල්බෙන දණ්ඩක් (සැරයටියක්) සංඝයාට දුනිමි.
# ´´එම චිත්ත ප්‍රසාදය කරණකොටගෙන මනා ව්‍රත ඇති සංඝයා මැද සැරයටියද දී උතුරු දිගට මුහුණලා ගියෙමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් සැරයටි දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. සැරයටි දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් දණ්ඩදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අම්බයාගු ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කරගත්, පස්මරුන් දිනූ සතරංසී නම් සර්‍වඥ තෙමේ සමාධියෙන් නැගිට මා වෙත පිඬු පිණිස එළඹි සේක.
# ´´එම පසේ බුදුන් දැක මම පිරිසිදු සිතින් යුත් උන්වහන්සේට පැහැදි සිතින්ම අඹ කැඳක් දුනිමි.
# ´´මින් සිව් අනූවන කපෙහිවූ එකල්හි යම් කර්මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. අඹ කැඳ පිදීමෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අම්බයාගු තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සුපුටක පූජක ථෙරාපදානය.
# ´´දිවා විහරණයෙන් නික්මුණු ඒ විපස්සී බුදුරජාණන් වහන්සේ පිඬු පිණිස හැසිරෙන්නේ මාගේ සමීපයට එළඹි සේක.
# ´´ඉක්බිති (එයින්) සතුටුවූ, සොම්නස්වූ මම, තාදී ගුණ ඇති බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේට ලුණුගොටුවක් දී කල්පයක් ස්වර්‍ගයෙහි සතුටුවීමි.
# ´´මින් එක්අනූවන කපෙහි මම (යම්) ලුණු ගොටු දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (ලුණු) ගොටු දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුපුටක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මඤ්චදායක ථෙරාපදානය.
# ´´ලොවට දෙටුවූ, තාදී ගුණ ඇති විපස්සී බුදුරජුන්හට පහන් සිතින් සියතින්ම මා විසින් ඇඳක් දෙන ලදී.
# ´´එයින් හස්ති යානයන්ද, අශ්වයන්ද, දිව්‍යමය යානයන්ද ලැබීමි. එම මඤ්චදානයෙන්ම (ඇඳ දන් දීමෙන්) ආශ්‍රවක්‍ෂය (රහත් බවටද) පැමිණියේ වෙමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් මඤ්චක දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මඤ්චක දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මඤ්චදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සරණගමණීය ථෙරාපදානය.
# ´´එකල්හි ආජීවකයෙක්වූ මමද, භික්‍ෂුවක්ද, නැවකට නැගුණෙමු. නැව බිඳෙන කල්හි ආජීවකයෙක්වූ මට භික්‍ෂුව සරණ දුන්නේය.
# ´´මින් එක් තිස්වන කපෙහි වූ එකල්හි ඒ භික්‍ෂු තෙමේ මට යම් සරණයක් දුන්නේද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මේ සරණාගමණයෙහි විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සරණගමණීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පිණ්ඩපාතික ථෙරාපදානය.
# ´´තිස්ස නම් සම්බුදුවරයෙක් විය. එකල්හි උන්වහන්සේ වනයෙහි විසූ සේක. මම තුසිතයෙන් මෙහි පැමිණ උන්වහන්සේට පිණ්ඩපාත දානයක් දුනිමි.
# ´´මහත්වූ ගුණ ඝෝෂාවක් ඇති, තිස්ස නම් බුදුරජුන්හට සිය සිත පහදවා වැඳ, මම (ආපසු) තුෂිත දෙව්ලොවට ගියෙමි.
# ´´මින් දෙයානූවන කපෙහි වූ එකල්හි යම් පිණ්ඩපාත දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිණ්ඩපාත දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පිණ්ඩපාතික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
