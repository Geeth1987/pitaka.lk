
 සුධාපිණ්ඩිය වග්ග. 
# සුධාපිණ්ඩිය ථෙරාපදානය. 

# ´´පිදීමට සුදුසුවූ, කෙළෙස් (පමා දොස්) ඉක්මවූ, තරණය කළ (තුනී කළ) ශෝක පරිදේව (දුක්) ඇති බුදුරජුන් කෙරෙහි හෝ සව්වන් කෙරහි හෝ පුදන්නහුට (හෙවත්) (කෙළස් පිරිනිවනින්) පිරිනිවි බිය නැති එබඳු උතුමන් කෙරෙහි පුදන්නාවූ ඔවුනට (ලැබෙන) පින මෙපමණයයි කිසිවකින් පමණ කිරීමට (මැණීමට) නොහැකිය.
#
# ´´යමෙක් (මේ ලෙව්හි) සිව්මහ දිවයිනට ඉසුරු රජය කරවන්නේද, (එය) මෙබඳු එක පියුමක් සොළොස්වෙනි කලාවකටද නොඅගී.
# ´´සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේගේ සෑයෙහි පැළුම් අතර මා විසින් පැහැදිලි සිතින් සුණු පිඩක් දෙන ලදී. (පැළුම් අතරෙහි සුණු ගසා කවන ලදී.)
# ´´මින් සිව් අනූවන කපෙහිවූ එකල්හි යම් (පින්) කමක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිළිසකරයෙහි (ප්‍රතිසංස්කරණයෙහි) මේ විපාකයයි.
# ´´මින් තිස්වන කපෙහි සත්රුවනින් යුත් පටිසංඛාර නම්වූ සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුධාපිණ්ඩිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සුචින්තික ථෙරාපදානය. 
# ´´මම සියල්ල දත් තිස්ස බුදුරජුන්හට තුටුවූයේ, තුටු සිතින්ම (සිලිටි දැව මුවා අනගි) පිරිසිදු පුටුවක් පිදීමි.
# ´´මින් අට තිස්වන කපෙහි මහාරුචි නම් රජෙක් වීමි. (ඔහුට) බොහෝවූ සයනාසනද, මහත්වූ භෝගද ඇත්තේ විය.
# ´´පිරිසිදු සිතින් බුදුරජාණන් වහන්සේට පුටුවක් දී තමා විසින් පෙර මැනවින් කළ (ඒ) ස්වකීය (පුණ්‍ය) කර්‍මය අනුභව කරමි.
# ´´මින් දෙනූවන කපෙහිවූ එකල්හි යම් පුටුවක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පීඨ දානයාගේ මේ විපාකයයි.
# ´´මින් අටතිස්වන කපෙහි රුචිය, උපරුචිය, මහා රුචිය යයි සක්විති රජවරු තිදෙනෙක් වුහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුචින්තික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අඩ්ඪචෙලක ථෙරාපදානය. 
# ´´මම සියල්ල දත් තිස්ස බුදුරජාණන් වහන්සේට (සිව්රු පිණිස) රෙදි කැබැල්ලක් දුනිමි. දුගඳින් යුත් මම අතිශය දුකට පත්වූයේ වෙමි.
# ´´මම (සිව්රු පිණිස මෙම) රෙදි කැබැල්ල පුදා කල්පයක් මුළුල්ලෙහි ස්වර්‍ගයෙහි සතුටුවීමි. සෙසු කල්පයන්හි මා විසින් කුශල් කරණ ලදී.
# ´´මින් දෙයානූවන කපෙහි වූ එකල්හි යම් වස්ත්‍රයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. වස්ත්‍ර දානයාගේ මේ විපාකයයි.
# ´´එකුන් පණස්වන කපෙහි ජනාධිපතිවූ සමන්ත ඡදන නම් දොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අඩ්ඪාචලක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සූචිදායක ථෙරාපදානය. 
# ´´මම පෙර බන්‍ධුමා නම් උතුම් පුරයෙහි කඹුරෙක් වීමි. මා විසින් විපස්සී බුදුරජාණන් වහන්සේට හිදි (ඉදිකටු) දනක් දෙන ලදී.
# ´´එම හිදි දන කරණකොට එවැනිමවූ විදුරග වැනි නුවණ වේ. පහවූ රාගය ඇත්තේ වෙමි. (කෙළෙසුන්ගෙන්) මිදුනේ වෙමි. ආශ්‍රවක්‍ෂයට පැමිණියේ වෙමි.
# ´´අතීත භවයෙහිද, වර්‍තමාන භවයෙහිද, අනාගත භවයෙහිද සියල්ල නුවණින් සොයා ගතිමි. හිඳි දනෙහි මේ විපාකයයි.
# ´´මින් එකානූවන කපෙහි සත්රුවනින් යුත් මහත් බල ඇති වජිර නම්වූ සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සූචිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ගන්‍ඩමාලිය ථෙරාපදානය. 
# ´´මම සිද්‍ධාර්ත්‍ථ භාග්‍යවතුන් වහන්සේට (සඳුන්, අගිල්, කපුරු, කස්තුරි ආදිය වඩා) සුගන්‍ධස්ර්‍ථපයක් කෙළෙමි. බුදුරජාණන් වහන්සේට සුදුසු සේ කරණලද එය සමන් මල්වලින් වසන ලදී.

# ´´රන් ඇගෑවක් මෙන් බබළන, ලෝකයෙහි අසහාය නායකවූ, සර්‍වඥවූ, උපුල් මලක් මෙන් බබළන, ඇවිලෙන ගින්නක් වැනි, ශ්‍රේෂ්ඨ ව්‍යාඝ්‍රයෙකු මෙන් උතුම්වූ, විශිෂ්ඨ ජන්ම ඇති, කේශර සිංහයෙකු වැනි, භික්‍ෂු සංඝයා විසින් පිරිවරණ ලද, ශ්‍රමණයනට අග්‍රවූ බුදුරජාණන් වහන්සේද, (එම) බුදුරජාණන් වහන්සේගේ ශ්‍රීපාද යුග්මයද වැඳ උතුරුදිගට මුහුණ ලා ගියෙමි.

#
# ´´යම්හෙයකින් මින් සිව් අනූවන කපෙහි මේ ගඳමල් පූජාවක් කරණ ලදද, බුදුරජුන් විෂයෙහි කරණ (ලද) පුණය කර්‍මයාගේ විපාක කරණකොටගෙන මම විශේෂයෙන් දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´එකුන් සතලිස්වන කපෙහි දෙවගන්‍ධ නම්වූ ඒ සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ගන්‍ධමාලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිපුප්ඵිය ථෙරාපදානය. 
# ´´මම පෙර මහ වෙනෙහි වැද්දෙක්වීමි. (විපස්සී බුදුරජුන්ගේ) නිල්වන්වූ පළෝල් බෝධිය දැක (එයට) මල් තුනක් විසිරවීමි. (පිදීමි.)

# ´´මම එකල්හි (බෝ) කොළ සතක් (අවුලා) ගෙන පිටත දැමීමි. ඇතුළත පිරිසිදුවූ පිටත (ද) පිරිසිදුවූ (කෙළෙසුන්ගෙන්) මැනවින් මිදුණු ආශ්‍රව රහිතවූ ලෝක නායකවූ විපස්සී බුදුරජුන් ඉදිරියෙහිදී මෙන් පළොල් බෝධිය වැඳ මම එහි කළුරිය කෙළෙමි.
#
# ´´මින් එක් අනූවන කපෙහි යම් බෝධි පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් තෙතිස්වන කපෙහි මහත් බල ඇති සමන්ත පාසාදිකා නම් වූ සක්විති රජවරු තෙළෙස් දෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මධුපිණ්ඩික ථෙරාපදානය. 

# ´´අල්ප ශබ්ද ඇති, නිරාකූලවූ මහ වනයෙහිදී, ඎෂීනට ශ්‍රේෂ්ඨවූ, ලෝකයාගේ පූජාවන් පිළිගන්නාවූ නොහොත් පූජාවනට ගෙයක් වැනිවූ, නිවීගිය ආත්මභාව ඇති, අති ශ්‍රේෂඨවූ උත්තම ආජානීයත්‍වය ඇති, ඔසධී තරුව මෙන් බබළන්නාවූ, දිව්‍ය සමූහයා විසින් නමස්කාර කරණ ලද සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේ දැක, එකෙණෙහි මට සොම්නස් සහගතවූ (සිතට) අමුතුවූ නුවණක් උපන්නේය.

#
# ´´සමාධියෙන් නැගී සිටි (ඒ) බුදුරජාණන් වහන්සේට මී දී, එම බුදුරජාණන් වහන්සේගේ ශ්‍රී පාද යුග්මයද වැඳ නැගෙණහිර දෙසට මුහුණ ලා ගියෙමි.
# ´´සූතිස්වන කපෙහි සුදස්සන නම් රජෙක් වීමි. එකල්හි (අනුභව කරණ) භෝජනයෙහිද නෙළුඹු පැනි වැගිරේ. මිහි වැසිද වැස්සේය. පූර්‍ව කර්‍මයාගේ මේ විපාකයයි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් මී වදයක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මධු දානයාගේ මේ විපාකයයි.
# ´´මින් සූතිස්වන කපෙහි සත්රුවනින් යුත් මහත් බල ඇති ඒ සුදස්සන නම් සිව් සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මධුපිණ්ඩික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# සේනාසනදායක ථෙරාපදානය. 
# ´´(මම) සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට පන් ඇතිරියක් දුනිමි. හාත්පස අසෙනිය කුසුම්ද විසිරවිය.
# ´´මෙසේ පහයෙහි (මොලොක්) ගුණ ඇති මනරම් අගනා (ඇතිරි) පරිහරණය කරමි. මාගේ සයනයෙහි මාහැඟි මල්ද වැටේ.
# ´´මම විසිතුරුවූ මල් අතුළ සයනයෙහි එකෙණෙහි හොවීමි. එකෙණෙහි සයනයෙහි මල් වැසිද වසින්නේය.
# ´´මින් සිව් අනූවන කපෙහි පත් ඇතිරියක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඇතිරි දනෙහි මේ විපාකයයි.
# ´´මින් පස්වෙනි කපෙහි ජනාධිපතිවූ තිණසන්ථරක නම් මේ සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සේනාසනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වෙය්‍යාවච්චක ථෙරාපදානය. 
# ´´විපස්සී බුදුරජාණන් වහන්සේගේ (භක්තිමත්) මහා ජන සමූහයා විය. මම (එකල්හි) සියළු කටයුතුවල යෙදුනු වතාවත් කරන්නෙක් වීමි.
# ´´මහර්ෂීවූ බුදුරජාණන් වහන්සේට පිදියයුතු දෙයක්ද මට නැත. පිරිසිදු සිතින් බුදුරජාණන් වහන්සේගේ සිරිපා වැන්දෙමි.
# ´´මම මින් සිව් අනූවන කපෙහි (බුදුරජුන්ට) වතාවත් කෙළෙම්ද, (ඒ හේතුවෙන්) වතාවත් කිරීමෙහි මේ ප්‍රතිඵලයයි.
# ´´මින් අටවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති සුචින්තිත නම් මේ සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වෙය්‍යාවච්චක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# බුද්‍ධුපට්ඨාන ථෙරාපදානය. 
# ´´(මම) විපස්සී බුදුරජුන්ගේ (කාලයෙහි) සක් පිඹින්නෙක් වීමි. මහර්ෂීවූ සුගතයන් වහන්සේට නිතර (සක් පිඹීමෙන්) උවටැනෙහි යෙදුනේ වීමි.
# ´´තාදී ගුණ ඇති ලෝකනාථයන් වහන්සේට උවටැන් කිරීමෙහි විපාකය බලව්. හැමකල්හි සැට දහසක් තූර්‍ය්‍යයෝ මා පිරිවරත්.
# ´´මින් එක් අනූවන කපෙහි මහර්ෂීවූ බුදුරජුන්ට (සක් පිඹීමෙන් යම්) උවටැනක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. උවටැනක් කිරීමෙහි මේ විපාකයයි.
# ´´මින් සූවිසිවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති මහා නිග්ඝෝෂ නම් වූ සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් බුද්‍ධුපට්ඨාන තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
