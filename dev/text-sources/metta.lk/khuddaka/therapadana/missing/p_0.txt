# ´´මින් තිස්එක්වන කපෙහි වූ එකල්හි යම් වර්‍ණයක් (පින් පිණිස) පෙවූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි.
# ´´මින් තෙවිසිවන කපෙහි සත්රුවනින් යුත් මහත් බල ඇති චන්දුපම නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වණ්ණකාර තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පියාල පුප්ඵිය ථෙරාපදානය. 
# ´´මම පෙර මහ වෙනෙහි මුව වැද්දෙක්වීමි. මම පිපුණු පියාල ගසක් දැක, බුදුරජුන් වැඩිය මගෙහි (එම මල් ගෙන) දැමීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පියාල පිප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අම්බයාග දායක ථෙරාපදානය. 
# ´´ස්වකීය (තර්‍කව්‍යාකරණාදී) ශිල්පයෙහි දක්‍ෂවූයේ මම වනයට ගියෙමි. මම මග වඩින්නාවූ බුදුරජුන් දැක අඹ දනක් දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. අඹ දනෙහි මේ විපාකයයි. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අම්බයාග දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ජගතිකාරක ථෙරාපදානය. 
# ´´මා විසින් අත්‍ථදස්සී බුදුරජුන් පිරිනිවි කල්හි එම බුදුරජුන්ගේ උතුම් සෑයෙහි පිළි දැකුම් කරණ ලදී.
# ´´එක් දහස් අටසියවන කපෙහි වූ එකල්හි යම් කර්‍මයක් කරණ ලදද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චෛත්‍ය ප්‍රතිජග්ගනයෙහි මේ විපාකයයි. 
# ´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ජගතිකාරක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වාසිදායක ථෙරාපදානය. 
# ´´මම පෙර උතුම් තිවරා පුරයෙහි කඹුරෙක් වීමි. මා විසින් සියල්ල දත් පස් මරුන් දිනූ බුදුරජුන්හට එක් දැලිපිහියක් දුනිමි.
# මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් දැලි පිහියා දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. දැලි පිහියා දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වාසිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
