 ජගතිදායක වග්ග.  
# ජගතිදායක ථෙරාපදානය. 
# ´´මම දම්මදස්සී බුදුරජුන්ගේ උතුම් බොධි වෘක්‍ෂය (පිහිටි තන්හි) පෘථිවිය පහන් සිත් ඇත්තේ, සොම්නස්වූයේ කරවීමි.
# ´´මම පර්‍වත විවරයකින් හෝ, පර්‍වතයකින් හෝ වැටුනේ, (එකල්හි පවා) පිහිටක් ලබමි. පෘථිවිය සකස්කොට දීමෙහි මේ විපාකයයි.
# ´´හොරු මාගේ (වස්තුව) පැහැර නොගනිත්. රජතෙම (මට) අවමන් නොකෙරේ. සියළු පස මිතුරන් ඉක්මවීමි. පොළොව සකස්කොට දීමෙහි මේ විපාකයයි.
# ´´දිව්‍යවූ හෝ, මනුෂ්‍යවූ හෝ යම් යම් යෝනියක උපදිම්ද, ඒ සියළු තන්හි පුදන ලදුයේ වෙමි. පොළොව සකස්කොට දීමෙහි මේ විපාකයයි.
# ´´මම අටළොස් සියවෙනි කපෙහි පොළොව (සකස්) කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පොළොව සකස්කොට දීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් ජගතිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# මොරහත්‍ථිය ථෙරාපදානය. 
# ´´මම මොණර පියාපතක් ගෙන, පහන් සිත් ඇත්තේ, සොම්නස්වූයේ ලෝකනායකවූ බුදුරජුන් වෙත එළඹියෙමි. (එළඹ) මොණර පියාපත පිදීමි.
# ´´මෙම මොණර පියාපත් දන කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, ත්‍රිවිධාග්නීහු නිවී ගියහ. මහත්වූ සැපද ලබමි.
# ´´ බුදුවරයෝ ආශ්චර්‍ය්‍යයහ. (නෛර්‍ය්‍යාණීක) ධර්‍මයෝද. ආශ්වර්‍ය්‍යයහ. අපගේ ශාස්තෘ සම්පත්තියද ආශ්වර්‍ය්‍යයකි. මම මොණර පියාපතක් දී මහත් සුව ලබමි.
# ´´මාගේ ත්‍රිවිධාග්නීහු නිවී ගියහ. සියළු භවයෝ නසන ලදහ. සියළු ආශ්‍රවයෝ ක්‍ෂය කරන ලදී. දැන් නැවත ඉපදීමක් නැත.
# ´´මින් එක් තිස්වන කපෙහිවූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මොණර පියාපත් දනෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් මොරහත්‍ථිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# සීහාසන වීජක ථෙරාපදානය. 
# ´´මම තිස්ස බුදුරජුන්ගේ බෝධි වෘක්‍ෂයට වැන්දෙමි. විජිනිපතක් ගෙන එහි සිහසුනට පවන් සැලීමි. 
# ´´මින් දෙයානූවන කපෙහි සිහසුනට පවන් සැලූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පවන් සැලීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් සීහාසන වීජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ති උක්කාධාරක ථෙරාපදානය. 
# ´´පදුමුත්තර බුදුරජුන්ගේ උතුම් බොධි වෘක්‍ෂයෙහි (ආලෝකය පිණිස) පහන් සිත් ඇති, සොම්නස්වූ (මම) පන්දම් තුණක් දැරීමි.
# ´´ඒ මම මින් සියක් දහස්වන කපෙහි (යම්) පන්දම් (තුණක්) දැරූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පන්දම් දනෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් ති උක්කාධාරක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# අක්කමන දායක ථෙරාපදානය. 
# ´´වැස නිමවන ලද බඹසර ඇති, බ්‍රාහ්මණ වංශයෙහිවූ, දිවා විහරණයෙහි හැසිරෙන්නාවූ කකුසඳ බුදුරජුන්හට මම පා තබන පුවරුවක් දුනිමි.
# ´´මෙම කල්පයෙහිමවූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පා තබන පුවරු දනෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් අක්කමන දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

# වන කොරණ්ඩිය ථෙරාපදානය. 
# ´´ලෝකජ්‍යෙෂ්ඨවූ, තාදී ගුණ ඇති සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට වන කරඬු (මලක්) ගෙන පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් වන කොරණ්ඩිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

#
# එක ඡත්තිය ථෙරාපදානය. 
# ´´පෘථිවි තොමෝ හටගත් අඟුරු ඇත්තීය. එසේම මහී තොමෝ උණු අළු සහිතවූවාය. පදුමුත්තර බුදුරජතෙමේද එලිමහනෙහි සක්මන් කළ සේක.
# ´´මම සුදු කුඩයක් ගෙන මගට පිලිපන්නෙමි. එහි සම්බුදුරජුන් දැක ප්‍රීතිය ඇතිවිය.
# ´´භූමිතොමෝ රැස්වූ මිරිඟුදිය ඇත්තීය. (එසේම) මේ පෘථිවිය දිලිසෙන අඟුරු මෙනි. සිරුරෙහි ආයුෂ පිරිහෙළන්නාවූ මහා වාතයෝද හමති.
# ´´ශීතය හා ඌෂ්ණය නසන්නාවූ, වාතය හා අව්ව වළකන්නාවූ මේ කුඩය පිළිගනු මැනව. මම (ඒ හේතුවෙන්) නිවන් ලබන්නෙමි´´යි කුඩය පිළිගැන්විය. 
# ´´එකල්හි (ලෝකයා කෙරෙහි) අනුකම්පා ඇති කාරුණීකවූ, මහර්ධිවූ පදුමුත්තර බුදුරජතෙම මාගේ අදහස් දැන (එය) පිළිගත් සේක. 
# ´´තිස් කපක් මුලුල්ලෙහි දෙවනුව දිව්‍ය රාජ්‍යය කරවීමි. (එසේම) මම පන්සිය වරක් සක්විති රජ වීමි.
# ´´කරවන ලද මහත්වූ ප්‍රදේශ රාජ්‍යය ගණන් වශයෙන් (මෙතෙකැයි) ගිණිය නොහැකිය. පෙර තමා විසින් මැනවින් කළ පුණ්‍ය කර්‍ම විපාකය අනුභව කරමි.
# ´´මාගේ මේ පශ්චිම ආත්මයයි. මේ අන්තිම භවය පවතී. අද පවා මට හැම කල්හි සේසතක් දරයි.
# ´´මින් සියක් දහස්වන කපෙහිවූ එකල්හි යම් ඡත්‍ර දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඡත්‍ර දානයාගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.

#මෙහි මේ අයුරින් ආයුෂ්මත් එක ඡත්තිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ජාති පුප්ඵිය ථෙරාපදානය. 
# ´´මහත් යසස් ඇති පදුමුත්තර බුදුරජුන් පිරිනිවි කල්හි (මම) හිසෙහි පළඳින දෑසමන් මල් මාලාවක් කොට ශ්‍රී දේහයට පිදීමි.
# ´´මම එහි සිත පහදවා නිර්මාණරතී දෙව්ලොවට ගියෙමි. මම දෙව්ලොවට ගියේ (එම) පින්කම සිහිකරමි.
# ´´හැමකල්හි මට අහසින් මල් වැසි වසී. ඉදින් මිනිස් ලොවෙහි සැරිසරම් නම් මහත් යශස් ඇති රජකෙනෙක් වීමි.
# ´´සර්‍වදර්ශිවූ බුදුරජුන්හට කරනලද එම පුෂ්ප පූජාවෙහි බලයෙන් එහි (මිනිස් ලොවෙහි)ද මට මල් වැසි වසී.
# ´´මේ මාගේ පශ්චිම භවයයි. මේ අන්තිම භවය පවතී. අදද හැමකල්හි මට මල් වැසි වසී.
# ´´(මම) මින් සියක් දහස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. (සර්‍වඥ) දෙහ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් ජාති පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# පටි පුප්ඵිය ථෙරාපදානය. 
# ´´ශරීරය (ආදාහනය සඳහා) ගෙන යන කල්හි බෙර වයන කල්හි (මම) පහන් සිත් ඇත්තේ, සොම්නස්වූයේ රත්ලොත් මල් පිදීමි.
# ´´(මම) මින් සියක් දහස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක (ආදාහන) පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් පටි පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ගන්‍ධපූජක ථෙරාපදානය. 
# ´´(ආදාහනය සඳහා) චිතකය කරන කල්හි නානා සුගන්‍ධයන් රැස්කරන කල්හි මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ සුවඳ මිටක් පිදීමි.
# ´´මින් සියක් දහස්වන කපෙහි යම් චිතක පූජාවක් කෙළම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් ගන්‍ධපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.
