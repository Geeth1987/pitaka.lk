කුරඤ්ඡිය වග්ග.  
# කුරඤ්ඡියඵලදායක ථෙරාපදානය. 
# ´´මම පෙර වනයෙහි සැරිසරන්නාවූ වැද්දෙක් වීමි. සියළු දහම්හි කෙළපත්, කෙළෙස් රජස් විරහිත බුදුරජුන් දුටිමි.
# ´´මම කුරඤ්ඡිය ඵලයක් ගෙන වීරවූ, පුණ්‍යක්‍ෂෙත්‍රවූ බුද්ධශ්‍රේෂ්ඨයන් වහන්සේට පහන්වූයේ, සියතින් දුනිමි.
# ´´මින් එක්තිස්වන කපෙහි යම් ඵලදානයක් දුනිම්ද, (ඒ හේතුවෙන්) මම දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
# 
# මෙහි මේ අයුරින් ආයුෂ්මත් කුරඤ්ඡියඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කපිත්‍ථඵලදායක ථෙරාපදානය. 
# ´´ලෝකයාගේ පූජාවන් පිළිගන්නාවූ, ස්වර්‍ණවර්‍ණවූ, වීථියට පිළිපන්නාවූ බුදුරජුන් දැක ගිවුළුඵලයක් (දිවුල් ගෙඩියක්) දුනිමි.
# ´´මින් එක් අනුවන කපෙහි වූ එකල්හි යම් (ගිවුළු) ඵලයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් කපිත්‍ථඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කොසුම්භඵල දායක ථෙරාපදානය. 
# ´´කුඹුක් රුකක් මෙන් බලන, දේවාතිදේවවූ, නරශ්‍රේෂ්ඨවූ, වීථියට පිළිපන් බුදුරජුන්හට කොසුම්භ ඵලයක් දුනිමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් කොසුම්භඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කෙතක පුප්ඵිය ථෙරාපදානය. 
# ´´පුරුෂොත්තමවූ බුදුරජතෙම විනතා නදී තීරයෙහි වැඩහුන්නාහ. (මම) මැනවින් සන්සුන්වූ, එකඟවූ කෙළෙස් රජස් විරහිත බුදුරජුන් දුටිමි.
# ´´එකල්හි මම මී සුවඳ වහනය වන වැටකෙයියා මලකින් පහන් සිත් ඇත්තේ, සොම්නස්වූයේ බුද්ධශ්‍රේෂ්ඨයන් වහන්සේ පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් කෙලක පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# නාගපුප්ඵිය ථෙරාපදානය. 
# ´´ලෝකයාගේ පූජාවන් පිළිගන්නාවූ, ස්වර්‍ණවර්‍ණවූ, වීථියට පිළිපන්, සම්බුදුරජුන්හට නා මලක් පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් නාගපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# අජ්ජුන පුප්ඵිය ථෙරාපදානය. 
# ´´එකල්හි (මම) චන්‍ද්‍රභාගා නදී තීරයෙහි කිඳුරෙක් වීමි. (එහිදී) කෙළෙස් රජස් විරහිත, සියල්ල තෙමේම දත්, පස්මරුන් දිනූ බුදුරජුන් දුටිමි. 
# ´´පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, හටගත් ප්‍රීති ඇත්තේ, කුඹුක් මලක් ගෙන බුදුරජුන් පිදීමි.
# ´´මැනවින් කරන ලද කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද මම කිඳුරු සිරුර හැර තව්තිසා දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි.
# ´´සතිස් වරක් දෙවෙන්‍ද්‍රව දිව්‍ය රාජ්‍යය කරවීමි. දස වරක් චක්‍රවර්ති මහා රාජ්‍යය කරවීමි.
# ´´(කරවන ලද) මහත්වූ ප්‍රදේශ රාජ්‍යය (මෙතෙකැයි) ගණන් වශයෙන් ගිණිය නොහැකිය. සර්‍වඥවූ මහා පින්කෙතෙහි මා විසින් වපුල බිජුවට ආශ්චර්‍ය්‍යයකි. 
# ´´මාගේ කුශලය (තවත්) විද්‍යමානවේ. අනගාර්‍ය්‍ය නම්වු සසුනෙහි පැවිදිවීමි. මම අද ශාක්‍යපුත්‍රවූ බුදුරජුන්ගේ සසුනෙහි පිදිය යුත්තෙක් වෙමි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් අජ්ජුන පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# කුටජ පුප්ඵිය ථෙරාපදානය. 
# ´´හිමවතට නුදුරෙහි චාවල නම් පර්‍වතයක් විය. (එම) අතරෙහි සුදස්සන නම් බුදුරජතෙමේ වැඩ වෙසෙති. 
# ´´මම හිමවතෙහි හටගත් මල් ගෙන අහසට (නැගී) ගියෙමි. එහිදී චතුරෝඝයන් තරණය කළ, ආශ්‍රව රහිතවූ සම්බුදුරජුන් දුටුවෙමි.
# ´´කෙළිඳ මල් ගෙන, හිසෙහි ඇඳිලි බැඳ, සියල්ල තෙමේම දත් මහර්ෂිවූ බුදුරජුන්හට පිදීමි.
# ´´මින් එක් තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් කුටජ පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ඝොස සඤ්ඤක ථෙරාපදානය. 
#
# ´´මම පෙර මහවෙනෙහි මුව වැද්දෙක් වීමි. (එහිදී) කෙළෙස් රජස් විරහිත, දිව්‍ය සමූහයා විසින් පිරිවරණ ලද, චතුස්සත්‍යය ප්‍රකාශ කරන්නාවූ, අමාපදය දෙසන්නාවූ බුදුරජුන් දුටිමි. ලෝකබන්ධුවූ සිඛී බුදුරජුන්ගේ මිහිරි ධර්‍මය ඇසීමි.
# ´´අසමවූ, අප්‍රති පුද්ගලවූ, (බුදුරජුන්) කෙරෙහිද, (උන්වහන්සේගේ බ්‍රහ්ම) ස්වරයෙහිද සිත පැහැදවීමි. එහි සිත පහදවා, තරණය කිරීමෙහි දුෂ්කරවූ භවය තරණය කෙළෙමි.
# ´´මින් එක් තිස්වන කපෙහි වු එකල්හි යම් (ධර්‍ම දේශනා ශබ්දයෙක්හි) සංඥාවක් ලැබූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බ්‍රහ්ම ස්වර සංඥාවෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
#
# මෙහි මේ අයුරින් ආයුෂ්මත් ඝොස සඤ්ඤක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# සබ්බ ඵලදායක ථෙරාපදානය. 
# ´´එකල්හි (මම) මන්ත්‍රයන්හි කෙළ පැමිණි, නම්න් වරුණ නම්වූ බ්‍රාහ්මණයෙක් වීමි. පුත්‍රයන් දසදෙනෙකු හැර දමා වනයට පිවිසියෙමි.
# ´´මම මනොඥ ලෙස කොටස් කරන ලද අසපුවක් සාදා පන්සැලක්ද කොට වනයෙහි වසමි.
# ´´ලෝකයාගේ පූජාවන් පිළිගන්නාවූ ඒ පදුමුත්තර බුදුරජතෙමේ මා සසරින් ගොඩලනු කැමැත්තේ මාගේ අසපුවට පැමිණි සේක.
# ´´වන ලැහැබෙහි ආලෝකය යම්තාක් මහත්වීද, එකල්හි බුදුරජුන්ගේ ආනුභාවයෙන් වනය ඇවිල ගත්තාක් මෙන් විය.
# ´´මම තාදී ගුණ ඇති බුද්‍ධශ්‍රේෂ්ඨයන් වහන්සේගේ ඒ පෙළහර දැක කොළ ගොටුවක් ඵලයෙන් පිරවීමි.
# ´´මම බුදුරජුන් වෙත එළඹ, කද සහිතවම දුනිමි. බුදුරජතෙම මට අනුකම්පාවෙන් මේ වචනය වදාළ සේක.
# ´´ඔබ (පළතුරු) කත් බර රැගෙන මාගේ පසු පසින් එව. සංඝයා එය වැළඳු කල්හි තොපට පින් (සිදු) වන්නේය´´ කියායි. 
# ´´මම එම (පළතුරු) ගොටුව ගෙන භික්‍ෂු සංඝයාට දුනිමි. මම එහි සිත පහදවා තුෂිත දෙව්ලොවෙහි උපනිමි.
# ´´එම දිව්‍ය ලෝකයෙහි දිව්‍යමයවූ නෘත්‍යයන් කරණකොටගෙනද, ගීතයන් කරණකොටගෙනද, වාදිතයන් කරණකොටගෙනද (එකී) කුශල කර්‍ම හා සංයුක්තවූ සුවය හැමකල්හි විඳිමි.
# ´´දිව්‍යවූද, යලි මනුෂ්‍යවූද යම් යෝනියක උපදිම් නම් මාගේ භෝගයෝ අඩු බවක් නැත. ඵල දානයාගේ මේ විපාකයයි.
# ´´මම බුදුරජුන්හට (මේ) ඵල (දානය) දී, සමුදුරු සහිතවූ සිව් මහ දිවයින් යම් පමණ නම් (එපමණ) තන්හි මාගේ ආධිපත්‍යය කරමි. 
# ´´මේ පක්‍ෂි සමූහයා අහසෙහි යම් පමණ දුර (උඩ) පියාඹත්ද, (එපමණ තන්හි අහසෙහිද) ඔවුහු (මාගේ) වසඟයට පැමිණෙත්. ඵල දානයාගේ මේ විපාකයයි.
# ´´වන සණ්ඩයෙහි (කැලෑ රොදෙහි) යම්තාක් යක්‍ෂයෝද, භූතයෝද, රාක්‍ෂසයෝද, කුම්භාණ්ඩයෝද, ගුරුලෝද වෙත් නම් ඔවුහු මාගේ පරිචරියාවට (මෙහෙවරට) පැමිණෙත්.
# ´´ඉදිබුවෝද, බල්ලෝද, මීමැස්සෝද, ඩැහැලේ බොන මැස්සන් හා මදුරුවෝද යන උභය ප්‍රාණීහුදැයි ඔවුහු මා වසඟයට පැමිණෙත්. ඵල දානයාගේ මේ විපාකයයි.
# ´´පක්‍ෂි යෝනියෙහි උපන්, මහත් බල ඇති සුපර්‍ණ නම්වූ ගුරුළු පක්‍ෂීහු වෙත් නම් ඔවුහුද මා සරණකොට යෙත්. ඵල දානයාගේ මේ විපාකයයි. 
# ´´දීර්‍ඝායුෂ්කවූ, ඎද්‍ධිමත්වූ, මහත් යසස් ඇති යම් නාග කෙනෙක් වෙත් නම් ඔවුහුද මාගේ වසඟයට එත්. ඵල දානයාගේ මේ විපාකයයි.
# ´´සිංහයෝද, ව්‍යාඝ්‍රයෝද, දිවියෝද, වලස්සුද, වෘකයෝද, කරබානා වලස්සුද යන මොවුහු මාගේ වසඟයට එත්. ඵල දානයාගේ මේ විපාකයයි. 
# ´´ඔසධී-තාණ (වෘක්‍ෂාදියෙහි) වාසය කරන්නාවූද, අහසෙහි වාසය කරන්නාවූද යම් (දෙවි කෙනෙක්) වෙත් නම් ඒ සියල්ලෝ මා සරණකොට යෙත්. ඵල දානයාගේ මේ විපාකයයි. 
# ´´අතිශයින් දැකුමට (අවබෝධයට) දුෂ්කරවූ ඉතා සියුම්වූ, ගැඹුරුවූ ධර්‍මයෙකැයි පවසන ලද (නිවණ) අවබෝධ කොට වෙසෙමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මම අෂ්ට විමොක්‍ෂයන්ද ලබා, ආශ්‍රව රහිතවූයේ, ප්‍රඥා ඇත්තේ වෙසෙමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මහත් යශස් ඇති, ක්‍ෂය කරන ලද දොස් ඇති, ඵලස්ථවූ යම් බුද්‍ධ පුත්‍ර කෙනෙක් වෙත් නම් ඔවුනතුරෙන් මමද එක්තරා කෙනෙක් වෙමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´කුශල මූලයෙන් මෙහෙයන ලදුයේ, අභිඥාවන්හි කෙළ පැමිණ, සියළු ආශ්‍රවයන් (ප්‍රහාණ පරිඥා වශයෙන්) දැන ආශ්‍රව රහිතව වෙසෙමි.
# ´´ත්‍රිවිද්‍යා ඇත්තාවූද, ඎද්‍ධි ප්‍රාප්තවූද, මහත් යසස් ඇත්තාවූද, දිව කණ ලද්දාවූද යම් බුද්‍ධ පුත්‍ර කෙනෙක් වෙත් නම් මමද ඔවුනතුරෙන් එක්තරා කෙනෙක් වෙමි.
# ´´මින් සියක් දහස්වන කපෙහි වු එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.

# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.

# ´´මම භවනයෙන් නික්ම (අවුත්) උන්වහන්සේට පියුමක් (නෙළුම් මලක්) දරා සිටියෙමි. එක් දිනක් (එසේ) දරා සිට නැවත භවනයට ආමි. 
# ´´මින් තිස් එක්වන කපෙහි යම් බුද්‍ධ පූජාවක් කෙළම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී.
