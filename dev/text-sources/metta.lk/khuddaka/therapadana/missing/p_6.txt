 බන්ධු ජීවක වග්ග.  
# බන්ධු ජීවක ථෙරාපදානය. 

# ´´(උපක්ලේශ විරහිත) චන්‍ද්‍රයා මෙන් නිර්‍මලවූ (කෙළෙස් කිළිටි නොමැති බැවින්) පිරිසිදුවූ, පැහැදුනාවූ, (කෙළෙස් මළ වලින්) නොකැළඹුනාවූ, (නන්‍දිභව නම්වූ) තෘෂ්ණාව හා භවය (පල විශේෂයෙන් හාත්පසින්ම) ක්‍ෂය කරණ ලද්දාවූ, ත්‍රිවිධ ලෝකයෙහි ඇල්ම නැතිකළ, ජන සමූහයා කෙළෙස් ගිනි නිවන්නාවූ සසර සයුරින් එතෙරවූ, එසේ එතෙරවූවන් අතුරෙන් උතුම්වූ, වනයෙහි ධ්‍යාන කරන්නාවූ, ඒ එකලාවූ, සන්සුන්වූ (සිඛී බුදුරජුන් දුටිමි.) 
#
# ´´මම බඳුවද මල් නූලක එල්වා සිඛී බුදුරජාණන් වහන්සේට පිදීමි.
# ´´මින් එක්තිස්වන කපෙහිවූ එකල්හි යම් කර්‍මයක් කරණ ලදද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් සත්වන කපෙහි මනුජෙන්‍ද්‍රවූ, වැඩි දියුණුවූ යසස ඇති, මහා බලවත් සමන්තචක්ඛු නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් බන්ධු ජීවක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
#
# තම්බ පුප්ඵික ථෙරාපදානය. 
# ´´අනුනට බැලමෙහෙ කිරීමෙහි යෙදුනේ මම අපරාධයක් කෙළෙමි. (ඒ හේතුවෙන්) භය ජනක අරමුණුවලින් තැතිගත්තේ වනයට පලා ගියෙමි.
# ´´පොකුරු බැඳි, මනාව නිර්මිතව පිපුණු ගසක් දැක මම (එහි) තඹවන් මල්ගෙන බෝධියට ඉසිමි. 
# ´´ඒ උතුම් පලොල් බෝ නැමද එම බෝ මුලෙහි පළඟ බැඳ (එරමිනියා ගොතාගෙන) (වාඩිවී) හුනිමි.
# ´´(අපරාධකොට) පලාගිය මග සොයා යන්නෝ මාගේ සමීපයට පැමිණියාහුය. මම ඔවුන් දැක උතුම් බෝධියම සිහිපත් කෙළෙමි.
# ´´මම පිරිසිදු සිතින් බෝධිය වැඳ (නැවත පහව යන්නෝ) බොහෝ තල්රුක් පමණින් ගිනිය යුතුවූ (ගැඹුරු) භයානක ගිරි දුර්‍ගයෙක්හි වැටුනෙමි.
# ´´මින් එක් අනූ වන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බෝධි පූජාවෙහි මේ විපාකයයි.
# ´´මින් තෙවන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති සම්ඵුසිත නම් සක්විති රජෙක් වීමි. 
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තම්බ පුප්ඵික තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# විථිසම්මජ්ජක ථෙරාපදානය. 
# ´´උදාවන්නාවූ සූර්‍ය්‍යයා මෙන්ද, (උදාවූ) අතිශයින් දීප්ති රැස් ඇති හිරු මෙන්ද, පුණු රැස් සඳ මෙන්ද, නික්මෙන්නාවූ බුදුරජුන් දිටිමි.
# ´´අට සැට දහසක්වු සියල්ලෝ ක්‍ෂීණාශ්‍රවයෝ වූහ. (ඔවුහු) විපදෝත්තමවූ, නරශ්‍රේෂ්ඨවූ බුදුරජුන් පිරිවැරූහ.
# ´´බුදුරජුන් නික්මෙන කල්හි (මම) එම වීථිය හැමද එහි පිරිසිදු ධජයක් එසවීමි.
# ´´මින් එක් අනූවන කපෙහි යම් ධජයක් පිදූයෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ධජ දානයාගේ මේ විපාකයයි.
# ´´මින් සිව්වන කපෙහි සර්වාකාරයෙන් සම්පූර්‍ණවූ සුධජයයි ප්‍රසිද්‍ධ රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් විථිසම්මජ්ජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කක්කාරු පූජක ථෙරාපදානය. 
# ´´දිව්‍යපුත්‍රයෙකුවූ මම සිඛී බුදුරජුන් පිදීමි. කැකිරි මලක්ද ගෙන (එම) බුදුරජුන්හට පිදීමි.
# ´´මින් එක්තිස්වන කපෙහි යම් මලක් පිදුයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් නවවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති සත්තුත්තම නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කක්කාරු පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මන්දාරව පූජක ථෙරාපදානය. 
# ´´දිව්‍යපුත්‍රයෙකුවූ මම සිඛී බුදුරජුන් පිදීමි. මදාරා මලක්ද ගෙන (එම) බුදුරජුන්හට පිදීමි.
# ´´සත් දිනක් මුළුල්ලෙහිම තථාගතයන් කෙරෙහි (එය) දිව්‍යමය වියනක් විය. සියලු ජනයෝ එක්ව තථාගතයන් වහන්සේට නමස්කාර කළාහුය.
# ´´මින් එක්තිස්වන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´මින් දසවෙනි කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති ජුතින්‍ධර නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මන්දාරව පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කදම්බ පුප්ඵිය ථෙරාපදානය. 
# ´´හිමවතට නුදුරෙහි කුක්කුට නම් පර්වතයක් විය. එම පර්‍වත පාදයෙහි (පසේ) බුදුවරු සත්නමක් වෙසෙත්.
# ´´උඩට නැංගාවූ සඳු වැනිව (බබළන) පිපුනු කොළොම් ගසක් දැක (මල්) දොහොතින් ගෙන සත් පසේබුදුවරයන් කෙරෙහි පිදීමි.
# ´´මින් අනූවන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි. 
# ´´මින් දෙඅනූවන කපෙහි සත්රුවනින් යුත් මහත් බල ඇති පුප්ඵ නම්වු සත් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කදම්බ පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිණසුලක ථෙරාපදානය. 
# ´´හිමවතට නුදුරෙහි භූතගණ නම් පර්වතයක් විය. එහි සියල්ල තෙමේම දැන ගත් ලොව හා නොගැටුණු එක් (සිඛී නම්) සර්‍වඥවරයෙක් වැඩ වෙසෙති. 
# ´´බෝලිද්ද මල් ගෙන (එම) බුදුරජුන්හට පිදීමි. (ඒ හේතුවෙන්) එකුන් සියක් දහස් කපක් මුලුල්ලෙහි දුගතියකට නොගියේ වෙමි. 
# ´´මින් එකොලොස්වන කපෙහි සත්රුවනින් යුත් මහත් බල ඇති ධරණීරුග නම් එක් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිණසුලක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# නාගපුප්ඵිය ථෙරාපදානය. 
# ´´මන්ත්‍රයන්හි පරතෙරෙට ගිය නම්න් සවච්ඡ නම්වූ බමුණෙක් ස්වකීය ශිෂ්‍යයන් සමග පර්වතාන්තරයෙහි වෙසෙයි.
# ´´ලෝකයාගේ පූජාවන් පිළිගන්නාවූ පදුමුත්තර නම් ඒ සර්‍වඥ තෙමේ මා (සසර සයුරින්) ගොඩනගනු කැමැත්තේ මා සමීපයට පැමිණිසේක.
# ´´(පැමිණ) සක්මන් කරයි. දුවයි. එසේම ඇවිලෙයි. (එසේකොට) මාගේ සතුට දැන ප්‍රාචීන දේශාභිමුඛව වැඩිසේක.
# ´´විෂ්මයජනක, ලොමු දැහැගන්නාවූ එම ආශ්චර්‍ය්‍ය දැක නාමල් ගෙන, 
# ´´මින් සියක් දහස්වන කපෙහි මම යම් මල් විසිරවීමක් කෙළෙම්ද, එම චිත්ත ප්‍රසාදය කරණකොටගෙන මම දුගතියක නොඉපදුනෙමි.
# ´´තුන්දහස් සත් සියවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති මහාරථ නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් නාගපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පුන්නාහපුප්ඵිය ථෙරාපදානය. 
# ´´වැද්දෙකු වූ මම මහා වනයට පිවිස වෙසෙමි. පිපුණු දොඹ ගසක් දැක බුද්ධශ්‍රේෂ්ඨයන් වහන්සේ සිහි කෙළෙමි.
# ´´එම සුවඳ මල් නෙලා සුභවූ, සුවඳවත්වූ මල් වැල්ලෙහි සෑයක් කොට බුදුරදුන්හට පිදීමි.
# ´´මින් දෙයානූවන කපෙහි මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´එක් අනූවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති තමොනුද නම් එක් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පුන්නාහපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# කුමුදුදායක ථෙරාපදානය. 
# ´´හිමවතට නුදුරෙහි හෙල පියුම් හා උපුල්වලින් ගැවසීගත් රත් පියුමින් සම්පූර්‍ණවූ මහත්වූ ස්වයංජාත විලක් විය.
# ´´එකල්හි එහි සිල්වත්වූ, බුද්ධිමත්වූ, කුසල් අකුසල් මේ මේ යයි දැනගැනීමෙහි දක්‍ෂවූ මම එහි කුකුත්‍ථක පක්‍ෂියෙක් වීමි.
# ´´ලෝකයාගේ පූජාවන් පිළිගැනීමට සුදුසුවූ නොහොත් පූජාවන් (රැස්වන) ගෙයක් වැනිවූ, මහාමුනිවූ පදුමුත්තර බුදුරජාණන් වහන්සේ (එම) ස්වයංජාත විලට නුදුරෙහි සක්මනෙහි යෙදුනු සේක.
# ´´ජලයෙහි හටගන්නාවූ කුමුදු මලක් කඩා බුදුරජුන් වෙත ගෙන ගියෙමි. මහාමුනිවූ සර්වඥතෙමේ මාගේ අදහස දැන එය පිළිගත් සේක.
# ´´එම දන දී කුශල මූලයෙන් මෙහෙයන ලද මම කල්ප ලක්‍ෂයක් මුළුල්ලෙහි දුගතියක නොඉපදුනිමි.
# ´´මින් එක් දහස් හය සියවන කපෙහි මහත් බල ඇති වරුණ නම් අට සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කුමුදුදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
