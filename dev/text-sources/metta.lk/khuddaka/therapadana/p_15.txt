# අධිඡත්තිය ථෙරාපදානය.
# ´´භාග්‍යවත්වූ අත්‍ථදස්සී බුදුරජාණන් පිරිනිවි කල්හි (උන්වහන්සේගේ) ධාතු චෛත්‍යයෙහි ඡත්‍රය මතුයෙහි ඡත්‍රයක් කරවා පිදීමි.
# ´´කලින් කල අවුත් සාස්තෲන් වහන්සේගේ චෛත්‍යයට නමස්කාර කෙළෙමි. මල් වියනක්ද කොට ඡත්‍රයෙහි (මත) නගා තැබීමි.
# ´´(ඒ හේතුවෙන්) එක් දහස් එක් සිය වන කපෙහි දිව්‍ය රාජ්‍යය කරවීමි. මිනිසත් බවට නොයමි. ස්ථූප පූජාවෙහි මේ විපාකයයි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අධිඡත්තිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ථම්භාරොපක ථෙරාපදානය.
# ´´ධම්මදස්සී බුදුරජුන් පිරිනිවි කල්හි (එම) බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ චෛත්‍යයෙහි ධජස්ථම්බයක් (කොඩියක්) නැංවීමි.
# ´´ඉණිමගක් සාදවා උතුම් සෑයට නැගුනෙමි. දෑසමන් මල් ගෙන සෑයට පිදීමි.
# ´´සර්‍වඥ තෙම ආශ්චර්‍ය්‍යයකි. ධර්‍මය ආශ්චර්‍ය්‍යයකි. අපගේ ශාස්තෘ සම්පත්තියද ආශ්චර්‍ය්‍යයකි. (එකී පුණ්‍ය හේතුවෙන්) දුගතියක් නොදනිමි. ථූප පූජාවෙහි මේ විපාකයයි.
# ´´මින් සිව් අනුවන කපෙහි මහත් බල ඇති ථූප සිඛර නම්වූ සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ථම්භාරොපක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වේදිකාරක ථෙරාපදානය.
# ´´ලෝකනායකවූ පියදස්සී බුදුරජුන් පිරිනිවි කල්හි මම පහන් සිත් ඇත්තේ සොම්නස්වූයේ, මුතු වේදිකාවක් කෙළෙමි.
# ´´මැණික්වලින් පිරිවරකොට උතුම් වේදිකාව කෙළෙමි. මම එම වේදිකාව පිදීමේ උත්සවය කොට එහි කළුරිය කෙළෙමි.
# ´´(ඒ හේතුවෙන්) දිව්‍යවූ හෝ, මනුෂ්‍යවූ හෝ යම් යෝනියක උපදිම්ද, අහසෙහි මිණි දරත්. පුණ්‍ය කර්‍මයාගේ මේ විපාකයයි.
# ´´මින් එක් දහස් හසියවන කපෙහි මහත් බල ඇති මනිප්පභාස නම් දෙතිස් සක්විති රජ කෙනෙක්වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වේදිකාරක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සපරිවාරිය ථෙරාපදානය.
# ´´පස්මරුන් දිනූ, ලෝක ජ්‍යෙෂ්ඨවූ, නරශ්‍රේෂ්ඨවූ, පදුමුත්තර බුදුරජ තෙම ගිනි කඳක් මෙන් දිලිසී පිරිනිවන් පානා ලදුයේ විය.
# ´´මහා වීරවූ බුදුරජාණන් වහන්සේ පිරිනිවි කල්හි (උන්වහන්සේගේ ධාතු නිදන් කළ) මහත් සෑයක් විය. අති උතුම් ධාතු ගෘහයෙහි දිව රෑ දෙක්හි උවටැන් කෙරෙත්.
# ´´පහන් සිත් ඇත්තේ, සොම්නස් වූයේ, (එහි) සඳුන් වේදිකාවක් (චේතියඝරයක්) කෙළෙමි. එකල්හි (එය) සෑයට සුදුසු වූයේ, දුම්කඳක් මෙන් පෙනේ.
# ´´දිව්‍යවූ හෝ, මනුෂ්‍යවූ හෝ භවයෙහි උපදිනා කල්හි මාගේ පහත් බවක් නොදකිමි. පූර්‍ව කර්‍මයාගේ මේ විපාකයයි.
# ´´මින් එක් දහස් පන්සියවන කපෙහි මහත් බල ඇති ඒ සියලු සක්විති රජවරු අට දෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සපරිවාරිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උම්මා පුප්ඵිය ථෙරාපදානය.
# ´´ලෝවාසීන්ගේ පූජාවන් පිළිගත්තාවූ නොහොත් පූජාවනට ගෙයක් වැනිවූ ලෝකනාථවූ සිද්‍ධාර්ත්‍ථ භාග්‍යවතුන් වහන්සේ පිරිනිවි කල්හි මහා සෑ පූජාවක් විය.
# ´´සිදුහත් බුදුරජුන්ගේ පූජාව පවත්නා කල්හි දියබෙරලිය මල් ගෙන සෑයට පිදීමි.
# ´´මින් සිව්අනූවන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ස්ථූප පූජාවෙහි මේ විපාකයයි.
# ´´මින් නවවන කපෙහි මහත් බල ඇති සෝමදේව නම්වූ පස් අසූ සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උම්මා පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# අනුලේපදායක ථෙරාපදානය.
# ´´මම අනොමදස්සී බුදුරජාණන් වහන්සේගේ බෝධියට (පරිමණ්ඩලවූ) වේදිකාවක් කරවීමි. (එසේම) මම සුණු පිඬුදී පිළිසකරද කෙළෙමි.
# ´´ශාස්තෘවූ අනොමදස්සී නරශ්‍රේෂ්ඨ තෙම මැනවින් කරණ ලද එම කර්‍මය දැක භික්‍ෂු සංඝයා මැද හිඳ මෙම ගාථාවන් වදාළ සේක.
# ´´මෙම සුණුකම කරණකොටගෙනද, චිත්තප්‍රණිධිය කරණකොටගෙනද සම්පත් අනුභවකොට දුක් කෙළවර කරන්නේය කියායි.
# ´´පහන්වූ මුහුණු පැහැ ඇත්තේ වීමි. එකඟවූයේ, සන්සුන්වූයේ, සම්බුදු සසුනෙහි අන්තිම ශරීරය දරමි.
# ´´මින් නොඅඩුව කප් සියයක් සම්පුර්‍ණවූ කල්හි මහත් බල ඇති සබ්බඝන නම් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අනුලේපදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# මග්ගදායක ථෙරාපදානය.
# ´´පසැස් අති බුදුරජ තෙම ගඟකින් එතරව වනයට යයි. මම උතුම් ලකුණු ඇති එම සිද්‍ධාර්ත්‍ථ නම් සම්බුදු රජුන් දිටිමි.
# ´´උදැල්ල හා කූඩයද ගෙත ඒ මග සමකොට ශාස්තෲන් වහන්සේ වැඳ, සිත පැහැදවීමි.
# ´´මින් සිව් අනූවන කපෙහිවූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. මාර්‍ග ඥානයාගේ මේ විපාකයයි.
# ´´සත් පණස්වන කපෙහි ජනාධිපතිවූ නම්න් සුප්පබුද්‍ධ නම්වූ නරශ්‍රේෂ්ඨ එක් සක්විති රජෙක් වීමි.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මග්ගදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ඵලදායක ථෙරාපදානය.
# ´´පෙර වඩු කමෙහි මැනවින් හික්මුණු රිය සදන්නෙකු වීමි. (මම) සඳුන් ලෑලිකොට, ලෝකබන්‍ධුවූ බුදුරජාණන් වහන්සේට දුනිමි.
# ´´රනින් නිමවන ලද මේ විමන බබළයි. හස්තියානද, අශ්වයානද, දිව්‍යමයයානද එළඹ සිටියේ වෙයි.
# ´´ප්‍රාසාදයෝද සිවිකාවෝ (දෝලාවෝ) යන යමක් කැමැති වේ නම් (ඒවා) උපදිත්. මාගේ රුවන් නිශ්චලය (නැති නොවේ.) (සඳුන්) පවුරු දීමෙහි මේ විපාකයයි.
# ´´මම මින් එක් අනූවන කපෙහි යම් පුවරු දනක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුවරු දනෙහි මේ විපාකයයි.
# ´´සත්පණස්වන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති නිම්මිත නම් සිව් සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# වටං සකිය ථෙරාපදානය.
# ´´සියල්ල තෙමේම දත්තාවූ පස්මරුන් දිනූ නම්න් සුමේධ නම්වූ (සර්‍වඥ තෙම) විවේක සුව වඩනුයේ මහ වනයට පිවිසි සේක.
# ´´පිපුනු සල්ල ගසක් දැක හිස් පළඳින මාලාවක් සේ ගොතා ලෝක නායකයන් වහන්සේ ඉදිරියෙහි (එම) බුදුරජාණන් වහන්සේටම පිදීමි.
# ´´තිස් දහසක් වන කපෙහි යම් මලක් පිදූයෙම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´එක් දහස් නවසියවන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති නිම්මිත නම්වූ සොලොස් සක්විති රජෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් වටං සකිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# පල්ලඞ්කදායක ථෙරාපදානය.
# ´´ලෝක ජ්‍යෙෂ්ඨවූ, නරශ්‍රේෂ්ඨවූ, තාදී ගුණ ඇති සුමේධ භාග්‍යවතුන් වහන්සේට උඩුවියන් (තේන්දය) සහිත ඇතිරිළි සහිත පළහක් (පුටුවක්) මා විසින් දෙන ලදී.
# ´´එකල්හි එය සත්රුවනින් යුත් පළඟක් විය. මාගේ අදහස දැන (ඒ අනුව) හැම කල්හි මට එබඳු (පර්‍ය්‍යංක) උපදියි.
# ´´තිස් දහස්වන කපෙහි වූ එකල්හි (යම්) පළඟක් දුනිම්ද (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´විස් දහස්වන කපෙහි සත් රුවනින් යුත් මහත් බල ඇති සුවණ්ණ නාභ නම් තුන් සක්විති රජකෙනෙක් වූහ.
# ´´මේ සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පල්ලඞ්කදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
