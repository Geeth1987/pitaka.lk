 බොධිවන්‍දක වග්ග.   
# බොධිවන්දක ථෙරාපදානය.
# ´´මම පළාවන්වූ පලොල් ගසක් දැක, එකස්කොට ඇඳිලි බැඳ, එම පළොල් බෝධියට වැන්දෙමි. (පළොල් ගස විපස්සී බුදුරජුන්ගේ බෝධියයි.

# ´´ඇඳිලි බැඳගෙන, සිතින්ද ගරුකොට, ඇතුළත පිරිසිදුවූ, පිටතද පිරිසිදුවූ, මැනවින් (සියළු කෙළෙසුන්ගෙන්) මිදුනාවූ, ආශ්‍රව රහිතවූ, ලෝවැස්සන් විසින් පුදන ලද, කරුණාවට හා ඥානයට සාගරයක් වැනිවූ, විපස්සී බුදුරජුන් හමුයෙහි මෙන් එම පළොල් ගසට වැන්දෙමි.
#
# ´´මින් එක් අනූවන කපෙහි යම් බෝධියකට වැන්දෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. වන්‍දනාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් බොධිවන්දක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පාටලීපුප්ඵිය ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කළවූ, අග්‍ර පුද්ගලවූ, පස්මරුන් ජයගත්, විපස්සී නම් භාග්‍යවතුන් වහන්සේ, ස්වකීය ශිෂ්‍යයන් විසින් පිරිවරණ ලදුයේ, බන්‍ධුම නගරයට පිවිසි සේක.
# ´´මා විසින් පළොල් මල් තුනක් ඔඩොක්කුවෙහි තබන ලද්දේ. (ඒ) මම හිස පටන් නගනු කැමැත්තේ ගංතෙරට ගියෙමි.

# ´´බන්‍ධුමතී නගරයෙන් නික්ම වඩින්නාවූ, මහනිල් මලක් මෙන් බබලන, ඇවිලගත් ගින්නක් වැනිවූ, ශ්‍රෙෂ්ඨ ව්‍යාඝ්‍රයෙකු මෙන්වූ, උතුම්වූ, විශිෂ්ඨ ජන්ම ඇති, සිංහයෙකු වැනිවූ, ශ්‍රමණයනට අග්‍රවූ, භික්‍ෂු සංඝයා විසින් පිරිවරන ලද ලෝකනායකවූ බුදුරජුන් දුටිමි.
#
# ´´කෙළස් මල දෝනාවූ, ඒ බුදුරජුන් කෙරෙහි පැහැදුනේ, (එම පළොල්) මල් තුන ගෙන බුද්ධ හ්‍රෙෂ්ඨයන් වහන්සේ පිදීමි.
# ´´මින් එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පාටලීපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිඋප්පලමාලිය ථෙරාපදානය.
# ´´මම චන්‍ද්‍රභාග නදී තීරයෙහි වඳුරෙක් වීමි. එකල්හි කඳු අතරෙහි වැඩහුන් කෙළෙස් රජස් නැති බුදුරජුන් දුටිමි.
# ´´සියළු දිගුන් බබුළුවන්නාවූ, පිපුණු සල් රුකක් වැනිවූ, (දෙතිස් මහ පුරිස්) ලකුණින් හා (අසූ) අනුව්‍යඤ්ජනයන්ගෙන් යුත් (උන්වහන්සේ) දැක සතුටුවූ මම,
# ´´ඔදවැඩි සිත් ඇත්තේ, සොම්නස්වූයේ, ප්‍රීතියෙන් පිනාගිය සිත් ඇත්තේ, උපුල් මල් තුනක් හිස මත (අහසෙහි දමා) පිදීමි.
# ´´මම ඵුස්ස බුදුරජුන්හට (එම) මල් පුදා, ගෞරව සහිතව උතුරු දෙසට මුහුණ ලා ගියෙමි.
# ´´පැහැදි සිතින් යුතුව යන්නේ, කඳු අතරෙහි වැටී ජීවිතක්‍ෂයට පත්වීමි.
# ´´මැනවින් කරණ ලද එම පුණ්‍ය කර්‍මය කරණකොටගෙනද, සිතෙහි ඇතිවූ ප්‍රාර්ත්‍ථනාවන් කරණකොටගෙනද, මම පළමු ජාතිය හෙවත් එම ආත්මය හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි.
# ´´තුන්සිය වරක් දිව්‍ය රාජ්‍යය කරවීමි. (එසේම) මම පන්සිය වරක් සක්විති රජ වීමි.
# ´´මින් දෙයානූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිඋප්පලමාලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පට්ටිපුප්ඵිය ථෙරාපදානය.
# ´´යම් කලෙක්හි මහර්ෂීවූ පදුමුත්තර බුදුරජ තෙම පිරිනිවන් පෑ සේක්ද, එකල්හි සියළුම මහජනයා එකතුව උන්වහන්සේගේ ශ්‍රී දේහය ගෙන යත්.
# ´´(එසේ) බෙර වයද්දී ශ්‍රී ශරීරය ගෙය යන කල්හි පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, මම පට්ටි පුෂ්පයක් පිදීමි.
# ´´මින් සියක් දහස් (ලක්‍ෂයක්) වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ශ්‍රී දේහය පිදූ කල්හි (ලැබෙන මේ) විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පට්ටිපුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සත්තපණ්ණිය ථෙරාපදානය.
# ´´ලෝකනායකවූ සුමන නම් සම්බුදු රජ තෙමේ ලොව පහළ වූහ. (මම) පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, රුක්අත්තන (මල්) පිදීමි.
# ´´මින් සියක් දහස් (ලක්‍ෂයක්) වන කපෙහි වූ එකල්හි යම් රුක් අත්තන මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. රුක්අත්තන මල් පිදීමේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සත්තපණ්ණිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# ගන්‍ධමුට්ඨිය ථෙරාපදානය.
# ´´නොයෙක් සුවඳ වර්‍ග රැස්කොට සෑයක් (චිතකයක්) කරනු ලබන කල්හි මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, සුවඳ මිටක් (මම සෑයට) පිදීමි.
# ´´මින් ලක්‍ෂයක්වන කකල්පයෙහි යම් චිතක පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් ගන්‍ධමුට්ඨිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# චිතක පූජක ථෙරාපදානය.
# ´´පදුමුත්තර නම් බුදුරජුන් පිරිනිවි කල්හි (උන්වහන්සේගේ ශ්‍රී දේහය) චිතකයට නැගූ කල්හි (මම) සල් මල් පිදීමි.
# ´´මින් ලක්‍ෂයක්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් චිතක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සුමනතාලවණ්ටිය ථෙරාපදානය.
# ´´මම සමන් මල්වලින් මසා කළ මාහැඟි විජිනිපතක් දරමි. මම එම විජිනිපත සිද්‍ධාර්ත්‍ථ භාගයවතුන් වහන්සේට දුනිමි.
# ´´මම මින් සිව් අනූවන කපෙහි යම් විජිනි පතක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. විජිනිපත් දනෙහි මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුමනතාලවණ්ටිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# සුමනදාසිය ථෙරාපදානය.
# ´´(චතුර් මාර්‍ග ඥාන ජලයෙන් කෙළෙස් කසට සේදී යන සේ) ස්නානය කළාවූ තපොගුණ ඇති සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට ඉදිරියෙහි සිටියේ සමන්මල් දමක් කොට දැරීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් (සමන් මල්) දමක් දැරූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. සුමන මල් දම් දැරීමෙහි මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සුමනදාසිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# කාසුමාරිඵලදායක ථෙරාපදානය.
# ´´කිණිහිරි රුකක් මෙන් බබළන්නාවූ කඳු, අතරෙහි වැඩහුන් කෙළෙස් රජස් විරහිත, ලොවට දෙටුවූ, නර ශ්‍රේෂ්ඨවූ බුදුරජුන් දුටිමි.
# ´´(දැක) පහන් සිත් ඇති, සොම්නස්වූ මම හිසෙහි ඇඳිලි බැඳ, කාසුමාරි ඵල (ඇත්දෙමට ගෙඩි) ගෙන බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේට දුනිමි.
# ´´මින් එක් තිස්වෙනි කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියත් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන්ම මාගේ යහපත් පැමිණීමක් විය. ත්‍රිවිද්‍යාවෝ පැමිණෙන ලද්දාහු වෙති. ත්‍රිවිද්‍යා (ලාබීමි.) බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.
# ´´මා විසින් කෙළෙස් දවන ලදී. සියළු භවයෝ නසන ලදහ. හස්තියෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කාසුමාරිඵලදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
