# එකාසනදායක ථෙරාපදානය.
# ´´දෙව්ලොව සිටියේ, විපස්සී බුදුරජුන්ගේ ධර්‍මය අසා සතුටුවූයේ මම මේ වචනය කීමි.
# ´´පුරුෂ ශ්‍රේෂඨයන් වහන්ස, ඔබ වහන්සේට නමස්කාර වේවා. පුරුෂොත්තමයන් වහන්ස, ඔබ වහන්සේට නමස්කාර වේවා. ඔබ අමා පදය (ධර්‍මය) දෙසමින් බොහෝ දෙන (සසර සයුරින්) එතර කරහිය කියායි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි (යම් බුදුරජුන්හට ස්තූති) වචනයක් කීවෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ ස්තූති කරණයෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් එකාසනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# එකාසනදායක ථෙරාපදානය.
# ´´බුද්‍ධ ශ්‍රේෂඨයන් වහන්සේගේ සසුනෙහි කුසල් කරනු කැමතිව දෙව් වෙස හැර බිරිය සමග මෙහි පැමිණියෙමි.
# ´´(එකල්හි) දෙවල නම් පදුමුත්තර බුදුරජුන්ගේ ශ්‍රාවකයෙක් විය. මා විසින් උන් වහන්සේට පහන් සිතින් භික්‍ෂාවක් දෙන ලදී.
# ´´මින් සියක් දහස්වන කපෙහි වූ එකල්හි යම් කර්‍මයක් කරණ ලදද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිණ්ඩපාත දානයාගේ මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් එකාසන දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# චිතකපූජක ථෙරාපදානය.
# ´´සියල්ල තෙමේම අවබෝධ කළාවූ, පස් මරුන් දිනූ ආනන්‍ද නම් සම්බුදු රජ තෙමේ අමනුෂ්‍ය පරිගෘහිත වනයෙහි පිරිනිවන් පා වදාළ සේක.
# ´´මම එකල්හි දෙව්ලොවින් මෙහි පැමිණ, චිතකයක් කොට (එම බුද්‍ධ ශරීරය) දැවීමි. සත්කාරද කෙළෙමි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් චිතකපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# තිචම්පක ථෙරාපදානය.
# ´´හිමවතට නුදුරෙහි චිතක නම් පර්‍වතයෙක් විය. එම පර්‍වතය මධ්‍යයෙහි වඩන ලද ඉඳුරන් ඇති ශ්‍රමණයෙක් තෙම වෙසෙයි.
# ´´උන්වහන්සේගේ සන්සුන්බව දැක පැහැදි සිතින් සපුමල් තුනක් ගෙන විසිරවීමි. (පිදීමි.)
# ´´මම එක් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් තිචම්පක පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# සත්ත පාටලිය ථෙරාපදානය.
# ´´පර්වතාන්තරයෙහි වැඩහුන්නාවූ, පිපුණු කිණිහිරි රුකක් මෙන් බබලන බුදුරජුන්හට පළොල් මල් සතක් පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් සත් පාටලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# උපාහනදායක ථෙරාපදානය.
# ´´එකල්හි චන්‍දන නම් බුද්‍ධපුත්‍රයෙක් විය. මා විසින් උන්වහන්සේට එක් පාවහන් සඟලක් දුනිමි. (දීමෙන් පැතීමි. එම්බා පාවහන් සඟල, ´´ඔබ මා (ශ්‍රාවක) බෝධියට පමුණුවන්නෙහිය´´ කියායි.
# ´´මින් එක් අනූවන කපෙහි වූ එකල්හි යම් පාවහන් දෙකක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පාවහන් දනෙහි මේ විපාකයයි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් උපාහනදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´

# මඤ්ජරී පූජක ථෙරාපදානය.
# ´´මම මදිටි මල් ගොටුවක් කොට වීදියට පිවිසියෙමි. (එහිදී) භික්‍ෂු සංඝයා විසින් පිරිවරණ ලද ශ්‍රමණයන්ට අග්‍රවූ බුදුරජාණන් වහන්සේ දුටිමි.
# ´´උතුම් ප්‍රීතියෙන් පහන් සිත් ඇත්තේ, සොම්නස් වූයේ, දෝතින් ගෙන බුදුරජුන් හට පිදීමි.
# ´´මින් දෙයානූවන කපෙහි යම් මල් පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප පූජාවෙහි මේ විපාකයයි.
# ´´මින් තෙසැත්තෑවන කපෙහි පෘථිවීශ්වරවූ, මහත් බල ඇති ජොතිය නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් මඤ්ජරීපූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# පණ්ණදායක ථෙරාපදානය.
# ´´මම හිමවතට වැහැරි සිවුරු දරණ, ලුණු නොමැතිව කොළ අනුභව කරන්නාවූ, නියමි නම්වූ, පඤ්ච සීලයෙහි හික්මුනාවූ තවුසෙක් වෙමි.
# ´´පෙරවරුව පැමිණි කල්හි සිද්‍ධාර්ත්‍ථ බුදුරජ තෙම මාවෙත වැඩි සේක. මම පහන්වූයේ සියතින් බුදුන්හට පිදීමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් කොළ දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. කොළ දානයාගේ මේ විපාකයයි.
# ´´සත්තිස්වන කපෙහි සත් රුවනින් යුත්, මහත් බල ඇති ´´යදත්‍ථිය´´ නම් නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් පණ්ණදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# කුටිදායක ථෙරාපදානය.
# ´´වනයෙහි හැසිරෙන්නාවූ සර්‍වඥ තෙමේ එකල්හි රුක්මුලක විසූහ. මම පස්මරුන් දිනූ උන්වහන්සේට පන්සලක් කරවා දුනිමි.
# ´´මින් එක් අනූවන කපෙහි යම් පර්‍ණකුටියක් දුනිම්ද, එකල්හි (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. කුටි දානයාගේ මේ විපාකයයි.
# ´´මින් අටවිසිවන කපෙහි, සබ්බත්‍ථ අභි වස්සි නම් සොළොස් සක්විති රජ කෙනෙක් වූහ.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් කුටිදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´ 
# 
# අග්ගජ පුප්ඵිය ථෙරාපදානය.
# ´´පර්‍වතාන්තරයෙහි වැඩහුන්නාවූ, ස්වර්‍ණවර්‍ණවූ රශ්මිය කරණකොටගෙන ගිනි කඳක් මෙන් බබළන කිඛී බුදුරජුන් (දුටිමි.)
# ´´මම අග්ගජ නම්වු මල් ගෙන බුදුරජුන් වෙත එළඹියෙමි. පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, (එය) බුදුරජුන්හට පිදීමි.
# ´´මින් එක්තිස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි.
# ´´පස්විසිවන කපෙහි සත්රුවනින් යුත්, මහත් බල ඇති අමිත නම් සක්විති රජෙක් වීමි.
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදී. බුදුරජාණන් වහන්සේගේ සසුනද කරණලදී. මෙහි මේ අයුරින් ආයුෂ්මත් අග්ගජ පුප්ඵිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.´´
