සාලකුසුමිය වග්ග.   
# සාලකුසුමිය ථෙරාපදානය.
# ´´පදුමුත්තර බුදුරජතෙම පිරිනිවි කල්හි (උන්වහන්සේගේ) ශ්‍රී දේහය චිතකයට නැංවූ කල්හි සල් මල් පිදීමි.
# ´´මින් සියක් දහස්වන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් සාල කුසුමිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# චිතක පූජක ථෙරාපදානය.
# ´´ධ්‍යාන වඩන්නාවූ, ලෝකබන්ධුවූ සිඛී බුදුරජාණන් වහන්සේගේ චිතකයට සපු මල් අටක් පිදීමි.
# ´´මින් එක් තිස්වන කපෙහිවූ එකල්හි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. චිතක පූජාවෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් චිතක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# චිතක නිබ්බාපක ථෙරාපදානය.
# ´´වෙස්සභූ බුදුරජුන්ගේ සිරුර දැවෙන කල්හි මම සුවඳ පැන් ගෙන චිතකය නිවීමි.
# ´´මින් එක් තිස්වන කපෙහි ( බුදුරජුන් ගේ) චතකය නිවූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. සුවඳ දිය දීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් චිතක පූජක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# සෙතුදායක ථෙරාපදානය.
# ´´සක්මන් කරන්නාවූ විපස්සී බුදුරජුන් හමුවෙහි මම පහන් සිත් ඇත්තේ, සොම්නස්වූයේ එදණ්ඩක් කරවීමි.
# ´´මින් එක් අසූවන කපෙහි (යම්) හෙයක් කරවූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. සෙතු (ඒ දඬු) දානයෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් සෙතුදායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# සුමනකාල චණ්ඩිය ථෙරාපදානය.
# ´´මම සිද්‍ධාර්ත්‍ථ බුදුරජාණන් වහන්සේට සමන් මල්වලින් වසන ලද විජිනිපතක් දුනිමි. (එයින් මම) මහත් යශස් දරමි.
# ´´මම මින් සිව් අනූවන කපෙහි යම් විජිනි දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. තල්වැට (වටාපත්) දීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් සුමනකාල චණ්ඩිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# අවණ්ට ඵලිය ථෙරාපදානය.
# ´´පස්මරුන් දිනූ, සියල්ල තෙමේම අවබෝධ කළාවූ සතරංසී නම් භාග්‍යවත් බුදුරජතෙම විවේක සුව කැමැත්තේ පිඬු පිණිස නික්මුණ සේක.
# ´´අතින් ගත් ගෙඩි (වර්‍ගයක්) ඇති මම බුදුරජුන් දැක (එහි) එළඹියෙමි. පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, එම (වටවු) ගෙඩි (වර්‍ගය) උන්වහන්සේට දුනිමි.
# ´´මින් සිව් අනූවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් අවන්ට ඵලිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# ලබුජ දායක ථෙරාපදානය.
# ´´එකල්හි (මම) බන්ධුමතී නගරයෙහි ආරාමිකයෙක් (අරමක වතාවත් කරන්නෙක්) වීමි.
#
# ´´මම දෙල් ගෙඩියක් ගෙන, බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේට දුනිමි. මහත් යසස් ඇති, මාගේ ප්‍රීතිය ඇති කරන්නාවූ, මෙලෙව්හිදීම සුව ගෙන දෙන්නාවූ බුදුරජතෙම අහසෙහි සිටියේ එය පිළිගත් සේක. පැහැදි සිතින් (එම) ඵලය බුදුරජුන්හට දී,
# ´´එකල්හි (මම) මහත්වූ ප්‍රීතියද, උතුම්වූ සැපයද ලැබීමි. ඒ ඒ තන්හි උපන්නාවූ මට රුවන්ද උපදී.
# ´´මින් එක් අනූවනවන කපෙහිවූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් ලබුජ දායක තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# පිලක්ඛඵල දායක ථෙරාපදානය.
# ´´වනයෙහිදී මහත් යසස් ඇති අත්‍ථදස්සී බුදුරජුන් දැක, පහන් සිත් ඇත්තේ, සොම්නස්වූයේ, පුලිල ගෙඩියක් උන්වහන්සේට දුනිමි.
# ´´අටළොස් සියවන කපෙහි වූ එකල්හි යම් ඵල දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ඵල දානයාගේ මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් පිලක්ඛඵල තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# සයම්පටි භාණිය ථෙරාපදානය.
# ´´බබලන්නාවූ (පිපුනු) කුඹුක් ගසක් වැනිවූ, දේවාතිදේවවූ, නරශ්‍රේෂ්ඨවූ, වීථියට පිළිපන්නාවූ බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´(මොහ නමැති) ඝනාන්‍ධකාරය නසා බොහෝ දෙන (සසරින්) එතරකොට ඥාන ලෝකයෙන් බබලන්නාවූ බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´රහතුන් ලක්‍ෂයක් සමග වඩින්නාවූ, ලෝක නායකවූ, බොහෝ සත්ත්‍වයන් දුකින් ගොඩලන්නාවූ බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´දහම් බෙරය ගසන්නාවූ, තීර්ත්‍ථක සමූහයා මඩින්නාවූ, සිහනද (ශ්‍රේෂඨ නාද) කරන්නාවූ බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´බඹලොවින් යම්තාක් බඹුන් සහිත (දෙවියෝ) අවුත් සියුම් පැණ විචාරත්ද, (ඒ බුදුරජුන්) දැක කවරෙක් නම් නොපහදිත්ද?
# ´´දෙවියන් සහිත ලෝකයා යම් උතුමෙකුට ඇඳිලි බැඳ අයදිත්ද, එයින් (ඔවුහු) පින් (ඵල) අනුභව කෙරෙත්ද, ඒ බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´සියලු දෙන රැස්ව බුදුරජුන් පවරත්. එසේ ආරාධනය කරන ලදුයේ කම්පා නොවේ. (එම බුදුරජුන්) දැක කවරෙක් නම් නොපහදීද?
# ´´නුවරට පිවිසෙන්නාවූ යමෙකුට බොහෝ භෙරීහු (බෙර) නාද කෙරෙත්ද, මත්ත හස්තීහු විශෙෂයෙන් නාද කෙරෙත්ද, (එම බුදුරජුන්) දැක කවරෙක් නම් නොපහදීද?
# ´´හැම කල්හි වීථියෙහි ගමන් කරන්නාවූ යමෙකුගේ (සිරුරින්) සියළු පැහැ බබලමින් විහිදේද, (භූමියෙහි) උස් ස්ථානයෙහි (පා තබනවිට) කවරෙක් නම් නොපහදීද?
# ´´බුදුරජුන් දහම් දෙසන කල්හි (මුළු) සක්වළෙහිම ඇසේද, ඒ බුදුරජතෙම සියළු සත්ත්‍වයන් (එම) ධර්‍මයෙන් අවබෝධ කරවති. (එම) බුදුරජුන් දැක කවරෙක් නම් නොපහදීද?
# ´´මින් සියක් දහස්වන කපෙහි බුදුරජුන්හට (මෙසේ) යම් ස්තූතියක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. බුදුරජුන්හට පැසසීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් සයම්පටි භාණිය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
# 
# නිමිත්ත ව්‍යාකරණීය ථෙරාපදානය.
# ´´මම එකල්හි හිමවතට පිවිස මන්ත්‍ර කියවමි. සූපණස් දහසක් ශිෂ්‍යයෝ මට උවටැන් කළාහුය.
# ´´වේද හදාරණ ලද්දාවූ, සියලු වේදය දත්, ෂඩඞ්ගයෙහි වෙදාඞ්ග පරතෙරෙට ගියාවූ ස්වකීය විද්‍යාවන්ගෙන් ස්ථිරබවට පැමිණියාවූ ඔවුහු හිමවතෙහි වසත්.
# ´´මහත් යසස් ඇති (බොධිසත්ත්‍ව) දිව්‍යපුත්‍රතෙම තුසිත දෙව්ලොවින් ච්‍යුතව, මනා නුවණැත්තේ, සිහියෙන් යුතුවූයේ මව්කුසෙහි උපන්නාහ.
# ´´සම්බුදු රජුන් උපදනා කල්හි දශසහශ්‍රී ලෝකධාතුව කම්පා විය. එම ලෝක නායකයන් වහන්සේ උපදනා කල්හි අන්‍ධයෝ ඇස් ලැබූහ.
# ´´මේ සම්පූර්‍ණ පෘථිවිය හැම අයුරකින්ම කම්පාවිය. (ඒ) මහ හඬට මහාජනයෝ කලබල වූහ.
# ´´සියළුදෙන එකතුව මාගේ සමීපයට පැමිණියාහුය. ´´මේ පොළොව කම්පා විය. කවර විපතක් වන්නේදැ´´ යි (විචාළේය.)
# ´´එකල්හි ඔවුන්හට (මම) මෙසේ කීයෙමි. ´´බිය නොවව්. තොපට (මින්) බියෙක් නැත. සියල්ලෝ විශ්වාස ඇත්තෝ වව්. මෙය සැප ඇති උපතෙකි. (එබඳු උපතක ලකුණකි.)
# ´´හේතු අටකින් ගැටුණු මේ පොළොව කම්පා කළේය. එබඳු නිමිතීහු පැණෙති. (මේ) අතිමහත් ආලෝකයකි.
# ´´පසැස් ඇති බුදුරජතෙම නිසැකවම (ලොව) උපදින්නේයැ´´ යි ජනසමූහයාට දන්වා මම ඔවුනට පන්සිල් කීමි.
# ´´පඤචසීලයද, දුර්ලභවූ බුද්ධොත්පාදයද (ගැන) අසා ඔවුහු සොම්නස්වූවාහු, තුටු පහටු වූවාහු, හටගත් කළකිරීම් ඇත්තෝද වූහ.
# ´´මින් දෙයානූවන කපෙහි (බුදුරජුන් උපදින්නේයයි) යම් නිමිත්තක් පැවසූයෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. නිමිති පැවසීමෙහි මේ විපාකයයි.
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතෙකු මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි.
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ පැමිණීමක් විය. පමුණුවන ලද ත්‍රිවිද්‍යා ඇත්තෝය. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
#
#මෙහි මේ අයුරින් ආයුෂ්මත් නිමිත්ත ව්‍යාකරණීය තෙරුන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.
