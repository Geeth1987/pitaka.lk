# කුණ්ඩලකෙසාපදානය.  
# ´´මින් (පෙර) සියක් දහස්වන කපෙහි ලෝක නායකවූ, සියළු ධර්‍මයන්හි පරතෙරට ගිය පදුමුත්තර නම් බුදුරජතෙම ලොව පහළවිය. 
# ´´එකල්හි මම හංසවතී නගරයෙහි නන්රුවනින් බබලන සිටු කුලයක උපන්නී, බොහෝ සුව විඳින්නියක් වීමි. 
# ´´එම මහාවීරවූ බුදුරජුන් වෙත එළඹ දම් දෙසුම ඇසීමි. ඉන් හටගත් පැහැදීම ඇත්තාවූ මම බුදුරජුන් සරණකොට ගියෙමි. 
# ´´එකල්හි මහා කාරුණීකවූ පදුමුත්තර නම් බුදුරජතෙම සුභ නම් භික්‍ෂුණියක් වහා දහම් අවබෝධ කරන්නාවූ භික්‍ෂුණීන් අතුරෙන් අගතැන්හි තැබූහ. 
# ´´එය අසා ප්‍රමුදිතව බුදුරජුන්හට දන් දී, සිරසින් පාමුල වැටී එම තනතුර පැතීය. 
# ´´මහාවීරවූ බුදුරජතෙමේ, ´´සොඳුර, තී විසින් පතන ලද යමක් වේ නම්, ඒ සියල්ල සමෘද්‍ධ වන්නේය. නිවනට පැමිණියා සුව ඇත්තියක් වනු මැනවැ´´ යි අනුමෝදනා කළ සේක. 
# ´´මින් මතු සියක් දහස්වන කපෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ නම්න් ගෞතම නම්වූ ශාස්තෘ තෙමේ ලොව පහල වන්නේය. 
# ´´උන්වහන්සේගේ ධර්‍මයෙහි දායාදීවූ, ධර්‍මයෙන් නිර්මිත කළාක් වැනිවූ, ඖරස බුද්ධපුත්‍රීවූ, ´´භද්දකුණ්ඩලකෙසී´´ යයි බුද්‍ධශ්‍රාවිකාවක් වන්නෙහිය.´´ 
# ´´මැනවින් කරන ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්තප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´ඉන් චුතව යාම දෙව්ලොවට ගියෙමි. මම ඉන් තුසිතයට ගියා, එතැනින් නිර්‍මාණරතියටද, ඉන් වසවර්ති පුරයටද ගියෙමි. 
# ´´එම පුණ්‍ය කර්‍මයාගේ බලයෙන් යම් යම් තැනක උපදිම්ද, ඒ ඒ තන්හි රජුන්ගේ මෙහෙසිබව කෙළෙමි. 
# ´´ඉන් චුතව මිනිසත් බැව්හි සක්විති රජුන්ගේද, මණ්ඩලීක රජුන්ගේද මෙහෙසිබව කෙළෙමි. 
# ´´දෙව් මිනිස් දෙගතියෙහි සම්පත් අනුභවකොට, හැම තන්හි සුව ඇත්තීව, නොයෙක් කල්පයන්හි සැරිසැරීමි. 
# ´´කථා කරන්නවුන් අතරින් ශ්‍රේෂ්ඨවූ, බ්‍රාහ්මණ වංශයෙහිවූ, මහත් යසස් ඇති, නම්න් කාශ්‍යප නම්වූ බුදුරජතෙම ලොව පහළවිය. 
# ´´එකල්හි උතුම්වූ බාරාණසී නගරයෙහි මිනිසුනට අධිපතිවූ ´´කිකී´´ නම් කසීරජ තෙමේ මහර්ෂීවූ බුදුරජුන්ගේ උපස්ථායකයා විය. 
# ´´එම (කිකී) රජුගේ සිව්වන දියණියවූ ´´භික්ඛුදාසී´´ යයි ප්‍රකටවූ මම බුදුරජාණන් වහන්සේගේ ධර්‍මය අසා පැවිද්ද රුචි කෙළෙමි.

# ´´(පැවිදිවීම සඳහා) පිය (රජ) තෙම අපට නොඅනුදත්තේය. එකල්හි රාජ කන්‍යාවන්වූ, සුවසේ වැඩුණාවූ, බුදේ‍ධාපස්ථානයෙහි ඇලුනාවූ, ප්‍රමුදිතවූ (එම රජුගේ) දූවරුන් සත්දෙනෙකුවූ අපි ගිහිගෙහිදීම විසිදහසක් හවුරුදු කෞමාරී බ්‍රහ්මචර්‍ය්‍යයෙහි හැසුරුණෙමු. 
# ´´(එම සත් දෙන නම්) සමණීද, සමණ ගුත්තාද, භික්‍ෂුණීද, භික්ඛුදායිකාද, ධම්මාද, සුධම්මාද, සත්වැනිවූ සංඝදාසියද, 
# ´´(එම සත්දෙන පිළිවෙළින් අප බුදුන් කල) ඛෙමාවද, උප්පලවණ්ණාවද, පටාචාරාවද, එසේම මමද, කිසා ගෝතමීයද, ධම්මදින්නාවද, සත්වැනිවූ විශාඛාවද වේ. 
# ´´මැනවින් කරණ ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´දැන් මේ අන්තිම භවයෙහි උතුම්වූ ගිරිව්‍රජ නගරයෙහි සමෘද්ධවූ සිටු කුලයක උපන්නී යම් කලෙක්හි මම යොවුන් බැව්හි සිටියාද, 
# ´´(එකල්හි) මම වධය පිණිස ගෙනයන්නාවූ සොරෙකු දැක ඔහු කෙරෙහි ඇළුනී, මාගේ පිය තෙම ඔහු එම වධයෙන් දහසක් දීමෙන් මුදා 
# ´´පියතෙම මාගේ අදහස දැන ඕහට පාවා දුන්නේය. මම ඕහට ඉතා විශ්වාසවූ, දයාවත්වූ, හිතවත්වූ තැනැත්තියක් වීමි. 
# ´´දොස් සහිත අදහස් ඇති ඒ සතුරු තෙමේ මාගේ ආභරණ ලෝභයෙන් චොරප්පාත නම් පර්‍වතයට (මා) ගෙන ගොස් මැරීමට සිතීය. 
# ´´එකල්හි මැනවින් කළ ඇඳිලි ඇති මම ස්වාමියාට වැඳ ජීවිතය රකින්නී මෙම වචනය කීමි.

# ´´එනු මැනව. මම පැදකුණු කොට වැළඳ ගන්නෙමි. දැන් නැවත ඔබගේ හා මගේ සඞ්ගමයක් නැත.´´

# ´´සියළු තන්හි පුරිස් තෙමේම නුවණැත්තේ නොවේ. වහා අභිවෘද්ධිය ගැන සිතා බලනසුළු ස්ත්‍රීහුද නුවණැත්තියෝ වෙති. 
# ´´සැබවින්ම කෙටියෙන්ද, ඉක්මණින්ද, නොයෙක් අදහස් සිතීමි. ඌර්‍ණමෘගයා මෙන් එකල්හි මම මෙසේ සතුරා නැසීමි. 
# ´´යමෙක් තෙම මතුවූ අදහස වහා අවබෝධ නොකෙරේ නම්, ඒ මඳ නුවණැත්තේ පර්‍වතයෙහි (වැටි නැසෙන) සොරෙකු මෙන් නැසේ. 
# ´´යමෙක් තෙම මතුවූ අදහස වහා අවබෝධ කෙරේ නම්, එකල්හි මම සතුරාගෙන් මිදුනාක් මෙන් (හෙතෙම) සතුරු අවහිරයෙන් මිදේ. 
# ´´එකල්හි එම සතුරා පර්‍වත ප්‍රපාතයෙහි හෙළා මම සෙතාම්බර (සුදු වස්ත්‍ර හඳිනා) නිවටුන්ගේ සමීපයට එළඹ පැවිදිවීමි. 
# ´´එකල්හි මාගේ කෙස් සර්‍වාකාරයෙන් අඬුවෙන් උදුරා පැවිදිකරවා නිරතුරුව ස්වකීය ධර්‍මය කීහ.

# ´´ඉක්බිති මම එය ඉගෙන හුදකලාව හිඳ, එම ධර්‍මය මෙනෙහි කෙළෙමි. සුනඛයෙක් තෙම කපා දමන ලද මිනිස් අතක් (ඩැහැ)ගෙනවුත් මාගේ සමිපයෙහි හෙලා ඉවත්ව ගියේය. ඒ පණුවන්ගෙන් ගැවසී ගත් අත දැක (පුලවක) පණුවන්ය යන නිමිත්ත ලැබීමි. 
# ´´කලකිරුනී ඉන් නැගිට සහ ධාර්මිකයන්ගෙන් (එය) විචාළෙමි. ´එම කරුණ ශාක්‍ය (පුත්‍ර) භික්‍ෂූහු දනිත්´ යයි ඔවුහු කීහ. 
# ´´ඒ මම එම කරුණ බුදුසව්වන් වෙත එළඹ විචාරන්නෙමැ´´ යි (කීමි.) ඔවුහු මා ගෙන බුද්‍ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ගියාහුය. 
# ´´ඒ ලෝකනායක තෙමේ ස්කන්‍ධ, ආයතන ධාතූහු අශුභයහ, අනිත්‍යයහ, දුඃඛයහ, අනාත්මයහ´´ යි මට දහම් දෙසූහ.

# ´´මම උන්වහන්සේගේ ධර්‍මය අසා දහම් ඇස පිරිසිදු කෙළෙමි. ඉක්බිති දැනගන්නා ලද සදහම් ඇත්තී, පැවිද්දද, උපසම්පදාවද ඉල්වීමි. එකල්හි ලෝකනායක තෙමේ, ´´සොඳුරුවූ තැනැත්තිය මෙහි එව´´ යි වදාළ සේක. එකල්හි මම උපසපන්වූවා ජලය ස්වල්පයක් දුටිමි. 
# ´´මම (ඒ ජලය ගෙන) පා සේදීමෙන් ඇතිවීම සහිතවූ විනාශය දැක එකල්හි සියළු සංස්කාරයෝ එසේයයි සිතීමි. 
# ´´ඉක්බිති (ආශ්‍රවයන් තෘෂ්ණාදීන්ගේ වශයෙන්) නොගෙන මාගේ සිත සර්‍වාකාරයෙන් මිදුනේය. 
# ´´බුදුසසුන කරන්නීවු (මම) ඎද්‍ධීන්හිද, දිව්‍ය ශ්‍රොත ධාතු ඥානයෙහිද වශීවෙමි. පර සිත් දනිමි. 
# ´´පෙර විසූ කඳ පිළිවෙළ දනිමි. දිවැස පිරිසිදු කරන ලදී. සියලු ආශ්‍රවයන් ක්‍ෂයකොට අති නිර්‍මලවූවා, පිරිසිදුවූවාද වෙමි. 
# ´´මා විසින් ශාස්තෘ තෙමේ ඇසුරු කරන ලදී. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. මහත්වූ බර බහා තබන ලදී. භව තෘෂ්ණාව නසන ලදී. 
# ´´යමක් පිණිස ගිහිගෙන් නික්ම අනගාර්‍ය්‍ය නම්වූ සසුනෙහි පැවිදිවූවාද, මා විසින් සියළු සංයෝජනයන් ක්‍ෂයකිරීම නම්වූ නොවිසිරෙන ලද එම අර්ත්‍ථය ඇත්තේද වෙමි. 
# ´´බුද්‍ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සසුනෙහි අර්ත්‍ථ, ධර්‍ම, නිරුක්තීන්හිද, එසේම ප්‍රතිභානයෙහිද, මාගේ ඤාණය මහත්ය, පිරිසිදුය.

# ´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.

# කිසා ගෝතමී අපදානය. 
# ´´මින් (පෙර) සියක් දහස්වන කපෙහි ලෝකනායකවූ, සියළු ධර්‍මයන්හි පරතෙරට ගිය පදුමුත්තර නම් බුදුරජතෙම ලොව පහලවිය. 
# ´´එකල්හි මම හංසවතී නගරයෙහි එක්තරා කුල ගෙයක උපන්නී, තථාගතයන් වෙත් එළඹ උන්වහන්සේ සරණකොට ගියෙමි. 
# ´´මිහිරිවූ, උත්තම ආස්වාදයවූ, සසර දුක සන්සිඳීමේ සුවය එළවන්නාවූ, චතුස්සත්‍යයෙන් යුක්තවූ උන්වහන්සේගේ ධර්‍මයද ඇසීමි. 
# ´´එකල්හි පුරුෂෝත්තමවූ (සර්‍වඥ) වීර තෙමේ රූක්‍ෂ චීවරධාරීවූ භික්‍ෂුණියක් එතදග්‍රස්ථානයෙහි තබමින් වර්‍ණනය කළ සේක. 
# ´´භික්‍ෂුණියගේ ගුණය අසා අපමණ ප්‍රීතිය උපදවා බුදුරජුන්හට ශක්තියවූ පරිද්දෙන්, බලයවූ පරිද්දෙන් සත්කාරකොට, 
# ´´එම බුදුරජුන් (පාමුල) වැටී එකී තනතුර පැතීමි. (ලෝක)නායකවූ බුදුරජතෙමේ එකල්හි (මතු එම) තනතුර ලැබීමේ හේතුවෙන් අනුමෙවෙනි බණ දෙසූහ. 
# ´´මින් (මතු) සියක් දහස්වන කපෙහි ඔක්කාක වංශයෙහි උපදින්නාවූ, නම්න් ගෞතම නම්වූ ශාස්තෘතෙමේ ලොව පහළ වන්නේය. 
# ´´(තී) උන්වහන්සේගේ දහම්හි දායාදීවූ, ධර්‍මයෙන් නිර්මිත කළාක් වැනිවූ, ඖරස බුද්‍ධපුත්‍රීවූ, නම්න් ගෞතමී නම්වූ ශාස්තෘ ශ්‍රාවිකාවක් වන්නෙහිය. 
# ´´එය අසා සතුටුව, එකල්හි දිවි ඇතිතාක් මෙත් සිත් ඇත්තී, (සිව්) පසයෙන් පස්මරුන් දිනූ බුදුරජුන්හට පරිවරණය කෙළෙමි. 
# ´´මැනවින් කරන ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්තප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´කථා කරන්නවුන් අතරින් ශ්‍රේෂ්ඨවූ, බ්‍රාහ්මණ වංශයෙහිවූ, මහත් යසස් ඇති, නම්න් කාශ්‍යප නම්වූ බුදුරජතෙම ලොව පහළවිය. 
# ´´එකල්හි උතුම්වූ බාරාණසී නගරයෙහි මිනිසුනට අධිපතිවූ ´´කිකී´´ නම් කසීරජතෙමේ මහර්ෂීවූ බුදුරජුන්ගේ උපස්ථායකයා විය. 
# ´´එම රජුගේ ´ධම්මා´ නම්න් ප්‍රකටවූ පස්වෙනි දුව වීමි. උතුම්වූ බුදුරජාණන් වහන්සේගේ ධර්‍මය අසා පැවිද්ද රුචි කෙළෙමි.

# ´´පිය රජ තෙම අපට නොඅනුදත්තේය. එකල්හි රාජ කන්‍යාවන්වූ, සුවසේ වැඩුනාවූ, බුදේ‍ධාපස්ථානයෙහි ඇළුනාවූ, ප්‍රමුදිතවූ දූවරුන් සත්දෙනෙකු වූ අපි ගිහිගෙහිම විසි දහසක් හවුරුදු කුමරි බඹසරෙහි හැසිරුනෙමු. 
# ´´සමණීද, සමණ ගුත්තාද, භික්ඛුණීද, භික්ඛුදායිකාද, ධම්මාද, සුධම්මාද, සත්වැනිවූ සංඝ දාසියද, 
# ´´ඛොමාවද, උප්පලවණ්ණාවද, පටාචාරාවද, කුණ්ඩලාවද, මමද, ධම්මදින්නාවද, සත්වැනිවූ විශාඛාවද වේ. 
# ´´මැනවින් කරන ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්තප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´දැන් මේ අන්තිම භවයෙහි මම දුගීවූ, ධනය නැත්තාවූ, නටුවාවූ සිටු කුලයක උපන්නී, ධනවත් කුලයකට ගියාය. 
# ´´ස්වාමියා හැර සෙස්සෝ මට ධන නැත්තියයයි කියත්. යම් කලෙක්හි (මම) පුතෙකු සහිත වූයෙම් නම් එකල්හි සියල්ලන්ගේ දයාව ඇත්තියක් වීමි. 
# ´´යම් කලෙක්හි මාගේ එම ලදරු පුත් තෙමේ කොමල අවයව ඇත්තේ, සුවසේ වඩන ලද්දේ, සිය පණ මෙන් ලැදිවූයේ වේද, එකල්හි මරු වසඟයට ගියේය. 
# ´´ශෝකයෙන් පීඩිතවූවා, බැගෑවූ මුහුණු ඇත්තී, කඳුළු (පිරුණු) නෙත් ඇත්තී, හඬන මුඛ ඇත්තී, මම මළකුණ ගෙන විලාප කියමින් ඇවිදිමි. 
# ´´එකල්හි එක් අයෙකු විසින් දක්වන ලද්දීවූ (මම) බුදුරජුන් වෙත එළඹ, ´´පින්වතුන් වහන්ස, පුතු ජීවත් කරන්නාවූ බෙහෙත් දෙනු මැනවැ´´ යි කීමි. 
# ´´යම් ගෙයක මළ කෙණෙක් නොවෙත්නම් එම ගෙයින් අබ ඇටයක් ගෙන එව´´ යි හික්මවීමේ උපායයෛහි දක්‍ෂවූ බුදුරජතෙම කීහ. 
# ´´එකල්හි සැවතට ගොස් එබඳු ගෘහයක් නොලැබීමි. එබඳු ගෙයකින් (ගතයුතු) අබ ඇටයක් නම් කොයින්ද, එබැවින් මම සිහි ලැබුවා,

# ´´යමෙක් තෙම උදය-ව්‍යය නොදකිමින් හවුරුදු සීයක් ජීවත් වන්නේ නම් (එයට වඩා) උදය-ව්‍යය දක්නහුගේ එක දිනක් ජීවත්වීම ශ්‍රේෂ්ඨය.

# ´´නොඋපන් මග උපදවන්නාවූ තථාගත තෙම නොදෙසන ලද්දක් දෙසූහ. නොඋපදවන ලද්දක් ඉපදවූහ. 
# ´´මාර්‍ගය ගැන අවබෝධය ඇති, මාර්‍ගය දන්නාවූ, මාර්‍ගය දේශනා කරන්නාවූ නර ශ්‍රේෂ්ඨ තෙමේ, මාර්‍ගයෙහි දක්‍ෂවූ, නරදම් සැරීන් අතුරෙන් උතුම්වූ, ශාස්තෘවූ, 
# ´´මාර්‍ග දේශකවූ මහා කාරුණික තෙමේ එකල්හි දහම් දෙසති. කාම මඩෙහි ගිලුනු සත්ත්‍වයන් (ඉන්) ගොඩදමති. 
# ´´එකල්හි මම හංසවතී නගරයෙහි මනා රූ ඇත්තාවූ, ධනවත්වූ, දයාවත්වූ, ශ්‍රීමත්වූ රජදුවක් වූවා, 
# ´´ආනන්‍ද මහරජුගේ අතිශය ශෝභනවූ දුවණියද, පදුමුත්තර නමැත්තහුගේ (පදුමුත්තර බුදුරජුගේ) වෙන මව් දරු සොහොයුරියද වීමි.

# ´´එකල්හි එම තථාගත තෙම පිරිස් මැදදී දිවැස් ඇති භික්‍ෂුණියකට පසසමින් ඇය අග්‍රස්ථානයෙහි තැබූහ. 
# ´´මම එය අසා සතුටුවූවා, බුදුරජුන්හට දනක් දී, සම්බුදුරජුන් පුදා, දිවැස පැතීමි.

# ´´මැනවින් කරන ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්තප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.)

# ´´මැනවින් කරන ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්තප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´එම කර්‍මයාගේ බලයෙන් යම්-යම් තැනක උපදිම්ද, ඒ ඒ තන්හි (උත්පත්ති වශයෙන්) ගිය මට මහත්වූ ප්‍රදීපයෝ දැල්වෙත්.

# ´´ඔදවැඩි සිත් ඇත්තී, සොම්නස්වූවා පැවිද්ද රුචි කෙළෙමි. නොබෝ කලකින්ම රහත් බවට පැමිණියෙමි.

# ´´පෙර විසූ කඳ පිළිවෙළ දනිමි. දිවැස පිරිසිදු කරන ලදි. සියළු ආශ්‍රවයන් ක්‍ෂය කොට අති නිර්‍මල වූවා, පිරිසිදුවූවාද වෙමි. 
# ´´මා විසින් ශාස්තෘ තෙමේ ඇසුරු කරන ලදී. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. මහත්වූ බර බහා තබන ලදී. භව තෘෂ්ණාව නසන ලදී. 
# ´´යමක් පිණිස ගිහිගෙන් නික්ම අනගාර්‍ය්‍ය නම්වූ සසුනෙහි පැවිදිවූවාද, මා විසින් සියළු සංයෝජනයන් ක්‍ෂය කිරීම නම්වූ පැමිණෙන ලද එම අර්ත්‍ථය ඇත්තේද වෙමි. 
# ´´ඉක්බිතිව මහා කාරුණිකවූ බුදුරජතෙම, ´´සකුලා තොමෝ දිවැස් ඇති භික්‍ෂුණීන් අතුරෙන් අග්‍රය´´ යි මා එතදග්‍රස්ථානයෙහි තැබූහ.

# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
මෙහි මේ අයුරින් ධම්මදින්නා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

# නන්‍දා ජනපද කල්‍යාණි අපදානය. 
# ´´මින් (පෙර) සියක් දහස්වන කපෙහි (කෙළෙස් සතුරන්) ජයගත්තාවූ, සියළු දහම්හි පරතෙරෙට ගිය පදුමුත්තර නම් බුදුරජතෙම ලොව පහළවිය. 
# ´´සව් සත්නට ඔවා දෙන්නාවූ, දහම් අවබෝධ කරවන්නාවූ, (සත්ත්‍වයන්) සසරින් එතර කරවන්නාවූ, දේශනාවෙහි දක්‍ෂවූ බුදුරජතෙමේ බොහෝ ජන සමූහයා සසරින් එතර කරවූහ. 
# ´´සව්සත්නට අනුකම්පා කරන්නාවූ, කාරුණිකවූ, හිත කැමැත්තාවූ (එම බුදුරජතෙම) පැමිණි සියලු තීර්ත්‍ථකයන් පන්සිල්හි පිහිටවූ සේක. 
# ´´මෙසේ (එම කාලය) තීර්ත්‍ථකයන්ගෙන් හිස්වූ, නිරවුල්වූ, තාදී ගුණ ඇති, (පඤ්චවශිතාවන්ගෙන්) වශීභූතවූ, රහතුන්ගෙන් විසිතුරු වූවක් විය. 
# ´´එම බුදුරජතෙම අටපණස් රියනක් උසය. රන් ඇගෑවක් මෙන් බබලන්නේය. දෙතිස් මහපුරිස් ලකුණු ඇත්තේය. 
# ´´එකල්හි විසි ලක්‍ෂයක් ආයුෂ වෙයි. ඒතාක් කල් වැඩහුන්නාවූ මෙම සර්‍වඥතෙම බොහෝ ජන සමූහයා (සසරින්) එතෙර කළ සේක. 
# ´´එකල්හි මම හංසවතී නගරයෙහි නන් රුවනින් බබලන, මහත් සැපතින් යුත් සිටු කුලයක උපන්නී වීමි. 
# ´´එම මහාවීරවූ බුදුරජුන් වෙත එළඹ අමෘතවූ, උතුම් අස්වැසිලි දෙනසුළුවූ, පරමාර්ත්‍ථය මේයයි දන්වන්නාවූ දම් දෙසුම් ඇසීමි. 
# ´´එකල්හි සංඝයා සහිත බුදුරජුන් පවරා පැවිදි සිත් ඇත්තී, සියතින් උන්වහන්සේට මහදන් දී, 
# ´´සංඝයා සහිත ලෝකනායකවූ බුදුරජුන් පාමුල සිරසින් වැටී, ධ්‍යාන කරන්නාවූ භික්‍ෂුණීන් අතුරින් අග තනතුර පැතීමි. 
# ´´එකල්හි නොදැමුනවුන් දමනය කරන්නාවූ, තුන් ලොවට පිහිටවූ, (දුක් නැතිකිරීමෙහි) සමත්වූ, නර ශ්‍රේෂ්ඨවූ තථාගත තෙම, ´´ඒ මැනවින් පතන ලද්ද ලබන්නෙහිය´´ යි පැවසූ සේක. 
# ´´මින් මතු සියක් දහස්වන කපෙහි ඔක්කාක කුලයෙහි උපදින්නාවූ, නම්න් ගෞතම නම්වූ ශාස්තෘතෙමේ ලොව පහළවන්නාහ. 
# ´´උන්වහන්සේගේ ධර්‍මයෙහි දායාදිවූ, ධර්‍මයෙන් නිර්මිත කළාක් වැනිවූ, ඖරස (බුද්‍ධ) පුත්‍රී වූ, නම්න් ´´නන්දා´´ නම් ශාස්තෘ ශ්‍රාවිකාවක් වන්නෙහිය.´´ 
# ´´එකල්හි එය අසා ප්‍රමුදිතව, දිවි ඇතිතාක් විනායකවූ බුදුරජුන්හට මෙත් සිත් ඇත්තී (සිව්) පසයෙන් පරිවරණය කෙළෙමි. 
# ´´මැනවින් කරණ ලද එම කර්‍මය කරණකොටගෙනද, චිත්ත ප්‍රණිධීන් කරණකොටගෙනද, මම මිනිස් සිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියා (වෙමි.) 
# ´´මම ඉන් චුතවූවා යාම දෙව්ලොවට ගියෙමි. ඉන් තුෂිත දෙව්ලොවට ගියා, එතැනින් නිර්‍මාණරතියටද, ඉන් (චුතව) වශවර්ති (දිව්‍ය) පුරයටද (උත්පත්ති වශයෙන්) ගියෙමි. 
# ´´එම කර්‍මයාගේ බලයෙන් යම් යම් තැනක උපදිම් නම් ඒ ඒ තන්හි රජවරුන්ගේ මෙහෙසිබව කෙළෙමි. 
# ´´ඉන් චුතව මිනිසත් බැව්හි සක්විති රජවරුන්ගේද, මණ්ඩලීක රජවරුන්ගේද මෙහෙසිබව කෙළෙමි.

# ´´එකල්හිද මම ඔහුගේ සමවූ අදහස් ඇති බැමිණිය වීමි. කිසියම් දිනෙක්හි එම බමුණු තෙමේ ජන සමුහයා මැද හුන්නාවූ අමාපිය (නිවන) දෙසන්නාවූ බුදුරජුන් වෙත එළඹියේය. දහම් අසා තුටුවූයේ එකම සළුව පිදීය.

# ´´එකල්හි මම අතිශයින් පිනා ගියා, ඇඳිලි බැඳ, ´´හිමියෙනි, තාදී ගුණ ඇති බුදුරජුන්හට සළුව මැනවින් දෙන ලදය´´ යි අනුමෝදන් වීමි. 
# ´´සුව ඇත්තේ, සැරසුනේව, කුදු මහත් භවයෙහි සැරිසරන්නේ, රම්‍යවූ බරණැස් නගරයෙහි මිහිපතිවූ රජෙක් විය. 
# ´´එකල්හි මම ඔහුගේ අන්තඃපුරයට උතුම්වූ මෙහෙසියවූවා, පූර්‍වස්නේහය කරණකොටගෙන එම ස්වාමියාහට ඉතා දයා ඇත්තී වීමි. 
# ´´හෙතෙම පිඬු පිණිස හැසිරෙන්නාවූ පසේ බුදුවරයන් අට නමක් දැක ප්‍රමුදිතව මාහැඟි පිණ්ඩපාතය දී, 
# ´´යලි පවරා රුවන් මඩුවක් කොට රියන් සියයක්වූ ස්වර්‍ණමය කුඩයක් කඹුරන් ලවා කරවන ලදී.

# ´´එකල්හි මම කාසිරාජයන් හා එම දනද එකතුවම දුනිමි. නැවත මම බරණැස් නගරයෙහි දොරගමක උපනිමි.

# ´´මහා වීරයන් වහන්ස, අර්ත්‍ථ, ධර්‍ම නිරුක්තීන්හිද, එසේම ප්‍රතිභාණයෙහිද, මාගේ ඥානය ඔබ වහන්සේගේ සමිපයෙහිදී උපන්නේය. 
# ´´ලොකනාථවූ බුදුවරයන්ගේ එක්වීම (කෙබඳුදැයි) ඔබ වහන්සේ විසින් මැනවින් දක්වන ලදි. මහා මුනීන් වහන්ස, ඔබ වහන්සේ සඳහා මා (කළ) පින්කම් බොහෝය. 
# ´´මුනීන්‍ද්‍රයන් වහන්ස, මා විසින් යම් කුශලයක් (කරන ලද නම්) මාගේ (ඒ) පෙර කර්‍මය සිහි කරන්න. මහා වීරයන් වහන්ස, මා විසින් ඔබ පිණිසම පින් රැස් කරන ලදී. 
# ´´වීරයන් වහන්ස, නුසුදුසු තැන් අත්හැර, අනාචාරයෙන් වැළකී, ඔබ වහන්සේ පිණිස මාගේ ජීවිතය පරිත්‍යාග කරන ලදි. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට මා (පරිවාර) ස්ත්‍රිය පිණිස දුන්නේය. එහිලා මම නොසතුටු නොවෙමි. මහා මුනීන් වහන්ස, (එය) ඔබ වහන්සේ සඳහාය. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට මා උපකාර පිණිස දුන්නේය. එහිලා මම නොසතුටු නොවෙමි. මහා මුනීන් වහන්ස, (එය) ඔබ වහන්සේ සඳහාය. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට මා භොජනය (සැපයීම) පිණිස දුන්නේය. එහිලා මම නොසතුටු නොවෙමි. මහා මුනීන් වහන්ස, (එය) ඔබ වහන්සේ සඳහාය. 
# ´´නොයෙක් කෙළ දහස් ගණන් ජීවිත පරිත්‍යාග කෙළෙමි. භයින් මිදීම ඇත්තියක් කරන්නෙමැයි (සිතා) මාගේ ජීවිතය හැරියෙමි. 
# ´´ශරීරාවයන්හිවූ නානාවිධ අලඞ්කාර වස්ත්‍රද, බොහෝ ස්ත්‍රී ආභරණද, මහා මුනීන් වහන්ස! ඔබ වහන්සේ සඳහා නොසඟවමි. 
# ´´මහා මුනීන් වහන්ස, ධන ධාන්‍ය පරිත්‍යාගයද, ගම්, නියම් ගම්, කෙත්, පුත් හා දූවරුන්ද පරිත්‍යාග කරන ලදී. 
# ´´මහා වීරයන් වහන්ස, අනන්තවූ ඇත්, අස්, ගව යන මොවුහුද, පිරිවර දාසියෝද, ඔබ වහන්සේ සඳහා පරිත්‍යාග කරන ලදී. 
# ´´යාචකයනට දනක් දෙමැ´´ යි යමක් මා සමග මන්ත්‍රණය කළේද, උතුම් දනක් දෙන්නහුට මාගේ දොම්නසක් නොදකිමි. 
# ´´මහා වීරයන් වහන්ස, බහු විධවූ සසරෙහි නන් වැදෑරුම්වූ බොහෝ දුක අපමණව ඔබ වහන්සේ උදෙසා අනුභව කරන ලදී. 
# ´´සැපයට පැමිණියා සතුටු නොවෙමි. දුක් පැමිණි කල්හි දොම්නස් ඇත්තීද නොවෙමි. මහා මුනීන් වහන්ස, ඔබ වහන්සේ පිණිස හැම තන්හි සැප දුක්ද දෙකෙහි සමව උපෙක්‍ෂක වෙමි. 
# ´´සම්බුදුරජතෙම අනුගමනය කළ මගින් යම් ධර්‍මයක් පැතීද, මහාමුනිතෙමේ සැපද දුක් අනුභවකොට (ඒ) සම්‍යක් සම්බෝධියට පැමිණි සේක. 
# ´´බ්‍රහ්මදේවවූ, ලෝකනායකවූ ගෞතම බුදුරජුන්ගේද, ඒ බොහෝ සෙසු බුදුවරයන්ගේද හමුවීම මට විය. 
# ´´මහා මුනීන් වහන්ස, ඔබ පිණිස මාගේ බොහෝ පින්කම් (විය.) මම බුදු දහම සොයන්නාවූ ඔබගේ පරිචාරිකාව (වීමි.) 
# ´´සාරාසංඛ්‍ය කල්ප ලක්‍ෂයකින් පෙර ලෝකනායකවූ දිවකුරු බුදුරජතෙම ලොව පහළවිය. 
# ´´පසල් දනව්වෙහි සොම්නස්වූ මිනිස්සු බුදුරජුන් පවරා, උන්වහන්සේ වැඩමවන මග පිරිසිදු කෙරෙත්. 
# ´´එකල්හි හෙතෙම සුමේධ නම් බමුණෙක් විය. සියල්ල දක්නාවූ බුදුරජුන්හට දීර්‍ඝ මාර්‍ගයද පිළියෙළ කළේය. 
# ´´එසමයෙහි මම බමුණු කුලයෙහි උපන්, නම්න් සුමිත්තා නම්වූ කන්‍යාවක් වීමි. (මෙම) රැස්වීමට(ද) එළඹියෙමි. 
# ´´ශාස්තෲන් වහන්සේට පිදීම පිණිස උපුල් මල් අට මිටක් ගෙන (එද්දී) ජන සමූහයා මැදෙහි වැහැරි සිවුර මත හුන්නාවූ, අතිශයින් සිත්කලුවූ, මනහරවූ, ඔදවැඩුනු සිත් ඇති ඎෂිවරයා දුටිමි. දැක එකල්හි මාගේ ජීවිතය සඵලයයි සිතීමි.

# ´´එකල්හි ඎෂිහුගේ ඒ උත්සාහය සඵලයයි දුටිමි. මාගේ පූර්‍ව කර්‍ම (බල)යෙන් සම්බුදුරජුන් කෙරෙහි සිතද පැහැදුනේය. 
# ´´ඔදවැඩි සිත් ඇති ඎෂිවරයා කෙරෙහි බෙහෙවින් සිත පැහැදවීමී. මහර්ෂී බුදුරජාණන් වහන්ස, ඔබ වහන්සේට දියයුතු අනිකක් නොදකිමි. (උපුල්) මල් දෙමි. 
# (මින් මල්) පස් මිටක් ඔබවහන්සේට වේවා. මාගේ ඎෂිවරයා කෙරෙහි තෙමිටක් වේවා. බුදුරජාණන් වහන්ස, ඔබවහන්සේ (ප්‍රත්‍යක්‍ෂ කළ ධර්‍මය) අවබෝධය පිණිස එම ඎෂිවරයා සමග (මාගේ අදහස්) සමවේවා.

# ´´එම ඎෂිතෙම මල් (තෙමිට) ගෙන වඩින්නාවූ මහත් යසස් ඇති බුදුරජුන්ට ජන සමූහයා මැදදී සර්‍වඥතාඥානාවබෝධය පිණිස පිදීය. 
# ´´මහාවීරවූ දීපඞ්කර බුදුරජතෙමේ ජන සමූහයා මැදදී ඔද වැඩි සිත් ඇති තවුසා දැක (මෙසේ) පැවැසූහ. 
# ´´මින් අපරිමෙය්‍ය කල්පයෙහි හෙවත් සාරාසංඛ්‍ය කල්ප ලක්‍ෂයකින් මතුයෙහි මහාමුනිවූ දීපඞ්කර බුදුරජතෙමේ මාගේ කර්‍මයද ඎජුබවද පැවසූහ. 
# ´´මහා ඎෂිවරය, ඔබ පිණිස (කළ) මෙම කර්‍මය කරණකොටගෙන (ඇය ඔබ හා) සම කර්‍ම ඇති, සමව කටයුතු කරන සුළු තැනැත්තියක් වන්නීය. ප්‍රියවූ තැනැත්තියක්ද වන්නීය. 
# ´´ඔහුගේ ධර්‍මයෙහි කොටස්කාරියක්වූ, මනා දැකුම් ඇති, ඉතා ප්‍රියවූ, මන වඩන්නාවූ, ප්‍රිය වචන කියන්නාවූ, ප්‍රිය ගති ඇති ස්ත්‍රියක්ද වන්නීය. 
# ´´ස්වාමියාගේ බඩු පෙට්ටියක් යම්සේ රකිත්ද, එසේම මෝතොමෝ කුසල් දහම් රකින්නේය. 
# ´´ඒ (අනුගාමික) ජනයෝද ඔහුට අනුකම්පා කෙරෙත්. (හෙතෙම) පෙරුම් පුරන්නේය. මැදිරිය (කූඩුව) බිඳ (නිදහස්වූ) සිංහයෙකු මෙන් සම්‍යක් සම්බෝධියට පැමිණෙන්නේය. 
# ´´මින් අපරිමෙයවූ කපෙහි බුදුරජතෙම මට යමක් ප්‍රකාශ කෙළේද, එම වචනයට අනුව සතුටුවන්නී මම මෙසේ කරනසුළු තැනැත්තියක් වීමි. 
# ´´එහිදී මැනවින් කරන ලද එම කර්‍මයට සිත පැහැදවීමි. දෙවි මිනිස් දෙගතියෙහි අපමණ (සැප) අනුභවකොට, 
# ´´මම දෙවියන් අතුරෙහිද, මිනිසුන් අතුරෙහිද, (මෙසේ) සැප දුක් අනුභවකොට අන්තිම භවය පැමිණි කල්හි ශාක්‍ය කුලයෙහි උපනිමි. 
# ´´මනා රූ ඇති, භෝග සම්පත් ඇති, කීර්තියෙන් හා ශීලයෙන් යුක්තවූ, යලි සියළු (පිරිපුන්) අඟ පසඟ සම්පත් ඇති කුලයන්හි අතිශයින් සත්කාර කරන ලද තැනැත්තෙක් වෙමි. 
# ´´ලාභ, කීර්තිය, සත්කාරය, (අට) ලෝ දහමින් ගැටීමය යන මේවා පිළිබඳ දුක් සහිත සිතක් නැත. බියක් නොමැතිව වසමි.

# ´´සසරෙහි කලකිරී, අනගාර්‍ය්‍ය නම්වූ සසසුනෙහි පැවිදිවීමි. දහසක් පිරිවර සහිතව පැවිදිව කිසිත් (වස්තුවක්) නැති තැනැත්තියක් වුවාය. 
# ´´මෙසේ ගිහිගෙය අතහැර දමා අනගාර්‍ය්‍ය නම්වූ සසුනෙහි පැවිදිවීමි. (පැවිදිව) අඩමසක් නොපැමිණි කල්හිම චතුස්සත්‍යාවබෝධයට පැමිණියෙමි. 
# ´´චීවරද, පිණ්ඩපාතද, ගිලානප්‍රත්‍යයද, සේනාසනද (යන මේ සිව්පසය) සයුරට රැළ (පමුණුවන්නාක්) මෙන් බොහෝ දෙන නානාවිධව පමුණුවත්. 
# ´´මා විසින් ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. (මා විසින්) පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇතියහ. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ටවිමොක්‍ෂයෝද, ෂඩභිඥාවෝද ශාක්‍ෂාත් කරන ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
# ´´මෙසේ නන්වැදෑරුම්වූ දුක්ද, නන්වැදෑරුම්වූ සම්පත්ද (විඳ) රහත් බවට පැමිණියා සියළු සම්පත් ලබමි. 
# ´´යම් ස්ත්‍රියක් තොමෝ බුදුරජුන්ට පින් පිණිස ස්වකීය ආත්මය දේද, (ඔහුට) සහාය සම්පත්තිහු වෙත්. අසංස්කෘතවූ නිර්‍වාණපදයද වේ. 
# ´´අතීතවූද, වර්‍තමානවූද, අනාගතවූද, මාගේ සියළු කර්‍මය ක්‍ෂය විය. බුදුරජාණන් වහන්ස, ඔබ වහන්සේගේ සිරිපා වඳිමි. 
මෙහි මේ අයුරින් යශෝධරා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

# යශෝධරා ප්‍රමුඛ දසදහසක් භික්‍ෂුණීන්ගේ අපදානය. 
# ´´සාරාසංඛ්‍ය කල්ප ලක්‍ෂයකින් පෙර ලෝකනායකවූ දීපඞ්කර නම් බුදුරජතෙම ලොව පහළවිය. 

# ´´මහාවීරවූ දීපඞ්කර බුදුරජතෙම සුමෙධ සුමිත්ත (යුවල) දෙවියන් සහිත ලෝකයෙහි හැසිරෙන්නාහු, දෙවි මිනිස් සුව දක්නාහු, සමාන සැප දුක් ඇතියවුන් වන බව පැවසූහ. ඔවුන්ගේ (අනාගත තත්ත්‍වය) පවසන කල්හි එම රැස්වීමට අපි(දු) එළඹ,

# ´´මතු එක්වීමෙහිදී (ඔබ) අප සියල්ලන්ගේ හිමියා වනු මැනව. අපි සියල්ලෝම ඔබගේ ප්‍රිය වචන කියන්නාවු, මන වඩන්නාවූ ස්ත්‍රීහු (වෙමු) යි (පැතීමු.) 
# ´´දාන සීලමයවූ සියළු කුසල්ද, භාවනාද මැනවින් වඩන ලදී. බොහෝ කාලයක් මුළුල්ලෙහි අපගේ සියල්ල බුදුරජාණන් වහන්සේට පරිත්‍යාග කරන ලදී. 
# ´´සුවඳ විලවුන්ද, මල්ද, රුවන්මුවා පහන්ද (යන) පතන ලද යම් කිසිවක් නම් ඒ සියල්ල බුදුරජාණන් වහන්සේට පරිත්‍යාග කරන ලදී. 
# ´´අනිකුත් කරන ලද කටයුතුද, මනුෂ්‍ය පරිභෝගයට සුදුසු දැයද යන අපගේ සියල්ල බොහෝ කාලයක් මුළුල්ලෙහි බුදුරජාණන් වහන්සේට පරිත්‍යාග කරන ලදී. 
# ´´සසර නොයෙක් ජාතීන්හි අප විසින් බොහෝ පින් කරන ලදී. කුදු මහත් භවයන්හි සැරිසරා සම්පත් අනුභවකොට, 
# ´´අන්තිම භවය පැමිණි කල්හි ශාක්‍යපුත්‍ර (ගෞතම)යන් වහන්සේගේ මැදුරෙහි නොයෙක් කුලයන්හි උපන්නාවූ, කමනීය වර්‍ණ ඇත්තාවූ අප්සරාවෝ වීමු. 
# ´´ලාභයෙන්ද, යසසින්ද අගතැන් පැමිණියාහු, සියල්ලන් විසින් සත්කාර කරන ලද්දාහු, හැම කල්හි ආහාර පාන ලාභී වූවාහු, අපි (සියල්ලන් විසින්) බුහුමන් කරන ලද්දියෝද වීමු. 
# ´´ගිහිගෙය අතහැර අනගාර්‍ය්‍ය නම්වූ සසුනෙහි පැවිදිවූවාහු, අඩ මසක් නොපැමිණි කල්හිම (අපි) සියල්ලෝ නිවනට පැමිණියෝ වෙමු. 
# ´´ආහාරපානද, වස්ත්‍ර හා සේනාසනද ලබන්නියෝ වූවාහු, සියළු ප්‍රත්‍යයෝ (තමන්) වෙතට පැමිණෙත්. හැම කල්හි (සියල්ලන් විසින්) සත්කාර කරන ලද්දියෝද, පූජා කරන ලද්දියෝද (වෙමු.) 
# ´´අපගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ දවන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිත වූවාහු වෙසෙමු. 
# ´´බුද්‍ධශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට ඒකාන්තයෙන් අපගේ පැමිණීමක් විය. (අප විසින්) පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇතියහ. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
# ´´ප්‍රතිසම්භිදාවෝ සිව්ිපිළිසිඹියාවෝද, අෂ්ට විමෝක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
මෙහි මේ අයුරින් යශොදරා ප්‍රමුඛවූ දසදහසක් භික්‍ෂුණින් වහන්සේ බුදුරජුන් හමුවෙහි මෙම ගාථාවන් වදාළ සේක.

# යශොධරා ප්‍රමුඛ අටළොස් දහසක් භික්‍ෂුණින්ගේ අපදානය. 
# ´´ශාක්‍ය කුලයෙහි උපන්නාවූ යශොධරා ප්‍රමුඛවූ අටළොස් දහසක් භික්‍ෂුණීහු සම්බුදුරජුන් වෙත එළඹියාහුය. 
# ´´අටළොස් දහසක්වූ ඒ සියළු භික්‍ෂුණීහු මහත් ඎද්‍ධි ඇත්තාහු වෙත්. (ස්වකීය) බලය තත්වූ පරිදි ආරොචනය කරමින් බුදුරජාණන් වහන්සේගේ සිරිපා වඳිත්. 
# ´´මහා මුනීන් වහන්ස, (අපගේ) ජාතියද, ජරාවද, ව්‍යාධියද, මරණයද ක්‍ෂය විය. ලෝකනායකයන් වහන්ස, ආශ්‍රව රහිතවූ අමෘතවූ ශාන්ත පදයට යමු. 
# ´´මහා මුනීන් වහන්ස, ඉදින් පෙර (අප) සියල්ලන්ගේ වැරදිමක් ඇත්තේ වි නම්, බුදුරජාණන් වහන්ස, එම අපරාධය නොදන්නියෝ (වෙමු.) අපගේ (එම අපරාධයට) ක්‍ෂමා වනු මැනවි.´´ 
# ´´මාගේ සසුන කරන්නාවූ (තෙපි) ඎද්‍ධියද දක්වව්. සියළු පිරිසගේ යම්තාක් සැක (වෙත් නම්) එයද සිඳිව්.´´ 
# ´´මහා වීරයන් වහන්ස, යශෝධරා තෙමෝ මන වඩන්නීය. පියවූ දැකුම් ඇත්තීය. මහා වීරයන් වහන්ස, සියළු භික්‍ෂුණීහු ගිහිගෙහිදී ඔබ වහන්සේගේ (පරිවාර) ස්ත්‍රීහුයි. 
# ´´වීරයන් වහන්ස, සානූකෙළක් ස්ත්‍රීන් අතුරෙන් ඒ අපි ගිහිගෙයි සියල්ලනට අධිපතිවූ ප්‍රමුඛ තැනැත්තියෝ වෙමු. 
# ´´රූපාචාර ගුණයන්ගෙන් යුක්තවූ, යෞවන බැව්හි සිටියාවූ, ප්‍රිය වචන කියන්නාවූ, මිනිස් දෙව් දූවරුන් වැනි සියල්ලෝ (අපට) ගරු කෙරෙත්. 
# ´´එකල්හි ශාක්‍ය වංශයෙහි උපන්නාවූ යශොධරා ප්‍රමුඛ අටළොස් දහසක්වූ සියල්ලෝ (හැමදෙනාහට) ප්‍රමුඛයහ. ඊශ්වරයහ. 
# ´´මහා මුනීන් වහන්ස. කාමධාතුව ඉක්මවූ, රූප ධාතුවෙහි සිටියවුන් අතුරෙනුදු රූපයෙන් අප හා සමානවූ තැනැත්තියක් නැත.

# ´´මහා මුණීන් වහන්ස, ඎද්ධීන්හිද, දිව්‍ය ශ්‍රෝත ධාතු ඥානයෙහිද වශී වෙමු. චෙතො පරියඤාණයාගේද වශී වෙමු. 
# ´´පෙර විසූ කඳ පිළිවෙළ දනිමු. දිවැස පිරිසිදු කරන ලදී. සියළු ආශ්‍රවයෝ ක්‍ෂය කරන ලදහ. දැන් නැවත ඉපදිමක් නැත. 
# ´´මහා වීරයන් වහන්ස, අර්ත්‍ථ, ධර්‍ම, නිරුක්තීන්හිද, එසේම ප්‍රතිභාණයෙහිද, අපගේ ඤාණය ඔබ වහන්සේ සමීපයෙහිදී උපන්නේය.

# ´´මුනීන්ද්‍රයන් වහන්ස, අප විසින් පෙර කුශලයක් කරන ලද නම් (එය ඔබ) සිහි කෙරෙහිය. මහා වීරයන් වහන්ස, අප විසින් ඔබ පිණිස පින් රැස් කරන ලදී. 
# ´´නුසුදුසු තැන් හැර අනාචාරයෙන් වැළකීමු. මහා වීරයන් වහන්ස, ඔබ වහන්සේ පිණිස අපගේ ජීවිත පරිත්‍යාග කරන ලදී. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට අප (පරිවාර) ස්ත්‍රීන් පිණිස දුන්නේය. මහා මුනීන් වහන්ස, ඔබ වහන්සේ සඳහා එහිලා අපි නොසතුටු නොවෙමු. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට අප උපකාර පිණිස දුන්නේය. එහිලා අපි නොසතුටු නොවෙමු. මහා මුනීන් වහන්ස, (එය) ඔබ වහන්සේ සඳහාය. 
# ´´නොයෙක් කෙළ දහස් ගණන් අයට අප ආහාර සැපයීම පිණිස දුන්නේය. එහිලා අපි නොසතුටු නොවෙමු. මහා මුනීන් වහන්ස, (එය) ඔබ වහන්සේ සඳහාය. 
# ´´අපගේ නොයෙක් කෙළ දහස් ගණන් ජීවිත පරිත්‍යාග කෙළෙමු. භයින් මිදීම කරන්නෙමු´´ යයි එසේ දිවි හළෙමු. 
# ´´මහා මුනීන් වහන්ස, ශරීර අවයවයන්හිවූ නා නා විධ අලංකාර වස්ත්‍රද, බොහෝ ස්ත්‍රී භාණ්ඩද ඔබ වහන්සේ සඳහා නොසඟවමු. 
# ´´ මහා මුනීන් වහන්ස, ධන ධාන්‍ය පරිත්‍යාගයද, ගම්, නියම්ගම්, කෙත්, පුතුන් හා දූවරුන්ද, පරිත්‍යාග කරන ලදී. 
# ´´ මහා වීරයන් වහන්ස, අනන්තවූ ඇත්, අස්, ගව යන මොවුහුද, පිරිවර දාසියෝද, ඔබ වහන්සේ සඳහා පරිත්‍යාග කරන ලදී. 
# ´´යාචකයනට දනක් දෙමැයි අප සමග යම්සේ මන්ත්‍රණය කළේද, උතුම් දනක් දෙන්නහුට අපගේ දොම්නසක් නොදකිමු. 
# ´´මහා වීරයන් වහන්ස, බහුවිධවූ සසරෙහි නන් වැදෑරුම්වූ බොහෝ දුක් අපමණව ඔබ වහන්සේ උදෙසා අනුභව කරන ලදී. 
# ´´සැපතට පැමිණියාහු සතුටු නොවෙමු. දුක් පැමිණි කල්හි දොම්නස් ඇත්තියෝද නොවෙමු. මහා මුනීන් වහන්ස, ඔබ වහන්සේ පිණිස හැම තන්හි උපෙක්‍ෂා වීමු.

# ´´සාරා සංඛ්‍ය කල්ප ලක්‍ෂයකින් පෙර ලෝකනායකවූ දිවකුරු බුදු තෙමේ ලොව පහළවිය. 
# ´´පසල් දනව්වෙහි සොම්නස්වූ මිනිස්සු බුදුරජුන් පවරා උන්වහන්සේ වැඩමවන මග පිරිසිදු කෙරෙත්. 
# ´´එකල්හි හෙතෙම සුමේධ නම් බමුණෙක් විය. සියල්ල දක්නාවූ බුදුරජුන්හට දීර්ඝ මාර්‍ගයද පිළියෙළ කළේය. 
# ´´එකල්හි (අපි) සියල්ලෝ බමුණු කුලයෙහි උපන්නියෝ වීමු. එම සමාගමයට ගොඩ දිය හටගන්නාවූ මල් ගෙන ආවෙමු. 
# ´´එසමයෙහි මහත් යසස් ඇති, මහාවිරවූ එම දීපඞ්කර බුදුරජතෙම ඔදවැඩි සිත් ඇති තවුසා දක (මෙසේ) පැවසූසේක. 
# ´´ඔදවැඩි සිත් ඇති තවුසාහට ඔහුගේ මාර්‍ගය විමසන කල්හි පෘථිවිය සෙලවෙයි. නාද කෙරෙත්. දෙව්ලෝ සහිතව කම්පා වෙයි. 
# ´´දිව්‍ය කන්‍යාවෝද, මනුෂ්‍යයෝද, දෙවියන් සහිතවූ අපිදු පිදිය යුතු නා නා භාණ්ඩ පුදා පැතූහ. 
# ´´දීපඞ්කර නම්වූ බුදුරජතෙම ඔවුනට මෙසේ පැවසූහ. ´´අද යමෙක් තුටු කරන ලද ප්‍රාර්ත්‍ථනාවක් ඇද්ද, ඔවුහු හමුවන්නාහුය´´ (කියායි.) 
# ´´මින් අපරිමෙය්‍යවූ කල්පයෙහි හෙවත් සාරාසංඛ්‍ය කල්ප ලක්‍ෂයකින් පෙර බුදුරජතෙම අපට යමක් පැවසීද, එම වචනයට අනුව සතුටුවන්නාවූ අපි මෙසේ කරනසුළු තැනැත්තියෝ වීමු. 
# ´´එහිදී මැනවින් කරන ලද එම කර්‍මයට සිත පැහැදවූහ. දෙව් මිනිස් දෙගතියෙහි අපමණව (සම්පත්) අනුභවකොට, 
# ´´දෙවියන් අතුරෙහිද, මිනිසුන් අතුරෙහිද (මෙසේ) සැපදුක් අනුභවකොට අන්තිම භවය පැමිණි කල්හි ශාක්‍ය කුලයෙහි උපනිමු. 
# ´´රූමත්වූ, භෝග සම්පත් ඇති, කීර්තියෙන් හා සීලයෙන් යුක්තවූ, යලි සියළු (පිරිපුන්) අඟ පසඟ සම්පත් ඇති, කුලයෙහි අතිශයින් සත්කාර කරන ලද තැනැත්තියෝ වෙමු. 
# ´´ලාභය, කීර්තිය, සත්කාරය, අටලෝ දහමින් ගැටීමය (යන මේවා පිළිබඳ) දුක් සහිත සිතක් නැත. බියක් නොමැතිව වසමු. 
# එකල්හි රජුගේ අන්තඃපුරයෙහිදී භාග්‍යවතුන් වහන්සේ විසින් මෙය දේශනා කරන ලදී. බුද්‍ධ වීරතෙමේ පෙර කැත් කුමරියන්ගේ උපකාරයන්ද දැක්වීය. 
# ´´උපකාර කරන්නාවූ යම් ස්ත්‍රී කෙනෙක් වූවාහුද, සුවදුක් දෙකෙහි සහභාගීවූ යම් ස්ත්‍රී කෙනෙක් වූවාහුද, යහපත මේයයි කියා දෙන්නාවූ යම් ස්ත්‍රී කෙනෙක් වූවාහුද, අනුකම්පා කරන්නාවූ යම් ස්ත්‍රී කෙනෙක් වූවාහුද ඒ මේ ස්ත්‍රීහු වෙති. 
# ´´සුචරිත ධර්‍මයෙහි හැසිරෙන්නේය. දුශ්චරිතයෙහි නොහැසිරෙන්නේය. ධර්මයෙහි හැසිරෙන තැනැත්තේ මේ ලෙව්හිද, පරලෙව්හිද සුවසේ සැතපේ. 
# ´´ගිහිගෙය හැර දමා අනගාර්‍ය්‍ය නම්වු සසුනෙහි පැවිදිවීමු. (පැවිදිව) අඩමසක් නොපැමිණි කල්හිම අපි සිව්සස් දනිමු. 
# ´´චීවරද, පිණ්ඩපාතද, ගිලානප්‍රත්‍යයද, සේනාසනද සයුරට රැල මෙන් බොහෝ දෙන නානාවිධව පමුණුවත්. 
# ´´අප විසින් ක්ලේශයෝ දවන කදහ. සියළු ආශ්‍රවයෝ නසන ලදහ. ඇතින්නන් මෙන් බැඳුම් සිඳ ආශ්‍රව රහිතව වෙසෙමු. 
# ´´ඒකාන්තයෙන් බුද්‍ධ ශ්‍රෙ.්ෂ්ඨයන් වහන්සේගේ සමීපයට අපගේ යහපත් පැමිණීමක් විය. (අප විසින්) පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇතියහ. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමෝක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරන ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරන ලදී. 
# ´´මෙසේ රහත්බවට පත්වූ (අපි) නානාවිධවූ දුක්ද, නානාවිධවූ සම්පත්දැයි සියලු සම්පත් ලබමු. 
# ´´යම් ස්ත්‍රී කෙනෙක් බුදුරජුන්හට පින් පිණිස ස්වකීය ආත්මය දෙද්ද (ඔහුට) සහාය සම්පත්තීහු වෙති. අසංකෘතවූ නිර්‍වාණ පදයද වේ. 
# ´´අතීතවූද, වර්‍තමානවූද, අනාගතවූද (අපගේ) සියළු කර්‍මය ක්‍ෂය විය. බුදුරජාණන් වහන්ස, ඔබ වහන්සේගේ සිරිපා වඳිමු. 
# ´´පිරිනිවනට පැමිණෙන්නාවූ තොපට වැඩියෙන් කුමක් කියම්ද, සංස්කාර දොස් ඇත්මැයි. අමෘත පදයවූ නිවනට පැමිණෙව්.´´ 
මෙහි මේ අයුරින් යශෝධරා ප්‍රමුඛ අටළොස් දහසක් භික්‍ෂුණීන් වහන්සේ බුදුරජුන් හමුවෙහි මෙම ගාථාවන් වදාළ සේක.
