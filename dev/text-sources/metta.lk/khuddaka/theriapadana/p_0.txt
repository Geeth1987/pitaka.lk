ථෙරී අපදාන පාළිය.
# සුමෙධා වග්ග.  
# සුමෙධා ථෙරී අපදානය.  
#
# ´´කෝණාගමන බුදුරජුන් අභිනව නිවාසයක්වූ සංඝාරාමයෙහි වැඩවෙසෙන කල්හි යෙහෙළියන් තිදෙනෙකුවූ (අපි) විහාර දානයක් දුනිමු. 
# ´´ඒ හේතුවෙන් දස වරක්ද, සියක් වරක්ද, දහස් වරක්ද, සියයේ (වාර) සියවරක්ද, (හෙවත් එකොළොස් දහස් එක්සිය දස වරක්ද) දෙවියන් අතුරෙහි උපනිමු. මිනිස් බැව්හි උපන් වාර ගැන කවර කථාවක්ද? 
# ´´දෙවියන් අතුරෙහිදු මහත් ඎද්‍ධි ඇත්තියෝ වීමු. මිනිසත් බැව්හි (මහෙශාක්‍ය වීම ගැන) කවර කථාවෙක්ද? මම (සක්විති රජුගේ) සප්තරත්නයට අයත් මෙහෙසියවූ ස්ත්‍රී රත්නයද වීමි. 
# ´´ධනඤ්ජානියද, ඛෙමාවද, මමද යන අපි තිදෙන මෙහි රැස්කළ පින් ඇත්තාහු, මැනවින් සමෘධිමත්වූ කුල ප්‍රවේනි ඇත්තියෝ වීමු. 
# ´´සියළු අවයවයන්ගෙන් සරසන ලද ආරාමයක් මැනවින් කොට බුද්ධ ප්‍රමුඛ සංඝයාට පිළිගන්වා ප්‍රමුදිත වීමු. 
# ´´එම කර්මයාගේ බලයෙන් යම් යම් තැන උපදිම්ද, දෙවියන් අතුරෙහිදු ශ්‍රේෂ්ඨ බවට පැමිණියා, මිනිසුන් අතුරෙහිදු එසේම (ශ්‍රේෂ්ඨ බවට පැමිණියා) වීමි. 
# ´´කථා කරන්නවුන් අතුරෙන් ශ්‍රේෂ්ඨවූ, බ්‍රාහ්මණ වංශයෙහිවූ, මහත් යසස් ඇති, නම්න් කාශ්‍යප නම්වූ බුදුරජතෙම මෙම කල්පයෙහි ලොව පහළවූහ. 
# ´´එකල්හි උතුම්වූ බාරාණසී නගරයෙහි මිනිසුනට අධිපතිවූ ´´කිකී´´ නම් කසීරජතෙමේ මහර්ෂීවූ බුදුරජුන්ගේ උපස්ථායකයා විය. 
# ´´ඔහුට සුවසේ වැඩුණු රාජ කන්‍යාවන් සත්දෙනෙක් වූහ. ඒ දූවරු බුද්ධොපස්ථානයෙහි නිරතවූවාහු බ්‍රහ්මචර්‍ය්‍යාවෙහිද හැසුරුණාහුය. 
# ´´(මම) ඒ (සත්) දියණියන්ගේ යෙහෙළියක්ව සීලයන්හි මැනවින් හික්මුණුබවට පැමිණියා, සකසා දන් දී ගිහිගෙයිදීම ශීලයෙහි හැසුරුණෙමි. 
# ´´මැනවින් කරණ ලද එම කර්‍මය කරණකොටගෙනද, චිත්තප්‍රණිධීන් කරණකොටගෙනද, මම මිනිස් සිරුර හැර දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි. 
# ´´මම ඉන් චුතව, යාම දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි. ඉක්බිති තුෂිත දෙව්ලොවට ගියෙමි. අනතුරුව නිර්මාණරතියටද, යලි වශවර්තිපුරයටද ගියෙමි. 
# ´´පුණ්‍යකර්‍මය සතහිවූ මම යම් යම් තැනක උපදිම්ද, ඒ ඒ තන්හි රජවරුන්ගේ (අග) මෙහෙසි බවද කෙළෙමි. 
# ´´ඉන් චුතවූවා, මිනිසත් බැව්හි සක්විති රජවරුන්ගේද, මණ්ඩලීක රජවරුන්ගේද මෙහෙසි බව කෙළෙමි. 
# ´´දෙවියන් අතුරෙහිද, මිනිසුන් අතුරෙහිද, සම්පත් අනුභවකොට, සියළු තන්හි සුඛිතව, නොයෙක් ජාතීන්හි සැරිසැරීමි. 
# ´´ඒ දුඃඛ හේතුවවූ තෘෂ්ණාවද, ඒ තෘෂ්ණා හේතුව නිසා ප්‍රභව වන්නාවූ හෙවත් ඇතිවන්නාවූ පඤ්චස්කන්ධ නම් දුකද, ඒ (දුඃඛ-සමුදය විරහිත ශාන්ති ලක්‍ෂණ) මූලයවූ නිරෝධයද, ඒ සසුනෙහි අභිමත මධ්‍යම ප්‍රතිපදා මාර්‍ගයද, ඒ නිවණට ප්‍රථම පිවිසීමවූ සෝවාන් මාර්‍ගයද, ධර්මයෙහි ඇලීම කරණකොටගෙන ඒ (අර්හත් ඵලයවූ) නිවණද (දනිමි.) 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් සුමෙධා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
#
# මේඛලා දායිකාපදානය. 
# ´´මම සිද්‍ධාර්ත්‍ථ බුදුරජුන්ගේ ස්තූපය කරවන්නියක් වීමි. ශාස්තෲන් වහන්සේගේ සෑයෙහි නවකම් පිණිස මා විසින් මෙවුල් දමක් (ආපසු ගැනීම සඳහා) දෙන ලදි. 
# ´´මහසෑය නිමවූ කල්හි මම යලිත් එම මෙවුල්දම පහන් සිත් ඇත්තී, සියතින්ම ලෝනාවූ බුදුරජුන් උදෙසා (පූජා බුද්‍ධියෙන්) දුනිමි. 
# ´´මින් පෙර සිව් අනූවන කපෙහි වූ එකල්හි යම් මෙවුල් දමක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. ස්තූප කරණයෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් මෙඛලාදායිකා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
#
# මණ්ඩක දායිකාපදානය. 
# ´´කෝණාගමන බුදුරජුන්හට මා විසින් මණ්ඩපයක් කරවන ලදී. ලොවට හිතවූ බුදුරජුන්හට නිතර (නියමිත වාරයන්හිදී) තුන් සිවුරු දුනිමි. 
# ´´නියම්ගමය, රජදහන්ය යන යම් යම් දනව්වකට යෙම්ද, (ඒ) හැමතන්හි පුදන ලද්දී වෙමි. පුණ්‍යකර්‍මයාගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් මණ්ඩකදායිකා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
#
# සඞ්කමනත්‍ථාපදානය. 
# ´´සත්ත්‍වයන් (සසර සයුරින්) තරණය කරන්නාවූ, විථියට පිළිපන්නාවූ, ලොවට දෙටුවූ, (අට ලෝ දහම්හි නොසැලෙන බැවින්) තාදී ගුණ ඇති විපස්සී බුදුරජුන්හට, 
# ´´මම ගෙදරින් නික්ම (අවුත්) කිබිසිනිව හෙවත් මුනින් වැටී (හෙයක් ලෙසින් පාගාගන යාම සඳහා) පූජාවක් කෙළෙමි. අනුකම්පා ඇති බුදුරජතෙම මාගේ කෙස්වැටිය පෑගීය. 
# ´´ලෝක නායකයන් වහන්සේ (එසේ) කෙස්වැටිය පාගාගන වැඩිය සේක. එම චිත්තප්‍රසාදය කරණකොටගෙන මම තුසිත දිව්‍ය ලෝකයෙහි උපනිමි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් සඞ්කමනත්‍ථා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
#
# නළමාලිකාපදානය. 
# ´´එකල්හි මම චන්‍ද්‍රභාගා නදීතීරයෙහි කිඳුරියක් වීමි. (එහිදි) සියල්ල තෙමේම අවබෝධ කළාවූ, පස්මරුන් ජයගත්තාවූ, කෙළෙස් රජස් රහිතවූ නොහොත් කාමයන්හි නොඇළුනාවූ බුදුරජුන් දුටිමි. 
# ´´මම පහන් සිත් ඇත්තී, සොම්නස්වූවා, හටගත් ප්‍රීති ඇත්තී, බඳැඳිලි ඇත්තී බට මලක් ගෙන බුදුරජුන්හට පිදීමි. 
# ´´මැනවින් කරණලද එම කර්‍මය කරණකොටගෙන දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි. සක් දෙව් රජුන් සතිස් දෙනෙකුගේ මෙහෙසි බව කෙළෙමි. 
# ´´සක්විති රජුන් දස දෙනෙකුගේද මෙහෙසි බව කෙළෙමි. (ඉක්බිති) මාගේ සිත සංවේගයට පත්ව, අනගාර්‍ය්‍ය නම්වූ සසුනෙහි පැවිදිවීමි. 
#
# ´´මින් පෙර සිව් අනූවන කපෙහි යම් පුෂ්ප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂ්ප පූජාවෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් නළමාලිකා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

# එකපිණ්ඩපාත දායිකාපදානය. 
# ´´බන්ධුමතී නගරයෙහි බන්ධුම නම් ක්‍ෂත්‍රියයෙක් විය. මම යම් (ඒ) රජකෙනෙකුගේ භාර්‍ය්‍යාව වීම්ද (එම රජු හා) එක් සිත් ඇතිව හැසිරෙමි. 
# ´´එකල්හි මම රහසිගතව හිඳ මෙසේ සිතීමි. මා විසින් කරණලද (මැරෙනවිට) ගෙන යා යුතු කුශලයක් නැත. 
# ´´(මෙසේ ඇති කල්හි) මහත් තැවුල් ඇති, කටුකවූ, ඝොර ස්වරූප ඇති, ඉතා දරුණුවූ නිරයට ඒකාන්තයෙන්ම යමි. මෙහි මාගේ සැකයෙක් නැත´´ (කියායි.) 
# ´´මෙසේ සිතා රජු වෙත එළඹ මෙසේ කීමි. ´´රජතුමනි, මට එක ශ්‍රමණයන් වහන්සේ නමක් දෙනු මැනව. උන්වහන්සේ වළඳවන්නෙමි´´ යි (කියායි.) 
# ´´මහරජතෙම මට වඩනලද ඉඳුරන් ඇති ශ්‍රමණයන් වහන්සේ නමක් දුන්නේය. (මම) උන්වහන්සේගේ පාත්‍රය ගෙන උතුම්වූ ආහාරයෙන් පිරවීමි. 
# ´´මම උතුම්වූ ආහාරය පුරවා, සුවඳද ගැල්වීමි. දහසක් වටිනා වස්ත්‍ර දෙකකින්ද වැසීමි. 
# ´´ජීවිතාන්තය දක්වා මෙම අරමුණ මම සිහි කරමි. මම එහි සිත පහදවා තව්තිසා දෙව්ලොවට ගියෙමි. 
# ´´දෙව්රජුන් තිස් දෙනෙකුගේ මෙහෙසිබව කෙළෙමි. මාගේ සිතින් පතන ලද්ද සිත්වූ පරිදි පහළවේ. 
# ´´සක්විති රජුන් විසි දෙනෙකුගේ මෙහෙසි බව කෙළෙමි. මම රැස්කරනලද පින් ඇත්තියක්ව වී භවයන්හි සැරිසරමි. 
# ´´සියළු බැඳුම්වලින් මිදුණු මම මාගේ ස්කන්ධයන් නැවත ඇතිවීමේ හේතූන්ගෙන් වෙන්වූවා, ක්‍ෂය කරණ ලද සියළු ආශ්‍රවයන් ඇත්තී වෙමි. දැන් නැවත උපතක් නැත. 
# ´´මින් (පෙර) එක් අනූවන කපෙහි වු එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පිණ්ඩපාත දානයාගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් එකපිණ්ඩපාත දායිකා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක. 
#
# කටච්ඡුභික්ඛා දායිකාපදානය. 
# ´´මම පිණ්ඩචරියාවෙහි හැසිරෙන්නාවූ, ශාස්තෘවූ තිස්ස නම් බුද්ධශ්‍රේෂ්ඨයන් වහන්සේට සැන්දක් පමණවූ භික්‍ෂාව ගෙන දුනිමි. 
# ´´ශාස්තෘවූ, ලෝකනායකවූ තිස්ස බුදුරජතෙම (එය) පිළිගෙන මට අනුමෝදනා දම් දෙසුම කළ සේක. (කෙසේදයත්?) 
# ´´සැන්දක් පමණවූ භික්‍ෂාව දී තව්තිසා දෙව්ලොවට යන්නෙහිය. සතිස් දෙව්රජ කෙනෙකුට මෙහෙසිබව කරන්නෙහිය. 
# ´´පණස් සක්විති රජකෙනෙකුටද මෙහෙසිබව කරන්නෙහිය. හැම කල්හි සිතින් පතන ලද සියල්ලක් ලබන්නෙහිය 
# ´´සම්පත් අනුභවකොට, (සියළු ධනය අතහැර දමන බැවින්) අකිඤ්චනවූවා, පැවිදි වන්නෙහිය. සියළු ආශ්‍රවයන් ප්‍රහාණ පරිඥා වශයෙන් දැන, ආශ්‍රව රහිතවූවා පිරිනිවන්නෙහිය´´ (කියායි.) 
# ´´ලෝකයෙහි අසහාය නායකවූ, ප්‍රාඥවූ තිස්ස බුදුරජතෙම මෙය වදාරා අහසෙහි (යන) හංසරාජයෙකු මෙන් ආකාශයට නැංගාහ. 
# ´´මා විසින් උතුම් දානයක් දෙන ලදී. දානමය යාග සම්පත් තොමෝ මැනවින් යජනය කිරීම හෙවත් දීම කරණ ලද්දීය. මම සැන්දක් පමණවූ භික්‍ෂාව දී, අචලපදයවූ නිවණට පැමිණියේ වෙමි. 
# ´´මින් (පෙර) දෙයානූවන කපෙහි වූ එකල්හි යම් දනක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. භික්‍ෂා දානයාගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´ 
#මෙහි මේ අයුරින් කටච්ඡුභික්ඛා දායිකා භික්‍ෂුණීන් වහන්සේ මෙම ගාථාවන් වදාළ සේක.

# සත්තුප්පල මාලිකාපදානය. 
# ´´අරණවතී නගරයෙහි ´´අරුණවා´´ නම් ක්‍ෂත්‍රියයෙක් විය. මම යම් ඒ රජකෙනෙකුගේ භාර්‍ය්‍යාවක් වීම්ද, (එම රජුගේ) පාදපරිවරණය කරමි. 
# ´´එකල්හි මම දිව සුවඳවත්වූ උපුල් මල් සතක් ගෙන උතුම් පහයෙහි හිඳ මෙසේ සිතීමි. 
# ´´මෙම මල්වලින් මාගේ හිස සැරසීමේ කවර ප්‍රයෝජනයක්ද? මාගේ බුද්‍ධ ශ්‍රේස්ඨයන් වහන්සේගේ (සර්‍වඥතා) ඥානයට පිදීම (එයට වඩා) උතුම්ය. 
# ´´යම් කලෙක්හි සම්බුදුරජතෙම වඩින්නේද, (එකල්හි) මම (මෙම මල්) (එම) බුදුරජුන්හට පුදන්නෙමැ´´ යි තථාගතයන් (පැමිණෙනු) බලාපොරොත්තු වන්නී, දොරටුව අසල හුනිමි. 
# ´´බබලන්නාවූ කුඹුක් ගසක් මෙන්ද, කෙශර සිංහ රාජයෙකු මෙන්ද, බුදුරජතෙම භික්‍ෂු සංඝයා සමග (එම) වීථියෙන් පැමිණි සේක. 
# ´´ බුදුරජාණන් වහන්සේගේ රශ්මිය දැක සතුටුවූවා, (ප්‍රීතියෙන්) උදම්වූ සිත් ඇත්තී, දොරටුව හැර (එම) බුද්ධ ශ්‍රේෂ්ඨයන් පිදීමි. 
# ´´අහසෙහි විසුරුවන ලද උපුල් මල් සත තථාගතයනට වියනක් (මෙන්) කරන්නාහු, එම පුෂ්පයෝ මස්තකයෙහි දරත්. 
# ´´ඔදවැඩි සිත් ඇත්තී, සොම්නස්වූවා, හටගත් ප්‍රීති ඇත්තී, බඳැදිලි ඇත්තී, මම එහි සිත පහදවා තව්තිසා දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි. 
# ´´මාගේ මුදුනෙහි මහනෙල් මල් වියනක් දරත්. දිව සුවඳ පතුරුවමි. උපුල් මල් සතක් (පිදීමෙහි) මේ විපාකයයි. 
# ´´කිසියම් කලෙක්හි මාගේ නෑ සමූහයා විසින් (මම) කැඳවාගන යනු ලබන කල්හි මාගේ පිරිසද (ඒතාක්) මට නිල් (වියනක්ද) දරනු ලැබේ. 
# ´´දෙව්රජුන් සත් දෙනෙකුගේ මෙහෙසිබව කරවීමි. හැමතන්හි අධිපතිව, ඇත්තීව කුදුමහත් භවයෙහි සැරිසරමි. 
# ´´තෙසැටක් සක්විති රජවරුන්ගේ මෙහෙසිබව කරවීමි. සියල්ලෝ මා අනුව පවතිත්. පිළිගතයුතු වචන ඇත්තියක්ද වීමි. 
# ´´මාගේ වර්‍ණයද උපුල් මලක වර්‍ණයට බඳු විය. (එසේම උපුල් මලකට බඳු) සුවඳද හමයි. දුවර්‍ණ බවක් නොදනිමි. බුද්‍ධ පූජාවෙහි මේ විපාකයයි. 
# ´´(සතර) ඎද්‍ධිපාදයන්හි දක්‍ෂවූවාය. (සප්ත) බොධ්‍යඞ්ග භාවනාවෙහි ඇළුනාය. අභිඥාවන්හි තෙර පැමිණියාය. බුද්‍ධ පූජාවෙහි මේ විපාකයයි. 
# ´´(සතර) සතිපට්ඨාන ධර්‍මයන්හි දක්‍ෂවූවාය. චිත්ත සමාධියට ධ්‍යානය ගෝචර වූවාය. (හෙවත් සමාධියෙන් ලබන ලද ධ්‍යාන බල ඇත්තීය.) (සතර) සම්‍යක් ප්‍රධාන වීර්‍ය්‍යයෙහි යෙදුනීය. බුද්‍ධ පූජාවෙහි මේ විපාකයයි. 
# ´´(මා විසින්) (ප්‍රඥා) ධුරය උසුලන මාගේ (කායික, චෛතසික) වීර්‍ය්‍යය නිර්‍වාණාභිමුඛව පමුණුවනු ලබන්නේය. (මාගේ) සියළු ආශ්‍රවයෝ ක්‍ෂය කරණ ලදහ. දැන් (සසරෙහි) නැවත ඉපදීමක් නැත. 
# ´´මින් (පෙර) එක්තිස්වන කපෙහි වූ එකල්හි යම් කර්‍මයක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පුෂපදානයිගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´

# පඤච දීපිකාපදානය. 
# ´´මම එකල්හි හංසවතී නගරයෙහි සඤ්චාරිකාවක් වීමි. පින් කැමැත්තාවූ (මම) ආරාමයෙන් ආරාමයට හැසිරෙමි. 
# ´´කාලපක්‍ෂයෙහි (වූ) අමාවක් දිනයෙහි උතුම්වූ බෝධියක් දුටිමි. මම එහි සිත පහදවන බෝමුලෙහි හුනිමි. 
# ´´එවිට (බර) සිතක් උපදවා සිරසෙහි ඇඳිලි බැඳ, එකෙණෙහි සොම්නස් උපදවා මෙසේ සිතීමි. 
# ´´ඉදින් බුදුරජතෙම අපමණ ගුණ ඇත්තේ වේනම්, අසමවූ, අප්‍රති පුද්ගල වේනම් මට ප්‍රාතිහාර්‍ය්‍යයක් දක්වාවා. මේ බෝධිය ආලෝකවත් වේවා´´ යි (කියායි) 
# ´´මාගේ ආවර්‍ජනය (මෙනෙහි කිරීම) සමගම එකෙණෙහි බෝධිය දැල්විනි. (එය) සර්‍ව ස්වර්‍ණමය විය. සියළු දිගුන් බැබලුනි. 
# ´´මම රෑ දාවල් සතක් එහි බෝමුලෙහි හුනිමි. සත්වැනි දවස පැමිණි කල්හි මම පහන් පූජාවක් කෙළෙමි. 
# ´´ආසනය පිරිවරා (වටකොට) පහන් පසක් දැල්වීමි. හිරු උදාවන තුරු එකල්හි මාගේ ප්‍රදීපයෝ දැල්වුනහ. 
# ´´මැනවින් කරණ ලද එම කර්‍මය කරණ කොට ගෙනද, චිත්ත ප්‍රණිධීන් කරණ කොට ගෙනද, මම මිනිස් නිරුර හැර තව්තිසා දෙව්ලොවට උත්පත්ති වශයෙන් ගියෙමි. 
# ´´එහි මට මැනවින් කරණ ලද (පහළවූ) විමානය ´´පඤ්චදීපී´´ යයි කියනු ලැබේ. (එය) සියක් යොදුන් උසය. තිස් යොදුනක් පුළුළය. 
# ´´අපමණ ප්‍රදීපයෝ මාගේ හාත්පස දැල්වෙත්. දිව්‍ය භවනය මුළුල්ලම ප්‍රදීපාලෝකයෙන් බැබලුණේය. 
# ´´ඉදින් අනභිමුඛයෙහි හිඳ (යමක්) බලන්නට කැමති වෙම්ද, (එවිට) උඩද, යටද, සරසද සියල්ල ඇසින් දකිමි. 
# ´´යම්තාක් (තන්හි) සුගති-දුගති(වල උපන් සත්ත්‍වයන්) දකින්නට කැමති වෙම්ද, එහි වෘක්‍ෂයන්හි හෝ පර්වතයන්හි හෝ (ඒ හේතුවෙන් සිදුවන) ආවරණයක් නැත. 
# ´´අසූවක් දෙව් රජවරුන්ගේ මෙහෙසිබව කෙළෙමි. සියයක් සක්විති රජුන්ගේද මෙහෙසිබව කෙළෙමි. 
# ´´දිව්‍යවූද, යලි මනුෂ්‍යවූද යම් යම් යෝනියක උපදිම් නම් සියක් දහසක් ප්‍රදීපයෝ මා පිරිවරා දැල්වෙත්. 
# ´´දෙව්ලොවින් චුතව මව් කුසෙහි උපනිමි. මම කුසට (පිළිසිඳ ගැනීම් වශයෙන්) ගිය මාගේ ඇස් නොපියවේ. 
# ´´කරණ ලද පුණ්‍යකර්‍මය හා යෙදුනාවූ සියක් දහසක් ප්‍රදීපයෝ තිඹිරිගෙහි දැල්වෙත්. පහන් පසක් පිදීමෙහි මේ විපාකයයි. 
# ´´අන්තිම භවය පැමිණි කල්හි (තෘෂ්ණා සහිත) සිත (තෘෂ්ණාවෙන් ඉවතට) පෙරළා දැමීමි. මම ජරා-මරණ විරහිතවූ, සිසිල් බැව් ඇති නිවණ අවබෝධ කෙළෙමි. 
# ´´උත්පත්තියෙන් සත් හැවිරිදිවූ මම රහත්බවට පැමිණියෙමි. ගෞතම බුදුරජතෙමේ (මාගේ) ගුණ දැන (මා) උපසම්පදා කරවූ සේක. 
# ´´මඩුවක හෝ, රුක්මුලක හෝ, ප්‍රාසාදයන්හි හෝ, හිස් ගෙයක හෝ ධ්‍යන කරණ කල්හි මාගේ (හාත්පස) ප්‍රදීපයෝ දැල්වෙත්. 
# ´´මාගේ දිවැස පිරිසිදු විය. මම සමාධියෙහි දක්‍ෂවූවා වෙමි. අභිඥාවන්හි තෙර පැමිණියාද වෙමි. පහන් පසක් පිදීමෙහි මේ විපාකයයි. 
# ´´සියල්ල සම්පූර්‍ණ ලෙස අවසන් කළා, කොට නිම කරණ ලද කටයුතු ඇත්තී, ආශ්‍රව රහිත වූවාය. පසැස් ඇත්තාවූ මහා වීරයන් වහන්ස, පඤ්චදීපා තොමෝ (ඔබ වහන්සේගේ) පා වඳියි. 
# ´´මින් (පෙර) සියක් දහස්වන කපෙහි වු එකල්හි යම් ප්‍රදීප පූජාවක් කෙළෙම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. පහන් පසක් පිදීමෙහි මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´

# උදක දායිකාපදානය. 
# ´´මම බන්ධුමතී නගරයෙහි (දිය අදිනා) කුම්භ දාසියක් වීමි. දිය ඇදීමෙන් ජීවත් වෙමි. එයින් දරුවන් පෝෂණය කරමි. 
# ´´නිරුත්තරවූ පුණ්‍ය ක්‍ෂෙත්‍රයෙහි පිදියයුතු දෙයක්ද මට නැත. මම (සංඝයාගේ නහන) කොටුවට එළඹ පැන් (පුරවා) තැබීමි. 
# ´´මම මැනවින් කරණ ලද එම කර්‍මය කරණකොටගෙන තව්තිසා දෙව්ලොවට (උත්පත්ති වශයෙන්) ගියෙමි. එහි මට මැනවින් කරණ ලද දිය ඇදීමේ පුණ්‍යකර්‍මය කරණකොටගෙන නිර්මිතවූ විමානයක් (වේ.) 
# ´´එකල්හි මම දහසක් අප්සරාවන් අතුරෙන් උතුම් තැනැත්තිය වෙමි. මම හැමකල්හි ඒ සියළු දිව්‍යප්සරාවන් කරුණු දසයකින් අභිභවනය (යටපත්) කරමි. 
# ´´දෙව්රජුන් පන්සියයක් දෙනෙකුගේ මෙහෙසි බව කෙළෙමි. සක්විති රජවරුන් විසි දෙනෙකුගේද මෙහෙසි බව කෙළෙමි. 
# ´´දෙව් අත්බැව්හිද, යලි මිනිස් අත්බැව්හිද යන මේ දෙ අත්බැව්හි සැරිසරමි. දුගතියක් නොදනිමි. දිය දීමෙහි මේ විපාකයයි. 
# ´´කඳු මුදුනකදී හෝ, ගස් මුදුනකදී හෝ, අහසෙහිදී හෝ, පොළොවෙහිදී හෝ යම් කලෙක්හි මම ජලය කැමැත්තෙම්ද, වහා එය ලබමි. 
# ´´(මා යන්නාවූ) වැසි නැති දිශාවක් නම් නැත. (එම දිශාවන්හි) තැවුනු, කැකෑරුනු බවකුදු නැත. මාගේ අදහස දැන මහ වැසි වසී. 
# ´´මාගේ නෑ සමූහයා විසින් (මා) කැඳවාගෙන යනු ලබන කිසියම් දිනයක් වේ නම් මම යම් කලෙක්හි වැසි කැමැත්තෙම්ද, එකල්හි මහ වැසි වසී. 
# ´´මාගේ සිරුරෙහි උෂ්ණයක් හෝ දැවිල්ලක් හෝ නැත. මාගේ ශරීරයෙහි (දැලිකුණු) රජස්ද නැත. උදක දානයාගේ මේ විපාකයයි. 
# ´´අද (මම) පිරිසිදු සිත් ඇත්තී, ඉවත්කළ පව්මල ඇත්ති, සියළු ආශ්‍රවයන් ක්‍ෂය කරණ ලද්දී වෙමි. දැන් නැවත ඉපදීමක් නැත. 
# ´´මින් (පෙර) එක් අනූවන කපෙහි වු එකල්හි යම් උදක දානයක් දුනිම්ද, (ඒ හේතුවෙන්) දුගතියක් නොදනිමි. උදක දානයාගේ මේ විපාකයයි. 
# ´´මාගේ ක්ලේශයෝ දවන ලදහ. සියළු භවයෝ නසන ලදහ. ඇතින්නක් මෙන් බැඳුම් සිඳ, ආශ්‍රව රහිතව වෙසෙමි. 
# ´´ඒකාන්තයෙන් බුද්ධ ශ්‍රේෂ්ඨයන් වහන්සේගේ සමීපයට මාගේ යහපත් පැමිණීමක් විය. පැමිණෙන ලද ත්‍රිවිද්‍යාවෝ ඇත්තාහුය. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී. 
# ´´සිව් පිළිසිඹියාවෝද, අෂ්ට විමොක්‍ෂයෝද, ෂඩභිඥාවෝද සාක්‍ෂාත් කරණ ලදහ. බුදුරජාණන් වහන්සේගේ සසුන කරණ ලදී.´´
