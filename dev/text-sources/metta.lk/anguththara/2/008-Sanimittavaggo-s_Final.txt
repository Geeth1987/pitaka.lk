# සනිමිත්ත වර්‍ගය. 
# සනිමිත්ත සූත්‍රය. 
# සනිමිත්ත සූත්‍රය. 
´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ කරුණු සහිතවම උපදිත්. කරුණු රහිතව නූපදිත්. එම කාරණයාගේ දුරුකිරීමෙන් මෙසේ ලාමකවූ අකුශල ධර්‍මයෝ නූපදිත්.´´ 
# සනිදාන සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ නිදාන සහිතවම උපදිත් .නිදාන රහිතව නූපදිත් ඒ නිදානයාගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්. ´´ 
# සහෙතුක සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ හේතු සහිතවම උපදිත්, හේතු රහිතව නූපදිත් ඒ හේතුහුගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්. ´´ 
# සසංඛාර සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ සංස්කාර සහිතවම උපදිත් .සංස්කාර රහිතව නූපදිත්. ඒ සංස්කාරයාගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්. ´´ 
# සපච්චය සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ ප්‍රත්‍යය සහිතවම උපදිත්. ප්‍රත්‍යය රහිතව නූපදිත්. ඒ දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්. ´´ 
# සරූප සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ රූප (හේතු) සහිතවම උපදිත්. රූප රහිතව නූපදිත්. ඒ කාරණයාගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්.´´ 
# සවෙදනා සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ වේදනා සහිතවම උපදිත්. වේදනා රහිතව නූපදිත්. ඒ වේදනාවන්ගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්.´´ 
# සසඤ්ඤා සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ සංඥා සහිතවම උපදිත්. සංඥා රහිතව නූපදිත්. ඒ සංඥාවගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්.´´ 
# සවිඤ්ඤාණ සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ විඤ්ඤාණ සහිතවම උපදිත්. විඤ්ඤාණ රහිතව නූපදිත්. ඒ විඤ්ඤාණයාගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නූපදිත්.´´ 
# සංඛතාරම්මණ සූත්‍රය. 
 ´´මහණෙනි, ලාමකවූ අකුසල ධර්‍මයෝ සංඛතාරම්මණ සහිතවම (ප්‍රත්‍යයෙන් නිපන් සංඛත ධර්‍මයන් අරමුණුකොට)ම උපදිත්. අසංඛතාරම්මණ සහිතව (අසංඛතවූ නිර්‍වාණය අරමුණුකොට) නූපදිත්. ඒ සංඛත ධර්‍මයාගේම දුරුකිරීමෙන් මෙසේ ඒ ලාමකවූ අකුසල ධර්‍මයෝ නොවෙත්.´´ 
( තුන්වෙනි වර්‍ගය නිමි. )
