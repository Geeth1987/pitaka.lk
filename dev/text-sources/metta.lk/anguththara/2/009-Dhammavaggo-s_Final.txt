# ´´විත්ත විමුක්තිය (අර්හත් ඵල සමාධිය) ද, ප්‍රඥා විමුක්තිය (අර්හත් ඵල ප්‍රඥාව)ද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# පග්ගාහා සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´ප්‍රග්‍රහය (වීර්‍ය්‍යය)ද, අවික්‍ෂෙපය (සිතෙහි එකඟ බව)ද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්.

# නාමරූප සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´නාමයද, රූපයද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# විජ්ජාවිමුත්ති සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´විද්‍යාව (සතරඵල ඥානයද), විමුක්තියද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# භවදිට්ඨි සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´භව (ශාස්වත) දෘෂ්ටියද, විභව (උච්ඡේද) දෘෂ්ටියද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# අහිරික සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´පව් කිරීමෙහි ලජෆ්ජා රහිත බවද, පව් කිරීමෙහි බිය නොවන බවද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# හිරි සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´පව් කිරීමෙහි ලජ්ජා ඇති බවද, පව් කිරීමෙහි බියද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# දොවචස්ස සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´දුර්‍වච (අකීකරු) බවද, පාප මිත්‍රයන් ඇති බවද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්. 
# සොවචස්ස සූත්‍රය. 
# ´´මහණෙනි, මේ ධර්‍මයෝ දෙදෙනෙක් වෙත්. කවර දෙදෙනෙක්ද යත්, 
# ´´සුවච (කීකරු) බවද, කල්‍යාණ මිත්‍රයන් ඇති බවද (යන දෙදෙන) වෙත්. 
# ´´මහණෙනි, මේ වනාහි ධර්‍මයෝ දෙදෙන වෙත්.
