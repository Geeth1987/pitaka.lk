# සාධු වර්‍ගය 
# සාධු අසාධු සූත්‍රය 
# ´´මහණෙනි, තොපට යහපතද, අයහපතද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, අයහපත කවරේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය අයහපතයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, යහපත කවරේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය යහපතයයි කියනු ලැබේ.´´ 
# අරියානරිය සූත්‍රය 
# ´´මහණෙනි, තොපට ආර්‍ය්‍ය ධර්‍මයද, අනාර්‍ය්‍ය ධර්‍මයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, අනාර්‍ය්‍ය ධර්‍මය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය අනාර්‍ය්‍ය ධර්‍මයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, ආර්‍ය්‍ය ධර්‍මය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය ආර්‍ය්‍ය ධර්‍මයයි කියනු ලැබේ.´´ 
# කුසලාකුසල සූත්‍රය 
# ´´මහණෙනි, තොපට කුසලද, අකුසලයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, අකුසලය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය අකුසලයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, කුසලය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය කුසලයයි කියනු ලැබේ.´´ 
# අත්‍ථානත්‍ථ සූත්‍රය 
# ´´මහණෙනි, තොපට අර්ත්‍ථයද, අනර්ත්‍ථයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, අනර්ත්‍ථය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය අනර්ත්‍ථයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, අර්ත්‍ථය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අර්ත්‍ථයයි කියනු ලැබේ.´´ 
# ධම්මාධම්ම සූත්‍රය 
# ´´මහණෙනි, තොපට ධර්‍මයද, අධර්‍මයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, අධර්‍මය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය අධර්‍මයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, ධර්‍මය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය ධර්‍මයයි කියනු ලැබේ.´´ 
# ආසවානාසව සූත්‍රය 
# ´´මහණෙනි, තොපට ආශ්‍රවයද, අනාශ්‍රවයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, ආශ්‍රවය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය ආශ්‍රවයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, අනාශ්‍රවය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අනාශ්‍රවයයි කියනු ලැබේ.´´ 
# සාවජ්ජානවජ්ජ සූත්‍රය 
# ´´මහණෙනි, තොපට සාවජ්ජයද, අනවජ්ජයද දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, සාවජ්ජය කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය සාවජ්ජයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, අනවජ්ජය කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අනවජ්ජයයි කියනු ලැබේ.´´ 
# තපනීයාතපනීය සූත්‍රය 
# ´´මහණෙනි, තොපට තපණීය ධර්‍ම හා අතපණීය ධර්‍ම දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, තපණීය ධර්‍ම කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය තපණීය ධර්‍මයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, අතපණීය ධර්‍ම කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අතපණීය ධර්‍මයයි කියනු ලැබේ.´´ 
# ආචයාපචයගාමී සූත්‍රය 
# ´´මහණෙනි, තොපට ආචයගාමී ධර්‍ම හා අපචයගාමී ධර්‍ම දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, ආචයගාමී ධර්‍ම කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය ආචයගාමී ධර්‍මයයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, අපචයගාමී ධර්‍ම කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය අපචයගාමී ධර්‍මයයි කියනු ලැබේ.´´ 
# සුඛදුක්ඛුද්‍රය සූත්‍රය 
# ´´මහණෙනි, තොපට දුක්ඛුද්‍රය ධර්‍ම හා සුඛුද්‍රය ධර්‍ම දෙසන්නෙමි. එය හොඳට අසව්. මෙනෙහි කරවු. කියන්නෙමි.´´ ´´ස්වාමීනි, එසේය´´ යි ඒ භික්‍ෂූහු භාග්‍යවතුන් වහන්සේට පිළිතුරු දුන්හ. භාග්‍යවතුන් වහන්සේ මෙය වදාළ සේක.´´ 
# ´´මහණෙනි, දුක්ඛුද්‍රය ධර්‍ම කෙසේද? සතුන් මැරීම, හොරකම, කාමයන්හි වැරදි හැසිරීම, බොරුකීම, කේළාම් කීම, ඵරුෂ වචන කීම, හිස් වචන කීම, දැඩි ලෝභකම, ක්‍රෝධය, වැරදි දැකීමයි. මහණෙනි, මෙය දුක්ඛුද්‍රය ධර්‍ම යයි කියනු ලැබේ.´´ 
# ´´මහණෙනි, සුඛුද්‍රය ධර්‍ම කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය සුඛුද්‍රය ධර්‍ම යයි කියනු ලැබේ.´´

# ´´මහණෙනි, සුඛ විපාක ධර්‍ම කෙසේද? සතුන් මැරීමෙන් වෙන්වීම, හොරකමින් වෙන්වීම, කාමයන්හි වැරදි හැසිරීමෙන් වෙන්වීම, බොරුකීමෙන් වෙන්වීම, කේළාම් කීමෙන් වෙන්වීම, ඵරුෂ වචන කීමෙන් වෙන්වීම, හිස් වචන කීමෙන් වෙන්වීම, තද ලෝභකම නොවීම, ක්‍රෝධය නොවීම, නිවැරදි දැකීමයි. මහණෙනි, මෙය සුඛ විපාක ධර්‍ම යයි කියනු ලැබේ.´´
