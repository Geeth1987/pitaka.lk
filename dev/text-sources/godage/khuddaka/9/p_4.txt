පංචක නිපාතය 
# අප්‍රකට ථෙරී ගාථා
#“යම්හෙයකින් පැවිදිව විසිපස් වසක් මුළුල්ලෙහි අසුරු ගසන පමණකුදු සිතේ සන්සුන් කමක් නොලදිමි. 
# “මම, එසේ සිත සන්සුන් වීමක් නොලැබ කාම ආශායෙන් තෙත්ව හිස අත බැඳ හඬමින් විහාරයට ගියෙමි. 

# “මා විසින් ඇදහියයුතු ඒ ධම්මදින්නා නම් භික්ෂුණිය වෙතට පැමිණියෙමි.  ඕතොම ස්කන්ධායතන ධාතූන් ප්‍රකාශිත ධර්මය මට දේශනා කළාය. 
# “එම භික්ෂුණියගේ ධර්ම කථාව අසා එකත්පසෙක හුනිමි. පෙර ජාති දනිමි. දිවැස පිරිසිදු කරණ ලදී. 
# “පරසිත් දැනීමේ නුවණද, දිව කණද පිරිසිදු කරණ ලදී. මා විසින් පෙළහර පෑමේ නුවණ අත් දක්නා ලදී. මාගේ පව් ගෙවීයාම විය. විශෙෂ දැනීම් සය අත් දක්නා ලදී. බුද්ධාවවාදය කරණ ලදී.” 
#
# විමලා (පුරාණ ගණිකා) ථෙරී ගාථා
# “ශරීර පැහැයෙන්ද, රූපයෙන්ද, සැපතින්ද, යසසින්ද, තරුණ කමින්ද මත්වූවා තදවූවා අන්‍ය ස්ත්‍රීන් ඉක්මවා සිතමි. 
# “මෝඩයන් කථාකරණ මේ කය විසිතුරු කොට සරසා, වැද්දා මළපුඩුව හෙලා බලාසිටින සේ වෙශ්‍යාගෘහයෙහි සිටියෙමි. 
# “ආභරණ හැප්පීමෙන් බොහෝ රහස් තැන් දැක්වීමෙන්, මහජනයාට සිනාසෙමින් නොයෙක් රැවටීම් කෙළෙමි. 
# ඒ මම අද හිස මුඩුකොට දෙපොට සිවුරු පොරවා පිඬු පිණිස හැසිර ද්විතියධ්‍යානය පටන් අර්හත්වය ලබා ගස් මුලෙක හුන්නී වෙමි. 
# “දෙවි මිනිසුන් ඒ සියළු බැඳුම් සිඳ සියලු පව් ගෙවා සිහිල් වීමි. නිවුනෙමි.” 
#
# සීහා ථෙරී ගාථා
# “පෙර මා සිත වසඟ නැති කල්හි නුනුවණින් කල්පනා කිරීමෙන් කාමරාගයෙන් මැඩී ඇවිසුණු සිත් ඇත්තී වීමි. 
# “කෙලෙසුන්ගෙන් මඩනා ලද්දී, රූපාදීන්හි ශුභ හැඟීම් අනුව පවත්නා සුළුවූවා. රාග සිතට වසඟව සිතේ සන්සිඳුමක් නොලදිමි. 
# “කෙට්ටුවී, සුදුමැලිව, අවලක්ෂණව සත්වසක් මුළුල්ලෙහි හැසුරුනෙමි. රෑ දාවල දෙක්හිම පැමිණියා සැපයක් නම් නොවින්දෙමි. 
# නැවත ගිහි බවට පැමිණ හැසිරීමට වඩා ගෙල වැල් ලා මිය යාම හොඳය කියා රැහැණක් ගෙණ වනය තුලට පිවිසියෙමි. 
# මලය දැඩි කොට ගස අත්තක බැඳ එහි ගෙල ඇතුලු කෙළෙමි. එකෙණෙහි මා සිත මිදුනේය.” 
#
# සුන්දරී නන්දා ථෙරී ගාථා
# “නන්දාවෙනි, ගිලන්වූ අපවිත්‍රවූ කුණුවූ මේ ශරීරය බලව. එකඟව සිත මනාව අරමුණෙහි පිහිටුවා සිත අශුභ භාවනාවෙන් වඩව. 
# දුගඳවූ, සැරව වැහෙන, මෝඩයන් සතුටු කරණ මෙය යම්සේද, තීගේ මේ සිරුරත් එසේමයි. එය යම්සේද මෙයත් එසේය. 
# රෑ දාවල අනලසව මෙසේ මේ දකිමින් පසුව ස්වකීය ප්‍රඥාවෙන් කලකිරීම දිටිමි. 
# අප්‍රමාදව නුවණින් සලකන්නාවූ ඒ මා විසින් මේ කය තම ඇතුළත පිටත වශයෙන් තතුසේ දක්නා ලදී. 
# “එසේ ඇති කල්හි මම ශරීරයෙහි කලකිරුණෙමි. ඇතුළු කයෙහිද නොඇලුනෙමි. අප්‍රමාදව, සියලු බැඳුම් වලින් සන්සිඳුනෙමි නිවුනෙමි.” 
#
# නන්දුත්තරා ථෙරී ගාථා
# “ගින්නද සඳද, හිරුද දෙවියන්ද මම නමස්කාර කෙළෙමි. ගං තොටුපලවලට ගොස් දියට බසිමි. 
# බොහෝවත් සමාදානයෙන් අඩක් හිස මුඩුකෙළෙමි. ඇතිරිලි නැති බිම නිදාගනිමි. රෑ අහරින් වැළකුණෙමි. 
# සැරසීමෙන්ද, පැළඳීමෙන්ද, නෑමෙන් හා ඇඟ ඉලීමෙන්ද කාමරාගයෙන් පීඩිතව මේ කය නැගිටුවෙමි. 
# එයින් පසු ශ්‍රද්ධා ඇතිවැ සස්නෙහි පැවිදිවීමි. මේ ශරීරය තතුසේ දැක කාමරාගය ඉගිලවීමි. 
# ආශාවද, ප්‍රාර්ථනාවද, සියලු උත්පත්තීද සිදින ලදී. සියලු කාමාදී යෝගයන්ගෙන් වෙන්ව සිත සන්සිඳීමට පැමිණියෙමි.” 
#
# මිත්තාකාලී ථෙරී ගාථා
# “මම ගිහිගෙයින් සස්නෙහි ශ්‍රද්ධාවෙන් පැවිදිව එතැන මෙතැන ලාභ සත්කාරවල ඇලී හැසුරුනෙමි. 
# උතුම් අදහස ඉවතලා හීනාර්ථය සෙවීමි. කෙලෙසුන් වසයට පැමිණ මහණකමේ අදහස නොදනිමි. 
# පන්සලෙහි උන්නාවූ ඒ මට තෘෂ්ණාවට වසඟව නොමගට ගියෙමියි සංවේගයක් විය. 
# මාගේ ජීවිතය ඉතා ස්වල්පයි. ඒ ද ජරාව හා ලෙඩ මිරිකයි. කය ජරාවට පැමිණ බිඳේ. මට පමාවීමට දැන් කල් නොවෙයි. 
# “ස්කන්ධයන්ගේ ඇතිවීම නැතිවීම තතුසේ සලකන්නී මිදුනු සිත් ඇතිව නැඟී සිටියෙමි. බුද්ධ වචනයට අනුව ක්‍රියා කෙළෙමි.” 
#
# සකුලා ථෙරී ගාථා
# “ගිහි ගෙය වසන මම භික්ෂූන් වහන්සේ නමකගෙන් බණ අසා රාගාදීන් තොරවූ නොමැරෙන නිර්වාණයට යන දහම දිටිමි. 
# “ඒ මම පුතාද දුවද ධන සම්පත්ද හැර කෙහෙ සිඳවා ශාසනයෙහි පැවිදි වීමි. 
# “හික්මෙන්නකුවූ මම මතු මග වඩන්නී එයින් නැසියයුතු කෙලෙසුන්ද කාමය හා කොපයද දුරු කෙළෙමි. 
# “උපසපන් මෙහෙණියක්ව පූර්ව ජාතිය සිහිකිරීමි. පිරිසිදුව මනාව වඩන ලද දිවැස පිරිසිදු කෙළෙමි. 
# “හේතුවෙන් හටගත් වහා බිඳෙන සංස්කාරයන් සතුරු වශයෙන් දැක සියලු පව් දුරු කෙළෙමි. සිහිල් වීමි. නිවුනෙමි.” 
#
# සොණා ථෙරී ගාථා
# “මේ රූප කායයෙන් දරුවන් දසදෙනෙකු වදා, දුර්වලවූවා, මහලු වූවා, භික්ෂුණියක් වෙතට ගියෙමි. 
#  ඕතොම මට ස්කන්ධ ආයතන ධාතූන් ගැන ධර්ම දේශනා කළාය. ඇගේ ධර්මය අසා කෙහෙ සිඳ පැවිදිවීමි. 
# “හික්මෙන්නාවූ ඒ මාගේ දිවැස පිරිසිදු විය. මා විසූ පෙර ජාති දනිමි. 
# “එකඟව, මනාව සිත එකමුතු වූවා අනිමිත්ත භාවනාව වැඩීමි. තෘෂ්ණා වශයෙන් නොගෙණ නිවී, අනතුරුව මිදුනා වීමි. 
# “මුල් කැපූ, නුවණින් දන්නා ලද පංචස්කන්ධයෝ පවතිත්. ලාමකවූ දිරූ තිගේ ශරීරයට නින්දා වේවා. නැවත වීමක් දැන් නැත.” 
#
# භද්‍රා කුණ්ඩලකෙසී ථෙරී ගාථා
# “ඉගුලු හිසකෙහෙ ඇති, දත් මල දරන්නාවූ එක් සළුවක්ම හඳනා පොරෝණා කොට ඇත්තී, නිවැරැද්දෙහි වරදයයි කල්පනා ඇතිව, වරද නිවරදයයි කල්පනා ඇතිව පෙර හැසුරුනෙමි. 
# “ගිජුකුළු පව්වෙහි දවල් විවේකයෙන් නික්මුණ මම භික්ෂු සංඝයා විසින් පිරිවරණ ලද රාග නැති සර්වඥයන් වහන්සේ දිටිමි. 
# “වෙතට ගොස් දෙදණ බිම හැන දොහොත් මුදුන්දී වැන්දෙමි. භාග්‍යවතුන් වහන්සේ එව පින්වතියයි වදාළ සේක. එය මගේ උපසම්පදාව විය. 
# “අංග, මගධ, වජ්ජි කාසි, කෝසල යන ජනපද ඇසුරුවිය. ණය රහිතවීමි. පණස් වසක් රට වැසියන් දුන් පිණ්ඩාහාරය අනුභව කෙළෙමි. 
# “යමෙක් තෙම රහත්වූ භද්‍රාවට සිවුරක් දුන්නේද ඒ මේ නුවණැති උපාසක තෙම ඒකාන්තයෙන් බොහෝ පින් රැස් කර ගති.” 
#
# පටාචාරා ථෙරී ගාථා
# “මිනිස්සු නඟුලින් කෙත සා බිම බිජු වපුරා ඉන් අඹුදරුවන් පෝෂ්‍ය කරමින් ධනයද ලබති. 
# “සීල සම්පන්නවූ ශාස්තෘ වචනය කරන්නාවූ කුසීත නොවූ, උඩඟු නොවූ මම කිම නිර්වාණය නොලබම්ද? 
# “පා සෝදා සිට උස් බිමින් මිටි බිමට ගලා ආ පා සේදූ දිය දැකැ ඒ ජලය කෙරෙහි භාවනා කෙළෙමි. 
#“අසරුවකු ආජානීය අශ්වයා දමනය කරන්නා සේ සිත සන්සුන් කෙළෙමි. ඉක්බිති මම පහනක් ගෙන පන්සලට ගියෙමි. 
#“යහන බලා සිහි ඇඳෙහි වැද ගතිමි. ඉන්පසු ඉද්ද ගෙන පහන් තිරය ඇද්දෙමි. එකෙණෙහිම පහන නිවුනාක් මෙන් සිත නිවුනේය.” 
#
# තිංසමත්තා ථෙරී ගාථා
# “මනුෂ්‍යයෝ මොහොල් ගෙන වී කොටා අඹු දරුවන් පොෂ්‍ය කරමින් ධනවත්ද වෙති. 
# “යමක් කොට පසුව නොතැවේද ඒ බුද්ධ ශාසනය කරව. වහා පා සෝදා එකත්පසෙක හිඳිව්. සිත සන්සිඳවීමෙහි යෙදී බුද්ධාවවාදය කරව්. 
# “ඒ භික්ෂුණීහු පටාචාරා තෙරණියගේ දේශනාව අසා පා සෝදා එක පසෙක හුන්නාහුය. සිත සන්සුන් කිරීමෙහි යෙදී බුද්ධ නියමය කළහ. 
# “රෑ මුල් යාමයෙහි පෙරවිසූ කඳ පිළිවෙළ සිහි කරණ දැනීම ලැබූහ. රාත්‍රි මධ්‍යම යාමයෙහි දිවැස පිරිසිදු කළහ. රෑ පසුයම අඳුරු කඳ නැසූහ. 
# “නැගිට පා වැඳ ඔබගේ අනුශාසනය කෙළෙමු. යුද්ධයෙන් දිනූ සක්දෙව් රජු පිරිවරා සිටි තව්තිසා දෙවියන් මෙන් ඔබ පිරිවරා හැසිරෙමු. තුන් දැනීම් ඇත්තමු. පව් නැත්තමු.” 
# චන්ද්‍රා ථෙරී ගාථා
# “පෙර නෑදෑ මිත්‍රයින්ගෙන් තොරවූ වැන්දඹුවූ පුත්‍රයන් නැත්තාවූ බතක් වතකුදු නොලබන දිළිඳු තැනැත්තියක් වීමි. 
# “මැටි බඳුනක්ද, සැරයටියක්ද ගෙන ගෙන්ගෙට සිඟා යමින් ශීතයෙන් ඌෂ්ණයෙන් මිරිකෙමින් මම සත් අවුරුද්දක් හැසුරුනෙමි. 


# “ආහාර පාන ලබන භික්ෂුණියක දැක වෙතට පැමිණ ශාසනයෙහි පැවිදි කරවයි කීමි. 

# “ඒ පටාචාරාවද අනුකම්පාවෙන් මා පැවිදි කළාය. අනතුරුව මට අවවාද කොට පරමාර්ථයෙහි යෙදීය. 
# “මම ඇගේ වචනය අසා අනුශාසනා කෙළෙමි. ආර්යාවගේ අවවාදය සඵල විය. ත්‍රිවිද්‍යාලාභී වීමි. පව් නැත්තියක වීමි.”
