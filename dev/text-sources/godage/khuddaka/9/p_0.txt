ඛුද්දකනිකාය
#ථෙරීගාථා 
#එකක නිපාතය 
# අඤ්ඤතරා (ථෙරිකා) ථෙරී ගාථා
# “ථෙරිකාවෙනි, ගින්නෙන් වළඳෙහි මැලවෙන පලා මැල්ලුමක් මෙන් තොපගේ කාමාශාව අනාගාමි මාර්ගයෙන් මැලවුනේද, ඉතිරි කෙලෙස් රහත් මාර්ගයෙන් දවා පාංශුකූල වස්ත්‍ර හැඳ පොරවා සැපසේ ඉරියව් පවත්වව, අප්‍රසිද්ධ එක් භික්ෂුණියක් මෙසේ මේ ගාථාව කීවාය.”
 (සෑම ථෙරී ගාථාවකම අවසානයේ තැනට නියමිත නම යොදා මේ ඡෙදය යොදාගත යුතුයි.) 

# මුත්තා ථෙරී ගාථා
# “මුත්තා වෙනි, චන්ද්‍රයා රාහුගෙන් මිදෙන්නාසේ කාමාදී යොග සතරින් මිදෙව, ආර්ය මාර්ගයෙන් සම්පූර්ණයෙන් මිදුනු සිතින් ආහාර ණය නැතිව අනුභව කරව, භාග්‍යවතුන් වහන්සේ මුත්තා නම් හික්මෙන භික්ෂුණියට මෙසේ මේ ගාථාවම නිතර වදාළසේක.” 
# පුණ්ණාථෙරී ගාථා
#“පුණ්ණා වෙනි, පුරපස ළොස්වක්හි සඳමෙන් ධර්මයන්ගෙන් සම්පූර්ණ වෙව. සම්පූර්ණවූ රහත් මාර්ග ඤාණයෙන් නොදැනීම් රාශිය බිඳදමව.” 

# තිස්සාථෙරී ගාථා
#“තිස්සාවෙනි, තුන් වැදෑරුම් හික්මීමෙහි හික්මෙව. යහපත් උත්පත්තිය හිස් නොකරන්න. සියලු ආශා නමැති රැහැන් කඩා පඤ්චස්කන්ධ ලොකයෙහි කෙලෙස් නැතිව වසන්න.” 
#
# අඤ්ඤතරාතිස්සා ථෙරී ගාථා
#“තිස්සාවෙනි, ධර්මයන්හි යෙදෙව යහපත් උත්පත්තිය නොඉක්මවව. යහපත් උත්පත්තිය ඉක්මවූවෝ අපායෙහි දුක් විඳිත් 
#
# ධීරා ථෙරී ගාථා
#“ධීරාවෙනි, කෙලෙස් නැසීම ස්පර්ශ කරව. රාගාදී පාප අදහස් සංසිඳවීම ඇති කාමාදී රැහැන් සතරින් නිදහස්වූ ශ්‍රෙෂ්ඨ නිර්වාණය අත්පත් කරව.” 
#
# වීරා ථෙරී ගාථා
#“වීරාවෙනි, වීරවූ ආර්ය ධර්මයන්ගෙන් දියුණු කරණලද, ශ්‍රද්ධාදී ඉන්ද්‍රියයන් ඇති භික්ෂුණියක්වූ නුඹ පිරිවර සහිත කෙලෙස් මාරයා පරදවා අන්තිම ශරීරය දරව.” 
#
# මිත්තා ථෙරී ගාථා
#“මිත්තාවෙනි ශ්‍රද්ධාවෙන් පැවිදිව කල්‍යාණ මිත්‍රයන් කෙරෙහි සත්කාර කිරීමෙහි ඇලෙව. නිවන් ලැබීමට කුසල්වූ ආර්ය මාර්ගයන් දියුණු කරව.” 
#
# භද්‍රා ථෙරී ගාථා
#“භද්‍රාවෙනි, ශ්‍රද්ධාවෙන් පැවිදිව යහපතෙහි යහපත්ව ඇලෙන්නෙක් වෙව. කාමාදි රැහැන් වලින් නිර්භයවූ නිවන්යාමට කුසල් දියුණු කරව.” 

#
# උපසමා ථෙරී ගාථා
#“උපසමාවෙනි, මාරයාට අයත් එගොඩවන්නට නොහැකි කාමාදී සතර මහවතුර එතර කරව. සේනා සහිත මාරයා නසා අන්තිම ශරීරය දරන්නියක් වෙව.” 
#
# මුත්තා ථෙරී ගාථා
#“වංගෙඩිය, මොහොළ ස්වකීය කුදු ස්වාමියා, යන කුද තුනින් මිදුනෙම් මනාකොට මිදුනෙමි. භවාශාව නසා ජාති මරණ දෙකින් මිදුනෙමි.” 
#
# ධම්මදින්නා ථෙරී ගාථා
#“රහත්වීමට තද ආශාව ඇති අවසානය සොයන්නාවූ  ඕතොමෝ සෝවාන් ආදී මාර්ගයන් තුනින් නිවන ස්පර්ශ කළාය. කාමයන්හි නොබැඳුන සිත් ඇත්තේ අවිහයෙ සිට අකනිටාවට යන්නීය.” 
#
# විශාඛා ථෙරී ගාථා
#“යමක්කොට අපායයෙහි නොතැවේද ඒ බුද්ධාවවාදයට අනුව ක්‍රියා කරව. ඉක්මනට දෙපා සෝදා එකපැත්තක හිඳගනුව.” 
#
# සුමනා ථෙරී ගාථා
#“ඇස් කන් ආදී ධාතූහු දුක් වශයෙන් දැක නැවත උත්පත්තියකට නොපැමිණෙව, කාමාදී භවයන්හි ඇල්ම අත්හැර ශාන්තව හැසිරෙන්නෙහිය.” 
#
# උත්තරා ථෙරී ගාථා
#“ශරීරයේ වචනයේ හා සිතේ සංවරයෙන් යුක්තවෙව. මුල් සහිත තෘෂ්ණාව උදුරා දමා කෙලෙස් නැති කළේ වෙමි. නිවුනේ වෙමි.” 
#
# (මහළුව පැවිදිවූ) සුමනා ථෙරී ගාථා
#“සුමනාවෙනි, මහළුවූ ඔබ පාංශුකූල වස්ත්‍ර හැඳ පොරවා සුවසේ වසව. තොපගේ රාගය සංසිඳුනේය. සිහිල්විය, නිවුනේය.” 
#
# ධම්මා ථෙරී ගාථා
#“සැරයටියෙහි ආධාරයෙන් පිඬු පිණිස නුවර හැසිරී වෙවුලන ශරීරයෙන් යුක්තව එහිම බිම වැටුනෙමි. ශරීරයෙහි දොෂය දැක මාගේ සිත කෙළෙසුන්ගෙන් මිදුනේය.” 
#
# සංඝා ථෙරී ගාථා
#“ප්‍රියවූ ගෙයද පුත්‍රයන්ද, ගව මහිෂාදීන්ද, අත්හැර රාගයද ද්වෙෂයද අත්හැර, අවිද්‍යාවන් පරදවා මුල් සහිත තෘෂ්ණාව උගුල්ලා දමා සන්සිඳුනෙමි. නිවුනෙමි.”
