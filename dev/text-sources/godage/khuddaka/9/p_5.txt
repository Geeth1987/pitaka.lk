ඡක්ක නිපාතය. 
# පඤ්ච සත ථෙරී ගාථා
# “මෙහි ආවාවූද, මින් ගියාවූද යමකුගේ මාර්ගය නොදනියිද? කොයින්දෝ ආවකුම මගේ පුතයි හඬති. 
# “මෙහි ආවාවූද, මින් ගියාවූද ඔහුගේ ගමන් මග දනු නම් ඔහුට ශොක නොකරව. සතුන්ගේ ස්වභාවය මෙසේය. 
# “නොඉල්ලන ලදුව එයින් මෙහි ආය. එසේම නොදන්වා කොහේදෝ සිට අවුත් කීප දවසක් මෙහි වැසමින් ගියේය. 
# “මෙයිනුත් අන්භවයකට ගියේය. ඉනුත් අන් තැනකට යෙයි. මළ තැනැත්තේ මනුෂ්‍යාදී රූපයෙන් ඔබ මොබ හැසිරෙමින් යන්නේය. මෙහි ආලෙසම එසේ ගියේය. එහි කවර නම් වැලපීමක්ද? 
# “මගේ හෘදය ඇසුරු කළ දුකසේ දැක්කයුතු දුක් උල ඉගිලවීය. ශෝකයෙන් පීඩිත මගේ ඒ දුහිතෘ ශොකය දුරු කෙළේය. 
# “ඒ මම උපුටා කළ හුල් ඇත්තී, තෘප්තියට පැමිණියා සපුරා නිවුනෙම්ද මුනින්වහන්ස අද බුදුන් දහම් සඟුන් සරණ කොට යමි.” 
# වාසෙට්ඨි ථෙරී ගාථා
# “පුත්‍ර ශෝකයෙන් පීඩිතවූවා අවුල් සිත් ඇත්තී එයින්ම පිරිසිදු සිහිනැත්තී නිර්වස්ත්‍රවූවා විසිරුණු කෙස් ඇත්තී මම තැනින් තැන හැසුරුණෙමි. 
# “කසල ගොඩවලද, සොහොනෙහිද, වීථිවලද විසීමි. බඩසා පවසින් පෙළෙමින් තෙවසක් හැසුරුණෙමි. 

# “නොදැමුණවුන් දමනය කරවා සියල්ල දත් කිසිවකින් භයක් නැත්තාවූ, ශොභන ගමන් ඇත්තාවූ මිථිලාව බලා වඩින්නාවූ බුදුන් දුටිමි. 
# “ප්‍රකෘති සිහිකල්පනා ලැබ වැඳ උන්වහන්සේ වෙතට පැමිණියෙමි. ඒ ගෞතමයන් වහන්සේ අනුකම්පාවෙන් මට දහම් දෙසූසේක. 
# “උන්වහන්සේගේ ධර්මය අසා ශාසනයෙහි මහණවීමි. බුදුන්ගේ වචනයෙහි යෙදී නිර්වාණ පදය අත්කෙළෙමි. 
# “මේ අන්තකොට ඇත්තාවූ සියළු ශෝකයෝ සිඳින ලදහ. යමෙකින් ශොකයන්ගේ පහළවීම වේද, ඒ වස්තුව මා විසින් පිරිසිඳ දන්නාලදී.” 
#
# ඛෙමා ථෙරී ගාථා
# “නුඹත් රූමත් තරුණියෙකි. මම ඒ එසේම පැහැපත් තරුණයෙකි. පසඟතුරු වැයුමෙන් යුක්තව සිත් අලවමු. ඛෙමාවෙනි එන්න. 
# “ලෙඩවූ වහා බිඳෙනසුළුවූ කුණු කය කරණකොට ගෙන පීඩාවෙමි. ලජ්ජාවෙමි. කාම තෘෂ්ණාව නසනලදී. 
# “කාමයෝ ශරීරයන් බිඳීමට සැතක් හා හුලක් වැනියහ. තා කියන කාමරතිය මට නම් දැන් අරතියෙකි. 
# “සියල්ලෙහි තෘෂ්ණාව වනසන ලදී. මෝහ රාශිය පලන ලදී. වාමකය, තෝ නසනුලැබීය. මෙසේ දනුව. 
# “අඥාණයෝ චන්ද්‍ර සූර්යාදී තාරකාවන්ට වඳිමින් වනයෙහි ගිනි දෙවියන් පුදමින් තත්වාකාරය නොදැන සුද්ධිය නොදනිත්. 
# “මමද පුරුෂොත්තමවූ සම්බුදුන් වඳිමින් සියලු දුකින් මිදුනෙමි. ශාස්තෘ ශාසනය කරන්නියක් වීමි.” 
#
# සුජාතා ථෙරී ගාථා
# “අලංකාරව, මහඟු වස්ත්‍ර හැඳ, මල්දම් පැළඳ, සඳුන් හා සියළු ආභරණයන් ගවසා දාසී සමූහයා පිරිවරණ ලදුව, 
# “බොහෝ ආහාරද, ඛාද්‍යයද භොජ්‍යයද ගෙන්වා ගෙණ ගෙයින් නික්ම උයනට පැමිණියෙමි. 

# “ඒ උයනෙහි සිත් අලවා, සෙල්ලම්කර සිය ගෙට එන්නී. සාකේත නුවර අඤ්ජන වන විහාරය බලන්නට පිවිසියෙමි. 
# “එහි වැඩහුන් ලොවට පහනක්වූ සර්වඥයන් වහන්සේ දැක වැඳ උන්වහන්සේ වෙතට පැමිණියේය. ඒ ඇස් ඇති සර්වඥයන් වහන්සේ අනුකම්පාවෙන් මට ධර්ම දේශනා කළසේක. 
# “මහර්ෂිවූ භාග්‍යවතුන් වහන්සේගේ ධර්මය අසා සත්‍යයන් තේරුම් ගතිමි. එහිදීම පහවූ රාග රජස් ඇති නිර්වාණ පදය ස්පර්ශ කෙළෙමි. 
# “ඉන්පසු දහම්දත් මම සර්වඥ ශාසනයෙහි පැවිදිවීමි. තුන් දැකීමට පැමිණියා, සර්වඥ ශාසනය නොසිස්කොට දකිමි. 
#
# අනොපමා ථෙරී ගාථා
# “බොහෝ සම්පත් ඇති මහත් ධන ඇති උසස්වූ කුලයෙක උපන්නෙමි. පැහැයෙන් රූපයෙන් සම්පූර්ණවූ මම මේඝ නම් සිටුහට දුවක් වීමි. 
# “රාජපුත්‍රයන් විසින් පතනලදුව, සිටුපුත්‍රයන් විසින් කැමතිව මට අනොපමාව රඳවයි කියා මපියාණන් වෙතට දූතයන් එවූහ. ඔබේ දූ වූ මේ අනොපමාව යම් පමණකට කිරා බැලීමෙන් බරද, ඉන් අටගුණයක් රන් මැණික් දෙමි. 
#
# “ඒ මම ලොවට උසස්වූ අත්‍යුත්තම බුදුන් දැක ඒ බුදුන්ගේ සිරි පතුල් වැඳ එකත්පසෙක හුන්නෙමි. 
# “ඒ ගෞතම සර්වඥයන් වහන්සේ අනුකම්පාවෙන් මට දහම් දෙසූ සේක. හුනස්නෙහිම අනාගාමී ඵලය ලදිමි. 
# “ඉන්පසු ඉස කෙස් කපාහැර සස්නෙහි පැවිදිවීමි යමකින් තෘෂ්ණාව වියලා අද මේ සත්වන රැය වෙයි.” 
#
# මහා ප්‍රජාපතීගොතමී ථෙරී ගාථා
#“යමෙක් මා සසර දුකින් මිදූද, අන්‍ය සත්වයිනුත් සසර දුකින් මිදූද, සියළු සත්වයන්ට උතුම්වූ බුද්ධ වීරයන්වහන්ස, ඒ ඔබට නම ස්කාරය වේවා, 
# “සියළු දුක මා දැනිණි, සසරට හේතුවූ තෘෂ්ණාව වියලනලදී. ආර්ය අෂ්ටාංගිකමාර්ගවූ නිවන ස්පර්ශ කළාය. 
# “පෙර මවුව, පුත්ව, පියාව, සහෝදරයාව පසුව මිත්තණියද විය. මෙලෙස ඇත්ත නොදැන නොලැබ සසර සැරීමි. 
# “මා විසින් ඒ භාග්‍යවතුන් වහන්සේ දක්නාලදී. මේ අන්තිම ශරීරයයි. ඉපදෙමින් හැසිරීම ක්ෂය විය. දැන් මතු ඉපදීමක් නැත. 
# “පටන්ගත් වීර්ය ඇති. කර්ම ක්ෂය කළ දැඩි පැරකුම් ඇති සමගිවූ ශ්‍රාවක සමූහයා දක්නෙම්ද, මේ බුදුවරයන්ට කරණ වන්දනාය. 
# “මහාමායා දේවීතොම බොහෝදෙනාට ඒකාන්ත වැඩ පිණිස ගෞතමයන් ඉපදවී, උන්වහන්සේ ව්‍යාධි මරණ වෙළුණවුන්ගේ දුක්ඛ රාශිය දුරුකළසේක. 
#
# ගුත්තා ථෙරී ගාථා
# “ගුත්තාවෙනි, දරුවන්ද, ඥාති සමූහයා, සම්පත්ද, හැර යමක් පිණිස පැවිදිවූයේද, ඒ වඩව සිතට වසඟ නොවව්. 
# “සිතින් රැවටුනු සත්වයෝ මාරවිෂයෙහි ඇළුනාහු නොයෙක් ඉපදීම් ඇති සසර දිව ඇවිදිත්. 
# “කාමයෙහි රුචියද ක්‍රොධයද ආත්මදෘෂ්ටියද ශීලව්‍රත පරාමර්ශනයද විචිකිත්සාවද යන, 
# “මෙතෙරට ගෙනෙන මේ බැඳුම් නසා, භික්ෂුණිය මෙයට නැවත නොපැමිණෙව. 
# “ඇල්ම, මානය, උඩඟුව, මුලාව යන බැඳුම් සිඳ දුක් කෙළවර කරන්නෙහිය. 
# “සසර හැසිරීම නිමවා, නැවත ඉපදීම දැන මේ දිවියේම සෑහී සන්සුන්ව වාසය කරව.” 
#
# විජයා ථෙරී ගාථා
# “සිත වසඟ නොවන කල්හි සිත එකමුතුවක් නොලැබ, නවවරක් විහාරයෙන් බැහැර ගියෙමි. 
# “භික්ෂුණිය, වෙතට පැමිණ සකසා ධර්මය විචාළෙමි.  ඕතොම ධාතු ආයතන යුතු ධර්මය මට දෙශනා කළාය. 
#
# “ඇගේ වචනය අසා ඒ අනුශාසනාව කරන්නී රෑ මුල් යාමයෙහි පෙර ජාති සිහි කෙළෙමි. රාත්‍රි මධ්‍යම යාමයෙහි දිවැස ලදිමි. පශ්චිම යාමයෙහි අඳුරු කඳ නැසීමි. 
#
# “එකල්හි ප්‍රීති සැපය ශරීරය පුරා පතුරවා වාසය කෙළෙමි. අඳුර නසා සත්වැනි දවස පය දිගහැරියෙමි.”
