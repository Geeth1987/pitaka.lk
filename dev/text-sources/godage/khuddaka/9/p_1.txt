දුක නිපාතය 
# අභිරූප (ඉතා රූමත්) නන්දා ථෙරී ගාථා
#“නන්දාවෙනි, ජරාවෙන් ලෙඩවූ කුණුවලින් අපවිත්‍රවූ කුණුවූ මේ ශරීරය බලව.” 
# රාගාදී වශයෙන් නොගෙන භාවනා කරව. ඉතිරි සහිත මානය දුරුකරව. එසේ මානය නැතිකිරීමෙන් සන්සුන්ව හැසිරෙව. භාග්‍යවතුන් වහන්සේ නන්දා නම් හික්මෙන භික්ෂුණියට මේ ගාථාවලින් නිතර අවවාද කරති.” 
# ජෙන්තී ථෙරී ගාථා
#“නිවන් දැකීමට මාර්ගයවූ යම් මේ බොජ්ක්ධංග හතක් වෙත්ද, බුදුන් වදාළ පරිදි ඒ ධර්මයෝ මා විසින් පුරුදු කරණ ලදහ.” 
# මා විසින් ඒ භාග්‍යවතුන් වහන්සේ දක්නා ලදී. මේ අන්තිම ශරීරයය. පුනරුත්පත්තිය ක්ෂය විය. නැවත ඉපදීමක් නැත.” 
# සුමංගල මාතා ථෙරී ගාථා
#“යහපත්ව කෙලෙසුන්ගෙන් මිදුනු තැනැත්තිය මොහොලින් වී කෙටීමෙන්ද ලජ්ජා භය නැති ස්වාමියාගෙන්ද පිළිකුල් බට කර්මාන්තයෙන්ද, පිළුණු ගඳ ගහන මගේ සැළියෙන්ද මිදුනෙමි. 
# මම රාගයද ද්වෙෂයද චිව්විට චිව්විට යන අනුකරණයෙන්ම නසමි. ඒ මම ගසමුලට පැමිණ අහෝ සැපයයි සැප වශයෙන් භාවනා කරමි.” 
#
# අඞ්ඪකාසී ථෙරී ගාථා
# “කසී රටින් ලැබෙන බද්ද යම්තරම් වීද මට ලැබෙන පඩිය එපමණ විය. නියම ගම් වැසි මහජනතෙමේ වටිනා මා අඩකින් සීමා කළේය. 
# “නැවත මම රූපයෙහි කලකිරුනෙමි. එහි නොඇලුනෙමි. නැවත සසර සැරීමක් නැත. ත්‍රිවිද්‍යාවන්ට පැමිණෙන ලදී. බුද්ධාවවාදය කෙළෙමි.” 
#
# චිත්තා ථෙරී ගාථා
# “ඉදින් මම ජරාවෙන් දිරා කෘශවීමි. ලෙඩව ඉතා දුර්වලව සැරයටියේ ආධාරයෙන් ඔබ මොබ යමි. 
# දෙපට සිවුර හා පාත්‍ර පසෙක තබා පර්වතයෙහි හිඳ ආත්මභාවය යටපත් කෙළෙමි. මොහ රාශියද දුරු කළෙමි.”
#
# මෙත්තිකා ථෙරී ගාථා
#“ඉදින් මම දුක් ඇතිව, දුර්වලව, ඉක්මගිය තරුණ වයස ඇතිව, සැරයටියේ ආධාරයෙන් යෙම්ද? 
# “පර්වතයට නැගී දෙපට සිවුර පසෙක තබා පාත්‍රයද තබා පර්වතයෙහි හුන්නෙමි. ඉක්බිති මාගේ සිත කෙලෙසුන්ගෙන් මිදුනේය. ත්‍රිවිද්‍යාවන්ට පැමිණියෙමි. බුද්ධාවවාදය කරණ ලදී.” 
#
# මෙත්තා ථෙරී ගාථා
# “පක්ෂය පිළිබඳ තුදුස්වක් දිනද, පසළොස්වක් දිනද, අටවක් දිනද, ප්‍රාතිහාර්ය කාලයෙහිද, දිව්‍ය සමූහයා සතුටු වන අටසිල් සමාදන්වීමි. 
# “ඒ මම බත් බුදින කාලය පමණ ස්වල්ප වේලාවක් හිස මුඩුකොට දෙපට සිවුර පෙරවීමි. හිතේ කෙලෙස් පිපාසය සන්සිඳුවා දිව්‍ය සැපය ප්‍රාර්ථනා නොකරමි.” 
#
# අභය මාතා ථෙරී ගාථා
# “මෑණියෙනි, පත්ලෙන් උඩ හිසකේ වැටියෙන් යට මේ ශරීරය අපවිත්‍ර වශයෙන් කුණු ගඳගහන්නක් වශයෙන් කල්පනා කරව. 
# “මෙසේ වාසය කරන්නාවූ මාගේ සියලු රාගය නැති විය. දැවිල්ල සන්සිඳුනේය. සිහිල්ව නිවුනෙමි.” 
#
# අභයා ථෙරී ගාථා
# “අභයාවෙනි, පෘථග්ජනයා යම් තැනක ඇලේද මේ ශරීරය වහා බිඳෙන සුළුය. යහපත් නුවණ හා සිහි ඇතිව මේ ශරීරයට ආශා දුරලමි. 
# බොහෝ දුකින් පෙළෙන අප්‍රමාදයෙහි ඇලුණු මගේ තෘෂ්ණාව පහවිය. බුදුන්ගේ අවවාදය කළෙමි.” 
#
# සාමා ථෙරී ගාථා
# “වීර්යයේ සමාන බවක් නැති හෙයින් සිතට සහනයක් නොලැබ නව වරක් විහාරයෙන් පිටතට ගියෙමි. ඒ මම අටවෙනි රාත්‍රියෙහි තෘෂ්ණාව නැසීමි. 
# “ප්‍රමාද නොවීමෙහි ඇලුනාවූ මට බොහෝ දුක් ස්වභාවයන්ගෙන් තණ්හා නැතිකරන්නට හැකිවිය. බුදුන්ගේ අවවාදය පිළිපදින ලදී.”
