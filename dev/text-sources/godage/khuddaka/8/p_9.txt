දසක නිපාතය. 
# කාළුදායී ථෙර ගාථා
# “පින්වතුන් වහන්ස, ඵල කැමැත්තහුට පරණ පත් සිඳීයාමෙන් අළුත් පත් හා ගෙඩි ඇති, ගින්නක් මෙන් බබලන වෘක්ෂයෝ වෙත්ද මහාවීරයන් වහන්ස, මේ අවස්ථාව එබඳු අවස්ථාවකි. 
# “වීරයන් වහන්ස, හාත්පස සියලු දිශාවන්හිම පිපුණු මල් ඇති වෘක්ෂයෝ වෙත්. ඵල නෙලාගනු කැමැත්තවුන්ට ඉතා සුදුසු කාලයෙකි. ඒ නිසා ඔබවහන්සේගේ වැඩමවීමට ඉතා සුදුසුය. 
# “පින්වතුන් වහන්ස, ඒකාන්තයෙන්ම අති ශීතත් නොවූ, අති උෂ්ණත් නොවූ සැපදායක ඍතු ගුණයක් පවතී. ඒ මඟ රෝහිණී ගංගාව පසුමුඛකොට වඩින්නාවූ ඔබවහන්සේට ශාක්‍ය කොළිය රජදරුවෝද දැක්ක හැකිය. 

# “ගොවියෝ ආශාවෙන් සී සාත්. ආශාවෙන් බිජු වපුරති. ධන ලබනු කැමැත්තහු සමුද්‍රය තරණයකොට වෙළඳාම් කරත් මමද ඔබවහන්සේගේ කපිලවස්තුපුර ගමන බලාපොරොත්තුවෙන් මෙහි සිටිමි. මගේ ආශාවද සමෘද්ධ වේවා. 
# “ගොවියෝ නැවත නැවතත් බිජු වපුරත්. වස්ස වලාහකතෙම වර්ෂාව නැවත නැවත පවත්වයි. සීසාන්නෝ නැවත නැවතත් සී සාත්. මිනිසුන් විසින් උපදවනලද ධාන්‍ය රටවලට යවත්. 
# “යාචකයෝ නැවත නැවත කුලයන්හි හැසිරෙත්. දානපතියෝ නැවත නැවතත් දන්දෙත්. දානපතියෝ නැවත නැවත දීමෙන් නැවත නැවත දෙව් සැප ලබත්. එහෙයින් මමද නැවත නැවත යාඥා කරමි. මගේ අදහසද ඉටුවේවා. 
# “වීර්යවන්තවූද, මහාප්‍රඥා ඇත්තාවූද පුරුෂයෙක් යම් කුලයෙක්හි උපදී නම් සත් පරපුරක් සම්‍යක් ප්‍රතිපත්තියෙන් පිරිසිදු කරයි. දේවාතිදේවවූ මුනිවරයෙක් හැටියට උපන් ඔබවහන්සේ සත්වයින්ට ඒකාන්ත හිතභාවය පිණිස පවතින නිසා එහි වැඩමවීම මම බලාපොරොත්තු වෙමි. 
# “මහර්ෂීවූ ඔබවහන්සේගේ පියා සුද්ධොදන රජුය. බෝධිසත්වවූ ඔබවහන්සේගේ කුසින් පරිහරණය කළ යම් මවක් වේනම් ඒ මහාමායා බිසව මරණින් මතු තුසිත දෙව් ලොව සතුටින් සිටින්නීය. 
# “ඒ ගෞතම ගෝත්‍රයෙහි උපන්නාවූ  ඕතොමෝ මෙයින් චුතව දිව්‍යමයවූ කාමයෙන් යුක්තව දිව්‍ය සමූහයා විසින් පිරිවරණලදුව පස්කම් සැප විඳී. 
# “අනිකෙකු විසින් අභිභවනය කළ නොහැකි අනුපමෙය්‍යවූ (උපමා කළ නොහැකි) අඟපසගින් ආලෝකය නික්මෙන්නාවූ තාදී ගුණයෙන් යුක්තවූ බුදුරජාණන් වහන්සේගේ මම පුත්‍රයෙක් වෙමි. ගෞතම ගොත්‍රයෙහි උපන් ශාක්‍ය රජතුමනි, තොපගේ පුත්‍රවූ බුදුරජාණන් වහන්සේ මගේ පියා වන්නේය. ස්වභාව වශයෙන් උන්වහන්සේ තොපටද පියා වන්නේය.” 
#
# එකවිහාරීය ථෙර ගාථා
# “තනිව වනයෙහි වසන්නා ඉදිරිපිට හා පසුපාර්ශවයෙහි අනිකෙක් නොමැති බැවින් බොහෝවූ චිත්ත සුඛයක් (සතුටක්) ඇතිව වෙසෙයි. 
# “බුදුන් විසින් වර්ණනා කරනලද වනයට මම තනිව යමි. නිවන් පිණිස මෙහෙයූ සිත් ඇති තනිවූ භික්ෂුවට සැප විහරණයක් ඇතිවිය. 
# “යෝගීන්ට ප්‍රීතිකරවූද, රම්‍යවූද මත් ඇතුන් හැසිරෙන්නාවූද, වනයක් වේ නම් ශ්‍රමණ ධර්මය පිණිස එහි වහා පිවිසෙමි. 
# “මනාකොට පිපියාවූ ගස් ඇති කඳු අතර සිහිල් ජලය ඇති ස්ථානයන්හි ස්නානයකොට හුදකලාව හැසිරෙමි. 
# “සහායවූ දෙවැන්නෙක් නැතුව රමණීය මහවනයෙහි ආශ්‍රව රහිතව කරණලද කිස ඇතුව මම කවදා වසම්ද? 
# “අරණ්‍යගතව භාවනා කරන්නහුට අදහස් සම්පූර්ණ වේද, මම ඒ අදහස මුල්කොට කටයුතු කරමි. අනිකෙකුට කරන අනිකෙක් නැත්තේය. 
# “ඒ මම වීර්යය නම්වූ සන්නාහය බඳිමි. වනයට පිවිසියෙමි. ආශ්‍රව දුරුනොකොට ඒ වනයෙන් නොනික්මෙන්නෙමි. 
# “සිහිල් වාතය හා සුවඳ හමන්නාවූ වනයෙහි වෙසෙමින් එහිම හුන්නේ අවිද්‍යාව නසමි. 
# “මලින් ගැවසීගත්තාවූද, සිහිල් පර්වත ඇත්තාවූද, පර්වත වලින් වටකරන ලද්දාවූද, වනයෙහි විමුක්ති සැපයෙන් වෙසෙමි. 
# “ඒ මම සම්පූර්ණ චන්ද්‍රයා මෙන් පරිපූර්ණ චිත්ත සංකල්පනා ඇතිව සියලු කෙලෙසුන් නසා සැපසේ වෙසෙමි. මට නැවත ඉපදීමක් නැත්තේය.” 
#
# මහාකප්පින ථෙර ගාථා
# “යමෙක් තමා කරා නොපැමිණි හිත අහිත යන දෙක පළමුකොට දකීද,  ඕහට අමිත්‍රයෝ හා මිත්‍රයෝ දොස් සොයන්නවුන් මෙන් දකී. එබඳු පුද්ගලයා නිවැරදි පුද්ගලයෙකි. 

# “යමෙකුට ආනාපානසති භාවනාව පරිපූර්ණද මනාකොට වඩන ලදද, බුදුන් විසින් දෙශනා කරනලද ඒ භාවනාව පිළිවෙළින් පුරුදුකරන ලද්දේද, ඒ යෝගාවචර තෙම උපක්ලේශ රහිත චන්ද්‍රයා මෙන් ලෝකය බබුලුවයි. 
# “මාගේ සිත නීවරණ නමැති මල නැතිවීමෙන් පිරිසිදුය. අප්‍රමාණ නීවරණ ධර්ම ප්‍රත්‍යක්ෂ කෙළෙමි. සියලු කෙලෙසුන් නැසුවෙමි. සියලු දිසාවන් බබුලුවන්නෙමි. 
# “වස්තුව නැතිකල්හිද ප්‍රඥාව ඇති පුද්ගලයා නිරවද්‍ය වූ ජීවිකාවෙන් සන්තොෂවේ. වස්තුව ඇතිකල්හිද ප්‍රඥාව නැති පුද්ගලයා සැපවත් ජීවිතයක් නොලබයි. 
# “ඇසුනාවූ කරුණු විනිශ්චය කරන යම් ප්‍රඥාවක් වේද, කීර්ති ප්‍රශංසා ලබන ප්‍රඥාවකුත්වේද; ඒ ප්‍රඥාවන්ගෙන් යුක්තවූ පුද්ගලයා ඒකාන්ත දුක් ඇති ස්කන්ධාදියෙන් යුක්තවුවද සම්‍යක් ප්‍රතිපත්තිය නිසා සැප ලබයි. 
# “සත්වයින්ගේ දිරීම මැරීම් ආදී ස්වභාව ධර්මයක් වේද, එය අළුත් දෙයක් නොවේ. ආශ්චර්ය අද්භූත දෙයක්ද නොවේ. ඒ කුමක් නිසා අද්භූත වන්නේද 
# “උපන්නාවූ තැනැත්තාට ජීවිතයට අනතුරුව මරණය ඒකාන්තයි. මෙහි උපදින උපදිනු ලබන්නාහු මරණයට පැමිණෙති. සත්වයින්ට මේ ධර්මය ඒකාන්තයි. 
# “යම්බඳුවූ අනුන්ගේ ජීවිත මළහුගේ ප්‍රයෝජනයට නොපැමිණෙයි. මැරුණු කල්හි ඇසීමෙන් ප්‍රයෝජනයක් නොවේ. එයින් මළ තැනැත්තා යස එෛශ්චර්ය සම්පත්තියක්ද නොලබයි. එමනිසා මහණ බමුණෝ ඇඬීම් ආදිය හොඳයයි වර්ණනා නොකරති. 
# “අඬන්නාවූ පුද්ගලයාගේ ඇස් හා ශරීරය වෙහෙසට පැමිණෙයි. ශරීර වර්ණය හා බලයද, ප්‍රඥාවද පිරිහෙයි. විරුද්ධකාරයෝ සතුටු වෙති. හිතෛෂීහු දුකට පත්වෙති. ඔහුට ඒ කිසිවකින් සැපතක් සිදුනොවෙයි. 

# “එම නිසා කුලයෙහි වසන බහුශ්‍රැත ඤාණවන්ත පුද්ගලයා මහොඝයෙන් (ජලය අධික වීමෙන්) පිරුණාවූ නදියෙහි නැවත තරණය කරන්නාක්මෙන් ප්‍රඥාවෙන් තමාගේ කටයුතු කිරීමට කැමැති වෙයි.” 
#
# චූලපන්ථක ථෙර ගාථා
# “මෝඩ ගතිය නිසා මම මුල් අවස්ථාවෙහි නින්දා ලැබුවෙමි. තොප ගෙදර යව කියා සහෝදරයාද පිටමන් කෙළේය. 
# “සහෝදරයා විසින් පිටමන් කරණලද ඒ මම සංඝාරාම දොරකඩ සමීපයෙහි දොම්නසින් ශාසන අපෙක්ෂාව ඇතුව සිටියෙමි. 
# “එකල්හි භාග්‍යවතුන් වහන්සේ මා කරා පැමිණි සේක. මගේ හිසද උන්වහන්සේ විසින් පිරිමදිනලදී. අතින් අල්ලා මා රැගෙන සංඝාරාමයට පිවිසිසේක. 
# “මට ශාස්තෲන් වහන්සේ අනුකම්පාකොට රෙදි කැබෙල්ලක් දුන්සේක. එකත්පස්ව සිට මෙහි පිරිසිදුකම අධිෂ්ඨාන කරව්. 
# “මම උන්වහන්සේගේ වචනය අසා ශාසනයෙහි ඇල්ම ඇත්තෙක්ව උත්තමාර්ථය පිණිස පවතින සමාධිය ඉපදවීමි. 
# “පෙර විසූ කඳ පිළිවෙළ දැනගතිමි. දිවැස පිරිසිදු කෙළෙමි. ත්‍රිවිද්‍යාව ලදිමි. බුදුන්ගේ අනුශාසනාවද සම්පූර්ණ කෙළෙමි. 
# “පන්ථක ස්ථවිරතෙමේ තමා දහස්දෙනෙක් කොට මවා කල් පැමිණෙනතුරු රම්‍යවූ අඹවනයෙහි හුන්නේය. 
# “කල් දැන ශාස්තෲන් වහන්සේ මා කරා දූතයෙකු එවූසේක. මමද කල් දැන අහසින් එහි ගියෙමි. 
# “මම ශාස්තෲන් වහන්සේගේ පාද වන්දනාකොට එකත්පසෙක සිටියෙමි. එසේ සිටියාවූ මා ශාස්තෲන් වහන්සේ විසින් පිළිගන්නාලදී. 
# “සියලු ලෝකයා විසින් පිං බලාපොරොත්තුවෙන් දෙනුලබන දක්ෂිණාව පිංකෙතක්ව සිට මමද පිළිගතිමි.” 
#
# කප්ප ථෙර ගාථා
# “නානාවිධ අසුන් කොට්ඨාශයන් ගෙන් යුක්තවද මහ වැසිකිළි වලක් හා සමානවූද, පැසෙන ගවරවලක් වැනිවූද, මහ ගඬුවක් වැනිවූද, මහ වනයක් වැනිවූද මේ ශරීරය. 
# “ලේ සැරවින් හා අශුචියෙන් පිරුණේ හැමකල්හි වැගිරෙන ස්වභාව ඇත්තේ මේ ශරීරය පිළිකුල් බවට පත්වෙයි. 
# “මහා නහරවැල් වලින් බඳනාලද්දේ මාංශාලේපයෙන් අලවන ලද්දේ සමින් වසනාලද්දේ නිරර්ථකවූ මේ කුණුකය, 
# “ඇටකෝටුවලින් ගටනාලද්දේ සුලු නහරවැල්වලින් බඳනාලද්දේ යන්ත්‍රයක් මෙන් ඉරියව් පවත්වයි. 
# “මරණය කරා ගමන්කරනුයේ මරහුගේ සමීපයෙහිම සිටිනුයේ මොහු තමාගේ ශරීරය හැර අන් තැනකට පැමිණේ. 
# “මේ ශරීරය අවිද්‍යාවෙන් වසනලදුව සතර ආකාර ගැට වලින් ගැටගසනුලැබූ සතර සැඩවතුරෙන් ගසායනලදුව අනුසය දැලින් යටපත්කරන ලදුව, 
#“පඤ්චනීවරණ ධර්මයන්ගෙන් යුක්තව කාමවිතර්කාදියෙන් යුක්තව තෘෂ්ණා නමැති භවමූලයෙන් බඳිනලදුව මෝහ වැස්මෙන් වසනලදුව මේ ශරීරය පවතී. 
# ‘කර්ම නමැති යන්ත්‍රයෙන් සම්පත්ති විපත්ති ආදී නානා භව වලට පැමිණේ. 
# “යම්බඳුවූ අන්ධ බාල පෘථග්ජනයෝ මෙම ශරීරයට මමත්වය කෙරේද, කටසී නම්වූ සංසාරය වඩත්. නැවත නැවත උපදිත්. 
# “යමෙක් ගූථගත සර්පයෙකු මෙන් මේ ශරීරය දුරු කෙරේද, ඔහු අවිද්‍යා තෘෂ්ණා දෙක හැරදමා කෙලෙස් නැති වීමෙන් පිරිනිවන් පාන්නාහුය.” 
#
# උපසෙන ථෙර ගාථා
# “ජනයාගෙන් වෙන්වූ, නිශ්ශබ්දවූ, වල්සතුන් විසින් සෙවුනාලද සෙනසුන භික්ෂුව විසින් හිත මනාකොට පිහිටුවා ගැණීමට සේවනය කටයුතුය. 

# “කසල ස්ථානයෙන් හෝ සොහොන් පිට්ටනියෙන් හෝ වෙන අන් තැනකින් හෝ ගනුලබන රෙදි කැබලිවලින් මසාගන්නාලද රූක්ෂ චීවරයක් දැරිය යුතුයි. 
# “නිහතමාන සිතින් යුක්තව පිළිවෙළින් භික්ෂුතෙමේ වසනලද සදොර ඇතුව මනා සංවරයෙන් යුක්තව පිඬු පිණිස හැසිරිය යුතුය. 
# “ස්වල්පවූ දෙයින්ද සතුටු වියයුතුය. අන්‍යවූ මිහිරි බොහෝ දේවල් බලාපොරොත්තු නොවිය යුතුය. රසයෙහි ගිජුවූ පුද්ගලයාගේ සිත ධ්‍යානයෙහි නොඇලේ. 
# “ඥානවන්ත පුද්ගලයා ස්වල්ප ආශා ගුණයෙන්ද, යථාලාභාදී වශයෙන් පවත්නා සන්තොෂ ගුණයෙන්ද, හුදකලා වාසයෙන්ද වෙසෙයි. ඒ ඥානවන්ත පුද්ගලයා ගිහියන් හා පැවිද්දන් සමගත් එකතුවීම් නැති වූයේ වෙසෙයි. 
# “තමා ගොළු බිහිරි නොවූයේම, ගොළු බිහිරි භාවය පෙන්නුම් කළ යුතුය. පණ්ඩිතතෙම සංඝයා සමගද මද පමණට බිණියයුතුය. 
# “කියනලද පිළිවෙතින් යුක්තවූ භික්ෂුව කිසිවෙක්හට අපවාද නොකටයුතුය. කයින් වෙහෙසීමද නොකළයුතුය. ප්‍රාතිමොක්ෂ සංවරයෙන් යුක්තවිය යුතුය. ඔහු විසින් භොජනයෙහිද පමණ දැනගත යුතුය. 
# “මනාකොට ගන්නාලද නිමිති ඇත්තෙක්ව යහපත් සිතිවිලි ඉපදවීමෙහි දක්ෂයෙක්ව සමථ භාවනාවෙහි යෙදිය යුතුය. සුදුසු කාලයෙහි විදර්ශනා භාවනාද වැඩිය යුතුය. 
# “ඒ පණ්ඩිත භික්ෂුව නිතර පවත්නා වීර්යයෙන් යුක්තව හැමකල්හි භාවනානුයෝගීව විසිය යුතුය. දුක් කෙළවර නොකොට විශ්වාසයකට නොපැමිණිය යුතුය. 
# “මෙසේ විවේක සෙනසුන්හි වාසය කරන්නාවූද, ඥානදර්ශන විශුද්ධිය කැමැත්තාවූද, භික්ෂුවගේ සියලු ආශ්‍රවයෝ ක්ෂය බවට යෙත්. (නැතිවෙයි) හෙතෙම දෙයාකාරවූ නිවනටම පැමිණෙයි.” 
#
# ගොතම ථෙර ගාථා
# “ඤාණවන්ත පුද්ගලයා විසින් තමන්ගේ යහපත දැනගතයුතුය. බුදුන් විසින් වදාරණලද වචනයද බැලිය යුතුය. පැවිදිබවට පැමිණියාවූ කුලපුත්‍රයා එහිම සුදුස්සෙක්ව විසිය යුතුයි. 
# “මෙහි කල්‍යාණ මිත්‍ර ආශ්‍රයෙන් ශික්ෂා සමාදානය ලැබ ගුරුන්ට කීකරුව වෙසේ නම් මෙය ශ්‍රමණයාට සුදුසු වෙයි. 
# “බුදුනට ගරුකරන්නෙක්ද, ධර්මයට ගරුකරන්නෙක්ද සංඝයාට සම්මානකරන්නෙක්ද පිළිවෙලින් විය යුතුය. මෙය ශ්‍රමණයාට සුදුසුවෙයි. 
# “අගොචරය දුරුකොට ගෝචර සම්පන්නව පිරිසිදු ජීවිකාවෙන් යුක්තව ඥානවන්තයන්ගෙන් ගැරහීම් නොලබන්නෙක්ව සිත මනාකොට පිහිටුවා වෙසේ නම් මෙය ශ්‍රමණයාට සුදුසුයි. 
# “චාරිත්‍ර සීලයෙන් හා වාරිත්‍ර සීලයෙන් යුක්තව පැහැදීම ඇති කරන ඉරියා පථයෙන්ද යුක්තව සමථ විදර්ශනා භාවනා කළයුතුය. මෙය ශ්‍රමණයාට සුදුසුයි. 
# “ඥානවන්තයා විසින් විවේක වූද, ශබ්ද නැත්තාවූද වන සෙනසුන් ඇසුරුකළ යුතුය. මෙය ශ්‍රමණයාට සුදුසුයි. 
# “සිල්වත් බව හා බහුශ්‍රැතබව ඇතිව රූප අරූප ධර්ම තත්වූ පරිදි විමසා නම් සත්‍යාවබෝධයවේ. මෙය ශ්‍රමණයාට සුදුසුයි. 
# “සියලු සංස්කාරයන්ගේ අනිත්‍ය බව හා අසුභබවද පුරුදු කළ යුතුය. ලෝක වස්තු පිළිබඳ නොඇලීමද පුරුදුකළ යුතුය. මෙය ශ්‍රමණයාට සුදුසුයි. 
# “බෝධ්‍යංග ධර්මයෝද, ඉර්ධිපාද ධර්මයෝද ඉන්ද්‍රිය ධර්මයෝද, බාල ධර්මයෝද, ආර්යඅෂ්ටාංගික මාර්ගයෝද වැඩිය යුතුය. මෙය ශ්‍රමණයාට සුදුසුයි. 
# “ඥානවන්තයා සියලුම තෘෂ්ණාව සිඳදැමිය යුතුය. සියලුම ආශ්‍රවයෝ මුල් සහිතව සිඳිය යුතුය. කෙලෙසුන්ගෙන් මිදුනාවූ ඒ පුද්ගලයා නිවන් අරමුණුකොට වාසය කරන්නේය. මෙය ශ්‍රමණයාට සුදුසුයි.”
