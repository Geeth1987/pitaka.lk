අට්ඨක නිපාතය. 
# මහා කච්චාන ථෙර ගාථා
#“මහා නව කර්මාන්ත නොකළ යුතුයි. ඇසුරු කිරීමෙන් කුසල් දහම් පිරිහීමට හේතු වන පුද්ගලයෝ දුරුකළ යුතුයි. කුලයාට සංග්‍රහ කිරීමට උත්සාහ නොකළ යුතුයි. මක්නිසාද? රසයෙහි ගිජුවූ හෙතෙම කුලයට උනන්දුවක් දක්වන හෙයිනි. සැපය බලාපොරොත්තුවන ඒ පුද්ගලයා අර්ථය හිස් කරයි. 
# “කුලයා විසින් කරන යම්බඳු වැඳුම් පිදුම් ඇද්ද, ඒ බුද්ධාදී උතුමෝ දියසෙවෙල් සහිත මඩක් කොට පෙන්නුම්කර වදාළහ. එයම දුරුකළ නොහැකි සියුම් ආයුධයක් හා සමානය. ලාභසත්කාරය නරක අදහස් ඇති පුද්ගලයාට දුරු නොකළ හැකිය. 
# “අන්‍යයාට වධබන්ධනාදිය අනුන් ලවා නොකර විය යුතුය. තමා විසින්ද, ඒ පව්කම් නොකළ යුතුයි. මක්නිසාද? සත්වයෝ කර්මය දායාද කොට ඇති හෙයිනි. 
# “තමන් සොරකම් නොකොට අනුන්ගේ වචන මාත්‍රයෙන් සොරෙකුනොවේ අනුන්ගේ වචනයෙන් මුනිවරයකුද නොවේ. තමාගේ පිරිසිදුකම හෝ අපිරිසිදුකම තමාම දන්නේය. පර සිත් දන්නා දෙවියෝද, ඒ බව දනිත්. 

# “කුසලාකුසල ඵල විපාක නොදන්නාවූ අන්‍යයෝ මෙලොවම සිටිනු මිස මැරෙන බව හා පරලොව යන බව නොදනිති. යම්බදු පණ්ඩිතයෝ වෙත්නම් ඔව්හු පමණක් මැරෙණ බව හා පරලොව යන බව දනිති. ඒ දන්නාවූ තැනැත්තෝ අනුන්ට හිංසා කිරීම් ආදිය නොකරත්. 
# “ප්‍රඥාවන්තතෙම වස්තුව විනාශවූ කල්හිත් ජීවිකාව පිණිස වැරදි නොකරයි. නුවණ නැති පුද්ගලයා මෙලොව පරලොව වරදවා වස්තුව ඇති කල්හිත් සැප නොලබයි. 
# “සියල්ල කණින් අසත්. සියල්ල ඇසින් දකිත්. ප්‍රඥාවන්ත පුද්ගලයා දිට්ඨ, සුත, මුතාදී, අරමුණු විෂයෙහි ඇලීම හා කිපීම ඇතිකර නොගනියි. 
# “දැකීමෙහි අන්ධයෙක් මෙන්ද, ඇසීමෙහි බිහිරෙක් මෙන්ද, දැකීමෙහි මෝඩයෙක් මෙන්ද, බලවත් බැව්හි දුර්වලයෙක් මෙන්ද, වෙයි. කළයුතු කටයුතු ඇති කල්හි මරඇඳෙහි සිටියද, ඒ කළ යුතුය. නොකටයුත්තෙහිදී මරනින්ද ලැබූවෙකු මෙන් නිසොල්මන් විය යුතුයි. 
#
# සිරි මිත්ත ථෙර ගාථා
# “නොකිපෙන්නාවූද, බද්ධවෛරයෙන් තොරවූද, රැවටිලි නැත්තාවූද, පිසුණුබස් කීමෙන් තොරවූවාවූද, භික්ෂුතෙමේ පරලොව ශෝක නොකරයි. 
# “නොකිපෙන්නාවූද, බද්ධ වෛරයෙන් තොරවූද, රැවටිලි නැත්තාවූද, පිසුණුබස් කීමෙන් තොරවූද, භික්ෂු තෙමේ හැමකල්හිම වසන ලද දොරටු ඇත්තේ එසේම පරලොව ශෝක නොකරයි. 
# “නොකිපෙන්නාවූද, බද්ධවෛරයෙන් තොරවූද, රැවටිලි නැත්තාවූද, පිසුණුබස් කීමෙන් තොරවූද, පිරිසිදු සිල් ඇති ඒ භික්ෂුතෙමේ එසේම ශෝක නොකරයි. 
# “නොකිපෙන්නාවූද, බද්ධ වෛරයෙන් තොරවූද, රැවටිලි නැත්තාවූද, පිසුණුබස් කීමෙන් තොරවූද, කල්‍යාණ මිත්‍රයන් ඇති ඒ භික්ෂුතෙමෙ එසේම ශෝක නොකරයි. 

# “නොකිපෙන්නාවූද, බද්ධ වෛරයෙන් තොරවූද, රැවටිලි නැත්තාවූද, පිසුණුබස් කීමෙන් තොරවූද, යහපත් ප්‍රඥාව ඇති ඒ භික්ෂුතෙම එසේම ශෝක නොකරයි. 
# “යමෙකුට බුදුන්කෙරෙහි අචලවූද, මනාකොට පිහිටියාවූද, ශ්‍රද්ධාවක්වේද, යමෙකුට ආර්යයන් විසින් කැමතිවන ලදද, ප්‍රශංසනීයවූද, සීලයකුත් වේද 
# “එසේම යමෙක්හට සංඝයා කෙරෙහි ප්‍රසාදයක්වේද, සෘජුවූ දැකීමකුත් වේද, ඒ පුද්ගලයා දිළින්දෙක් නොවේ. ඔහුගේ ජීවිතය හිස් නොවේ. 
# “ඒ නිසා බුදුන්ගේ අනුශාසනාව සිහිකරන්නාවූද, ඤාණවන්ත පුද්ගලයා ශ්‍රද්ධාවෙහිද, සීලයෙහිද, ප්‍රසාදයෙහිද, ධර්ම දර්ශනයෙහිද, යෙදිය යුතුයි. 
#
# මහාපන්ථක ථෙර ගාථා
# “යම්කලෙක්හි පළමුකොට නිර්භයවූ, ශාස්තෲන්වහන්සේ දැකී නම් පුරිසොත්තමවූ උන්වහන්සේ දැක සංවේගයක් උපණ. 
# “යමෙකු කරා ආවාවූ එෛශ්චර්ය සම්පත් අත්වලින් ගසා හැර දමද්ද, එෛශ්චර්යක් බඳුවූ ශාස්තෲන් වහන්සේ කරා පැමිණ අවවාද පැහැර හැරීම නොකරමි. 
# “ඉක්බිති මම අඹුදරුවන්ද, ධනධාන්‍යයද, හැර දැමීමි. කෙස් රැවුල් සිඳ අනගාරික ප්‍රව්‍රජ්‍යාවෙන් පැවිදිවීමි. 
# “ශික්ෂා සහිත ආජීවයෙන් යුක්තවූයේ ඉන්ද්‍රිය සංවර ඇතිව බුදුරජාණන් වහන්සේට නමස්කාර කරමින් අපරාජිතව (නොපැරදී) පසුවෙමි. 
# “එහෙයින් මගේ සිත පිළිබඳ කෙබඳු සිතක් ඇති පුද්ගලයෙක්ද යන බව කියතොත් තෘෂ්ණා නැමැති හුල ඇනීමෙන් මොහොතකුදු සැනසීමක් නොලබන්නෙකු වැනිය. 
# “එබඳු විහරණයක් ඇතුව සිටි මට වීර්ය පරාක්‍රමය නිසාවූ යහපත බලව, ත්‍රිවිද්‍යාවද ලදිමි. බුදුන්ගේ අනුශාසනාවද කෙළෙමි. 

# “පෙර කඳ පිළිවෙල දැනීමේ නුවණ (පූර්වෙනිවාසානුස්මෘති ඥානය) ලදිමි. දිව ඇස පිරිසිදු කෙළෙමි. රහත්වූ දක්ෂිණාර්හයෙක් වූයෙමි. කෙලෙසුන්ගෙන් මිදී උපධි රහිත නිර්වාණය ලදිමි. 
# “ඉක්බිති රාත්‍රිය ගතවීමෙන් හිර පායන කල්හි සියලුම තෘෂ්ණාව වියලවා පළඟ බැඳ හුන්නෙමි. 

#
#අටවෙනි නිපාතය නිමි.
