
සොළසක නිපාතය 
# අඤ්ඤාකොණ්ඩඤ්ඤ ථෙර ගාථා
# “ඒ රූපාදී කිසිවක් තෘෂ්ණා දෘෂ්ටි වශයෙන් නොගන්නාවූ විරාගය මුල් කොට දේශනා කරනලද යම් ධර්මයක්වේද, මහා රසයෙන් යුක්තවූ ඒ ධර්මය අසා බොහෝකොට පැහැදුනෙමි. 
# “මේ මිනිස් ලොව ඇතුළු සියලු ලෝකයන්හි විචිත්‍ර විලාශයෙන් පවත්නා බොහෝවූ අරමුණුවෙත්ද, ඒ අරමුණු නිසා ශුභාදි වශයෙන් ඇතිවෙන මිථ්‍යා සංකල්පනාවෝ මර්දනය කළාක් මෙන් සිටියහ. 
# “වාතයෙන් උඩට නගින්නාවූ රජස් මහා මේඝයෙන් සමනය වන්නේද, එසේම ආර්ය ශ්‍රාවකතෙම සියලු වැරදි කල්පනාවෝ ප්‍රඥාවෙන් දැකීම නිසා නිවැරදි කල්පනාවෙන් සමනය කරයි. 

# “සියලුම සංස්කාරයෝ අනිත්‍යයයි විදර්ශනා ප්‍රඥාවෙන් දකීද, එකල්හි සසර දුකෙහි කලකිරෙයි. විශුද්ධිය පිණිස එයම මාර්ගයද වෙයි. 
# “සියලුම සංස්කාරයෝ දුකයයි විදර්ශනා ප්‍රඥාවෙන් දකීද, එකල්හි සසර දුකෙහි කලකිරෙයි. විශුද්ධිය පිණිස එයම මාර්ගයද වෙයි. 
# “සියලුම භූමිතුනට අයත් ධර්මයෝ අනාත්මයයි විදර්ශනා ප්‍රඥාවෙන් දකීද, එකල්හි සසර දුකෙහි කලකිරෙයි. විශුද්ධිය පිණිස එයම මාර්ගයද වෙයි. 
# “බුදුන්ට අනුව අවබොධකළාවූද, දැඩි වීර්ය ඇත්තාවූද ඉපදීම හා මරණය දුරුකළාවූද, යම් කොණ්ඩඤ්ඤ ස්ථවිර නමක් වේ නම් උන්වහන්සේ මාර්ග බ්‍රහ්මචරියාව සම්පූර්ණ කළහ. 
# “යම්රාග පාසයකුත්වේද, (දැල) පැලීමට දුෂ්කරවූ පර්වතයක් මෙන් පවතින සිතට උල් ධර්මයකුත්වේද, ඒ දෙවැදෑරුම් ධර්මයන්ම සිඳ දමා අඥානකම නැමැති පර්වතය විදුරු නුවණින් බිඳදමා එතරවුනාවූද ධ්‍යානකරන්නාවූද ඒ පුද්ගලයා මාරබන්ධනයෙන් මිදුනේය. 
# “උඩඟු බැව් ඇත්තාවුද චපල බැව් ඇත්තාවූද, පාපමිත්‍රයන් ඇත්තාවූද, භික්ෂුව දිය රැළින් යටපත්කොට මහවතුරෙහි ගොඩවීමට නොහැකිව සිටියි. 
# “උඩඟු බැව් නැත්තාවූද, චපල බැව් නැත්තාවූද, කල්‍යාණ මිත්‍රයන් ඇත්තාවුද, සංවරකළ ඉඳුරන් ඇත්තාවූද, ප්‍රඥාවෙන් යුක්තවූවාවූද භික්ෂුව දුක් කෙළවර කරයි. 
# “කළුපුරුක් වැළකට බඳුවූ අවයව ඇතිව කෘශවූයේ. ඉපිලී ගිය නහරවැල් ඇත්තේ ආහාරයෙහි පමණ දැනගන්නා සුලුවුයේ කුසීත නොවූ සිත් ඇත්තේ පුරුෂයෙක් වූයෙහිය. 
# “මහාවනයෙහි මැසි මදුරු උවදුරු ලබමින් යුද්ධ භූමියකට එළඹි හස්තියෙක් මෙන් සිහියෙන් යුක්තව එහිම විසීමි. 
# [“බ්‍රහ්මයා යම්බඳුවූ ධ්‍යාන සැපයකින් වාසය කෙරේද, එසේම භික්ෂුවද විවේක සැපයෙන් වාසය කරයි. දෙවියෝ යම්බඳුවූ වාසයක යෙදේද භික්ෂුවද එබඳුවූ සහවාසයක යෙදෙයි. එසේම තුන්දෙනෙකුන් සමග විසීම ග්‍රාමවාසය වෙයි. ඊට වැඩිදෙනෙකුන් සමග වසනුයේ ජනසන්නිපාතයක් බඳුයි. ]
# “මරණයටද සතුටු නොවෙමි. ජීවත්වීමටද සතුටු නොවෙමි. මා විසින් ශාස්තෲන් වහන්සේ ඇසුරු කරන ලදී. උන්වහන්සේගේ අනුශාසනාවද සම්පූර්ණ කෙළෙමි. 
#
# “යම් අර්ථයක් නිසා ගිහිගෙයින් නොගිහි බැව් නම්වූ ප්‍රව්‍රජාවට පැමිණියෙම්ද, මා විසින් සියලු සංයෝජනයන් නසා ඒ අර්ථය සිද්ධකරන ලදී.” 
#
# උදායි ථෙර ගාථා
# “මනුෂ්‍යයෙක්වූද ආත්ම දමනයෙන් යුක්තවූද එකඟවූ සිත් ඇත්තාවූද ශ්‍රෙෂ්ඨ විහරණ ඇත්තාවූද නිවනෙහි ඇලුනාවූද යම් බුදුවරයෙක් වේද? 
# “සියලු ධර්මයන්ගේ පරතෙරට පැමුණුනාවූ ඒ බුදුරජාණන් වහන්සේට මනුෂ්‍යයෝ නමස්කාර කරත්. උන් වහන්සේට දෙවියෝද නමස්කාර කරත්. මේ වනාහි මම රහතුන්ගෙන් ඇසුවෙමි. 
# “සියලු සංයෝජනයන් ඉක්මෙව්වාවූද කෙලෙස් වනයෙන් නිවනට පැමිණියාවුද, කාමයෙන් නික්ම නෛෂ් ක්‍රම්‍යයෙහි ඇලුනාවූද පර්වතයෙන් වෙන්කොට ගත් රන්රුවනක් හා සමාන විය. 
# “නාගයෙක් සමානවූ බුදුරජාණන් වහන්සේ තමන්ගේ කැමැත්තට අනුවම ශොභාවත් විය. අන් පර්වත අභිභවනය කොට බබලන හිමාල පර්වතය මෙනි. නාගනාමය ඇති සියල්ලන්ටම වඩා සත්‍යවූ නාගනාමය ඇත්තේ නිරුත්තරවූ බුදුරජාණන් වහන්සේටය. 
# “තොපට නාගයා පෙන්නුම්කර දෙමි. පව් නොකරනුයේ හෙතෙම නාග නම් වේ. සීලයත් කරුණාවත් දෙක ඒ නාගයාගේ ප්‍රථම පාදයෝය. 

# “සිහියත්, ප්‍රඥාවත් ඒ නාගයාගේ චරණයෝය. (පසු පාදයෝය.) නිවැරදි ධර්ම ගැණීමෙහි ශ්‍රද්ධා හස්තයක්ද, උපෙක්ෂා නමැති සුදු දළද වෙත්. 
# “සිහි නමැති ග්‍රීවාද, ප්‍රඥා නමැති සිරසද, කුශල් දහම් විමසන සිතීමේ ශක්තියද සමථ විදර්ශනා නම් දහම් කුසද, ඒ බුද්ධ නාගයාගේ උපධි විවේකය වාලධියද වෙයි. 
# “හෙතෙම ධ්‍යාන කරණසුලු වූයේ නිවනෙහි ඇලුණේ ඵල සමාපත්තියෙන් සමාහිතවූයේ ඒ නාගයා ගමනෙහිද සිටීමෙහිද සන්සුන් ස්වභාවය දක්වයි. 
# “ඒ සංසුන්වූ නාගයා හිඳින කල්හිද, නිදන කල්හිද, සංසුන්ව හිඳියි. නාගයා සියලු තන්හිම සංවරව වෙසෙයි. මේ වනාහි නාගයාගේ සම්පත්ති ගුණය වෙයි. 
# “වැරදි සහිත දේ නුබුදියි. නිවැරදි දේ වළඳයි. බඩගිනි නිවාගැණීම උතුම් කොට ඇත්තේ රැස්කිරීම නොකරයි. 
# “කුඩාවුද, මහත්වූද, සියලු සයෝජන බන්ධනයන් සිඳ දමා යම් යම් තැනකට යේ නම් බලාපොරොත්තුවක් නැතුවම ගමන් කරයි. 
# “සුවඳ ඇති මනොරම්‍යවූ දියෙහි හටගත් පියුම දියෙහි තැවරීමක් නැතුව යම්සේ වැඬේද? 
# “එසේම ලෝකයෙහි උපන්නාවූ බුදුන්වහන්සේද ලෝකයා සමග නොගැටී වෙසෙයි. දියෙහි වැඩුන පියුමක් වැනිය. 
# “මහ ගින්න වුවද උපකරණ නොලැබීමෙන් සංසිඳෙයි. අඟුරු නිවිකල්හි නිවීමද වෙයි. 
# “මේ අර්ථය දැනගැණීමට උපමාවකි. ඥානවන්තවූ බුදුන් විසින් දේශනා කරන ලද ධර්මය, ඥානවන්තයෝම දැන ගනිති. තථාගත මහා නාගයා ඇසුරු කරන්නේද, ක්ෂීණාශ්‍රවවූම නාගයෝය. 

# “පහකළ රාගය ඇත්තාවූද පහකළ ද්වෙෂය ඇත්තාවූද, පහකළ මෝහය ඇත්තාවූද, ආශ්‍රවයෝ ක්ෂය කළාවූද, ක්ෂීණාශ්‍රව නාගතෙමි ශරීරය හරිනුයේ පරිනිර්වාණය කරන්නාහුය.
