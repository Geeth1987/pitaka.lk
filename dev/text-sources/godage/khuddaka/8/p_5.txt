ඡක්ක නිපාතය 
#උරුවෙල කස්සප ථෙර ගාථා
# “පතළාවූ කීර්ති ඇති ගෞතම බුදුන්ගේ ප්‍රාතිහාර්ය දැකද, ඊර්ෂ්‍යා මානයෙන් වඤ්චා කරණලද බැවින් මම උන්වහන්සේට නොවැන්දෙමි. 
# “මගේ වැරදි කල්පනාව නිසා පුරිසදම්ම සාරථිවූ බුදුන්ට චොදනා කෙළෙමි. එහෙයින් මට අද්භූතවූද, ලොමෝද්ගතවූද, සංවේගයක් උපන. 
# “පෙර තාපසයෙක්ව සිටි මට සවල්පමාත්‍රවූ ලාභසත්කාරමය ඉර්ධියක්වීද මම එය හැරදමා සර්වඥ ශාසනයෙහි පැවිදිවීමි. 
# “පෙර යාග නම් වූ යැදීමෙන් සතුටුවූයේ කාම සුගතිය පිළිබඳ ආශාව ඇත්තෙක්ව විසුවෙමි. පසුව රාග, දොෂ, මෝහ, මාර්ග පිළිවෙළින් නැසුවෙමි. 
# “(ඒ මම) පෙර විසූ කඳ පිළිවෙල දන්නා නුවණද, දිව ඇසද, දිව කණද, ඉර්ධි බලයද පිරිසිදු කෙළෙමි. 
# “යම් අර්ථයක් නිසා ගිහි ගෙයින් පැවිදි බවට පැමිණියේද, සියලුම සංයෝජන ධර්මයන් නැති කිරීමෙන් ඒ අර්ථය සිද්ධකර ගතිමි.”
# ආයුෂ්මත් උරුවෙල කාශ්‍යප තෙරණුවෝ මෙසේ මේ ගාථා කීහ. (ථෙර ගාථා අවසානය දක්වාම ගාථා කී ඒ ඒ තෙරුන් වහන්සේලාගේ නම යොදා මේ ඡෙදය අගට යොදා ගත යුතුයි.) 
# තෙකිච්ඡකානි ථෙර ගාථා
# “වී හාල් ආදී ධාන්‍ය වර්ග ගෙවලත් ගම්වලත් පිරී ඇති කල්හි භික්ෂා මාත්‍රයක් නොමලදිමි, ඒ මම කෙසේ කෘෂිකර්මයෙහි යෙදෙම්ද? 
# “අපරිමාණ ගුණ ඇති බුද්ධ, ධර්ම, සංඝ යන තෙරුවන් සිහිකර පැහැදුනේ උසස් ප්‍රීතියෙන් යුක්තව සාපිපාසා දෙකින් නොමඩනා ලදුව විසුවෙමි. 
#
#
# “ලනලද දොර අගුල් ඇති සෙනසුන්හි විසීම සැපයට හේතුවක්කොට සිතා, සීතලෙන් යුක්තවූ හිම වැටෙන රාත්‍රිවල එළිමහන් ස්ථානයන්හි විසුවෙමි. එසේ සීතලෙන් මඩනා ලදුයේ වෙහෙසට නොපැමිණියෙමි. 
# “සතර බ්‍රහ්ම විහරණ සමාපත්තීන්ට සමවදිමින් විසුවෙමි. එයින්ම සැපයට පැමිණියේ සතර ඉරියව් පැවැත්වීමි. සීතාදී උපද්‍රව නැතිව සිත නොකැළඹීමෙන් විසුවෙමි.” 
#
# මහානාග ථෙර ගාථා
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කිරීමක් නැත්තේද, හෙතෙම සද්ධර්මයෙන් පිරිහෙයි. ඒ කෙසේදයත්, දියසිඳුන වළක වසන මත්ස්‍යයා මෙනි. 
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කිරීමක් නොමැත්තේද හෙතෙම සදහම්හි දියුණුවක් නොලබයි. කෙතෙහි වපුල කුණු බිජු වටක් මෙනි. 
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කිරීමක් නො මැත්තේද හෙතෙම සර්වඥ ශාසනයෙහි දැක්වෙන නිවනට කිට්ටු නොවෙයි. 
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කෙරේද හෙතෙම සදහමින් නොපිරිහෙයි. දිය පිරී තිබෙන වළක වෙසෙන මත්ස්‍යයා මෙනි. 
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කෙරේද, හෙතෙම සද්ධර්මයෙහි දියුණුවට පැමිණේ. කෙතෙහි වපුළ යහපත් ධාන්‍ය වැඩෙන්නාක් මෙනි. 
# “යමෙකු සබ්‍රම්සරුන් කෙරෙහි ගරු කෙරේද, හෙතෙම බුද්ධ ශාසනයෙහි පෙන්නුම් කළ නුවණට කිට්ටුවූ තැනැත්තෙක් වෙයි.” 
#
# කුල්ල ථෙර ගාථා
# කුල්ල ස්ථවිරයන් වහන්සේ සොහොනට ගොස් උඩුකුරුව හෝනා පණුවන් විසින් කනු ලබන ස්ත්‍රියක් පිළිබඳ මෘත ශරීරයක් දුටුයේය. 
# “බාලයන්ගේ සතුටට හේතුවූ උඩ වන මුඛයන්ගෙන් හා පහත වන මුඛයන්ගෙන් අශුචි වැගිරෙන්නාවූද, දුගඳ හමන්නාවූද, දුකෙන් පෙළෙන්නාවූද ශරීරය කුල්ල ස්ථවිරයෙනි බලව. 
# “ඤාණය පහළකර ගැණීමට ධර්ම නැමැති කණ්ණාඩිය ගෙණ ඇතුළතත්, පිටතත් පිළිකුල්වූ මේ ශරීරය ඤාණයෙන් සිහිකොට බැලීමි. 
# “මේ ශරීරය යම්සේද අන්‍ය ශරීරත් එසේමැයි අන්‍ය ශරීර යම්සේද මේ ශරීරයත් එසේමැයි. යට කොටස යම්සේද උඩ කොටසත් එසේමැයි. උඩ කොටස යම්සේද යට කොටසත් එසේමැයි. 
# “දහවල් යම්සේද රාත්‍රියත් එසේමැයි. රාත්‍රිය යම්සේද, දහවලත් එසේමැයි බාල තරුණ කාලවල යම්සේද මහළු කලත් එසේමැයි. මහළු කල යම්සේද බාල තරුණ කලත් එසේමැයි. 
# “එකඟවූ සිතින් ධර්මය බලන්නහුට ඇතිවන රතිය (ආශාව) පඤ්චාංගික තූර්ය නාදයෙන්ද ඇති නොවේ.” 
#
# මාලුංක්‍යපුත්ත ථෙර ගාථා
# “ප්‍රමාද විහරණයෙන් යුක්ත සත්වයා මාලුවා නැමැති වැල වැඩෙන්නාක් මෙන් තණ්හාවෙන් වැඬේ. හෙතෙම වනයෙහි වසන ඵල (ගෙඩි) කැමැති වඳුරෙක් මෙන් භවයෙන් භවයට දිවේ. 
# “යම් පුද්ගලයෙක් ලාමක තණ්හාවෙන් වැඬේද හෙතෙම එම තෘෂ්ණාව දුරුනොකට හැකිව ඊටම යටපත්ව වෙසේ. ඔහු කරා වැටෙන ශෝකයෝ සැවැන්න පදුරෙහි වැටෙන ජල බින්දු මෙන් එකතුව සිටිති. 
# “යහපත්වූ තොපට කියමි. තොප විසින් මෙහි එක් රැස්ව තෘෂ්ණාවේ මුල සිඳිව්. කෙසේදයත් සැවැන්න පඳුරක් ගලවන්නාක් මෙනි. 
# “තද දිය පාරකින් ගඟබඩ බට පඳුරු විනාශ කරන්නාක් මෙන් ක්ලෙශ මාරයාගේ පහරට අසු නොවව්. බුදුන්ගේ වචනය ඉෂ්ට කරව්. දුර්ලභව ලැබූ මේ ක්ෂණ සම්පත්තිය පැහැර නොහරිව්. ක්ෂණ සම්පත්තිය පැහැර හැරියාහු නරකයන්හි දක්නා ලැබේ. 
## “ප්‍රමාද දොෂය (රජස්) දූවිල්ලක් වැනිය. රාගාදී (රජස්) දූවිලි ප්‍රමාදය නිසා උපදී. අප්‍රමාදයෙනුත් මාර්ග විද්‍යාවෙනුත් රාගාදී හුල් (උල්) උදුරා දමව්.” 
#
# සප්පදාස ථෙර ගාථා
# “මම විසිපස් අවුරුද්දක් පැවිදිව සිටියේද අසුරුසනක් ගසන කාලය පමණකුදු චිත්ත ශාන්තියක් නොලදිමි. 
# “සිතේ එකඟ බවක් නොලැබ කාම රාගයෙන් පීඩිතව දෙඅත් බැඳගෙණ අඬමින් විහාරයෙන් නික්ම” 
# “ආයුධයක් ගෙනෙමි. මගේ ජීවිතය කවර ප්‍රයෝජනයක්ද? ශික්ෂාව හැර කාලක්‍රියා කරන්නේ කෙසේද? 
# “ඒ මම ආයුධය ගෙණ ඇඳෙහි හුන්නෙමි. ගෙල සිඳීමට ආයුධය බෙල්ලෙහි තැබුවෙමි. 
# “ඉක්බිති සීලය මෙනෙහි කරණ කල්හි නුවණ පහළ විය. මාර්ගඵල ප්‍රත්‍යවේක්ෂා කරණ ඤාණයද පහළ විය. 
# “මගේ සිත කෙලෙසුන්ගෙන් මිදුනේය. ධර්මයෙහි ඇති යහපත් බව බලව, ත්‍රිවිද්‍යාවද ලදිමි. බුදුන්ගේ අනුශාසනාවද කරණ ලදී.” 
#
# කාතියාන ථෙර ගාථා
# “කාතියාන ස්ථවිරයෙනි උට්ඨාන වීර්යය කරව, සිහියෙන් යුක්තව හිඳගනුව, නිදි බහුලව නොසිට නිදි වර්ජිත කිරීමෙන් යුක්තවව්. අලසවුව හොත් මාරයා උපක්‍රමයෙන් යටපත් කෙරේ. 
# “කෙසේදයත් මහා සමුද්‍රයාගේ රැළ වේගයකින් යටපත් කරණ පුරුෂයෙකු මෙන් ජාති ජරාදී දුකින් යටපත් කරයි. තොපි අර්හත් ඵල නම්වූ ප්‍රදීපය ආලොක කරව්, තොපට වෙන පිළිසරණක් නැත්තේය. 
# “ශාස්තෲන් වහන්සේ වනාහි මේ මාර්ගය ජය ගත් සේක. උන්වහන්සේ විසින් ජාති ජරාදිය ඉක්මවන ලදී. පූර්ව අපර යාමයන්හි අප්‍රමාදව භාවනාවෙහි යෙදෙව්. 
# “කාතියාන ස්ථවිරයෙනි, කාමගුණ බන්ධනයෝ දුරු කරව, හිස මුඩු කරන ලද්දේ භික්ෂාචාරය ගනුව, ක්‍රීඩාවෙහිද නිදීමෙහිද යන කරණුවල නොයෙදී ධ්‍යානයෙහි යෙදෙව. 
# “කාතියාන ස්ථවිරයෙනි, ධ්‍යාන වැඩීමෙන් ජයගනුව. බොධිපාක්ෂික ධර්මයන්හි දක්ෂවව, නිරුත්තරවූ පිරිසිදු බවට පැමිණ ජලයෙන් ගිනි කඳක් නිවන්නාක් මෙන් පරිනිර්වාණය කරව. 
# ලපටි වැලක් මලානික වන්නාක්මෙන් ස්වල්ප මාත්‍ර ආලොකයක් ඇති පහන සුළඟින් නිවීයේද, එපරිද්දෙන් ඉන්ද්‍රගොත්‍ර මාරසේනාව විනාශ කරව, සියලු වේදනාවන්හි දුරුකරන ලද්දේ සිහිල්වූ තමාගේ පරිනිර්වාණ කාලය බලාපොරොත්තුවව.” 
#
# මිගජාල ථෙර ගාථා
# “සූර්ය වංසයෙහි උපන්නා වූද, පසැස් ඇත්තාවූද, බුදුරජාණන් වහන්සේ විසින් මනා කොට දේශනා කරනලද ධර්මය සියලුම සංයෝජනයන් ඉක්මවීමෙන් සියලු සසරදුක නැසුවෙහිය. 
# “තෘෂ්ණාමූලය වියලවා සත්වයන්ගේ ඇතිවීමට හා නැතිවීමට හේතුවවූ කර්මක්ලෙශයන් සිඳ එතරවූයේ නෛයර්යාණික නිවනට පැමිණියෙහිය. 
# “භව චක්‍රය නසන්නා වූද, ආත්ම භාව නම්වූ යන්ත්‍රය නැතිකරන්නාවූද, ඤාණ නැමති වජ්රායුධය හෙළමින්, 
# “ත්‍රිවිධ වේදනාවන් දුක් වශයෙන් සලකා උපාදානයන්ගෙන් මිදුනේ කාමභව ආදී නවවිධ භවය ගිනිගෙණ දිලිසෙන අගුරු වළක්සේ ඤාණයෙන් ප්‍රත්‍යවෙක්ෂා කොට, 
# “මහ සැපතක් වැනිවූද ගැඹුරක් වැනිවූද, ජරාමරණ වළකන්නාවූ ආර්ය අෂ්ටාංගික මාර්ගය දුක් සමනය කරණ ක්ෂේමස්ථානයක් වැනිය. 
# “කර්මය කර්මය හැටියටත් විපාකය විපාකය හැටියටත් දැන පටිච්චසමුප්පාද ධර්මයන්ගේ යම්බඳු ආලොකයක් ඇද්ද, ඒ ආලොකය දිටිමි. ශාන්තවූ නිර්වාණය අවසාන යහපතසේ දිටිමි.” 
#
# ජෙන්ත ථෙර ගාථා
# “මම භොග මද, ජාතී මද, ඓශ්චර්ය මද ආදියෙන් මත්වූයේ ආරොහපරිනාහ (අංග ප්‍රත්‍යංග ශොභන තාවය) සම්පත්තියෙන් මදයට පැමිණියේ හැසුරුනෙමි. 
# “බාල බැවින් හා අතිමානයෙන් තද වූයේ නොනැමෙන ස්වභාව ඇත්තේ කිසිවෙකු උසස්කොටද, සමකොටද නොසිතුවෙමි. 
# “ගරුකළ යුත්තාට ගරු නොකොට මානයෙන් තද වූයේ කිසිවෙකුට ආදර නැතුව මවුපියන් ආදී ගරුකටයුත්තන්ට නොවැන්දෙමි. 
# “උත්තම පුරිස දම්ම සාරථිවූද, සූර්යයාමෙන් බබළන භික්ෂූන් වහන්සේගෙන් පිරිවරා ගන්නාලද අග්‍රවූද නායකවූද බුදුරජාණන් වහන්සේ දැක, 
# “මාන්‍යයත්, මදයත් හැර දමා පැහැදුනාවූ සිතින් සියලුම සත්වයින්ට උත්තමවූ බුදුරජාණන් වහන්සේට මස්තකයෙන් නමස්කාර කරෙමි. (ඔලුව බිම තබා වැන්දෙමි.) 
# “අතිමානයත් හීන මානයත් දෙක ප්‍රහීණ කෙළෙමි. සිත එකඟ කෙළෙමි. මමය මාගේය යන මානය සිඳින ලදී සියලුම මානයෝ නැති කෙළෙමි.” 
#
# සුමන ථෙර ගාථා
# “යම් කලෙක්හි සත් හැවිරිදි වයස් ඇති අළුත් පැවිද්දෙක්ව ඉර්ධි උපදවා බල සම්පන්න නාග රාජයෙකු දමනය කොට 
# “අනොතත්ත විලින් ජලය රැගෙණ උපාධ්‍යායන් වහන්සේට දුනිමි. ඒ දැක බුදුරජාණන් වහන්සේ මෙසේ වදාළසේක. 
# “සාරිපුත්‍රය, අග්‍රඵල සමාපත්තියෙන් සිත තැන්පත් කළාවූ දිය කළයක් රැගෙන එන කුමාරයා බලව. 
# “බලන්නවුන්ට ප්‍රශාදය ඇතිකරන ආචාරවතින්ද යහපත් ඉරියව්වෙන්ද, අනුරුද්ධ ස්ථවිරයන් වැනි විශාරද ඉර්ධියෙන්ද යුක්තවූ සාමණේරයෙක්වේද, හෙතෙම 
[“උත්පත්තියෙන් සත් අවුරුදු වයස් ඇති අළුත් පැවිද්දෙක්ව ඉර්ධි බලයෙන් බලසම්පන්නවූ නාගරාජයෙකු පරදවා ජයගෙණ ]
# “ශ්‍රෙෂ්ඨ පුරුෂයෙකු වශයෙන් උත්තමවූයේ යහපත් බවට පැමිණියේ හික්මුණේ කළ කටයුතු ඇති අනුරුද්ධ ස්ථවිරයන් විසින් හික්මවන ලද්දේ හෙතෙම, 
# “චිත්ත විමුක්තිය උත්තමකොට ඇති පරම ශාන්ති සැපයට පැමිණි ඒ සුමන සාමණෙර නම්වූ මම රහත් කෙනෙක් හැටියට කිසිවෙක් දැනගනුවයි නොසිතුවෙමි.” 
#
# නහාතකමුනි ථෙර ගාථා
# “වාත රෝගයෙන් මඩනා ලද්දේ මහ වනයෙහි වාසය කරන්නේ ප්‍රත්‍ය දුර්ලභවූ වාත රෝගයට අහිතවූ ස්ථානයෙහි මහණ තොප කුමක් කරන්නෙහිද? 
# “මහත්වූ ප්‍රීති සැපයෙන් මුළු ශරීරයම සැපවත් කොට අරණ්‍ය වාසයෙහි සැහැල්ලු පැවතුම් ඇතුව සියල්ල ඉවසා වනයෙහි වාසය කෙළෙමි. 
# “සතිපට්ඨාන ඉන්ද්‍රිය බල බොඤ්ජංග ධර්මයන් වඩමින් කෙලෙස් නැතිව වාසය කරමි. 
# “සියලු කෙලෙසුන්ගෙන් වෙන්ව පිරිසිදු සිතක් ඇතිව නොකැළඹුනාවූ අදහසින් නිතර නුවණින් සිහි කරමින් ආශ්‍රව ධර්ම රහිතව වාසය කෙළෙමි. 
# “ආධ්‍යාත්මික බාහිර වශයෙන් පැවති සියලුම ක්ලේශයෝ මතුනූපදනා පරිද්දෙන් විනාශකර දැමීමි. 
# “ස්කන්ධ පංචකය පිරිසිඳීමෙන් සියලු මුල් සිඳ දුක් නැතිකළ බවට පැමිණියෙමි. මින්මතු නැවත ඉපදීමක් මට නැත.” 
#
# බ්‍රහ්මදත්ත ථෙර ගාථා
# “උත්තම දමනයෙන් දමනයවූ කය සමනය කළ මනාකොට දැන කෙළෙසුන්ගෙන් මිදුනාවූ උපශාන්ත ගුණයෙන් අටලෝ දහමෙහි කම්පා නොවන ගුණයෙනුත් යුක්තවූ ක්‍රොධ නොකරන මට ක්‍රොධයක් කොයින්ද? 
# “යමෙක් කිපෙන්නෙකුට කිපේද, ඔහුට එයින් පව් ඇතිවේ. කිපෙන්නාට නොකිපෙන තැනැත්තා දිනිය නොහැකි යුද්ධයකින් ජය ගත්තෙක් මෙනි. 
# “අනුන් කිපි බව දැන තමා සංසිඳී නොකිපී සිටියොත් තමාටත් අනුන්ටත් කියන දෙපක්ෂයටම ඔහු යහපතක් කළ කෙනෙකි. 
# “තමාගේත් අනුන්ගේත් යහපතට හේතුවන ක්‍රොධ නොකිරීමේ ප්‍රතිකර්මය කළ තැනැත්තා ධර්මය නූගත් අඥාන ජනයා මෝඩයෙක්කොට සලකයි. 
# “ඉදින් ක්‍රොධය උපදීනම් කියතෙහි උපමා අවවාදය සිහිපත්කරව, රස තෘෂ්ණාව උපදී නම් පුත්‍රමාංශොපමාව සිහිකරව. 
# “ඉදින් සිත කාම සැප භව සැප යනාදියෙහි දිවේද, සිහිය උපදවා ඒ සිතට නිග්‍රහ කළයුතුය. කුළු ගොනෙක් කණුවක බැඳ හික්මවන්නාක් මෙනි.” 
#
# සිරිමන්ද ථෙර ගාථා
# “(දුශ්චරිත) වැසීමෙන් කෙලෙස් වැඩිවෙයි. නොවැසීමෙන් කෙලෙස් අඩුවෙයි. ඒ නිසා නොවසා විවෘත කරව. එසේ කල්හි කෙලෙස් වැඩි නොවේ. 
# “මේ සත්ව ලොකය මරණයෙන් යටපත් කෙරේ. ජරාවෙන් පිරිවරණ ලද්දේවෙයි. තෘෂ්ණා නම් වූ හුලෙන් ඇතුළත විදින ලද්දේවෙයි. හැමකල්හි අරමුණ කැමැත්තේ දුම් දැමීමක් වෙයි. 
# “සත්වලොකය මරුවා විසින් යටපත් කරනලදුව, ජරාවෙන් පිරිවරණ ලදුව, අසරණව අපරාධ කළ සොරෙක් රාජ දඬුවම් ලබා තැවෙන්නාක් මෙන් තැවීමට පත්වෙයි. 
# “වනයෙහි හටගත් ලැව්ගින්නක් මෙන් ජරා ව්‍යාධි මරණ යන තුන අභිභවනය කරන්ට නොහැකිය. ජය ලබා පලායාමටද නොහැකිය. 
# “විදර්ශනා භාවනාවෙන් තොරව දවස් නොහරිව්, ස්වල්පමාත්‍ර කාලයකුදු හිස් නොකොට බොහෝ වශයෙන්ම භාවනාදියෙහි යෙදුනේ රෑ දවල් හිස්නොකොට ගත කෙරේ නම් ඔහුගේ ජීවිතය හිස් නොවේ. 
# “යාම, හිඳීම, සිටීම, සයනය කිරීම යන සතර ඉරියව්වෙහිම කාලය හිස් නොකොට අප්‍රමාද විහරණයෙන් යුක්තව වසව.” 
#
# සබ්බකාමී ථෙර ගාථා
# “දුගඳ හමන්නාවූ මේ දෙපා ඇති ශරීරය නොයෙක් කුණපයෙන් පිරුණේ නව දොරින් වැගිරෙන දුගඳ ඇත්තේ සුවඳවත් කිරීමට නොහැකිය. 
# “වැද්දා කොළඅතු ආදියෙන් මුවාව සිට මුවන් ගන්නාක් මෙන්ද, බිලියෙන් මසුන් ගන්නාක් මෙන්ද, ලාටුවෙන් වඳුරන් ගන්නාක් මෙන්ද, පෘථග්ජනයා පෙළයි. 
# “රූප, ශබ්ද, ගන්ධ, රස, ස්පර්ශ යන රමණීය පංචකාම ගුණයෝ ස්ත්‍රී රූපයෙහි දක්නා ලැබේ. 
# “රාගයෙන් මත්වූ පෘථග්ජන සත්වයෝ මේ කාම වස්තු සේවනය කෙරෙද්ද, ඔව්හු ජාති ජරාදී දුක් සමූහයත් නිරායාදී අපාය භයත් සංසාර දුකත් නැවත නැවත ලබයි. 
# “යමෙක් මේ කාම වස්තූහු සර්ප ඔලුවක් දුටු කලෙක මෙන් දුරුකෙරේද, ඔහු විෂ සංඛ්‍යාතවූ ලොකයෙහි සිහියෙන් යුක්තව කල් යවයි. 
# “කාමයෙහි දොෂයත්, පැවිදිවීමෙහි ආනිසංසයත් දැක සියලුම කාමයන්ගෙන් වෙන්වූ මම කෙලෙස් නැති කිරීමෙන් රහත් බවට පැමිණියෙමි.”
