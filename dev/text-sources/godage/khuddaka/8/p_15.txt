
වීසති නිපාතය 
# අධිමුත්තක ථෙර ගාථා
# “යාග හෝමාදිය පිණිස ධන පැහැර ගැනීම් පිණිස හෝ අපි මට පළමුවෙන් යමෙක් නැසුවාහු නම් ඒ සියල්ලෝම බියපත් වූවාහුය. එසේම වෙවුළුවාහුය. එසේම වැලපුනාහුය. 
# “ඔබවහන්සේට භයකුත් නැත්තේය. ඔබවහන්සේගේ ශරීර වර්ණයද පැහැපත් විය. මෙබඳුවූ භයට හේතුවූ අවස්ථාවෙහි කුමක් හෙයින් නොහඬනාහුද? 
# “තෘෂ්ණාවෙන් ඇතිකරන අපේක්ෂාව නැති බැවින් සිත පිළිබඳ දුකක් නැත්තේය. සංයෝජන ධර්මයන් ක්ෂය කළාවූ මම සියලු භය නැති කෙළෙමි. 
# “තත්වූ පරිද්දෙන් චතුස්සත්‍යාවබොධ කොට භවයාගේ කෙළවර සිඳදමන ලද බැවින් බර බහා තැබීම කළ මරණයෙහි භයක් නැත. 
# “ශාසන බ්‍රහ්මචරියාවද පුරුදු කෙළෙමි. ආර්ය මාර්ගයද මනාකොට වැඩුවෙමි. රොගයකින් සුවවූවෙකුට මෙන් මට මරණයෙහි භයක් නැත. 
# “මා විසින් ශාසන බ්‍රහ්මචරියාව පුරුදු කරණ ලදී. ආර්ය මාර්ගයද මනාකොට වැඩුවෙමි. භවයාගේ නිසරුකමද දැක්කෙමි. ප්‍රයෝගයකින් හරණා ජීවිතයක් මෙන් මරණයෙහි භයක් නැත. 
# “මරණයෙන් නිදහස ලැබූ සොරෙකු මෙන් සසරින් එතරට පැමිණියාවූද කොට නිමවන ලද කටයුතු ඇත්තාවූද කිසිවක් තෘෂ්ණා දෘෂ්ටි දෙකින් නොගන්නාවූද කෙලෙස් නැත්තාවූද මම ආයුෂයාගේ කෙළවර වන මරණය සතුටින් බලාපොරොත්තු වෙමි. 

# “ධර්ම ස්වභාවය දැකීමෙන් අර්හත්වයට පැමිණියාවූද සියලු ලෝකයෙහි අපේක්ෂාව දුරුකළාවූද මම ගිනිගත් ගෙයකින් පිටවූවෙකුමෙන් මරණයෙහි ශෝක නොකරමි. 
# “ලෝකයෙහි ප්‍රත්‍යයෙන් උපන් යම්කිසිවක් වේද සත්ව නිකායයන්හි ලබන උත්පත්ති භාවයකුත් වේද, ඒ සියල්ලම ප්‍රධාන බවට හේතු නොවන බව මහර්ෂීවූ බුදුන් විසින් කියන ලද බැවින් මරණයෙහි ශොක නොකරමි. 
# “යම් කිසිවෙකු විසින් බුදුන් දේශනා කරනලද ධර්මය එපරිද්දෙන්ම දනී නම් ගිනි ගත් ලෝගුලියක් අතින් නොගන්නාක්මෙන් කිසි භවයක් නොගනී. 
# “අතීතයෙහි මම මෙයාකාරයෙන් සිටියේයයි කියාද, අනාගතයෙහි මෙබන්දෙක් වන්නෙමියි කියාද අදහස් නොකරමි. ප්‍රත්‍ය බලයෙන් පවත්නාවූ සංස්කාරයෝ විනාශයට පැමිණෙත්. මෙසේ දක්නාවූ මට කවර හැඬීමක්ද? 
# “ආත්ම සාරයෙන් තොරවූද, ප්‍රත්‍යයෙන් හටගත්තා වූද, ක්ලෙශාදී වශයෙන් බැඳී පවත්නාවූද, සංස්කාර තත්ත්වය තත්වූ පරිදි දක්නාවූ මට භයක් නැත්තේය. 
# “තණ හා දර අනිකෙකු විසින් ගනු ලබන කල්හි මා පිළිබඳ දෙයක් ගන්නේයයි අදහස් නොවේද, එසේම තණ හා දර සමවූ ලෝකය ප්‍රඥාවෙන් දකීනම් කිසිවකට මමත්වයක් නොකරයි. මේ දේ මට තිබී නැතිවීයයි. ශොක නොකරයි. 
# “ශරීරය කෙරෙහි කලකිරුනෙමි. භවයෙන්ද මට ප්‍රයෝජනයක් නැත. මේ ශරීරයද බිඳෙයි. මට අනික් කයක් මත්තෙහි ඇති නොවෙයි. 
# “මේ ශරීරයෙන් තොපට යමක් කැමැත්තාහුද ඒ කරව්. කරන කල්හිත් නොකරන කල්හිත් මට අනුකූල බව හෝ ක්‍රොධය යන දෙකම ඇති නොවේ. 

# “ඒ අධිමුත්ත ස්ථවිරයන්ගේ අද්භූතවුද, රෝමෝද් ගතවූද, වචන අසා ඒ හොරු ආයුධ බිමහෙලා උන්වහන්සේට මෙසේ සැළ කළහ. 
# “පිංවතුන් වහන්ස, කවර ආචාර්ය වරයෙකුගේ අවවාදයක් නිසා කෙබඳු තපස් කිරීමක් කොට කවරකුගේ ශාසනයකට පැමිණ මරණ අවස්ථාවෙහි ශෝක නොකිරීම ඔබවහන්සේ විසින් ලබන ලදද? 
# “සියල්ල දක්නාවූද, සියල්ල දන්නාවූද, ජයගත්තා වූද, මහා කාරුණිකවුද, කෙලෙස් රෝගයන්ට පිළියම් කරන්නාවූද, බුදුන් වන්සේ මාගේ ශාස්තෘවරයාණෝය. 
# “නිරුත්තරවූද නිර්වාණගාමීවූද ධර්මය ඒ බුදුන් වහන්සේ විසින් දේශනා කරණ ලදී. උන්වහන්සේගේ ශාසනයට පැමිණ ශොක නොකරන බැව් ලැබ ගතිමි. 
# “චෞරයෝ ඒ අධිමුත්තක ස්ථවිරයන්ගේ යහපත් වචන අසා ආයුධාදිය බිම දමා සමහර කෙනෙක් ඒ සොරකම් කිරීමෙන් වැළකුණාහුය. සමහරෙක් ප්‍රව්‍රජ්‍යාව ලැබ ගත්තාහුය. 
# “බුදු සස්නෙහි පැවිදි බව ලැබූ ඔවුහු බොජ්ඣංග ධර්ම වඩා ප්‍රීතියෙන් යුක්ත සිත් ඇතිව සොම්නසට පැමිණ අසංඛත නම්වූ නිර්වාණ පදය ලැබුවාහුය.” 
#
# පාරාසරිය ථෙර ගාථා
# “කාය විවේකයෙන් හා චිත්ත විවේකයෙන් යුක්තව ධ්‍යාන කරන්නාවූ පරාසර ගොත්‍රයෙහි උපන්නාවූ භික්ෂුවට ධර්ම චින්තාවක් විය. 
# “පුරුෂයා පළමුකොට කුමක් කළයුතුද කවර ශීලයක් අනුව හැසිරිය යුතුද තමහට කෘත්‍ය පිණිස පවතින කටයුතු කරන පුද්ගලයා අනුන්ට වෙහෙසීමක් නොකළ යුතුය. 
# “මිනිසුන්ට ඉන්ද්‍රියයෝ හිත පිණිසද අහිත පිණිසද වෙත්. නොරක්නා ලද ඉඳුරෝ අහිත පිණිසත්, රක්නා ලද ඉඳුරෝ හිත පිණිසත් පවතී. 

# “සිහියෙන් යුක්තව ආරක්ෂා කරන ඉන්ද්‍රියයෝ වෙත්ද, තමහට යහපත පිණිසම පවතී. අන්කිසිවෙකුට වෙහෙසක් පිණිස නොපවතී. 
# “රූපයන් කරා යනු ලබන ඇස නමැති ඉන්ද්‍රිය නොවලක්කා ලබන්නාවූද ආදීනව නොදක්නාවූද හෙතෙම දුකින් නොමිදේ, 
# “ශබ්ද අරමුණු කරා ගමන් කරන්නාවූ කණ නමැති ඉන්ද්‍රිය නොවලක්නාවූද ආදීනව නොදක්නාවූද හෙතෙම දුකින් නොමිදේ. 
# “ආදීනව නොදැකීමෙන් සුවඳ සේවනය කෙරේද, සුවඳෙහි මුසපත්වූ ඒ පුද්ගලයා දුකින් නොමිදේ. 
# “ඇඹුල් මිහිරි තිත්ත ආදී රසයන් පිළිබඳ මෙනෙහි කරනුයේ රස තෘෂ්ණාවෙන් බැඳුනාහූ දුක් කෙළවර කෙරෙමියි අදහසක් නොසිතයි. 
# “සුන්දරවූද මනොරම්‍යවූද ස්පර්ශ සිහිපත් කරනුයේ රාගයෙන් රත්වූයේ නානා ප්‍රකාර දුක් ලබයි. 
# “කියනලද පස්වැදෑරුම් ධර්මයන්ගෙන් හා ධර්මාරම්මණයන්ගෙනුත් හිත ගැටීමට යමෙකුට නොහැකිවූයේ නම් ඒ සෑම අරමුණකින්ම ඔහුට දුක් ඇතිවෙයි. 
# “ලේ සැරවින් පිරුණාවූද කුණපයන්ගෙන් පිරුණාවූද කරනලද සිතියමක් මෙන් ඔපමට්ටම්වූ මේ ශරීරය බාල ජනයන් විසින් මුලාවෙන් ගනී. 
# “කටුක දුක්වලට හේතුවන්නාවූද මිහිරි බවක් පෙන්නුම් කරන්නාවූද එහෙයින්ම ප්‍රිය භාවයෙන් බැඳෙන්නාවූද මේ ශරීරයෙහි ගිජු බව සිතාගත් පුද්ගලයා දැලි පිහි මූණත ලෙවකන්නකුමෙන් සැපයක් නොවිඳ දුකටම පත්වෙයි. 
# “ස්ත්‍රී රූපයෙහිද ස්ත්‍රී ස්වරයෙහිද ස්ත්‍රී ගන්ධයෙහිද ඇලුනාවූ විවිධ දුක් විඳිති. 

# “ස්ත්‍රියගෙන් වැගිරෙන්නාවූ තෘෂ්ණා දියපහරවල් පුරුෂයාගේ පංචද්වාරයෙහිම පවතී. ඒ සංවරණය කරන්ට හැකි පුද්ගලයෙක්වේ නම් ඔහු වීර්යවන්තයෙකි. 
# “යමෙක් පංචකාම රතියෙහි ඇලුනේවේද අර්ථයත් ධර්මයත් නොහැර ස්වකීය කටයුතු කෙරේ නම් හෙතෙම මේ ලෝකයෙහි අර්ථයෙන් යුක්තවූද ධර්මයෙන් යුක්තවූද දක්ෂයෙක් හා විචක්ෂණ පුද්ගලයෙක් වෙයි. 
# “යම්කිසිවෙක් මෙලොව අර්ථය පමණක් ගෙණ පරලොව හැරදමා නම් ඒ කෘත්‍යය හැරදමන්නාවූ පුද්ගලයා අප්‍රමාදී විචක්ෂණ පුද්ගලයෙක් නොවෙයි. 
# “යමක් අර්ථයෙන් යුක්ත නම් ධර්මයෙනුත් යුක්ත නම් එය මනාකොට කරව්. හේ උත්තම රතිය නම් වේ. 
# “නොයෙක් ආකාර උපායයන්ගෙන් අනුන් සන්තකදේ ගැණීමට කැමැතිවේද, මරා හෝ සැහැසිකම් කොට හෝ ශෝකයට පමුණුවා ගනු ලබන දෙයින් රතියක් ඇති නොවෙයි. 
# “කාය බලාදියෙන් යුක්තවූ වඩු තෙමේ ලොකු ඇණය ගසා කුඩා ඇණය ගලවද්ද, එසේම දක්ෂවූ භික්ෂුව චක්ෂුරාදී ඉන්ද්‍රියයන් විදර්ශනා බලයෙන් දමනය කරයි. 
# “සද්ධා, විරිය, සති, සමාධි ප්‍රඥා යන ධර්ම පස වඩමින් කෙලෙස් උපදවන චක්ෂුරාදි ඉන්ද්‍රියයන් පස නැති කොට නිදුක්වූයේ බ්‍රාහ්මණයෙක්වූ ඒ උත්තමයා නිවනට පැමිණෙයි. 
# “යථොක්ත පුද්ගලයා උත්තමාර්ථයෙන් යුක්තවූයේ ධර්මයෙහිම සිටියේ බුදුන්ගේ අනුශාසනාවට අනුව පිළිපැද නිවන් සැප ලබන්නේය.”

# තෙලකානි ථෙර ගාථා
# “බොහෝ කාලයක් මුළුල්ලෙහි වීර්යවන්තව ධර්මය සොයමින් ඇවිද්දෙමි. නොයෙක් මහණ බමුණන්ගෙන් විචාළේ නමුදු සිත සංසිඳුවන ආර්ය ධර්මය නොලදිමි. 

# “ලෝකයෙහි කවරෙක් පරතෙරට ගියාහුද නිවනට පැමිණියාහු කවරහුද, උත්තම අර්ථය දන්නාහු කවරහුද කියා සෙව්වෙමි. 
# “හෘදයගත කෙලෙස් ඇදයෙන් යුක්තවූවෙමි. කෙසේද, ඇමක් ගිලින මත්ස්‍යයෙක් මෙන් දුකට පත්වීමි. ශක්‍රයාගේ පාසයෙන් වේපචිත්ති අසුරෙන්ද්‍රයා මෙන් දුකට පත්වීමි. 
# “පාසගත මුවෙක් මිදීමට නොදන්නේ ඇදීමෙන් බන්ධනය තදකරගණීද එසේම මමද ක්ලේශ බන්ධනයෙන් මිදිය නොහැකිව කවරෙක් මට මිදීමේ උපාය කියන්නේදැයි සෙවීමි. 
# “කෙලෙස් බිඳ හරින්නාවූ ධර්මය ප්‍රකාශ කරන ශ්‍රමණ බ්‍රාහ්මණයෝ කවරහුද ජරාමරණ නැතිකරන කවර ධර්මයක් පිළිපදිමුද කියායි. 
# “විචිකිච්ඡා නම්වූ සැකයෙන් බැඳුනාවූද එකට එක කිරීමෙන් යුක්තවුනාවූද ක්‍රොධමානයෙන් තද බවට පැමිණියාහුද කැමති සේ ලැබීමේ අපේක්ෂාවෙන් තෘෂ්ණාවෙන් යුක්ත වුනාවූද, 
# “පපුවෙහි දැඩිකොට ඇති පවත්නා තණ්හා හුලද තිසක් පමණවූ ප්‍රභෙදගත දෘෂ්ටි හුල්ද බලව. 
# “ප්‍රහීණ නොවූ ආත්ම දෘෂ්ටියද උත්සාහවත් කරණ ලද මිථ්‍යා සංකල්පනාවෝද නිසා සුළඟින් වැනෙන කොළ මෙන් කම්පාවෙන් පවතී. 
# “ආත්ම භාවයෙහිම හටගෙණ සීඝ්‍රතරව දරාද සයාකාර ස්පර්ශ අරමුණු කෙරෙහි සෑම කල්හිම පවතී. 
# “නානාප්‍රකාර ආයුධ යොදා මන්ත්‍ර ප්‍රයෝගයකින් හෝ මාගේ ශරීරගත කෙලෙස් හුල් උදුරාදමා පිළියම් කිරීමට හැකි වෛද්‍යවරයෙක් නොදැක්කෙමි. 
# “ආයුධ නොයොදා තුවාලද නොකොට පීඩාවකුත් නොමකොට ශරීරාභ්‍යන්තරයෙහි පවත්නා මේ හුල් උදුරා දැමීමට හැකි පුද්ගලයා කවරෙක්ද? 

# “රාගාදී කෙලෙස් සුංකරන්නාවූ යම්කිසි ධර්ම නිමිත්තක්වේද, එයම ශ්‍රෙෂ්ඨයි. ගැඹුරුවූ සංසාර මහොඝයෙහි පතිතවූ මට නිර්වාණ කාලය පෙන්නුම් කරන්නේද ඒ පුද්ගලයා ශ්‍රෙෂ්ඨයි. 
# “යම්බඳුවූ ගිහිගේ නම් මඩින් යුක්තවූ ජලාශයක්වේද, එහි පතිතව මායාව, ඊර්ෂ්‍යාව, එකට එක කිරීම ථිනමිද්ධය යන මෙයින් යුත්තව විසීම. 
# “උද්ධච්ච නැමැති වැසි ධාරාවෙන් හා සංයෝජන නැමැති වලාවෙන් යුක්තව රාග නිශ්‍රිත සංකල්පනා නැමැති දියපාර ඇතිව අපාය නැමැති සමුද්‍රයට අදිමින් සිටියෙමි. 
# “තණ්හා සොත දෘෂ්ටි සොතාදි ලතාවෙන් වැළඳ ගන්නාලද්දේද ඒ ලතාව කවරෙක් සිඳ දමන්නේද? 
# “දියපහරින් වැළැක්වීමට ඒදණ්ඩක් කළ යුතුයි. තොපගේ මනෝමයවූ තෘෂ්ණා දියපහරින් ගං ඉවුරෙක පිහිටි ගසක් ඉදිරී වැටෙන්නාක් මෙන් ඇද වැටේ. 
# “මෙසේ සසර හටගත් භය ඇති මට ඒ භයින් මිදීමට නිවන සොයන්නේ ආරක්ෂා ස්ථානය වශයෙන් ප්‍රඥා යුධය ඇති ආර්ය සංඝයා විසින් පිරිවරණලද ශාස්තෲන් වහන්සේ හේතුවිය. 
# “මනාකොට කරන ලද්දාවූද උප ක්ලේශ විරහිතවූද ශ්‍රද්ධා ප්‍රඥාදී සාරයෙන් යුත්වූද මේ විදර්ශනා නැමැති හිණිමග ගසාගෙණ යනු ලබන මට ශාස්තෲන් වහන්සේ දුන්සේක. භය නොවේයයි කීහ. 
# “සතිපට්ඨානය නැමැති ප්‍රාසාදයට නැගී චතුරාර්ය සත්‍යය මාර්ගඥානයෙන් ප්‍රතිවෙධ කෙළෙමි. එසේ කළ මට මමය මාගේයයි යන දෘෂ්ටිගත් තීර්ථකාදී පිරිස පහත්කොට සිතුවෙමි. 
# “ආර්ය මාර්ග නැමැති නැවට නැගීමේ උපාය වශයෙන් විදර්ශනා මාර්ගය දැක්කෙමි. ඒ සියලු මාර්ගයන්ට වඩා උත්තම මාර්ගය හැටියට දැක්කෙමි. 

# “මමය මාගේය යන තෘෂ්ණා හුල්වෙත්ද ඒ පාප ධර්මයන්ගේ නොපැවැත්මට ශ්‍රෙෂ්ඨ ආර්ය මාර්ගය දෙසූසේක. 
# “බොහෝ කාලයක් සන්තානගතව සයනය කරන ලද ක්ලේශ දෝෂයෝ බුදුන්ගේ දේශනානුභාවයෙන් ඉතුරු නොකොට ප්‍රහීණ කෙළෙමි.” 
#
# රට්ඨපාල ථෙර ගාථා
#වස්ත්‍රාභරණාදියෙන් විසිතුරු කරනලද නවමුඛයකින් අසූචි වැගිරෙන තුන්සීයක් ඇට කෝටු ඇති නිතර ගිලන් ස්වභාවයෙන් යුක්තවූ බාල ජනයන් විසින් වරදවා කල්පනා කරන ස්ථිර භාවයක් නැති බිඳෙන ස්වභාවයෙන් යුක්තවූ මේ ශරීරය බලව. 
# “මාණික්‍යාදි ආභරණ විශෙෂයන්ගෙන් විසිතුරු කරන ලද්දාවූද ඇට සහ සමෙනුත් යුක්තවූද වස්ත්‍රයන්ගෙන් ශොභා සම්පන්න මේ ශරීරය බලව. 
# “ලාකඩ වතුර රැඳවූ විලේපනවලින් ඔපකරන ලද මුඛයක් ඇති මේ ශරීරය අඥාන පුද්ගලයාගේ මුලාවට හේතු වෙයි. නිවන් සොයන්නහුට මුලාවට කාරණයක් නොවේ. 
# “අෂ්ටපදාකාරයෙන් කරනලද කෙස් ඇත්තාවූද, අඳුන් තැවරූ ඇස් ඇත්තාවූද මේ ශරීරය අඥානයාගේ මුලාවට හේතුයි. නිවන් සොයන්නහුගේ මුලාවට හේතුවක් නොවේ. 
# “අලංකාර කරනලද මේ කුණු කය මාලාකර්ම දන්තාදියෙන් විසිතුරුය. ඒ බාලයාගේ මුලාවට හේතුවක් වෙයි. 
# “වැද්දා මුවා මැරීමට උගුල අටවා කෑම දමාතැබුවේද, දක්ෂ මුවෙක්වූ මම මිගබන්ධකයන් හඬද්දී පලාගියෙමි. 
# “වැද්දාගේ පාසය සිඳ දමනලදී. රන් රුවන්ද, ස්ත්‍රී සමූහයාද නමැති ඒ පාසයට අසුනොවීමි. පාසය ළඟ තැබූ කෑම පමණක් ගෙන මිගලුද්දකයන් හඬද්දීම ගියෙමි. 
# “ලෝකයෙහි මෝහයෙන් ධන රැස්කොට අනුන්ට නොදෙන්නාවූ ධනවත් මනුෂ්‍යයන් දැක්කෙමි. ඔව්හු ධන රැස්කොට බොහෝ සෙයින් කාම සැප පතන්නාහුය. 

# “රජවරුද වංශානුගත පෘථිවිය ජයගෙණ සාගරය කෙළවරකොට ලැබුවත් එයින් සෑහීමකට නොපැමිණ මුහුදෙන් එතර රටවල්ද බලාපොරොත්තු වෙති. 
# “රජවරුද අවශෙෂ මනුෂ්‍යයෝද තෘෂ්ණා දුරු නොකොට මරණයට පැමිණෙත්. සම්පූර්ණ නොවූ සිත් ඇත්තෝව ශරීරය හැර දමත්. ලෝකයෙහි කාමවස්තූන් පිළිබඳ තෘප්තියක් නැත්තේය. 
# “නෑදෑයෝ මළ පුරුෂයා උදෙසා කෙස් කඩාදා හඬත්. අපේ නෑයා දෙවිවන්නේද කියත්. වස්ත්‍රයෙන් ඔතා චිතකය කරා ගෙණ ගොස් දමත්. 
# “ඒ මළ සත්වයා දවන්නවුන් විසින් හුල්වලින් ඇණ දවන්නාහුය. භොගයන් සියල්ල වෙනුවට එක් වස්ත්‍රයක් ඇතිව දවනුලබන්නාවූ ඔහුගේ ආරක්ෂාවට ඤාති මිත්‍රාදි කිසිවෙක් නැත්තාහ. 
# “ඔහුගේ ධනය උරුමක්කාරයෝ ගනිත්. ඒ සත්වයා කර්මය අනුව ගියේය. අඹුදරුවෝද, ධනයද, රටද යන කිසිවක් මැරෙණ පුද්ගලයා අනුව නොයයි. 
# “ධනය නිසා දීර්ඝායුෂ නොලබයි. ජරාවද, නොවලක්වයි. අස්ථිරවූද වෙනස්වන ස්වභාව ඇත්තාවූද, ජීවිතය ප්‍රාඥයෝ ස්වල්ප දෙයක්කොට සලකති. 
# “ධනවන්තයෝද, දිළින්දෝද, ඉෂ්ටානිෂ්ට ස්පර්ශයන් ලබත්. බාලයෝද, ඥණවන්තයෝද, එසේම ලබත්. බාලයෝ වනාහි බාල්‍යතාවය නිසා පීඩාවෙන් වෙසෙත්. පණ්ඩිතයෝ ඉෂ්ටානිෂ්ට දෙකෙහි කම්පා නොවෙත්. 
# “ඒ නිසා ධනයට වඩා ප්‍රඥාව ශ්‍රෙෂ්ඨයි. ප්‍රඥාවෙන් භවය කෙළවර කරයි. අවසාන නිෂ්ටාව නොලැබුවාහු ලොකු කුඩා භවයන්හි මෝහයෙන් පව්කම් කරත්. 
# “එසේ පව්කම්කොට සසරට වැද ගර්භසෙය්‍යාව හා ලෝකෝත්පත්තිය යන දෙකින් නොමිදෙත්. අඥානවූ පුද්ගලයා පරලොවද ගර්භසෙය්‍යාවද ආදී දුක් ලබයි. 

# “ආරක්ෂක පුරුෂයන් විසින් ගනු ලබන සොරෙක් රාජපුරුෂයන් විසින් බඳිනලදුව කසවලින් තලමින් වෙහෙසට පත්කරත්ද, එසේම පාප ධර්මයන්ගෙන් යුක්තවුනාවූ සත්වයෝ පරලොවදී වෙහෙසට පැමිණෙත්. 
# “කාමයෝ වනාහි විචිත්‍රය, මධුරය, මනෝරම්‍යය, අනේකවිධ ස්වරූපයෙන් සිත නොමගට ගෙණ පෙළත්. මේ දෝෂයෝ කාම වස්තුවල ඇති බව දැන මහරජ මම පැවිදිබවට පත්වූවෙමි. 
# “පැසුනාවූද, නොපැසුනාවූද, ගෙඩි වැටෙත්. එසේම බාලයෝද, වෘද්ධයෝද, ශරීර භෙදයෙන් මරණයට පත්වෙත්. මේ බවද දැක මහරජ මම පැවිදිබවට පත්වූවෙමි. අවිරුද්ධ ශ්‍රමණ භාවයම ශ්‍රෙෂ්ඨයි. 
# “මම ශ්‍රද්ධාවෙන් බුදුන්ගේ ශාසනයෙහි පැවිදිබව ලැබූයේ මගේ පැවිද්ද හිස්නොවීය. රහත්බැව් ලබනලද බැවින් හිස්නොවීය. කෙලෙස් නැතිව ස්වාමිභාවයෙන් ප්‍රත්‍යය පරිභොග කරමි. 
# “වස්තුකාම, ක්ලේශ කාමයන් එකොළොස් ගින්නකින් ඇවිලෙනබව දැක කළ නොකළ සියලුම රන් රිදී රත්න විශෙෂයෝ අනර්ථ පිණිස පවතින බවද සියලු සසර පැවැත්ම දුකක් බවද නරකයන්හි මහත් භය ඇති බවද නුවණින් දුටුවෙමි. 
# “මේ ආදීනව දැන සංවේගය ලදිමි. ගිහි අවස්ථාවෙහි රාගාදී හුල්වලින් විදිනාලදුව පැවිදිබවට පැමිණ කෙලෙස් ශාන්තබවට පැමිණියෙමි. 
# “මා විසින් ශාස්තෲන් වහන්සේ අසුරු කරණලදී. බුදුන්ගේ අනුශාසනාවට අනුව කටයුතු කෙළෙමි. ගරුභාරය බහාතැබීමි. ගවයාගේ කෙළවරද බිඳ දැමූවෙමි. 
# “යම් අර්ථයක් නිසා ගිහිගෙයින් නොගිහි බැව් නම්වූ ප්‍රව්‍රජ්‍යාවට පැමිණියෙම්ද, සියලු සංයෝජනයන් නසා මා විසින් ඒ අර්ථය සිද්ධකරනලදී.”

#මාලුංක්‍ය ථෙර ගාථා
# “ඇසින් රූපය දැක ශුභ නිමිති වශයෙන් මෙනෙහි කරන්නාවූ මගේ සිහිය මුලාවිය. රාගයෙන් රත්වූ සිත් ඇත්තේ ඒ රූපාරම්මණයෙහිම බැසගන්නා ලද්දේ සැපය යන හැඟීම ඇති විය. 
# “නොයෙක් රූපයන් නිසා සැප වේදනාව වැඩෙත්ම ලෝභ, ද්වෙෂ දෙකෙන්ද සිත මඩනාලද්දීය. මෙසේ රසවිඳීම කරනු ලබන්නේ සංසාර දුක පවතී. ඔහුට නිවන ඉතා දුරවේ. 
# “කණින් ශබ්ද අසා ඒ ශබ්දාරමුණු ප්‍රිය වශයෙන් මෙනෙහිකරන්නාවූ පුද්ගලයාගේ සිහිය මුලාවේ. රාගයෙන් මඩනාලද සිත් ඇතුව එහිම බැසගත් පුද්ගලයාට නොයෙක් වේදනාවෝ ලැබෙත්. 
# “නොයෙක් ශබ්දාරමුණුවලින් වේදනාව වැඩෙන්නාවූ ඔහුට ලෝභයත්, ක්‍රෝධයත් නිසා ගැටිම ඇතිවේ. ඒ ඒ වෙදනාවන්ගේ රැස්වීමෙන් සංසාර දුක පවතී. නිවන ඔහුට ඈත්වේ. 
# නාසයෙන් ගන්ධාරමුණු ලැබ ඒ ප්‍රිය වශයෙන් මෙනෙහි කරන්නහුගේ සිහිය මුලාවෙයි. රාගයෙන් රත්වූ සිත් ඇති ඔහු එහිම බැසගන්නාලදුව නොයෙක් වේදනා විඳියි. 
# “නොයෙක් ගන්ධ අරමුණු නිසා ඇතිවෙන වේදනාවන් වැඩෙනුයේ ලෝභය, ක්‍රෝධය ඔහුගේ සිත මැඩගනියි. මෙසේ රැස්කරන්නාවූ කර්ම චෙතනා නිසා ඔහුට සංසාරදුක පවතී. නිවන බොහෝ ඈත්වේ. 
# “දිවෙන් රස ලබා එය සුභවශයෙන් මෙනෙහි කරන්නාවූ පුද්ගලයාගේ සිහිය මුලාවෙයි. රාගයෙන් රත්වූ සිත් ඇති එහිම බැසගන්නාලද ඔහුට නොයෙක් වේදනාවෝ ඇතිවෙත්. 
# “නොයෙක් වෙදනාවන් නිසා ඇතිවනු ලබන වේදනාවෝ වැඩෙත්ද, ලෝභය හා ක්‍රෝධය ඔහුගේ සිත මැඩගනියි. මෙසේ රැස්කරන කර්ම ඇති පුද්ගලයාට සංසාර දුක පවතී. නිර්වාණය ඔහුට ඈත්වේ. 
# “ශරීරයෙන් ස්පර්ශ අරමුණු ලැබ එය සුභ වශයෙන් මෙනෙහි කරනුලබන පුද්ගලයාගේ සිහිමුලාවෙයි. රාගයෙන් රත්වූ සිත් ඇති එහිම බැසගෙණ සිටින්නාවූ ඔහුට නොයෙක් වේදනාවෝ ඇතිවෙත්. 
# “අනේකවූ ස්පර්ශාරම්මණයන් නිසා ඔහුට වේදනාවන් ඇතිවෙත්. එය පිළිබඳ ලෝභයද ක්‍රෝධයද, ඔහුගේ සිත මැඩගනී. මෙසේ කර්ම රැස්කරන ඒ පුද්ගලයාට සංසාර දුක පවතී. නිර්වාණය ඔහුට ඈත්වේ. 
# “සිතින් ධර්ම අරමුණු ලැබ සුභ නිමිති මෙනෙහි කරන්නාවූ පුද්ගලයාගේ සිහිය මුලාවේ. රාගයෙන් රත්වූ සිත් ඇති එහිම බැසගන්නාලද ඔහුට නොයෙක් වේදනාවෝ ඇතිවෙත්. 
# “අනේකවූ ධර්ම අරමුණු නිසා ඔහුට නොයෙක් වේදනාවන් වැඩෙත්. ලෝභයද ක්‍රෝධයද ඔහුගේ සිත මැඩගනියි. මෙසේ කර්ම රැස්කරන පුද්ගලයාට සසර දුක පවතී. නිවන ඔහුට ඈත්වේ. 
# “රූප අරමුණු දැක ඒ ප්‍රඥාවෙන් බලන්නාවූ පුද්ගලයාගේ සිත රූපයෙහි නොඇලෙයි. නොඇලුනාවූ සිත් ඇති ඔහු එහි නොබැසගන්නාලද්දේ වේදනා නොලබයි. 
# “තත්වූ පරිද්දෙන් රූප අරමුණු දකින්නාවූද, සේවනය කරන්නාවූද ඔහුගේ වේදනාවන් ක්ෂයබවටයෙත්. ඔහු සිහියෙන් යුක්තව හැසිරෙයි. මෙසේ දුක් නැතිකරන ඔහුට නිර්වාණය කිට්ටුවෙයි. 
# “හෙතෙම කණින් ශබ්දඅසා ශබ්ද අරමුණෙහි නොඇලේද, නොඇලුනාවූ සිත් ඇති ඔහු එහි බැසනොගනියි. 
# “කණින් ශබ්ද අසන්නාවූ ඔහුට ඇතිවන වේදනාවෝ ක්ෂය බවට පැමිණෙත්. කර්ම රැස්නොකරන ඔහු සිහියෙන් හැසිරෙයි. මෙසේ දුක් පහකොට නිවන් ලැබිය හැකිය. 
# “නාසයෙන් ගන්ධ අරමුණු ලැබ එය නුවණින් නොඇලීසිටීද, නොඇලුනාවූ සිත් ඇති එහි නොබැසගත් ඔහුට යම්බඳු වේදනාවෝ වෙත්ද, 

# “ගන්ධ අරමුණු නිසා ඇතිවන වේදනාවෝ ක්ෂය බවට පැමිණෙත්. සිහියෙන් යුක්තව හැසිරෙන ඔහු කර්ම රැස්නොකරයි. මෙසේ රැස් නොකරන කර්ම ඇති ඔහු දුක් නසයි. නිවන ඔහුට ආසන්න වෙයි. 
# “දිවෙන් රස අරමුණු ලැබ සිහියෙන් යුක්තව එහි නොඇලී සිටීද, නොඇලුනාවූ සිත් ඇති එහි නොබැසගන්නා ලද වේදනාවෝ වෙත්ද, 
# “රසයෙහි ගිජුනොවී වේදනා නොලබන ඔහු කෙලෙස් ක්ෂය කරයි. හෙතෙම සිහියෙන් යුක්තව හැසිරෙයි. එසේ කර්ම රැස්නොකරන ඔහු දුක් නසයි. නිවන ඔහුගේ සමීපයෙහිය. 
# “කයින් ස්පර්ශ අරමුණු ලැබ සිහියෙන් යුක්තව නොඇලී සිටී නම් නොඇලුනාවූ සිත් ඇති එහි නොබැසගත්තාවූ ඔහුගේ වේදනාවෝ වෙත්ද, 
# “ස්පර්ශ අරමුණු ලැබීමෙන් එබඳු වේදනා ඇති පුද්ගලයා කෙලෙස් ක්ෂය කරයි. ඔහු සිහියෙන් යුක්තව හැසිරෙයි. ඔහු දුක් කෙළවර කරයි. නිවන් ඔහුගේ සමීපයෙහිය. 
# “සිතින් ධර්ම අරමුණු ලැබ සිහියෙන් යුක්තව එහි නොඇලී සිටින්නාවූ පුද්ගලයා නොඇලුනාවූ සිතින් යුක්තව එහි නොබැසගන්නාලදුව වේදනාවක් විඳීද, 
# “ධර්ම අරමුණු සේවනය කිරීමෙන් වේදනා ලැබූ ඔහු කෙලෙස් ක්ෂයකරයි. කර්ම රැස්නොකරන ඒ පුද්ගලයා සිහියෙන් යුක්තව හැසිරෙයි. කර්ම රැස්නොකරන ඒ පුද්ගලයාට දුක් නැසියහැකිය. නිවන ඔහු සමීපයෙහිය.” 
#
# සෙල ථෙර ගාථා
# “සම්පූර්ණ සිරුරක් ඇති, යහපත්ව උපන් වීර්යවත්වූ සිත සතුටුකරන දැකුම් ඇති, භාග්‍යවතුන් වහන්සේ රන්වන් සිරුරක් ඇත්තේය. සුදු දත් ඇත්තේය. 
# “යහපත්ව උපන් මනුෂ්‍යයාගේ යම් ලක්ෂණ කෙනෙක් වෙත්ද, ඒ සියලු මහා පුරුෂ ලක්ෂණයෝ ඔබ වහන්සේගේ ශරීරයෙහි ඇත්තාහ. 
# “පැහැදුනු ඇස ඇත්තේය. යහපත් මුව මඬලක් ඇත්තේය. උස මහතය. ඍජුය ආනුභාසම්පන්නය, ශ්‍රමණ පිරිස මැද සූර්යාමෙන් බබළයි. 
# “ශ්‍රමණයන්වහන්ස, යහපත් දැකුම් ඇති රන්කඳකට බඳු සිවියක් ඇති, මෙවැනි උතුම් වර්ණයෙන් යුත් ඔබ වහන්සේට පැවිදිකමින් කවර ප්‍රයෝජනද? 
# “රථයන්ට වෘෂභයකු බඳු ඔබවහන්සේ ජම්බුද්වීපයට අධිපති සතර මහා දිවයින දිනනසුළු සක්විති රජවන්නට සුදුසුය. 
# “ගෞතමයන් වහන්ස, මහත් භොග ඇති රජවරු ඔබට අනුකූල වෙත්. රජුන් විසින් පුදන මනුෂ්‍යයන් විසින් පුදන අධිපතියෙක්ව රජ කරනු මැනව.” 
# “භාග්‍යවතුන් වහන්සේ සෙලය ශ්‍රෙෂ්ඨ ධර්මරාජවූ මම රජෙක් වෙමි. යම් චක්‍රයක් කිසිවෙකු විසින් නොපැවැත්විය හැකිද ධර්මයෙන් ඒ ආඥාව පවත්වමි.” 
# “සෙලතෙම නිරුත්තරවූ ධර්මරාජතෙමේ සම්බුද්ධ යයි අඳුන්වන්නේය, ධර්මයෙන් ආඥාව පවත්වන්නෙමියි කියන්නේය. මෙය ගෞතමයන් වහන්සේ ප්‍රකාශ කරණ සේක්වායි” කීයේය. 
# “පින්වත්හු විසින් පවත්වනලද ධර්ම චක්‍රයට අනුව ඔබවහන්සේ අනුගමනය කරණ සේනාපතියාවූ ශ්‍රාවකයා කවරේද?” 
# “සෙලය නිරුත්තරවූ යම් ධර්ම චක්‍රයක් ඇත්තේද මා විසින් පවත්වන ලද චක්‍රය තථාගතයන් වහන්සේගේ අනුජාතවූ සැරියුත් තෙරුන් පවත්වයි. 
# “දතයුත්ත දන්නාලදී. වැඩියුත්ත වඩනලදී. දුරුකළ යුත්ත දුරුකරණලදී. බ්‍රාහ්මණය එහෙයින් මම බුද්ධයෙමි. 
# “බමුණ, මා කෙරෙහි සැකය දුරුකරව, ස්ථිරසේ සිත පිහිටුවාගනුව, නිතර බුදුවරුන්ගේ දර්ශනය දුර්ලභවේ. 
# “බ්‍රාහ්මණය, ලෝකයෙහි යම් කෙනෙකුන්ගේ නිතර පහළවීම දුර්ලභද, බ්‍රාහ්මණය, ඒ මම රාගාදී කෙලෙස් කපන්නෙක් වූයෙම් සම්බුද්ධවෙමි. 
# “බ්‍රහ්මයෙකු වැනි කිසිවෙකුට සමාන නොවූ මර සෙනඟ පරදවන (මම) බිය නැත්තේ සියලු අමිත්‍රයන් වසඟ කොට සතුටුවෙමි.” 
# “පසැස් ඇති කෙලෙසුන් කපාහරින මහාවීරතෙම සිංහයා මෙන් වනයෙහි නාදකරයි. උන්වහන්සේ යම්සේ දෙශනා කෙරෙද්ද, එයට කන් නමව්. 
# “බ්‍රහ්මයා බඳු උපමා ඇති මරසෙනඟ පරදවන බුදුන් දැක චණ්ඩලාදී නීච ජාති ඇත්තාවූද, කවරෙක් නොපහදින්නේද, 
# “යමෙක් මා කැමැත්තේද නවතීවා යමෙක් නොකැමැත්තේද යේවා මම ශ්‍රෙෂ්ඨ ප්‍රාඥයන් වහන්සේගේ සමීපයෙහි පැවිදිවන්නෙමි. 
# “පින්වත්හට ඉදින් මේ සම්‍යක් සම්බුද්ධයන්වහන්සේගේ ශාසනය කැමතිවේද, අපිද ශ්‍රෙෂ්ඨ ඤාණවන්තයා සමීපයෙහි පැවිදිවන්නෙමු. 
# “මේ තුන්සියයක් බ්‍රාහ්මණයෝ ඇඳිලිබැන්දාහු බඹසර ඉල්ලති. භාග්‍යවතුන් වහන්ස, ඔබවහන්සේ සමීපයෙහි බ්‍රහ්මචර්යයෙහි හැසිරෙමු.” 
# භාග්‍යවතුන් වහන්සේ, සේලය. බ්‍රහ්මචර්යය මනාකොට ප්‍රකාශකරන ලද්දේය. ඇස ඉදිරිපිට විපාක ඇත්තේය. කල්නොයවා දෙන විපාක ඇත්තේය. යම්තැනක අප්‍රමාදව හික්මෙන්නහුට ප්‍රව්‍රජ්‍යාව හිස එකක් නොවේ.” 
# “ඇස් ඇති භාග්‍යවතුන් වහන්ස, මින් අටවෙනි දිනයෙහි ඔබවහන්සේගේ පිහිට ලබා සප්ත රත්නයක් ඇති ශාසනයෙහි හික්මවූයේද, 
# “නුඹවහන්සේ බුද්ධය. නුඹවහන්සේ ශාස්තෘය. නුඹවහන්සේ මාරයන් මැඩපවත්වන මුනිය, නුඹවහන්සේ අනුශය සහිත කෙලෙස් නසා එතරවූයේ වාද පිරිසද එතර කෙළෙහිය. 
# “ඔබවහන්සේ විසින් කෙලෙස් ඉක්මවනලදී. නුඹවහන්සේ විසින් ආශ්‍රවයෝ පලන ලදහ. අත්හළ බිය හා භයානක අරමුණු ඇති තෘෂ්ණාදීන් නොගන්නාවූ සිංහයෙක් බඳු වූයෙහිය. 
# “තුන් සියයක් පමණ මේ භික්ෂූහු ඇඳිලි බැඳ සිටිත් වීරයන් වහන්ස, සිරිපා දික්කළ මැනව කෙලෙස් නොගන්නා නාගයෝ බුදුන් වඳිත්වා.”

# භද්දිය ථෙර ගාථා
# “මම මීට පළමුව ඇත්කඳ පිට සිට ගමන් කෙළෙමි. සියුම් කසී වස්ත්‍ර හැන්දෙමි. පවිත්‍රවූ මාංශයෙන් සෙවුනා ලද හැල් සාලින් පිසූ බත් අනුභව කෙළෙමි. 

# “කාලිගොධා නම්වූ ස්ත්‍රියගේ පුත්‍රවූ භද්දිය වන මම අද සිල්ගුණයෙන් යුක්තවූයේ, ශ්‍රමණ ධර්මයෙහි නිතර යෙදුනේ පිණ්ඩියාලොප භෝජනයෙන් යුක්තවූයේ ධ්‍යාන කරමින් සතුටින් වෙසෙමි. 
# “කාලිගොධා බැමිණියගේ පුත්‍රවූ භද්දිය ස්ථවිරයන් පාංශුකූලිකාංගය සමාදන් වීමෙන් හා පිණ්ඩපාතිකාංගය සමාදන් වීමෙන් කිසිවක් තෘෂ්ණාදෘෂ්ටියෙන් නොගෙණ ධ්‍යාන කරමින් වාසය කරයි. 
# එසේම පිණ්ඩපාතිකාංගය සමාදන් වීමෙන්ද, තෙචීවරිකාංගය සමාදන්වීමෙන්ද, සපදානචාරිකාවෙන්ද, යුක්තව එකාසනිකාංගයෙන් යුක්තව සතුටින් වාසය කරමි.

# එසේම පත්තපිණ්ඩිකාංගයෙන් යුක්තවද, ඛලුපච්ඡා භක්තිකාංගයෙන් යුක්තවද, ආරණ්‍යකාංගයෙන් යුක්තවද වෘක්ෂමූලිකාංගයෙන් යුක්තවද, සතුටින් වෙසෙමි.

# අබ්භොකාසිකාංගයෙන් යුක්තවද, සොසානිකාංගයෙන් යුක්තවද, යථාසන්තිතිකාංගයෙන් යුත්වද, නේසජ්ජිකාංගයෙන් යුක්තවද, සතුටින් වාසය කරමි.

# කාලිගොධා බැමිණියගේ පුත්‍රවූ භද්දිය ස්ථවිරයන් වහන්සේ පටන්ගන්නා ලද වීර්යය ඇත්තෙක්ව සතුටින් වෙසෙයි. 
# ස්වර්ණමයවූ රැස් විහිදෙන්නාවූ රන්තලිය හැර මැටි පාත්‍රය ගත්තෙමි. මේ දෙවන සේවනය වෙයි. 
# ප්‍රාකාරවලින් වටකරනලද ස්ථිරවූ දොර අගුළුවලින් යුක්ත මතුමහල් තලයෙහි කඩුගත් අත් ඇති පුරුෂයන් විසින් රකිනාලදුව ත්‍රාශ සහිතව මම පළමුව විසුවෙමි. 
# “කාලිගොධා බැමිණියගේ පුත්‍රවූ භද්දිය වන මම අද ත්‍රාශයන් නැතිව භය හා භයට හේතු දේ දුරුකොට වනයෙහි ධ්‍යාන කරමින් සතුටින් වෙසෙමි. 
# සීලස්ඛන්ධයෙහි පිහිටා ඵල සමාපත්ති නිරොධ සමාපත්ති වඩමින් පිළිවෙලින් සියලු සංයෝජන ධර්මයන් ක්ෂයකොට නිවන් සැපයට පැමිණියෙමි.”

# අංගුලිමාල ථෙර ගාථා
# “ශ්‍රමණය, ඔබ යන්නේම සිටියෙමියි කියයි මමද සිටියේ නොසිටියෙමියි කියන්නෙමි. ශ්‍රමණය මේ අර්ථය මම විචාරමි. ඔබ කෙසේ සිටියෙහිද මම කෙසේ නොසිටියෙම්ද? 
# “අංගුලිමාලය, මම සියලු සත්වයන් කෙරෙහි දඬු මුගුරු බහා තබා සැමකල්හි සිටියේ වෙමි. තෝ සත්වයන් කෙරෙහි අකරුණාවන්තව වසන්නෙහිය. එහෙයින් මම සිටියෙමි. තෝ නොසිටියේ වෙහි. 
# “මට අනුකම්පා පිණිස ලෝකයා විසින් පුදනු ලබන්නාවූ මහර්ෂීවූ ශ්‍රමණයන්වහන්සේ මෙහි වැඩිසේක. ඒ මම උන් වහන්සේගේ ධර්මයෙන් යුක්තවූ ගාථා ඇසුවෙමි. ඒ අසා බොහෝ කලක් පුරුදු කරන ලද පාප සහශ්‍රයන් දුරුකර හැරීමි. 
# “ඊට අනතුරුව අඟුමල් සොරතෙමේ කඩුව හා සෙසු ආයුධත් ප්‍රපාතවූ විවරස්ථානයක හෙලුවේය. ඉක්බිති අඟුල් මල් සොරතෙමේ සුගත්වූ බුදුන් වහන්සේගේ පාද වැන්දේය. එහිදීම ප්‍රව්‍රජ්‍යාවද ඉල්ලීය. 
# “දෙවියන් සහිත සත්ව ලෝකයට මහා කාරුණිකවූද මහර්ෂීවූද බුදුන් වහන්සේ ‘එව භික්ෂුවයි’ කියා වදාළහ. ඔහුට එයම එහි භික්ෂුභාව ප්‍රව්‍රජ්‍යාවට හේතු විය. 
# “යමෙක් පළමුව ප්‍රමාදවිහරණයෙන් යෙදී, පසුව අප්‍රමාදයෙන් යුක්ත වේද, උපක්ලේශ රහිත චන්ද්‍රයා නේ හෙතෙම මේ ලෝකය බබුළුවයි. 
# “යමෙක් කරන ලද පාපකර්ම, කුශල්වලින් වසාද, ඔහු උපක්ලේශ රහිත චන්ද්‍රයා මෙන් මේ ලෝකය බබුළුවයි. 
# “යම් භික්ෂුවක් තරුණවූද බුදුසසුන්හි යෙදුනේ නම් උපක්ලේශ රහිත චන්ද්‍රයා මෙන් මේ ලෝකය බබුළුවයි. 
# “මගේ විරුද්ධකාරයෝද, ධර්මය අසත්වා, මගේ විරුද්ධකාරයෝද බුද්ධශාසනයෙහි යෙදෙත්වා, මගේ විරුද්ධකාරයෝද සත්පුරුෂයන් ඇසුරු කරත්වා. සත්පුරුෂයෝ වනාහි යහපත් ධර්මවල පිහිටුවත්. 

# “මගේ විරුද්ධකාරයන් විසින් ශාන්තිය කියන්නාවූ මෛත්‍රියට ප්‍රශංසා කරන්නාවූ ධර්මය සුදුසු කාලයෙහි අසත්වා, එයම ඉගෙණ ඊට අනුව කටයුතු කෙරෙත්වා, 
# “විරුද්ධකාරයෝ මට හිංසා නොකරත්වා, අන්කිසිවෙකුටත් හිංසා නොකරත්වා, පරමශාන්තියට පැමිණ චංචල වූද, නිශ්චලවූද, සත්වයන් ආරක්ෂා ලබත්වා, 
# “දිය ගෙනයන්නෝ කැමති තැන්වලට ගෙණයත්. හී වඩුවෝ ඍජුකොට ඊතල සාදා ගනිත්. වඩුවෝ කැමැති පරිදි දැව ඇද හැර සැස ගනිත්. එසේම පණ්ඩිතයෝ තමන් දමනය කර ගනිත්. 
# “සමහරු දඬුවලින්ද, දමනය කර ගනිත්. සමහරු අංකුසයෙන්ද (හෙණ්ඩුවෙන්) සමහරු කසයෙන්ද දමනය කරත්. මම වනාහි එබඳුවූ දඬු නැතිවද, ආයුධ නැතිවද දමනය වීමි. 
# “පළමුවෙන් මම හිංසකයකුවූයේම අහිංසක නාමය ලදිමි. අද සත්‍යවශයෙන්ම මම අහිංසක වූයෙමි. කිසිවෙකුට මම හිංසා නොකරමි. 
# “මම මීට පළමුව මිනිසුන් මරණ සොරෙකු වීමි. කාමෝඝයෙන් ගසාගිය මම බුදුන් සරණ ගියෙමි. මම ලේ පෙරෙන අංගුලිමාල නමින් විසුවෙමි. පසුව සරණාගමනයට පැමිණ භවය සුන්කළ හැටි බලව. 
# “එබඳුවූ අගතිගාමී බොහෝ අකුසල කර්ම කොට ප්‍රහීනකරන ලද කර්මවිපාක ඇත්තේ, කෙලෙස් ක්ෂය කරන ලද්දේ, සිවුපසය වළඳමි. 
# “දුශ්ප්‍රාඥවූ අඥාන පුද්ගලයෝ ප්‍රමාදයෙන් කල්ගත කරත්. ඥානවන්තයෝ අප්‍රමාදය ශ්‍රෙෂ්ඨ ධනයක් මෙන් රකිත්. 
# “ප්‍රමාදයෙහි නොයෙදෙව්. කාමරති සන්ථවයද, නොකරව්. අප්‍රමාදව ධ්‍යාන කරන පුද්ගලයෝ උතුම් සැපයට පැමිණෙත්. 
# “මගේ ඊම යහපත් ඊමක් විය. නොආයේ නම් නපුරුය. පැවිදිවෙමියි සිතීමද, යහපත් සිතීමෙකි. එය නොසිතුවේ නම් නපුරෙකි. සාවද්‍ය අනවද්‍ය ධර්මයන් කෙරෙහි යම් නිර්වාණයක් ශ්‍රෙෂ්ඨවේද, එයට පැමිණියෙමි. 

# “මගේ ඊම යහපත් විය. නොඊම නපුරුය. මගේ සිතීම යහපත්ය. නොසිතීම නපුරුය. ත්‍රිවිද්‍යාවද ලදිමි. බුදුන්ගේ අනුශාසනාවද, කෙළෙමි. 
#
# “කැලයෙහිද, රුක් මුල්හිද, පර්වතයන්හිද, ගුහාවන්හිද, යන යම් යම් තැනෙක සිටියේ නම් පළමුව දුකෙන් විසුවෙමි. දැන් වනාහි ඒ ස්ථානවල සැපසේ වෙසෙමි. 
# “සැපයේ සයනය කරමි. සැපසේ සිටිමි. සැපසේ දිවි පවත්වමි. මාරයාට ගොදුරු නොවී සිටිමි. ශාස්තෲන්වහන්සේගේ අනුකම්පාව නිසයි. අහෝ! ඒ ආශ්චර්යවත්ය. 
# “පෙර බ්‍රහ්මයා විසින් දෙන ලද උසස් කුලයක ඉපදීමි. ඒ මම අද ධර්මරාජන්වූද, සුගතවූද, ශාස්තෲන් වහන්සේගේ පුත්‍රයෙකු වෙමි. 
# “දුරු කරන ලද තෘෂ්ණාව ඇතිව කිසිවක් තෘෂ්ණා දෘෂ්ටි දෙකින් නොගෙන චක්ෂුරාදී ඉඳුරන් සංවරකොට දුක් මුල් නසා මා විසින් කෙලෙස් නැති කරන ලදී. 
# “මා විසින් ශාස්තෲන් වහන්සේ ඇසුරු කරන ලදී. බුදුන්ගේ අනුශාසනාවද කෙළෙමි. ගරුභාරයද බහා තැබුවෙමි. භවයාගේ කෙළවරද නැති කෙළෙමි.” 
#
# අනුරුද්ධ ථෙර ගාථා
# “මවුපියන්ද, සහෝදර සහෝදරීන්ද, නෑ පිරිසද, හැර පඤ්චකාමයෙන් තොරව අනුරුද්ධ ස්ථවිරයන් වහන්සේ ධ්‍යාන කරයි. 
# “නෘත්‍ය ගීතාදියෙන් යුක්තව සුන්දර තාලශබ්දයෙන් ප්‍රබොධවනුයේ විසුවෙමි. මාරවිෂයෙහි ඇලුනේ එයින් ශුද්ධියක් නොවන බව දැන, 
# “එයද හැරදමා බුද්ධශාසනයෙහි ඇලුනේ සියලු සැඩ වතුර ඉක්මවා අනුරුද්ධ ස්ථවිරයන් වහන්සේ ධ්‍යාන කරයි. 

# “මනෝරම්‍යවූ රූපයෝද, ශබ්දයෝද, ගන්ධයෝද රසයෝද, ස්පර්ශයෝද යන මේ සියල්ලම හැරදමා අනුරුද්ධ ස්ථවිරයන් වහන්සේ ධ්‍යානකරයි. 
# “පිඬු සිගීමෙන් නැවතුනේ තනිවූයේ තෘෂ්ණාව නැත්තේ ඤාණවන්තවූයේ පාංශුකූල චීවරයන් සොයනුයේ අනුරුද්ධ ස්ථවිරයන් වහන්සේ ධ්‍යාන කරයි. 
# “සෙවීමෙන් පිළිකුල්කොට ගනුලබනුයේ එය ධොවනයකොට හා රජනයකොට (සෝදා පඬුපොවා) ඒ පාංශුකූල චීවරයම කැපකොට පොරවා අනුරුද්ධ ස්ථවිරයන් වහන්සේ කෙලෙස් නැතිව වෙසෙයි. 
# “බොහෝ දෙයක් කැමතිවීමද, ස්වල්පදෙයකින් සතුටු නොවීමද, ගිහි පැවිද්දන් හා සංසර්ග වීමද උඩඟුබවද, යන මේ කරුණු ලාමක කෙලෙසීම බවට හේතුවන ධර්මයෝය. 
# “යමෙක් සිහියෙන් යුක්තවේද, අල්පෙච්ඡ ගුණයෙන් යුක්තවේද, සන්තෝෂ ගුණයෙන් යුක්තවේද, සිත එකඟ ගුණයෙන් යුක්තවේද, විවේක ගුණයෙන් යුක්තවේද, පටන් ගන්නාලද වීර්ය ඇත්තේද, 
# “ඒ පුද්ගලයාට මේ කියන යහපත් බොධිපාක්ෂික ධර්මයෝ වෙත්ද; හෙතෙම අනාශ්‍රවද වෙයි. මහර්ෂීවූ බුදුරජාණන් වහන්සේ මේ බව වදාළහ. 
# “මගේ චිත්ත සංකල්පනාවෝ දැන ලෝකයෙහි නිරුත්තරවූ ශාස්තෲන් වහන්සේ මනෝමය ඉර්ධියෙන් යුක්ත කයින් මා කරා පැමිණිසේක. 
# “යම්කලෙක මට චිත්ත සංකල්පනාවෝ වූවාහුද එයින් මත්තෙහි සම්පූර්ණ කිරීම් වශයෙන් උන්වහන්සේ දෙසූසේක. ප්‍රමාදකරවන ධර්මයන්හි නො ඇලුනාවූ බුදුන් වහන්සේ ප්‍රමාද නොවන ධර්මය දෙසූසේක. 
# “මම උන්වහන්සේගේ ධර්මය දැන ශාසනයෙහි ඇලී විසුවෙමි. ත්‍රිවිද්‍යාවද ලදිමි. බුදුන්ගේ අනුශාසනාවද කෙළෙමි. 

# “මම පස් පණස් හවුරුද්දක් නේසජ්ජිකාංගය විසුවෙමි. විසිපස් හවුරුද්දක් නිදි වර්ජිතව විසුවෙමි. 
# “චතුර්ථධ්‍යානයෙහි නැවති සිත් ඇත්තාවූද තාදී ගුණයෙන් යුක්තවූද, තෘෂ්ණාව නැත්තාවූද, පසැස්ඇත්තාවූ බුදුරජාණන් වහන්සේගේ ආශ්වාස ප්‍රශ්වාසයෝ නැත්තාහුය උන්වහන්සේ නිවන් අරභයා පිරිනිවන් පෑසේක. 
# “කුසීත නොවූ සිතින් යුක්තව මරණාන්තික වේදනාව ඉවසූසේක. පහනක් නිවීගියාක්මෙන් සිත විමොක්ෂ භාවයට පත්විය. 
# “මුනීන්ද්‍රයන් වහන්සේගේ ස්පර්ශය පස්වෙනිකොට පවත්නාවූ රූපාදී ධර්මයෝ අවසන් වූවාහුය. පිරිනිවන් පෑ බුදුන් කෙරෙහි අන්‍ය ධර්මයෝද ඇතිනොවෙත්. 
# “යම්කිසි දේව නිකායක හටගැණීමක් හෝ නොවෙයි. ජාති සංසාරය ක්ෂය කරණලදහ. පුනර්භවයක් නැත්තේය. 
# යම් ක්ෂීණාශ්‍රව භික්ෂුවකට මොහොතකින් බඹලොව ඇතුළුව සියලු ලොකධාතුව දතහැක්කේද, එබඳු ඍද්ධිගුණ ඇත්තාවූද, චුති උත්පත්ති දෙක දක්නාවූද, ඒ භික්ෂුව දෙවියෝ දකිත්. 
# “මම පෙර කලෙක අන්නහාර නම් බැළමෙහෙකොට ජීවත්වූ දිළින්දෙක් වීමි. එකල්හි කීර්තිමත් උපරිට්ඨ නම් පසේබුදුවරයන් වහන්සේට දානයක් දුන්නෙමි. 
# “ඒ කුශලයෙන් මම ශාක්‍යකුලයෙහි උපන්නේ නෘත්‍ය ගීතාදියෙන් යුක්තවූයේ යහපත් තාල ශබ්දයෙන් ප්‍රබෝධවනුයේ අනුරුද්ධ නම් විය. 
# “ඒ මම ශාස්තෲවූ පහවූ භය ඇති සම්බුදුන් වහන්සේ දුටුවෙමි. උන්වහන්සේ කෙරෙහි සිත පහදවා අනගාරික නම්වූ ප්‍රව්‍රජ්‍යාවට පැමිණියෙමි. 
# “මම පළමුව යම් තැනක විසුයේද, පෙර පසු තැන් දන්නා නුවණින් ඒ ස්ථාන දනිමි. තව්තිසා දෙව්ලොවද, ශක්‍ර ජාතියෙහිද සිටියෙමි. 

# “ජම්බුද්වීපයට ප්‍රධානවූ මේ සතරමහා දිවයින් ජය ගත්තේ සියලු මනුෂ්‍යයන්ට ප්‍රධානවූයේ සත්වරක් සක්විති රජකම් කෙළෙමි. ඒ කාලයෙහි දඬුවලින් හා ආයුධවලින් තොරව ධර්මයෙන් අනුශාසනා කෙළෙමි. 
# “මේ මිනිස්ලොවින් චුතව දෙව්ලොවට අධිපතිභාවය සත්වරක්ද ඒ දෙව්ලොවින් චුතව මනුෂ්‍යලෝකයෙහි සක්විති රජ බව සත්වරක්ද, යන භවාන්තර දහසතරක් සැරිසැරුවෙමි. 
# “දිව්‍යලෝකයේ සිටියේම පෙරකඳ පිළිවෙළ දන්නා නුවණ ලදිමි. පසඟින් යුත් සමාධියද, ශාන්තභාවයද එකඟකමද, කෙලෙස් සිඳදැමීමද කොට දිව්‍යචක්ෂුර්භිඥානය පිරිසිදු කෙළෙමි. 
# “මනුෂ්‍යයින්ගේ ආගිය තැන්ද, අවශෙෂ තිරිසන් ගත සත්වයින්ගේත් ඉපදීමට පෙර සිටිතැන්ද, පඤ්චංගික ධ්‍යානයේ සිටියේ චුතිඋත්පත්ති දෙක දන්නා ඥාන බලයෙන් දනිමි. 
# “මා විසින් ශාස්තෲන් වහන්සේ ඇසුරුකරනලදී. බුදුන්ගේ අනුශාසනාවද කෙළෙමි. ගරුභාරයද බහාතැබුවෙමි. භවයාගේ කෙළවරට පැමිණියෙමි. 
# “මම අව්‍යාන මොහොතෙහිදී වජ්ජීන්ගේ වෙළුව නම් ගමෙහි උණපඳුරක් යට ආශ්‍රව රහිතව පිරිනිවන්පාමි.” 
#
# පාරාපරිය ථෙර ගාථා
#
# “පුරුෂොත්තමවූද ලෝකනාථවූද, බුදුන්වහන්සේ ජීවමානව වැඩසිටින කල්හි භික්ෂූන්ගේ ඉරියව් පැවැත්ම අන්පරිද්දෙකින් විය. දැන් වනාහි ඊට වඩා වෙනස් ආකාරයකින් පෙණේ. 
# “ඒ අවස්ථාවෙහිද සීත සුළඟින් ආරක්ෂාවීම පිණිසත් ලජ්ජා ස්ථාන වැසීම පිණිසත් සිවුරු පරිභොග කෙළෙමි. ප්‍රත්‍ය සන්තොෂයෙන් යුක්තවද වැළඳුවෙමි. 
# “ප්‍රණීතවූ හෝ අප්‍රණීතවූ හෝ ස්වල්පවූ හෝ බොහෝවූ හෝ ප්‍රත්‍යලැබීමෙහිදී ගිජුබැව් නොකොට එහි නොඇලී යැපෙන පමණක් ගෙණ වැළඳුවෙමි. 
# “දිවි පැවැත්මට හේතුවන බෙහෙත් පිරිකර ආදියෙහිද දැඩිඋනන්දුවක් නොදැක්වූවාහුය. ආශ්‍රව ධර්මයන් ක්ෂය කිරීමෙහිම උත්සාහවන්තවිය. 
# “කැලෑවද, රුක්මුල්හිද, කඳුවලද, ගුහාවලද විවේකයෙන් යුක්තව භාවනා කරනුයේ විවේකයම උතුම් කොට සැලකුවාහුය. 
# “යටහත් පැවතුම් ඇත්තේ, ශ්‍රද්ධා ඇත්තේ, පහසු පෝෂණයෙන් යුක්තවූයේ මෘදු බැව් ඇත්තේ, තද නොවූ සිත් ඇත්තේ සිහියෙන් යුක්තවූයේ මුඛර නොවූයේ යහපත කැමැති සිත් ඇත්තේ එහිම ඇලී විසීය. 
# “එහෙයින් ප්‍රසාදබවට පැමිණියේ කාය වාග්ප්‍රයෝග යොදා ප්‍රත්‍ය පරිභෝගයෝද ගෝචරාකාරයෙන් පැවතුනේ විය. ඉරියාපථයවනාහි සිනිඳු තෙල් ධාරාවක් ගලන්නාක් මෙන් විය. 
# “මහත්වූ ධ්‍යාන ඇත්තේද, මහත්වූ සිතින් යුක්ත වූයේද සියලු ආශ්‍රව ධර්මයන් ප්‍රහීණ කරනලද්දේ පිරිනිවන් පෑවාහුය. දැන් වනාහි එබඳු ගුණ ඇති ස්ථවිරයන්වහන්සේලා ස්වල්පයහ. 
# “කුශල ධර්මයන්ගේ හා ප්‍රඥාවගේ පිරිහීමෙන් සියලු ආකාරයන් ගෙන් උතුම්වූ සර්වඥශාසනය වැනසෙයි. 
# “පාප ධර්මයන්ගෙන් හා ක්ලේශයන්ගෙන් යුක්ත වූ කාලය පැමිණියාහුය. එබැවින් විවේකය දියුණුකොට ධාර්මික ප්‍රතිපත්තිය සිද්ධකර ගැනීමට නොහැකිවිය. 
# “කෙලෙස්වලින් වැඩෙන්නාවූ අන්ධ බාල ජනයන් අනුව හැසිරෙත්. උමතු රාක්ෂසයන්ගේ ක්‍රීඩා පරවශව බාලයන් මෙන් හැසිරෙත්. 
# “ක්ලේශයන්ගෙන් මඩනාලද සිත් ඇතිව ඒ ඒ තැන්හි දුවනුලබන්නාවූ ක්ලේශ වස්තූන්හිම ඇලී රන් රුවන් විසුරුවාගනුවයි කී කල්හි කැමැති කැමැති හැටියට ගනුලබන්නාක් මෙන් විය. 
# “සද්ධර්මය බැහැරකොට දෘෂ්ටීන්ට බැසගන්නා ලදුව මෙයම ශ්‍රෙෂ්ඨයයි ඔවුනොවුන් කලහ කරත්. පුත්‍රයෝද ධනයද භාර්යාවද හැරදමා එයින් නික්මී බත් සැන්දක් පමණකට නොකටයුතු කරත්. 
#
# “බඩ පිරෙනතුරු කා උඩුකුරුව හොවිත්. නැගිටීමෙන් පසුව බුදුන් විසින් ගරහනලද තිරිසන් කථාවන් කථා කරත්. 
# “කුඩා තල් අතු ආදී හස්ත ශිල්පයෝද සකස්කොට කිරීමෙන් නොහික්මී නොසන්සුන් සිත් ඇතිව ශ්‍රමණභාවය ඉක්මවත්. 
# “මැටි තෙල් සුණු යනාදියද, දිය භෝජන ආසන යනාදියද, වැඩියක් කැමැතිබැව් ඇතිව ගිහීන්ට දෙත්. 
# “දැහැටි දඬුද, දිවුල්ද, මල්ද, පිණ්ඩපාතයෙන් ලැබුණු කැවිලිද අඹ නෙල්ලි යනාදි ගෙඩිද වැඩිදෙයක් කැමැතිව ගිහියන්ට දෙත්. 
# “වෙදුන් මෙන් බෙහෙත් කිරීමෙහි යෙදෙයි. ගිහියන්ගේ ලොකු කුඩා කටයුතුවලදී ගිහියන්මෙන් කටයුතු කරයි. ශරීරය සැරසීමෙහි වෙසෙඟනන් මෙන් කටයුතු කරයි. එෛශ්චර්යයෙහි ක්ෂත්‍රියයන් මෙන් වැඩකරත්. 
# “රන් නොරන් ආදී වශයෙන් පෙන්වා කරණ විජ්ජාවලින්ද කූටමානාදී වශයෙන් කරන වංචනික ක්‍රියාවන්ගෙන්ද යන බොහෝ වැරදි ජීවිකාවන්ගෙන් ආමිෂ පරිභොගය කරයි. 
# “කැප යෙදීම්වල නොයෙදී අකැප වශයෙන් දිවි පවත්වන්නාහු බොහෝ ධනයද රැස්කරත්. 
# “ධර්මයෙන් තොරව නොමගින් පිරිසට සංග්‍රහකරත්. ධර්මය දෙශනා කරන්නේද, යහපත පිණිස නොව ලාභය පිණිසමයි. 
# “සංඝ ගතියෙන් බාහිරවූවාහු සංඝ ලාභයට කෝලාහල කරත්. අනුන්ගේ ලාභයෙන් ජීවත්වන්නාහු විලිබිය දෙකින් තොරව වාසය කරත්. 

# “ශ්‍රමණ ධර්මයන්හි නොයෙදී හිස මුඩුකොට සිවුරු දරා ලාභ සත්කාරයෙහි මුසපත්ව ගරු සැලකිල්ල ඉවසීමෙහි කැමැති වෙත්. 
# “නොලැබූ ධ්‍යාන විදර්ශනා ලැබීමට කිරීම හෝ රැකීමට නොහැකිව නොයෙක් බාහිර දෙය කරත්. 
# “පාවහන් නොමැතිව කටුස්ථානවල හැසිරෙන්නාක් මෙන් සිහියෙන් යුක්තව ඤාණවන්තයෝ ගමෙහි හැසිරෙන්නාහ. 
# “පූර්වයෝගීන් සිහිකරමින් ධර්මයට අනුව පිළිපදිමින් විදර්ශනා උපදවා නිවන් ලබාගත යුතුය. 
# “මඩනාලද ඉඳුරන් ඇති ඒ ශ්‍රමණයන් වහන්සේ මෙසේ කියා පිරිනිවන් පෑසේක. ඒ බ්‍රාහ්මණයා නැවත ඉපදීම ක්ෂයකරණලදී.”
