චතුක්ක නිපාතය. 
# නාගසමාල ථෙර ගාථා
# “අලංකාරවූ යහපත්ව ඇන්දාවූ මල් දරණ සඳුන් ගෑ ශරීර ඇති නැටුම් දත් ස්ත්‍රියක් තූර්ය ගසන කල මහමග මැද නටයි. 
# “මම පිඬු පිණිස හැසිරෙන්නේ අලංකාරවූ යහපත්ව ඇන්දාවූ දමන ලද මරු තොණ්ඩුවක් වැනි ඈ දුටුවෙමි. 
# “ඉක්බිති මාගේ මනසට සිතිවිල්ලක් පහළ විය. දොස් දැක්ම පහළ විය. කළකිරීම පැමිණ සිටියේය. 
# “ඉක්බිති මාගේ සිත කෙලෙසුන්ගෙන් මිදුණේය. ධර්මයෙහි යහපත් බව බලව. ත්‍රිවිද්‍යාවන්ට පැමුණුනෙමි. බුදුන්ගේ අනුශාසනය කරණ ලදී.”
# ආයුෂ්මත් නාගසමාල තෙර මෙසේ මේ ගාථා කීයේය. 
# භගු ථෙර ගාථා
# “මම නිදිබර ස්වභාව ඇත්තේ විහාරයෙන් නික්මුනෙමි. සක්මනට බසින්නේ එහිම බිම වැටුනෙමි. 
# “අත් පා අතගා නැවත සක්මනට බැස්සෙමි. ඒ මම ඇතුළත සන්සුන්ව සක්මනෙහි සක්මන් කෙළෙමි. 
# “ඉක්බිති මාගේ මනසට සිතිවිල්ලක් පහළ විය. දොස් දැක්ම පහළ විය. කළකිරීම පැමිණ සිටියේය. 
# “ඉක්බිති මාගේ සිත කෙලෙසුන්ගෙන් මිදුණේය. ධර්මයෙහි යහපත් බව බලව. ත්‍රිවිද්‍යාවන්ට පැමුණුනෙමි. බුදුන්ගේ අනුශාසනය කරණ ලදී.”
# ආයුෂ්මත් භගු තෙර මෙසේ මේ ගාථා කීයේය. 
# සභිය ථෙර ගාථා
# “අන්‍යයෝ අපි මෙහි මාරයා සමීපයට යමුයයි නොදනිත්. යම් කෙනෙක් එහි යන බව දනිත් නම් ඉන් කෝලාහල සන්සිඳෙත්. 
# “නොදන්නාහු යම් කලෙක්හි නොමැරෙන්නවුන් මෙන් ඉරියව් පවත්වත්ද යම් කෙනෙක් ධර්මය දනිත්ද ඔව්හු ලෙඩුන් අතරෙහි ලෙඩ නැතිව වසත්. 

# “ලිහිල් කළ යම් කර්මයක් වේද කිලිටිවූ යම් ව්‍රතයක් ඇත්තේද සැකයෙන් සිහි කටයුතු හැසිරීමක් වේද එය මහත් ඵල නොවේ. 
# “යමෙකුට සමාන සිල් ඇත්තන් කෙරෙහි ගෞරවයක් නොලැබේද, අහසට පොළොව මෙන් ධර්මයෙන් ඈත්වේ.”
# ආයුෂ්මත් සභිය තෙර මෙසේ මේ ගාථා කීයේය. 
# නන්දක ථෙර ගාථා
# “අශූචියෙන් පිරුණාවූ දුර්ගන්ධවූ මාරයාගේ පක්ෂයෙහිවූ කුණු වැගිරෙන ශරීරයට නින්දා වේවා. තොපගේ ශරීරයෙහි යමක් නිතර ගලා බසිත්ද, ඒ සිදුරු නවයකි. 
# “පැරණි මට ඇතිවූ අදහස් නොසිතනු මැනවි. තථාගතයන් වහන්සේ අවමන් නොකරන්නෙහිය. ඒ ශ්‍රාවකයෝ දිව්‍ය ලොකයෙහිත් ඇලෙන්නාහුය. මනුෂ්‍ය ලොකය ගැන කියනුම කවරේද? 
# “බාලවූ ප්‍රඥාව නැති නපුරු සිතිවිලි සිතන මොහයෙන් වැසුනාවූ එවැන්නෝ මාරයා විසින් දමන ලද බැම්මෙහි බැඳෙත්. 
# “යම් කෙනෙකුන් විසින් රාගයද ද්වෙෂයද නොදැනීමද සිඳින ලද්දේද නොසෙල්වෙන බැමි ඇති සර්වඥ පුත්‍රයෝ එහි නොඇලෙත්.”
# ආයුෂ්මත් නන්දක තෙර මෙසේ මේ ගාථා කීයේය. 
# ජම්බුක ථෙර ගාථා
# “පස් පනස් වසක් ධූලි හා අශුචි දැරුවෙමි. මසකට වරක් ආහාර දිවගා බලමින් අළු ඉසීමෙන් බුරුල්වූ කෙස් අතින් උගුල්ලන්නේ විය. 
# “එක් පයකින් සිටියෙමි. ආසන දුරු කෙළෙමි. වේලුණු අශූචි කෑවෙමි. ආරාධනා නොඉවසීමි. 
# “මෙබඳු බොහෝ අපායට යන දේ කොට අපායට යන්නේ බුදුන් සරණ ගියෙමි. 
# සරණයාම බලව. ධර්මයෙහි යහපත් බව බලව, ත්‍රිවිද්‍යාවන්ට පැමිණියෙමි. බුදුන්ගේ ශාසනය කරණලදී.”
# ආයුෂ්මත් ජම්බුක තෙර මෙසේ මේ ගාථා කීයේය. 

# සොනක ථෙර ගාථා
# “ගයාවෙහි මැදින් මස පොහොය දිනයෙහි එකාන්තයෙන් මට යහපත් පැමිණීමක් වේවා. උතුම්වූ ධර්මය දෙශනා කරණ බුදුන් දුටුවෙම්ද එහෙයිනි. 
# “මහත් ආලොක ඇති පිරිසට නායකවූ ශ්‍රෙෂ්ඨත්වයට පැමිණි තමහට වඩා ශ්‍රෙෂ්ඨයක නැති බුදුන් දෙවියන් සහිත ලොකයාට ශ්‍රෙෂ්ඨ දැකීමකි. 
# “මහත් ඇතෙකු වැනිවූ මහාවීරයෙකුවූ මහා ආලොකයක් ඇති, කෙලෙස් නැති, සියලු කෙලෙස් නැති කළාවූ කිසිතැනක බියක් නැත්තාවූ ශාස්තෘන් වහන්සේ යම් හෙයකින් දැක්කේද එය මගේ යහපත් දැකීමකි. 
# “ඒකාන්තයෙන්ම වැරදි විශ්වාස නමැති බැම්මෙන් බඳිනු ලදුව මා බොහෝ කාලයක් කිලිටිවූයේද, ඒ භාග්‍යවතුන් වහන්සේ සියළු ගැටවලින් සෝනක තෙරුන් මිදෙව්වේය.”
# ආයුෂ්මත් සෝනක තෙර මෙසේ මේ ගාථා කීයේය. 
# සම්භූත ථෙර ගාථා
# “යමෙක් තෙමේ සැක සහිත කාලයෙහි වෑයම් කරයිද එතර වියයුතු කාලය හිස් කරයිද, නුවණ නැති බව හෙතුකොට බාලයා දුකට පැමිණෙයි. 
# “එබඳු පුද්ගලයාගේ මේ ආත්මයෙහිවූ අභිවෘද්ධිය කළුවර පක්ෂයෙහි හඳ මෙන් පිරිහෙත්. අයසටද පැමිණෙයි. මිත්‍රයන් සමගද විරුද්ධ වේ. 
# “යමෙක් සැක කළයුතු කාලයෙහි සැක කරයිද එතර විය යුතු කාලයෙහි එතරවේද නුවණින් යුක්තවීමෙන් පණ්ඩිතතෙමේ සැපයට පැමිණෙයි. 
# “ඔහුගේ අභිවෘද්ධිය සඳ මෙන් පිරෙයි. යසසටද කීර්තියටද පැමිණෙයි. මිත්‍රයන් හා විරුද්ධ නොවේ.
# ආයුෂ්මත් සම්භූත ස්ථවිරතෙමේ මෙසේ මේ ගාථා කීයේය. 
# රාහුල ථෙර ගාථා
# “දෙපැත්තෙන් යහපත්වූ සොඳුරු රාහුලයයි මට කීහ. යම් මම බුදුන්ගේ පුත්‍රද, යම් මම ධර්මයන්හි ඇස් ඇත්තෙමි. 

# “මා විසින් යම් පව් දුරු කළාහුද, යම් නැවත ඉපදීමක් මට නැද්ද දක්ෂිණාවට සුදුසු රහත් කෙනෙක්මි. නිවන පෙනෙන ත්‍රිවිද්‍යා ඇත්තෙකි. 
# “කෙමින් කටෙහි අසුවූ මසුන් මෙන් කාමයන්හි අන්ධය. දැලින් වැසෙන ලදී. ආශා නැමැති පියස්සෙන් වසනලදී. ප්‍රමාදවවන්ගේ නෑවූ මාරයා විසින් බඳින ලදී. 
# “ඒ කාමය මම උදුරා මාරයාගේ බැමි සිඳ මුල් සහිත තෘෂ්ණාව උදුරා සිහිල් වූයෙමි. නිවුණෙමි.”
# ආයුෂ්මත් රාහුල තෙර මෙසේ මේ ගාථා කීයේය. 
# චන්දන ථෙර ගාථා
# “රනින් වැසුණු ආභරණ පැළඳි දාසී සමූහයා විසින් පෙරටු කරණ ලද්දේ, පුත්‍රයා ඇකයෙන් ගෙන භාර්යාව මාවෙත පැමිණියාය, 
# “අලංකාරවූ යහපත්ව ඇන්දාවූ දැමූ මරු පාසයක් වැනි තම පුත්‍රයාගේ මව දැක 
# “ඉක්බිති මට නුවණින් යුතු සිතක් උපන්නේය. දොෂය පෙනුණේය. කලකිරීම එළඹ සිටියේය. 
# “ඉක්බිති සිත කෙලෙසුන්ගෙන් මිදුනේය. ධර්මයෙහි යහපත් බව බලව. ත්‍රිවිද්‍යාවට පැමිණියෙමි. බුදුන්ගේ අවවාදය කරණලදී.
# ආයුෂ්මත් චන්දන තෙර මෙසේ මේ ගාථා කීයේය. 
# ධම්මික ථෙර ගාථා
# “දශකුශල ධර්මය ඒකාන්තයෙන් ධර්මචාරී පුද්ගලයා රකියි. යහපත්කොට පුරුදු කළ ධර්මය සැප ගෙනදෙයි. පුරුදු කළ ධර්මයෙහි මේ ආනිසංසය ධර්මචාරී පුද්ගලයා දුගතියකට නොයවයි. 
# “ධර්මය අධර්මය යන දෙක සමාන විපාක දෙන්නේ නොවේ. අධර්මය අපායට යවයි. ධර්මය නිවනට යවයි. 
# “එසේ හෙයින් ධර්මයෙහි ආශාව කරන්නේය. නොසෙල්වෙන බුදුන් කෙරෙහි සතුටුවන්නේ ධර්මයෙහි ආශාව කරන්නේය. ධර්මයෙහි සිටියාවූ ශ්‍රෙෂ්ඨ පිහිට අනුව යන ප්‍රඥා වන්තයෝ නිවනට පමුණුවත්. 

# “අවිද්‍යාව දුරුකරණලදී. තෘෂ්ණා දැල නසනලදී. ඒ මම ක්ෂය කළ සංසාරය ඇත්තෙම් දොස් නැති කලා සම්පූර්ණ සඳ මෙන් ක්ෂය කළ සංසාරය ඇත්තේ රාගාදී ක්ලේශයන් නැත්තෙක්වීමි.”
# ආයුෂ්මත් ධම්මික තෙර මෙසේ මේ ගාථාව කීයේය. 
# සප්පක ථෙර ගාථා
# පිරිසිදු සුදුපාට පියාපත් ඇති පක්ෂීහු යම් කලෙක්හි කළු පැහැ ඇති වලාකුලට බියෙන් තැතිගත්තාහු තමාගේ කූඩුව සොයා යති. එකල්හි අජකරණි නදිය මගේ සිත සතුටු කරයි. 
# “පිරිසිදු සුදුපාට පියාපත් ඇති පක්ෂීහු යම් කලෙක්හි කළු පැහැති වලාකුලට බියෙන් තැතිගත්තාහු සැඟවීමට තැනක් නැතිව ලෙනක් සොයත්ද, එකල්හි අජකරණි නම් නදිය මගේ සිත සතුටුකරයි. 
# එහි දෙපසෙහි දඹ ගස් කවරෙකුගේ සිත් නො අලවත්ද, මගේ ලෙන පිටිපස ගං ඉවුර අලංකාර කරත්. 
# “ඒ මැඩියෝ සර්ප සමූහයාගෙන් දුරුවූ ඒ ස්ථානයෙහි මිහිරි හඬින් නාදකරත්. අද පර්වතය නදීන්ගෙන් ඈත් විය යුතු දිනයක් නොවේ. බිය නැති අජකරණි නදිය ඉතා යහපත්ය.”
# ආයුෂ්මත් සප්පක තෙර මෙසේ මේ ගාථා කීයේය. 
# මුදිත ථෙර ගාථා
# “මම ජීවිකාව පිණිස පැවිදිවීමි. උපසම්පදාව ලබා ඊටත් වඩා ශ්‍රද්ධාව ලැබුවෙමි. තද වීර්ය ඇතිව භාවනා කෙළෙමි. 
# “ඒකාන්තයෙන් මේ ශරීරය බිඳේවා, මේ වීර්යයෙන් ශරීරයෙහි මස්පිඬු විසිරේවා, දනසන්ධි දෙකින් කෙණ්ඩා කැඩී වැටෙත්වා. 
# “මගේ හෘදයෙහි තණ්හා හුල නැති නොවනතුරු නොකන්නෙමි නොබොන්නෙමි. පන්සලෙන් පිට නොවන්නෙමි. පැත්තකින් නොනිදන්නෙමි. 
# “මෙසේ වාසය කරන්නාවූ මගේ තද උත්සාහය බලන්න. ත්‍රිවිද්‍යාවන් ලද්දෙමි. බුදුන්ගේ අවවාදය කරණ ලදී.”
# ආයුෂ්මත් මුදිත තෙර මෙසේ මේ ගාථා කීයේය.
