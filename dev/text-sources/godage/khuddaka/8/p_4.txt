
පඤ්චක නිපාතය 
# රාජදත්ත ථෙර ගාථා
# “මහණතෙම සොහොනට ගොස් සොහොනෙහි හැරදමනලද පණුවන් වටවී කන ස්ත්‍රියකු දුටුයේය. 
# “යම්හෙයකින් නින්දිතවූ මළහු දැක ඇතැම් කෙනෙක් පිළිකුල් කරත්ද, මට කාම ආශාව පහළවිය. අශුචි වැගිරෙන ශරීරයෙහි අන්ධයෙකු මෙන් වීමි. 
# “බතක් පැසෙන කාලයටත් අඩු කාලයක් ඒ ස්ථානයෙන් බැහැරවූයෙමි. මම සිහි ඇතිව යහපත් නුවණින් යුක්තව එක්පැත්තකට ගියෙමි. 
# “ඉක්බිති මට නුවණින් යුතු සිතක් උපන්නේය. දොෂය පෙනුණේය. කලකිරීම එළඹ සිටියේය. 
# “ඉක්බිති සිත කෙලෙසුන්ගෙන් මිදුනේය. ධර්මයෙහි යහපත් බව බලව. ත්‍රිවිද්‍යාවට පැමිණියෙමි. බුදුන්ගේ අවවාදය කරණලදී.
# ආයුෂ්මත් රාජදත්ත තෙර මෙසේ මේ ගාථා කීයේය. 

# සුභූති ථෙර ගාථා
# “තම යහපතද අනුන්ගේ යහපතද බලාපොරොත්තුවන පුරුෂතෙම තමා නොයෙදිය යුතු දෙයෙහි යෙදෙන්නේද හිත සැපයට නොපැමිණෙන්නේය. එය මගේ අවාසනාවකැයි සිතයි. 
# “රාග නමැති දුකට පැමිණි සංසාර නමැති රට වනසා නොදමා භාවනා නමැති දෙවැන්නා නැතිව තනිවම අත්හරින්නේද, එය පාපයක් වන්නේය. නිවන් මෝරවන ගුණධර්ම සියල්ල අත්හරින්නේය. සම විෂම නොදක්නා බැවින් අන්ධයෙක්ම වන්නේය. 
#“යමක් කරන්නේද එය කියන්නේය. යමක් නොකරන්නේද එය නොකියන්නේය. පණ්ඩිතයෝ කියන්නාවූ දෙය නොකරන්නාහු විශෙෂයෙන් දැනගනිත්.
#සිත්කලුවූ, මනාවූ පැහැ ඇති, සුවඳ රහිතවූ, මල් යම්සේ (සුවඳ නොපතුරුවා) ද එපරිද්දෙන්ම මනාකොට දෙසනලද ත්‍රිපිටක බුද්ධ වචනය (ඒ ධර්මය) නොකරන්නාහට සඵල නොවෙයි.
#සිත්කලුවූ, මනා පැහැ ඇති, සුවඳ සහිතවූ, මල් යම්සේ (සුවඳපතුරුවා) ද එපරිද්දෙන්ම මනාකොට දෙසන ලද ත්‍රිපිටක බුද්ධ වචනය (ඒ ධර්මය) කරන්නාහට සඵල නොවෙයි.
#ආයුෂ්මත් සුභූති තෙර මෙසේ මේ ගාථා කීයේය. 
# ගිරිමානන්ද ථෙර ගාථා
# “මෙඝය වලාකුලේ ගීතයට අනුව වසීද, මාගේ කුටිය සෙවෙලිකරණ ලදී. සැපය පහසුය. එහි සංසිඳී වාසය කරමි, වැස්ස ඉදින් ඔබට වුවමනානම් වර්ෂා කරන්න. 
# “මෙඝය වලාකුලේ ගීතයට අනුව වසීද, මාගේ කුටිය සෙවෙලිකරණ ලදී. සැපය පහසුය. එහි සන්සුන් සිත් ඇත්තේ වාසය කරමි, වැස්ස ඉදින් ඔබට වුවමනානම් වර්ෂා කරන්න. 
# “මෙඝය වලාකුලේ ගීතයට අනුව වසීද, මාගේ කුටිය සෙවෙලිකරණ ලදී. සැපය පහසුය. එහි පහවූ රාග ඇතිව වාසය කරමි, වැස්ස ඉදින් ඔබට වුවමනානම් වර්ෂා කරන්න. 
# “මෙඝය වලාකුලේ ගීතයට අනුව වසීද, මාගේ කුටිය සෙවෙලිකරණ ලදී. සැපය පහසුය. එහි පහවූ ද්වෙෂ ඇතිව වාසය කරමි, වැස්ස ඉදින් ඔබට වුවමනානම් වර්ෂා කරන්න. 
# “මෙඝය වලාකුලේ ගීතයට අනුව වසීද, මාගේ කුටිය සෙවෙලිකරණ ලදී. සැපය පහසුය. එහි පහවූ මෝහ ඇතිව වාසය කරමි, වැස්ස ඉදින් ඔබට වුවමනානම් වර්ෂා කරන්න. 
# ආයුෂ්මත් ගිරිමානන්ද තෙර මේ ගාථා මෙසේ කීයේය. 
# සුමන ථෙර ගාථා
# “නිවැරදි ධර්මයන්ගෙන් යම් ධර්මයක් මට ලැබේවායි බලාපොරොත්තුවන උපාධ්‍යායතෙමේ අමෘත මහා නිර්වාණය බලාපොරොත්තුවෙන් මට අනුකම්පා කළේය. මා විසින් කළ යුතු වැඩ අවසන් කරණලදී. 
# “උපද්‍රව නැති ධර්මයට පැමිණියෙමි. තෙමේම ප්‍රත්‍යක්ෂ කරණලදී. ඔබගේ සමීපයෙහි පිරිසිදු නුවණ ඇත්තේ සැක නැත්තේ ප්‍රකාශ කරමි. 
# “පෙර විසූ පරිදි දනිමි. දිවඇස පිරිසිදු කරණලදී. මා විසින් තමාගේ යහපතට පැමිණෙනලදී. බුදුන්ගේ අනුශාසනය කරණලදී. 
# “අප්‍රමාදවූ මට ඔබවහන්සේගේ සස්නෙහි හික්මීම අස්වනලදී. මා විසින් සියලු ආශ්‍රවයෝ ක්ෂය කරණ ලද්දාහ. දැන් නැවත ඉපදීමක් නැත. 
# “පිරිසිදු ශීලාදී වත් සමාදානයෙන් මට අනුශාසනා කරණලදී. අනුකම්පා කරණලදී. අනුග්‍රහකරණලදී. අතවැස්සා කෙරෙහි හික්මවනලද අවවාදය හිස්නොවීය.”
# ආයුෂ්මත් සුමනතෙර මේ ගාථා මෙසේ කීයේය. 
# වඞ්ඪ ථෙර ගාථා
# “මව මට අවවාද නමැති කෙවිටක් එල්ලකරන්නේද, යහපත්ය. මව විසින් අවවාද කරණලද මම යමෙකුගේ වචන අසා, 
# “පටන්ගන්නාලද වීර්ය ඇත්තේ රහත්වූයේ උතුම් බුද්ධත්වයට පැමිණියෙමි. ත්‍රිවිද්‍යා දත් නිවන් ප්‍රත්‍යක්ෂකරණලද දීමට සුදුසු රහත් කෙනෙක්මි. 
# “මාරසේනාව සිඳ කෙලෙස් නැතිව වසමි. ඇතුළතද පිටතද, යම් කෙලෙස් කෙනෙක් වූවාහුද, 
# “සියල්ලෝ ඉතිරි නැතිව සින්දාහ. නැවත නූපදී. ධර්මය නිපුන නැගනිතොමෝ මේ කරුණ කීවාය. 
# “ඒකාන්තයෙන් මා කෙරෙහිද තොපගේ පුත්‍රයා කෙරෙහිද සබඳකම් නැතැයි සිතමි. අවිද්‍යාදී කැලෑවක් තොප කෙරෙහි නැත. දුක කෙළවර කරණලදී. මේ අන්තිම ඉපදීමය. දැන් නැවත ඉපදීම යයි කියනලද උත්පත්තියක් මරණයක් සසරක් නැත.”
# ආයුෂ්මත් වඞ්ඪතෙරණුවෝ මෙසේ මේ ගාථා කීහ. 

# නදී කාශ්‍යප ථෙර ගාථා
# “බුදුරජතෙම ඒකාන්තයෙන් මගේ යහපත පිණිස නෙරංජරානදිය වෙත ගියේය. මම යමෙකුගේ ධර්මය අසා වැරදි විශ්වාසය අත්හලේද ඒ බුදුන් වහන්සේය.” 
# “අන්ධයෙකු මෙන් වූයේ පෘථග්ජනයෙක්වූයේ මෙය සුද්ධියයයි සිතමින් උස් පහත් යාග කෙළෙමි. මම එහි ගිනි දෙවියා පිදීමි. 
# “අන්ධයෙකු මෙන් නුවණ නැතිව දෘෂ්ටීන් වෙත දිව්වෙමි. තදින් ගැනීම් සංඛ්‍යාත අල්වා ගැනීමෙන් මුළා වූයේ අපිරිසිදු දෙය පිරිසිදු යයි සිතුවෙමි. 
# මාගේ වැරදි වැටහීම සියලු භව පලන ලදී. දීමට සුදුසු ගින්නෙහි පුදමි, තථාගතයන් වඳින්නෙමි. 
# “මාගේ සියලු මොහයෝ නැතිවූහ. භව ආශාව පලන ලදී. ඉපදීම් නමැති සසර ක්ෂයවිය. දැන් නැවත ඉපදීමක් නැත.” 
#නදී කාශ්‍යප තෙරණුවෝ මෙසේ මේ ගාථා කීහ. 

# ගයා කාශ්‍යප ථෙර ගාථා
# “ඒ මම දවසට තුන් වරක් උදෑසනද මද්දහනද සවසද උපවාස කාලයෙහි ගයාවෙහි දියට බැස්සෙමි. 
# “පෙර නොදන්නා කාලයෙහි මා විසින් යම් පාප කර්මයක් කරණ ලද්දේද එය දැන් මෙහි පා කර හරිමි. පෙර මෙබඳු දෘෂ්ටියක් විය. 
# “ධර්මයෙන් හා වැදගත් පදයන්ගෙන් යුත් යහපත් වචන අසා ඒ තත්වූ පරිදි තේරුම්ගෙන නුවණින් සිතුවෙමි. 
# “සෝදාහරණලද සියලු පව් ඇත්තෙමි. නිර්මලවෙමි. පරිශුද්ධවෙමි. සෝදන ලද්දේවෙමි. පිරිසිදුවූයේ පිරිසිදු වස්තු අයිති බුදුන්ගේ ළෙහි වැඩුණු පුත්‍රයෙක් වූයෙමි. 
# “අංග අටකින් යුත් මාර්ගයට බැස සියලු පව් පා කර හැරියෙමි. ත්‍රිවිද්‍යාවන් ලැබුවෙමි. බුදුන්ගේ අවවාදය කෙළෙමි.”
# ආයුෂ්මත් ගයා කාශ්‍යප තෙරණුවෝ මේ ගාථා මෙසේ කීහ. 

# වක්කලී ථෙර ගාථා
# “වාත ලෙඩකින් පෙළුනු මහා වනයෙහි වාසය කරන්නාවූ ප්‍රත්‍ය පහසු නැති මහණ ඔබ කෙසේ වාසය කරන්නෙහිද?” 
# “මහත්වූ ප්‍රීති සැපයෙන් ශරීරය පතුරුවමින් රළුවුවද ඉවසමින් වනයෙහි වාසය කරන්නෙමි. 
# “සතිපට්ඨානයන්ද, ඉන්ද්‍රියයන්ද, බල ධර්මද දියුණු කරමින් බොජ්ක්ධංගද වඩමින් වනයෙහි වාසය කරන්නෙමි. 
# “පටන් ගන්නාලද වීර්ය ඇති නිවන වෙත යොමු කළ සිත් ඇති නිතර දෘඪ වීර්යයෙන් යුක්තව සමගිවූ සමානයන් දැක වනයෙහි වාසය කරමි. 
# “ශ්‍රෙෂ්ඨවූ හික්මුනාවූ එකඟවූ සිත් ඇති බුදුන් සිහි කරමින් කම්මැලි නැතිව රෑ දාවල් වනයෙහි වාසය කරන්නෙමි.”
# ආයුෂ්මත් වක්කලී තෙරණුවෝ මේ ගාථා කීහ. 

# විජිතසෙන ථෙර ගාථා
# “චිත්තය තොපගේ මාර්ගය කුඩා දොරටුවකින් ඇතෙකු මෙන් වලක්වන්නෙමි. මාර දැලක්වූ සිත, තෝ පාපයෙහි නොයොදවන්නෙමි. 
# “දොරටුවක් නොලැබෙන ඇතෙකු මෙන් තෝ වැලැක්කේ නොයන්නෙහිය. සිත නැමැති කාලකණ්නිය, නැවත නැවත වලක්වන්නේ පාපයෙහි ඇලුනෙක් නොවන්නෙහිය. 
# “ඇත්ගොව්වා අලුත අල්වා ගත් නොහික්මුණු ඇතෙකු යම්සේ නොකැමැති පරිද්දෙන් නවත්වයිද, මෙසේ තොප නවත් වන්නෙමි. 
# “ශ්‍රෙෂ්ඨ අසුන් දමනය කිරීමෙහි දක්ෂවූ උතුම් අස්ගොව්වෙක් ආජානීය අශ්වරාජයෙකු දමනය කරයිද, ශ්‍රද්ධාදී පඤ්චබලයෙහි පිහිටා තොප මෙසේ දමනය කරන්නෙමි. 
# “තොප සිහියෙන් බඳින්නෙමි. පිරිසිදුවූයේ චොදනා කරන්නෙමි. වීර්යය නමැති වියගහෙන් කම්පා කෙළේ සිතමින් ඈතට නොයන්නේය.”
# ආයුෂ්මත් විජිතසෙන තෙරණුවෝ මේ ගාථා මෙසේ කීහ. 

# යසදන්ත ථෙර ගාථා
# “දොෂාරොපන කරණ අදහසින් මෝඩයා බුද්ධාවවාද අසයිද අහසට පොළොව මෙන් සද්ධර්මයෙන් ඈත්වූයේවේ. 
# “දොෂාරොපන කරණ අදහසින් මෝඩයා බුද්ධාවවාද අසයිද කළුවර පක්ෂයෙහි චන්ද්‍රයා මෙන් සද්ධර්මයෙන් පිරිහෙයි. 
# “දොෂාරොපන කරණ අදහසින් මෝඩයා බුද්ධාවවාද අසයිද දිය නැති විල්හි මසෙකු මෙන් සද්ධර්මයෙන් පිරිහේ.” 
# “දොෂාරොපන කරණ අදහසින් මෝඩයා බුද්ධාවවාද අසයිද කුඹුරෙහි කුණුවූ බීජ මෙන් සද්ධර්මයෙහි නොවැඬේ. 
# “යමෙක් සතුටු සිතින් බුද්ධාවවාදය අසයිද, සියලු පව් නසා රහත් බව ප්‍රත්‍යක්ෂකොට පරම ශාන්තියට පැමිණ කෙලෙස් රහිතව පිරිනිවන්පායි.”
# ආයුෂ්මත් යස දන්ත තෙරණුවෝ මෙසේ මේ ගාථා කීහ. 

# සොණකුටිකණ්ණ ථෙර ගාථා
# “මා විසින් උපසම්පදාව ලද්දේය. කෙලෙස් මිදීමෙන් මිදුනෙමි. කාමාශාදියෙන් තොරවූවෙමි. ඒ මා විසින් නුදුටු බුදුහු දක්නා ලදී විහාරයෙහි එකට විසීමි. 
# “භාග්‍යවතුන් වහන්සේ රාත්‍රියෙන් වැඩිහරියක් එලිමහනෙහි විසූසේක. විසීම්වල දක්ෂ බුදුහු ඊට පසු ගඳකිළියට ඇතුල්වූහ. 
# “නැති කළ බිය හා භයානක අරමුණු ඇති බුදුහු සඟල සිවුර අතුරා ගල්ගුහාවක සිංහයකු මෙන් සයනය කළ සේක. 
# “යහපත් වචන ඇති බුද්ධ ශ්‍රාවකවූ සොණතෙමේ බුදුරජාණන් වහන්සේ සමීපයෙහි සද්ධර්මය කීයේය. 
# “ස්කන්ධ පඤ්චකය දැන මාර්ග වඩා පරම ශාන්තියට පැමිණ කෙළෙස් රහිතව පිරිනිවන් පාත්.”
# ආයුෂ්මත් සොණ කුටි කණ්ණ තෙරණුවෝ මේ ගාථා මෙසේ කීහ. 

# කොසිය ථෙර ගාථා
# “යම් ඤාණවන්තයෙක් ඒකාන්තයෙන් ගුරුන්ගේ වචන දන්නේද උන්වහන්සේලාගේ වසඟයෙහි සිටීද ගෞරව උපදවාද ඒ පණ්ඩිත තෙමේ භක්ති ඇත්තේවේ. ධර්මයන් දැන විශෙෂ ගුණ උපදවන්නේය. 
# “මහත්වූ යම් විපත්තීහු පහළවූවාහුද, කම්පා නොවී ඉවසන්නේද, ඒ පණ්ඩිත තෙමේ ශක්තිමත් නම් වේ. ධර්මය දැන විශෙෂයට පැමිණෙන්නේය. 
# “ගැඹුරු මුහුද යම්සේ කම්පා නොවේද, ගැඹුරු ප්‍රඥා ඇති අර්ථයෙහි දක්ෂ පණ්ඩිතයා නොබෙදිය හැකිවේ. දැන ධර්මයන්හි විශෙෂ ගුණ උපදවන්නේයි. 
# “බහුශ්‍රැතවූ ධර්මධරවූ ධර්මය අනුව හැසිරෙන එබඳු පණ්ඩිතතෙමෙ ධර්මයන් දැන විශෙෂ පුද්ගලයෙක් වන්නේය. 
# “යමෙක් බුද්ධ වචනයාගේ අර්ථය දනීද අර්ථය දැන එසේ කරයිද, හෙතෙම අර්ථ දත් පමණකින් පණ්ඩිත වේ. ධර්මයන් දැන විශෙෂ පුද්ගලයෙක් වන්නේය.”
# ආයුෂ්මත් කොසිය තෙරණුවෝ මේ ගාථා මෙසේ කීහ. 

#පස් වැනි නිපාතය නිමි.
#
